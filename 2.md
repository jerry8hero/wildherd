  两种情况                                                                                                                                                                                    
                 
  情况一：模型下载到 Windows D 盘                                                                                                                                                             
                                                                                                                                                                                              
  1. 创建 D 盘目录

  在 Windows 资源管理器中，D 盘根目录新建一个文件夹：

  D:\framepack_models

  2. 设置 Windows 环境变量

  1. 按 Win + R，输入 sysdm.cpl，回车
  2. 点击 高级 → 环境变量
  3. 在 系统变量 中点击 新建：
  ┌─────────────┬───────────────────────┐
  │   变量名    │        变量值         │
  ├─────────────┼───────────────────────┤
  │ HF_HOME     │ D:\framepack_models   │
  ├─────────────┼───────────────────────┤
  │ HF_ENDPOINT │ https://hf-mirror.com │
  └─────────────┴───────────────────────┘
  HF_ENDPOINT 设置为镜像站点，下载会更快。

  4. 点击确定，重启电脑

  3. 在 WSL 中也设置对应路径

  打开 WSL 终端：

  # 在 ~/.bashrc 中添加
  echo 'export HF_HOME="/mnt/d/framepack_models"' >> ~/.zshrc
  echo 'export HF_ENDPOINT="https://hf-mirror.com"' >> ~/.zshrc
  source ~/.zshrc

  4. 验证设置

  echo $HF_HOME

  应该显示：/mnt/d/framepack_models

  ---
  情况二：模型下载到 WSL 内部 D 盘挂载点

  如果你的 D 盘在 WSL 中挂载为 /mnt/d，可以直接用这个路径：

  # 创建目录
  mkdir -p /mnt/d/framepack_models

  # 设置环境变量
  echo 'export HF_HOME="/mnt/d/framepack_models"' >> ~/.bashrc
  echo 'export HF_ENDPOINT="https://hf-mirror.com"' >> ~/.bashrc
  source ~/.bashrc

  ---
  下载模型

  设置完成后，模型会自动下载到 D 盘目录：

  D:\framepack_models\
  ├── hub\
  │   └── models--Comfy-Org--HunyuanVideo-repackaged\
  │       └── ...

  首次下载约 15-20GB，使用镜像站点会快很多。

  ---
  验证是否成功

  下载完成后，在 WSL 中检查：

  ls -la /mnt/d/framepack_models/hub/models--Comfy-Org--HunyuanVideo-repackaged/

  能看到文件说明下载成功。

  ---
  提示：设置环境变量后，重启电脑或重新打开 WSL 才会生效。
