# 【冷到你唔信】系列文案模板

## 系列定位

**系列名称**：冷到你唔信
**内容方向**：爬行动物冷知识
**语言风格**：粤语口语、幽默风趣、网感强
**目标受众**：B站宠物/动物爱好者

---

## 文案结构

### 1. 系列开场（每期都要有）

```
各位，今日开新系列——冷到你唔信！

呢个系列係搞咩嘅？就係帮你哋讲下爬行动物入面嗰啲超冷门知识。啲咩神奇习性、犀利技能、骑呢名字，保证你听完眼都碌埋，造物主真系乜都谂得出！
```

**注意**：
- 只在系列第一期使用完整开场
- 后续期数只需简短提及，如"今日继续冷到你唔信"

### 2. 品种引入

```
好，我哋从头讲起。今日吹下水，讲只超级神奇嘅[品种]——[品种名]！

[用一个问题或反差引出品种]
你可能話，[常见误解]？[反转让观众惊讶]
```

### 3. 核心知识点（3-5个）

每个知识点用一个小标题，穿插趣味表达：

```
你知唔知[知识点1]？

[展开讲解，用比喻、网络梗增加趣味]
[停顿/反问]——[金句]

---

讲到[主题]，就係[核心观点]。

[详细展开]
[再用比喻/例子说明]

但/不过[转折补充]
[增加信息丰富度]
```

### 4. 结尾三件套

```
各位，跟住落嚟讲嘅，就係[品种]最[形容词]嘅[内容分类]。

[最后一个知识点]

---

下期我哋继续冷到你唔信系列，你哋想听咩犀利嘅动物冷知识？评论区留言！

最后问问大家——睇完呢期，你最中意边个技能？係[技能A]？係[技能B]？定係[技能C]？评论区吹下水！

如果你觉得呢期可以学到嘢嘅话，记得三连！你哋嘅三连係我更新最大嘅动力！下期见！
```

## 常用表达库

### 开头/转折
| 普通话 | 粤语 |
|--------|------|
| 没想到 | 估你唔到 |
| 没想到 | 唔到你唔信 |
| 但是 | 但係 |
| 虽然 | 雖然 |
| 因为 | 因為 |

### 形容厉害
| 普通话 | 粤语 |
|--------|------|
| 厉害 | 劲/犀利 |
| 太厉害了 | 几劲啰 |
| 非常厉害 | 劲到呢 |
| 就这样 | 就係咁上下 |
| 差不多 | 就係咁上下 |

### 常用口头禅
| 普通话 | 粤语 |
|--------|------|
| 不知道 | 知唔知 |
| 好厉害啊 | 好嘢 |
| 保证你 | 包你 |
| 真的吗 | 唔係嘛 |
| 还有呢 | 仲有 |

### 结尾互动
| 普通话 | 粤语 |
|--------|------|
| 点赞 | 三连/掂个掂 |
| 评论区 | 评论区吹下水 |
| 下期见 | 下期见 |

## 禁止使用

- ❌ "兄弟们"（太书面）
- ❌ "龟龟们"（对观众称呼奇怪）
- ❌ 过于书面的"首先、其次"
- ❌ 绝对化用语（100%、所有）

```
docs/video/scripts/冷到你唔信/[序号]-[品种]/[序号]-[标题]_视频文案_粤语.md
```

示例：
```
docs/video/scripts/冷到你唔信/002-四爪陆龟/002-绕月球飞行的乌龟_视频文案_粤语.md
```

## 创作流程

1. **提取原始素材** → 从 epub 中获取品种内容
2. **提炼3-5个核心冷知识** → 选择最有话题性的点
3. **按模板生成文案** → 参考绿海龟的结构
4. **Review 修改** → 检查粤语表达是否地道
5. **保存文件** → 按命名规范存放
