# 视频制作需求描述模板

---

## 快速模板

直接复制粘贴使用：

```
[主题]，[龟种]系列，第[数字]期
```

### 示例

```
养龟黑话，通用系列，第1期
```

---

## 详细模板

```
帮我做一期"[主题]"视频
龟种：[龟种]
序号：第[数字]期
内容方向：[可选：重点讲XX/面向新手/进阶内容]
结构偏好：[可选：五站/三站/按内容定]
```

### 示例

```
帮我做一期"养龟黑话"视频
龟种：通用
序号：第1期
内容方向：龟圈常用术语讲解，面向新手
结构偏好：五站结构（品相/繁殖/健康/交易/饲养）
```

---

## 内容方向关键词

| 关键词 | 含义 |
|--------|------|
| 面向新手 | 基础知识为主，避免太专业术语 |
| 进阶内容 | 包含更多专业知识和细节 |
| 重点讲XX | 指定某个部分要详细展开 |
| 实操为主 | 包含具体操作步骤 |

---

## 结构偏好关键词

| 关键词 | 含义 |
|--------|------|
| 五站结构 | 分5个主题站讲解 |
| 三站结构 | 分3个主题站讲解 |
| 按内容定 | 由我根据内容决定结构 |

---

## 龟种分类

| 龟种 | 说明 |
|------|------|
| 鳄龟 | 归入 `docs/video/scripts/鳄龟/` |
| 草龟 | 归入 `docs/video/scripts/草龟/` |
| 通用 | 归入 `docs/video/scripts/通用/` |

---

## 输出文件

执行后会生成以下三个文件：

| 文件 | 内容 |
|------|------|
| `[序号]-[主题].md` | 完整口播稿（含口播技巧） |
| `[序号]-[主题]_剪映版.md` | 清理后的纯口播文案 |
| `[序号]-[主题]-B站发布内容.md` | B站发布内容 |

---

## 流程回顾

1. 你描述需求
2. 我生成完整三件套
3. 你确认内容或提出修改
4. 你 git add + commit

修改格式：`[序号]-[主题]-B站发布内容.md` → `[序号]-[主题]_剪映版.md` → `[序号]-[主题].md`
