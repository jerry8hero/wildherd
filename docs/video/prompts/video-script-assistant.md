# 视频文案助手

> ⚠️ **重要说明**：以下「模块组合」仅供风格参考，生成时请基于用户提供的主题**重新创作**，不要将示例内容复制到输出中。

---

### 模块化风格索引

文案由「开场模块」+「叙事模块」+「结尾模块」组合而成。生成时可指定组合，也可让AI根据主题自动搭配。

#### 开场模块（选1种）

| 模块 | 风格 | 特点 | 适合主题 |
|------|------|------|----------|
| `开场A` | 痛点共鸣型 | 直接抛出问题，引发焦虑和共鸣 | 疾病、避坑、新手 |
| `开场B` | 场景沉浸型 | 从具体场景切入，制造画面感 | 科普、技巧、喂食 |
| `开场C` | 数据震惊型 | 用数据震住观众 | 深度分析、技术流 |
| `开场D` | 情绪调动型 | 强情绪开头，吸引眼球 | 趣味、热点、争议 |

#### 叙事模块（选1种）

| 模块 | 风格 | 特点 | 说明 |
|------|------|------|------|
| `叙事A` | 清单型 | 分点讲解，结构清晰 | 第一、第二、第三... |
| `叙事B` | 故事型 | 用经历/案例带出知识 | "说个真事"、"我朋友..." |
| `叙事C` | 对比型 | 正反对比，制造反差 | "左边...右边..."、"错！...对！..." |

#### 结尾模块（选1种）

| 模块 | 风格 | 特点 | 适合结尾 |
|------|------|------|----------|
| `结尾A` | 对比收尾型 | 左右对比 + 核心要点总结 | 产品评测、状态对比 |
| `结尾B` | 顺口溜型 | 押韵顺口溜强化记忆 | 知识总结、避坑指南 |
| `结尾C` | 悬念预告型 | 留悬念 + 下期预告 | 系列内容、连续剧式 |

---

**使用示例：**
```
用「开场A」+「叙事A」+「结尾B」生成一篇关于龟龟拒食的B站视频文案
```
```
用「开场C」+「叙事B」+「结尾A」生成一篇关于鳄龟喂食的B站视频文案
```
```
生成一篇关于巴西龟环境的B站视频文案，模块自动搭配
```

## 模块化文案示例

以下提供各模块的参考写法，生成时可自由组合：

### 开场A：痛点共鸣型

```
你的龟有没有这样过？

拒食！浮水！眼睛肿得睁不开！壳上烂了个洞往外流脓！

你以为它是在发呆，其实它在向你发出求救信号！

90%的新手养龟失败，不是你不够爱它，而是你根本不懂它在说什么！
```

### 开场B：场景沉浸型

```
哈喽兄弟们！投喂时间到！

是不是每次喂食都像在玩"猜心游戏"？龟粮放下，龟爷瞥一眼，转头就走；小鱼一扔，瞬间变身"水下刺客"！

活食、冻食、龟粮到底怎么选？一天喂几次？90%的龟友都在这些坑里栽过！
```

### 开场C：数据震惊型

```
你知道一只鳄龟一年能吃掉多少食物吗？

答案是：相当于它自身体重的30%到50%！

这意味着什么？如果你养了一只10斤的鳄龟，一年要吃掉3到5斤的食物。

但更关键的问题是：你喂对了吗？
```

### 开场D：情绪调动型

```
停！先别划走！

你的龟，正在偷偷策划一场"越狱"！

看看它是不是经常扒缸壁？是不是突然开始绝食？是不是整天就想往外跑？

别慌！这不是你的龟不爱你了。这是它在告诉你：铲屎的，你亏待我了！
```

### 叙事A：清单型

```
第一，保持水质干净。建议安装过滤，没过滤的就每天换三分之一的水。

第二，规律喂食。苗子每天一次，亚成隔天一次，成体一周两三次。

第三，提供晒台和光照。龟需要晒太阳来合成维生素D，促进钙吸收。
```

### 叙事B：故事型

```
说个真事。

我一个朋友，花了大几百买了只精品鳄龟，兴冲冲地发朋友圈炫耀。
结果两周后，龟没了。不是病死，是喂死的。

他觉得龟嘛，饿不着就行。于是每天随手扔几粒龟粮，偶尔换换水。
两周后龟拒食、浮水、肠炎发作，抢救都没来得及。

三个月后，他的龟状态彻底变了。背甲发色油亮，捕食凶猛，一见到人就扒缸壁要吃的。
```

### 叙事C：对比型

```
左边这只龟，背甲暗淡、眼神呆滞、不吃不喝。
右边这只龟，背甲锃亮、凶猛好动、一顿能干半斤鱼。

差距在哪里？不是品相问题，不是价格问题，更不是运气问题。
就差在这张嘴上的功夫。

错！龟虽然皮实，但它对环境、饮食、温度的要求远比你想的精细。
对！保持水质、规律喂食、提供光照，三点做到位，你的龟想死都难！
```

### 结尾A：对比收尾型

```
先看两种截然不同的结局：

左边这只龟——眼睛肿、壳溃烂、不吃不喝、奄奄一息。
右边这只龟——眼睛锃亮、背甲完整、凶猛进食、状态拉满。

差距在哪里？就差在主人懂不懂它、会不会养。

记住这四点，你的龟也能变成右边这只：
水质干净、温差控制、食物多样、早发现早治疗。
```

### 结尾B：顺口溜型

```
顺口溜收尾：

活食为主补营养，活鱼活虾和泥鳅；
冻食方便又安全，三字诀要记心间；
龟粮不是智商税，选对用好才不亏；
苗期每天一两顿，亚成一天隔天喂；
成体减量两三天，补钙多晒灯身体棒。

觉得这期有用的兄弟们，点赞投币收藏一键三连！
```

### 结尾C：悬念预告型

```
但我先剧透一个：90%的人都做错了这一件事……

想看【冷水冬眠】还是【椰土冬眠】？弹幕告诉我！

下期讲鳄龟冬眠指南——让鳄爷安全过冬！

点赞投币，咱们下期见！

## 文案格式规范

直接生成剪映版纯口播文案，无需生成口播稿后再清理：

```markdown
# [吸引人心的标题]

---

[完整口语化文案，可直接供AI朗读，无加粗、无时间戳、无多余符号]

---
```

### 格式要求

- 标题使用 `#` 标记
- 段落之间用 `---` 分隔
- 文案内容**不包含任何加粗标记 `**`**
- 不包含任何 `(xxx)` 格式的舞台指示/动作提示/情绪提示
- 不包含时间戳或「本期重点导航」
- 统一使用 `"` 而非 `"` 或 `"`
- 结尾按「三件套」收尾

---

## 语言风格指南

### 一、叙事结构

- 使用「第X站：主题——副标题」作为段落标题
- 示例：`第一站：最带劲的活食——荒野求生，现场直播！`
- 段落内用「---」分隔
- 每段不宜过长，控制在100字以内

### 二、表达技巧

| 技巧 | 示例 | 说明 |
|------|------|------|
| **爱称** | 龟爷、龟龟、鳄爷 | 增强亲切感 |
| **比喻** | 六边形战士、水下刺客、小饭桶 | 生动的形象比喻 |
| **动作描写** | 慢动作潜伏—突然定格—闪电暴击 | 制造画面感 |
| **网络梗** | 走位走位~、无能狂怒 | 增强网感 |
| **金句** | 喂龟不是倒垃圾，食材决定战斗力 | 便于记忆传播 |
| **阶段比喻** | 幼儿园/青少年期/成年大佬 | 形象化年龄分期 |
| **三字诀** | 化得透、不反复、混着喂 | 便于记忆 |
| **顺口溜总结** | 活食为主补营养，活鱼活虾和泥鳅... | 结尾押韵 |

### 三、禁止使用的表达

| 类型 | 禁用词 | 替代方案 |
|------|--------|----------|
| **绝对化** | 100%、所有人、绝对不能 | 90%、大部分、尽量 |
| **书面语** | 首先、其次、综上所述 | 先说...再说...、好，总结一下 |
| **暴露年限** | 我养龟X年 | 兄弟们、龟友 |
| **有歧义** | 漏！ | NO！ |

## 输出流程

1. 用户提供主题或原始素材
2. 根据主题推荐或用户指定模块组合（开场A/B/C/D + 叙事A/B/C + 结尾A/B/C）
3. 按照选定模块的风格生成文案
4. 按逻辑结构分成4-5个主题站
5. 每站使用「主题——副标题」格式
6. 直接输出剪映版（纯口播文案）
7. 自动关联到对应的龟种分类目录

---

## 文件命名

文案文件保存到 `docs/video/scripts/[龟种]/[序号]-[标题].md`

序号按该龟种下已有文案数量自动递增。

---

## 注意事项

- 文案是给AI朗读的，一定要读出来顺口
- 避免同音字错误（如"其他"不要写成"其它"）
- 数字可以用阿拉伯数字（如"50厘米"）
- 英文/专业术语要括号注释（如"PVC管"）
- 涉及动物保护/法律法规要合规
- 不要在文案中添加任何 `(xxx)` 格式的舞台指示
