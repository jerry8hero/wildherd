# 评论区置顶

📌 对比总结：

✅ 1个月的你：
- 龟看到你就缩头，喂食等很久才敢吃
- 每天盯着龟看八百遍，少吃一口就紧张
- 换水好麻烦，每次都手忙脚乱
- 龟腐皮了！疯狂发帖求助
- 这只可爱！买！那只不错！买！
- 龟粮好贵啊！
- 朋友圈发龟视频没人点赞
- 龟怎么不动？是不是生病了？

✅ 1年的你：
- 龟一看到你就扒缸边，追着镊子跑
- 龟爱咋咋地，生病了也不慌
- 换水？不就十分钟的事，边刷手机边换
- 腐皮？干养+聚维酮碘，小问题
- 这次只买一只...（三天后又买了）
- 龟粮不是最大开支，缸才是，开始研究发色粮
- 龟友群发言最活跃，经常交换"资源"
- 龟是有灵性的动物，养龟真的会上瘾

😂 钱包变化：
- 1个月：95块（龟15+缸50+设备30）
- 1年：N只龟+大中小缸+隔离缸+过滤器+加热棒+晒背灯+温控器+一箱箱龟粮 = 不想算

💡 给新手的3个核心建议：
1. 先做功课再买龟，别冲动消费
2. 设备一步到位，后期省大钱
3. 养龟需要耐心，慢就是快
