# 评论区置顶

📌 新手建议总结：

1️⃣ 不要买太贵的龟，从便宜的开始
2️⃣ 养龟设备提前准备好
3️⃣ 自来水要困水才能用
4️⃣ 换水不要全换，换三分之一到二分之一
5️⃣ 龟晒太阳很重要
6️⃣ 多看视频多学习，遇到问题不要慌
