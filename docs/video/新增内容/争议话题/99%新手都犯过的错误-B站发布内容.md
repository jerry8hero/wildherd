# 评论区置顶

📌 错误纠正总结：

1️⃣ 买龟前先做功课了解品种
2️⃣ 自来水困水24小时再用
3️⃣ 每周换三分之一到二分之一水
4️⃣ 换水温差不超过2度
5️⃣ 每次喂头部大小的量
6️⃣ 禁止喂盐和调料
7️⃣ 每天晒1-2小时太阳
8️⃣ 龟缸越简单越好
