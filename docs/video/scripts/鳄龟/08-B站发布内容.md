特别提醒：

- 公龟5-6年、母龟4-5年才性成熟
- 体型达标才能繁殖（公30cm+，母25cm+）
- 产卵需要15cm以上厚度的土
- 孵化温度28度左右，60-90天出壳
- 卵黄囊完全收进后再喂食

置顶评论：
📌 补充一些视频里没讲到的：

1️⃣ 公母特征详细对比：
- 尾巴：公粗长有角质突出，母短小
- 腹甲：公凹陷，母平坦
- 体型：同龄公比母更大
- 气味：成体公龟有特殊气味

2️⃣ 合缸注意事项：
- 先隔缸相望几天，让龟熟悉彼此
- 合缸时有人在旁看守
- 准备好隔离网兜，防止打斗受伤
- 交配期间不要喂食

3️⃣ 产卵前兆：
- 食欲下降或拒食
- 后腿频繁挖土
- 变得焦躁不安
- 腹部隆起

4️⃣ 孵化盒制作：
- 容器：外卖盒或专业孵化盒
- 垫材：蛭石或椰土（蛭石:水=1:1）
- 标注：记录产卵日期，方便计算出壳时间

5️⃣ 常见问题：
- Q：蛋没受精怎么办？
  A：没受精的蛋不会发育，白斑不会扩大
- Q：孵化多久没反应是不是坏了？
  A：有的蛋发育慢，照蛋确认有血管就是活的
- Q：小龟性别由什么决定？
  A：孵化温度决定，28度左右母龟比例高

⚠️ 繁殖风险提醒：
- 母龟难产需要及时求助医生
- 孵化温度不能超过32度！
- 小龟卵黄囊没收完不能下水！
- 繁殖对龟体力消耗大，繁殖后要加强营养！
