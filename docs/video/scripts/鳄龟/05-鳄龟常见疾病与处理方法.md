# 鳄龟常见疾病与处理方法

---

你的鳄龟有没有这样过？眼睛肿、壳烂了、不吃东西——你以为它在发呆，其实它在叫救命！

九成的养龟新手失败，不是你不够爱它，而是你根本不懂它在说什么！今天这期，帮兄弟们一次性扫清所有误区，看完你也能成为龟龟的家庭医生！

不废话，先上硬货！

鳄龟爷虽然皮实好养，但该来的病一样都不会少。腐皮腐甲、白眼病、感冒肺炎、拒食问题，样样都有。而且因为鳄龟代谢快、水质恶化也快，一旦生病，恶化速度比别的龟快得多。所以今天这些知识，对养鳄龟的兄弟格外重要！

先说腐皮腐甲，水质问题是万恶之源！

先说腐皮。皮肤溃烂发炎，出现白色或黄色斑块，严重的直接出血露肉！原因就一个，水太脏！氨氮超标，鳄爷皮肤直接发炎抗议！

处理三步走。

第一，隔离换水。发现腐皮立刻捞出来，放干净水里静养，每天换水！

第二，干养疗法。轻微腐皮每天干养两到三小时，让皮肤干燥愈合。记得保温，别让龟龟感冒雪上加霜！

第三，用药处理。严重腐皮用聚维酮碘消毒，涂红霉素软膏或百多邦，每天一次，一周左右好转。

再说腐甲。这俩是难兄难弟，只不过腐甲长在壳上。症状三连，白色或褐色斑块，按下去软软的，渗血水脓液，腐臭味直冲天灵盖！

处理分两种。

轻度，牙签轻轻挑开腐烂部分，清理脓液，聚维酮碘消毒，涂药膏，干养让药物吸收。

重度，腐烂深入骨骼可能要注射抗生素，别硬撑，直接找专业爬宠医生！

重点记一下，腐皮腐甲预防方法一模一样，水质第一位！换水动作轻点，别让龟龟受惊吓！

然后说白眼病，龟爷睁不开眼了！

症状三连，眼睛发红肿胀，白色薄膜覆盖，完全睁不开眼。原因就两个，水质太差导致细菌感染，或者维生素A缺乏！

处理四连击。

第一，眼部清洁。棉签蘸生理盐水，轻轻擦拭眼睛，清理分泌物。

第二，滴眼药水。氯霉素滴眼液、氧氟沙星滴眼液都可以，每天两到三次。

第三，干养吸收。滴完干养半小时左右，让药物充分发挥作用。

第四，补营养。多喂富含维生素A的食物，虾、胡萝卜安排上！

鳄爷是纯肉食狂魔，喂活食的时候更要注意食材新鲜，不然变质肉分分钟让水质崩给你看！

预防关键，水质清洁加多样化饮食，别长期只喂一种食物！

接着说感冒肺炎，温差是最大杀手！

感冒和肺炎是呼吸道疾病兄弟，肺炎是感冒的升级版！

症状识别。普通感冒，流鼻涕、嘴巴张开呼吸、发出声音、不爱动、不吃东西。肺炎升级版，龟会浮水，姿势诡异，侧浮或者腹甲朝天漂！

原因只有一个，温差太大！换水时新水和老水温差大，或者室温忽冷忽热，龟龟直接中招。

处理方法。

第一，恒温。把水温调到二十八度左右，保持恒定。这是治疗基础，没有之一！

第二，隔离。多个龟的话，生病的龟必须单独隔离，防止传染！

第三，用药。轻微感冒不用药，保持恒温让龟自己恢复。严重感冒或肺炎需要抗生素，但用药必须谨慎，最好有专业指导！

第四，雾化。肺炎用雾化治疗效果比较好，小型雾化器加抗生素溶液，让龟龟吸入。

肯定有人问，我的龟只是轻微流鼻涕怎么办？

答，恒温二十八度观察两天，没有好转再用药！

预防核心，控制温差！换水时用温度计测量，温差不要超过三度！

最后说拒食，不吃东西是在报警！

重点，拒食不是病，是很多疾病的表现症状！龟龟不吃饭，一定有原因！

常见原因，应激、新环境、受惊吓、水质不好，温度不对、生病了。

处理思路，先排查原因，再对症下药！

第一，检查环境。水温合适吗？水质干净吗？缸里有啥让龟龟紧张的吗？

第二，检查龟龟。看有没有腐皮腐甲、白眼、流鼻涕等明显症状？

第三，隔离静养。放到安静避光的地方，水温二十八度，不喂食，观察两到三天。

第四，诱导开食。活食小鱼小虾诱导，或者喜马拉雅红虫膏。

鳄爷对肉食那是狂热分子，长期拒食绝对是大问题！这时候更要冷静排查，别一股脑硬塞！

如果都不行，可能生了大病，需要进一步检查或找医生治疗！

终极总结！

第一，水质干净。懒人建议上过滤，没过滤就每天换三分之一。

第二，温差控制。换水用温度计测量，新老水温差不超过三度。

第三，食物多样。龟粮为主，活食为辅，营养均衡才健康。

第四，早发现早治疗。龟龟异常别大意，发现问题立刻处理，拖越久越危险。
