置顶评论：
📌 补充一些视频里没讲到的：

1️⃣ 常备药品清单：
- 聚维酮碘（消毒用）
- 红霉素软膏/百多邦（腐皮腐甲）
- 氯霉素滴眼液（白眼）
- 阿莫西林（感冒轻症）
- 土霉素（肺炎）
- 妈咪爱（调理肠胃）

2️⃣ 消毒方法：
- 龟缸消毒：高锰酸钾或聚维酮碘浸泡
- 龟体消毒：聚维酮碘直接涂抹
- 器具消毒：沸水煮或高锰酸钾浸泡

3️⃣ 如何判断龟是否健康：
- 眼睛明亮，无分泌物
- 壳色正常，无异味
- 四肢有力，能撑起身体
- 呼吸无声，不张嘴
- 食欲正常，排便成型

4️⃣ 腐甲严重程度判断：
- 轻度：壳色改变，按压发软 → 门诊治疗
- 中度：壳下有脓液，有臭味 → 加强治疗
- 重度：深入骨骼，渗血 → 必须找医生

5️⃣ 肺炎紧急处理：
- 立刻恒温28度
- 隔离病龟
- 观察是否侧浮
- 谨慎用药，必要时求助医生

⚠️ 重要提醒：
- 用药要注意剂量，宁少勿多！
- 温差不能超过3度！
- 隔离病龟防止传染！
- 病情三天没好转要重视！
