## 【置顶评论】

```
🐢 鳄龟疾病实战经验总结

这期从腐皮腐甲到白眼病、从感冒肺炎到拒食问题全部讲清楚了，重点都在视频里！

大家最关心的问题统一回复：

1. 腐皮腐甲：关键在水质！发现后立刻隔离换水，轻微干养，严重用聚维酮碘消毒涂药膏
2. 白眼病：生理盐水清洁眼睛，滴氯霉素滴眼液，干养半小时让药物吸收
3. 感冒肺炎：恒温28度是治疗基础！温差不能超过3度，隔离防止传染
4. 拒食问题：拒食是症状不是病！先排查原因——水温、水质、是否受惊、生病
5. 预防核心：水质干净、温差控制、食物多样、早发现早治疗
```
