## 【置顶评论】

📋 入门品种速查表：

| 品种 | 一句话定位 | 适合谁 |
|---|---|---|
| 雨林蝎 | 霸气酷炫的艺术品 | 喜欢"酷"的新手 |
| 豹纹守宫 | 温顺可上手的小可爱 | 想要互动感的人 |
| 睫角守宫 | 缸里最有画面感的蜥蜴 | 喜欢观赏造景的人 |
| 角蛙 | 最省心的"活摆件" | 怕虫子/想省心的人 |
| 六角恐龙 | 永远在笑的外星人 | 有冷水条件的人 |
| 东方蝾螈 | 便宜皮实的国产蝾螈 | 南方/预算有限的人 |

⚡ 快速决策：
- 想要酷 → 雨林蝎
- 想要互动 → 豹纹守宫
- 想要好看 → 睫角守宫
- 想要省心 → 角蛙
- 有冷水 → 六角恐龙
- 没空调 → 东方蝾螈

⚠️ 五个新手必避的坑：买太小苗子/不做温度管理/混养/喂太多/忽视蜕皮
