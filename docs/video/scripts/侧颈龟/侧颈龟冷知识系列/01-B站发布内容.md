## 评论区置顶

🐢 侧颈龟冷知识要点回顾：

🔹 第一站：缩壳方式
→ 特点：歪向一侧、L 型折叠
→ 为什么：适应沼泽环境的保命绝活
→ 日常：检查壳缝、侧面逗龟

🔹 第二站：亚目不同
→ 侧颈龟亚目：脖子侧向折叠
→ 曲颈龟亚目：脖子 S 型垂直向下
→ 区别：相当于猫和狗那么大

🔹 第三站：冈瓦纳三兄弟
→ 非洲派：西非泥龟、巨头、沼泽
→ 南美派：蛇颈龟
→ 澳洲派：澳洲短颈龟

🔹 第四站：蛇颈龟
→ 脖子几乎等于体长
→ 南美沼泽捕食绝活

🔹 第五站：腹甲可闭合
→ 非洲泥龟、巨头、澳洲短颈龟
→ 腹甲铰链、自带"门锁"

🔹 第六站：加温过冬
→ 热带品种无冬眠本能
→ 必须配加热棒 + 温控
→ 水温维持在 25–28 度
