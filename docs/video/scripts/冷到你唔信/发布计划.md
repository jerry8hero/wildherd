## 排期逻辑

- **第1-2周**：先推顶流，建立账号调性（大熊猫、科莫多巨蜥、霸王龙、内陆太攀蛇）
- **第3-4周**：强化经典动物印象（虎、尼罗鳄、翻车鱼）
- **第5-6周**：加入爬行动物特辑（湾鳄、象龟）
- **第7-8周**：加入小众和新面孔，避免审美疲劳

---

## 周更节奏建议

```
周一：哺乳动物（温血，毛茸茸有亲切感）
周三：爬行动物（冷血，冲击感）
周五：恐龙/鱼类（话题性强，适合周末观看）
```

## 爆款物种清单（后续可选）

### 哺乳动物
- 大熊猫、北极熊、虎、狮子、非洲象、长江江豚、雪豹、金丝猴

### 爬行动物
- 科莫多巨蜥、尼罗鳄、湾鳄、内陆太攀蛇、黑曼巴蛇、加拉帕戈斯象龟

### 恐龙
- 霸王龙、腕龙、三角龙、剑龙、甲龙、奇翼龙

### 鸟类
- 巨嘴鸟、企鹅、猫头鹰、极乐鸟

### 鱼类
- 大白鲨、鲸鲨、翻车鱼、电鳗、蝠鲼、七彩神仙鱼

### 两栖动物
- 箭毒蛙、娃娃鱼、变色蛙

### 无脊椎动物
- 章鱼、箱形水母、蜜蚁、巨人捕鸟蛛
