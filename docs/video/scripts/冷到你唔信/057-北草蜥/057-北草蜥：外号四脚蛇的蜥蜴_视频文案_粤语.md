# 冷到你唔信 - 北草蜥

今日继续冷到你唔信！

好，我哋从头讲起。今日吹下水，讲只超级神奇嘅蜥蜴——北草蜥！

你可能話，蜥蜴我见得多啦，唔就四脚爬爬虫？你如果係咁諗，就晒你啦。北草蜥可唔係你想象中嗰种普通蜥蜴，佢竟然可以喺中国北到吉林、西到甘肃、华南地区都搵到佢！而且係中国特有品种——即係净係中国先有，外国想养都养唔到。

你知唔知北草蜥最大嘅特点係咩？就係条尾劲长！係几长？可以达到头体长度嘅2到3倍！即係如果头加身子係10厘米，条尾可以去到20甚至30厘米——比例夸张过好多减肥人士嘅梦想。

但係，如果讲到尾长，南草蜥就更离谱——佢条尾可以去到头体长度嘅3倍以上！南草蜥表示：比唔过你，我就超你。

讲到北草蜥，就唔得唔提佢个花名——四脚蛇。你望真啲，个样真系似蛇多啲啰，长长扁扁嘅身躯，但係又生埋四条腿——就似蛇仔著咗四条裤咁，好嘢。

亦因为呢个样，好多人见到佢都话"哇，四脚蛇！"但係呢个唔係蛇嚟嘅，虽然似，但係四肢健全，係正正经经嘅蜥蜴。

你知唔知北草蜥有几快？劲！佢哋有"风一般嘅速度"，抓猎物或者走佬都好敏捷。就似你楼下隻曱甴，一见到就飙走——但北草蜥快得过曱甴添！

呢种速度唔单止用嚟捕食，仲用嚟求生。因为条尾实在太长，好容易被人捉住——所以佢哋识得自己断尾！一被咬住，条尾就自动断开，佢就趁机走佬。呢个就係大自然嘅"金蝉脱壳"——丢车保帅，生存智慧。

讲到保护级别，北草蜥2000年就被列入《国家保护的有益的或者有重要经济、科学研究价值的陆生野生动物名录》——即係三有保护动物。

但係问题嚟啰，喺部分地区，有人认为"四脚蛇"可以制成药酒——呢个就係佢哋生存最大嘅威胁。不幸嘅係，虽然有保护名录，但係误解仍然存在。

各位，跟住落嚟讲嘅，就係北草蜥最犀利嘅生存技能。

北草蜥最犀利嘅唔单止係速度，仲有佢哋嘅适应能力——从吉林到甘肃到华南，祖国大江南北都有佢哋嘅足迹。呢种"全国都去得"嘅分布，证明了佢哋係几咁百搭。就似你嘅朋友，永远都话"我边度都去得"，但係永远都唔会主动约你。

但係正因为分布广，见到嘅人多，关于佢嘅误解都特别多。边度最多四脚蛇嘅传说，就边度最多奇怪用法——呢个就係"名氣大嘅代价"。
