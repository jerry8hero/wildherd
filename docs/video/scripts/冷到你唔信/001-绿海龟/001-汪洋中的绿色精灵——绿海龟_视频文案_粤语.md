# 冷到你唔信 - 绿海龟

各位，今日开新系列——冷到你唔信！

呢个系列係搞咩嘅？就係帮你哋讲下爬行动物入面嗰啲超冷门知识。啲咩神奇习性、犀利技能、骑呢名字，保证你听完眼都碌埋，造物主真系乜都谂得出！

好，我哋从头讲起。今日吹下水，讲只超级神奇嘅海龟——绿海龟！

你可能話，海龟我见得多啦，唔就绿色嘅咩？你如果係咁諗，就晒你啦。绿海龟可唔係你想象中嗰种普通绿色海龟，佢竟然可以落到几十米甚至上百米深！而且用肺呼吸噶，每隔一阵就要浮上嚟换气——嘢，呢个几攰啰，唔知以為佢做紧咩大工程。

更离谱嘅係，绿海龟一辈子几乎净係食素！餐餐都係海草同大型海藻，食到个身都变晒绿色——但呢个绿唔係外壳绿色，而係里面嘅脂肪同组织都染成淡绿色，呢个先係"绿海龟"嘅真正由嚟。呢个大概就係最健康嘅饮食方式啰，食吓食吓就将自己变成"绿色食品"。

你知唔知绿海龟可以潜几深？几十米甚至上百米！普通潜水员落到三四十米就劲啰，佢呢？两三倍啰。虽然潜得深，但係爬行动物嚟嘅嘛，始终要用肺呼吸，所以每隔一阵就要上水面换气。就似你打机打到兴起，突然要比人呉走——绿海龟心想：我就想做条安静嘅深海王者，點解要咁对我！

你知唔知？绿海龟係国家一级保护野生动物！喺我哋国家被评为一级，梗係有料到啰。

讲到食素，绿海龟就係代表。

佢餐餐都係海草同大型海藻，严格素食主义者。食得多绿嘢，叶绿素喺身体入面积聚，慢慢噉脂肪同组织都染成淡绿色。个名入面嘅"绿"，就係实打实食出嚟嘅！但唔好以为佢斋听唔打得喎，喺海里，佢游起身快过好多鱼！仲可以活到60到80岁——呢个就係养生之道喇：清淡饮食，保持心境开朗，想唔长寿都几难。

你知唔知？绿海龟大个咗之后，里面变晒样！

出世嗰时，背甲係黑色噶，黑蚊蚊咁，似块炭咁。等佢慢慢大个，因为开始食多咗海草，内脏同脂肪就慢慢染成淡绿色——但外壳颜色其实都係棕绿色为主，唔系完全变绿，不过呢个唔关公乸事嘅，绿海龟无论公乸都会咁样㗎。

所以下次见到黑色小海龟，千祈唔好认错，佢係绿海龟嘅"童年黑历史"！

讲到绿海龟最犀利嘅能力，就梗係导航——动物界嘅 GPS 就係咁劲！

每年繁殖季节，绿海龟就会从茫茫大海入面，精准噉游返自己出世嘅嗰片沙滩度产卵。係几劲？就似你喺广东，突然间唔洗导航就知边间係你楼下嘅奶茶店——而且係知到门牌号咁！

科学家觉得佢主要靠地球磁场嚟导航，兼夹会靠埋嗅觉同视觉地标，但呢种与生俱来嘅能力的确劲。

想象下，你喺海里游咗二三十年，有一日，凭住记忆入面某个神秘信号，你开始往某个方向游，历经千辛万苦，终于揾到你出世嘅沙滩——然后掘个窿，落完蛋，又游走。几浪漫。

各位，跟住落嚟讲嘅，就係绿海龟最刺激嘅生存故仔。

刚破壳嘅小海龟，想喺窿入面爬出嚟，要花3至7日！一个蛋入面嘅小生命，要用好几日先整通条逃生通道。

好唔容易爬出嚟，真正嘅挑战先开始——海岸上面全部都係等住开饭嘅捕食者！海鸟、螃蟹、各种肉食动物，全部盯实呢群刚出世嘅小家伙。小海龟想保命，只有一个方法：搏命往海里冲！

呢个时候佢会循住海浪声，锁定方向，然后——冲！浪花一嚟，就搏命往外海游。小家伙可以连续游24个钟唔停！游得越远越安全，到咗外海，捕食者先追唔上。
