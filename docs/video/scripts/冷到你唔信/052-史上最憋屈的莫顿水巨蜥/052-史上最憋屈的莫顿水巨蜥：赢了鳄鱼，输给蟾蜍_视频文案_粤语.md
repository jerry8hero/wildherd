# 冷到你唔信 - 莫顿水巨蜥

今日继续冷到你唔信！

好，我哋从头讲起。今日吹下水，讲只超级犀利嘅巨蜥——莫顿水巨蜥！

你可能話，巨蜥喔，唔就系放大版嘅四脚蛇？你如果係咁諗，就晒你啦。呢只嘢你可以得罪讲，係澳洲北部最恶死嘅水中霸主——鳄鱼见到佢都要忌三分。但係，佢竟然输咗俾一只癞蛤蟆？你没听错，就係蔗蟾蜍。呢个就係莫顿水巨蜥最离谱嘅生存故事。

你知唔知莫顿水巨蜥几傍派对？佢哋住喺澳洲北部，鍾意游水多过喺陆地度荡。普通巨蜥晒太阳搵块石头就得，呢只嘢就连做日光浴都要搵块浮木或者石头浸喺水度——十足十温泉爱好者。体长唔超过1米，唔算巨蜥入面最大只，但係游水身手就劲到呢，一转眼就唔见咗佢。

讲到打交，莫顿水巨蜥就係典型嘅得闲死人。

佢係超级凶悍嘅捕食者，鱼、两栖、鸟、哺乳动物，统统都係佢嘅菜。唔好以为佢细细只就没什么威胁性，佢咬落去力大无穷，一嘢就搞掂。有片拍到佢哋杀掉年幼嘅淡水鳄鱼——注意喔，係鳄鱼！咁大只嘢都搞掂，犀利到呢。

但係，牛皮唔系吹嘅，火车唔系推嘅，佢就係输得咁冤。

1935年，蔗蟾蜍引入澳洲，本来系想对付蔗园入面嘅害虫。点知？蟾蜍食多咗唔单止大只咗，重分泌毒素。莫顿水巨蜥唔知死穴喺边度，照食可也——食完就中毒身亡。被蟾蜍毒死成为近百嚇年最常见嘅死因之一，係呀，就系咁样慢慢被毒到绝种边缘。你话阴唔阴功？

呢个就係大自然教我哋嘅一课：唔系大只就恶晒，有时最危险嘅敌人唔系你面前条大嘢，而系你身后嗰只衰嘢。

各位，跟住落嚟讲嘅，就係莫顿水巨蜥最讽刺嘅生存之道。

莫顿水巨蜥游水犀利过鳄鱼，咬嘢劲过蛇，但係偏偏就唔知蔗蟾蜍有毒。呢个大概就係进化嘅嘨钱——佢祖先嗰时冇呢种蛤蟆，仔仔哋唔知咩叫套路。

所以下次见到佢，记得提醒下佢：只蛤蟆唔系咁易嘢！
