# 冷到你唔信 - 五趾双足蚓蜥

各位，今日继续冷到你唔信！

我哋今日讲只超级骑呢嘅物种——五趾双足蚓蜥！

你脑海中可能即刻浮现：係蜥蜴呀嘛？有啲似变色龙个种？如果你係咁諗，就大错特错啰！呢只嘢身体似蚯蚓咁又长又粉嫩，但係行行下突然有两只细脚仔伸出来，每只脚上面重有5只萌萌嘅小脚趾——係5只呀！见到佢你真系会諗：究竟係蚯蚓着咗一对拖鞋出嚟，定係蜥蜴去咗韩国整咗个缩骨手术？

你知唔知？呢只嘢唔係蜥蜴，亦都唔係蚯蚓，而係蚓蜥家族嘅成员——爬行动物嚟嘅！

对，你无听错，爬行动物。但係呢个家族就犀利啰，既唔係蜥蜴科，又唔係蛇科，自己一个科就叫做蚓蜥目。佢哋嘅祖先大约喺7000万年前就已经出现，经历过恐龙大灭绝都生存到今日——呢个先至係真正嘅老前辈。

讲到佢哋最得意嘅地方，就梗係对生仔嘅脚仔。每只后脚有5只脚趾，隻隻分明，睇起身就似迷你版嘅人类BB手仔——但係得两只，仲要好细只。科学家都觉得呢对脚仔几靓仔，几得意，但係实际用途就......呃，其实行路都係靠蠕动为主，脚仔只係辅助嘞。

你知唔知？五趾双足蚓蜥係点样行路嘅？

呢只嘢喺地下蠕动起身就似蚯蚓咁，一伸一缩咁样推进。但係！佢偶尔会伸出嗰对细脚仔嚟撑下地面，就似你瞓地下阵时想撑起身咁——但係呢对嘢就真係有几亿年历史嘅"撑地神功"。因为后腿已经完全退化晒，所以佢哋基本上係靠身体蠕动嚟移动，脚仔只係偶尔用嚟转向或者爬过障碍物。呢个就係点解佢叫"双足"——两对脚剩低一對，但係用嚟撑嘢唔係行路。

各位，跟住落嚟讲嘅，就係五趾双足蚓蜥最得意嘅生存技能。

全身粉嫩粉嫩，睇起身就似一条活生生嘅腊肠——但係唔好以为佢斋睇唔打得喎。喺地下，佢可以靠身体一节节咁蠕动嚟穿泥土，呢种技能就叫做"蠕动推进"。佢哋嘅鳞片好光滑，黏土见到都自动让路——係几爽呢，就似你喺健身房做完facial，滑一滑咁！

但係最估你唔到嘅係，佢哋嘅食谱入面竟然有蚯蚓！係蚯蚓呀！同自己样成一模一样的蚯蚓！呢个就似你突然发现有人锺意食素，但係佢自己就係素菜——究竟係乜嘢心态呢？科学家都觉得离之大谱，但係自然界就係咁残酷：基因唔同，就算样衰衰地都要照食。

你知唔知？五趾双足蚓蜥既係蚯蚓嘅天敌，又係蜥蜴嘅猎食者——就係咁都得！

佢哋嘅英文名叫"two-legged worm lizard"，直译就係"两条腿嘅虫样蜥蜴"。但係你实际见到佢嘅时候，你一定会问：两条腿？係呀，係有两只细脚仔，但你真系会忍唔住问：得两只是唔係整走咗呀？呢个就係大自然讲笑嘞：留两只就係为咗等你觉得佢可爱，其实佢根本係靠成条身蠕过去——呢个就係生物界嘅"呃人嘢"。

所以下次见到粉红色、似蚯蚓又似蜥蜴嘅嘢，千祈唔好话係蚯蚓——佢係有脚嘅爬行动物，虽然只係得两只，虽然几乎唔使用，但係真系有㗎！
