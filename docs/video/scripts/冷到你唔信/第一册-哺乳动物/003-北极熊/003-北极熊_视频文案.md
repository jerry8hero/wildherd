
# 冷到你唔信 - 北极熊

兄弟们！如果我话，有只熊一巴掌可以拍晕成吨海象，900公斤体重撞埋你，你会点？一巴掌KO！呢只北极圈嘅"白色巨熊"，白雪雪噉望落去，善良到不得了？呃你嘅！今日带你去片——睇清呢只勥到震嘅顶级杀手！

但系！你真系识北极熊？Too young too simple！你印象中嘅北极熊——白色、大只、可爱？错晒！北极熊竟然唔系白色！佢嘅毛发係透明中空嘅管状结构，光线喺里面折射，先至睇落白雪雪。至于皮肤？黑色！你望下佢哋眼眶、鼻头、耳仔入面——黑色皮肤清晰可见！所以北极熊其实係一只"黑熊"，你被骗咗几耐先？

更离谱嘅係：呢只"黑熊"重係地球上最大只嘅陆生食肉哺乳动物！成年雄性轻松超过500公斤，唔信？你去搵十几个人叠埋一齐，就差唔多咁重！巴閉唔巴閉？

你話："500公斤大只仔我见多咗，劲就劲喎？"——劲！劲到离谱！北极熊可以跑到40公里一小时，仲犀利嘅係，连续游超过300公里！对，你冇听错，三百公里！相当于广州游到深圳再游回去！佢哋闭气超过两分钟，喺零下四十度嘅冰海里面潜水揾嘢食。嗅觉劲到可以闻到1公里以外嘅海豹——就似你坐喺客厅闻到隔篱屋厨房紧喺整咩餸！

最残忍嘅真相——气候变暖令北极熊食物越来越少。以前日日系高质量高脂肪嘅海豹大餐，而家被迫要食生果、鸟蛋、垃圾桶。就似你本来日日系牛扒鱼生，突然要转去斋——生存难度直接拉满……

呢个就係北极熊最残酷嘅生存故事：最大只、最劲，但系最脆弱。
