# 冷到你唔信 - 南非穿山甲

兄弟们！知道全球走私量最大嘅哺乳动物係边个吗？就係呢只满身鳞片、识用两条腿跑路嘅——南非穿山甲！受威胁时卷成球，係，但真正劲嘅係——佢係唯一会用两条后腿加速嘅穿山甲！

南非穿山甲因发现于非洲南部而得名，主要分布在东非同中非地区。钟意喺灌木林地或者草地出没，独居动物，日头匿喺洞穴休息，一到夜晚就变身成为白蚁同蚂蚁嘅终极噩梦！

讲到搵食，呢只穿山甲真係犀利到爆！佢会用条长到离谱嘅鼻子四处搜寻，一旦发现蚁穴——就伸出又细又长嘅舌头直插进去！你知唔知？呢条舌头连体内部分算埋足足有70厘米长！边舔边吸疯狂输出，一次进食最多可以吞下上百只蚂蚁或白蚁！白蚁界嘅顶级杀手，呢个称号真係名不虚传！

但最夸张嘅係——南非穿山甲係唯一可以两足行走嘅穿山甲！平时会用四条腿慢慢行，但必要时会微微抬起身体，用两条后腿快步跑路！冇错，佢唔爬，佢係跑嘅！放弃爬行用后腿加速追敌——呢个就係南非穿山甲嘅生存之道！

想象下，你係一只穿山甲，但係你唔爬，你跑，用两条后腿喺非洲草原上冲刺——呢个画面係唔係几好笑但又几犀利？

总结时间！非洲特有穿山甲，舌头70厘米，一次食上百只蚂蚁，唯一两足行走的穿山甲——真正嘅穿山甲界跑步高手！
