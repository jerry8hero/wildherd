# 冷到你唔信 - 南非穿山甲

南非穿山甲——全球走私量最大嘅哺乳动物！但呢只鳞片嘅家伙，受到威胁时会卷缩成球添！

好喇兄弟們！今日呢期嘅主角，识两条腿行路嘅穿山甲——南非穿山甲！你以为佢净系一只普通嘅鳞片动物？但係啊，呢只嘢劲到你唔信！

南非穿山甲系另一种非洲特有嘅穿山甲——因发现于非洲南部而得名，东非同中非地区都有分布！

南非穿山甲钟意喺较多灌木嘅林地或草地活动——生活区域通常离水源较近！独居生活——日头躲洞穴休息，夜晚就出外捕食白蚁同蚂蚁！

不过呢，南非穿山甲搵食好犀利——用鼻子搜寻食物，找到蚁穴后就把又细又长嘅舌头伸入洞穴！（算上体内睇唔见部分足足有70厘米！）边舔边吸地大食一顿——一次进食最多可以吞下上百只蚂蚁或白蚁！

但係啊，最得意嘅系——南非穿山甲系唯一可以两足行走嘅穿山甲！前肢短小——必要时会微微抬起身体，用两条后腿走路上！为咗提高速度——真正嘅双足步行者！

想象下，你系一只穿山甲，但系你唔爬，你跑——用两条后腿快步走，呢个就係南非穿山甲嘅犀利之处！

所以呢，南非穿山甲就系穿山甲家族嘅异类——唔止识行，重识跑！

呢个就係南非穿山甲最犀利嘅生存之道：非洲特有穿山甲！舌头70厘米，一次食上百只蚂蚁！唯一两足行走嘅穿山甲——为提速用后腿行！真正嘅穿山甲界跑步高手！
