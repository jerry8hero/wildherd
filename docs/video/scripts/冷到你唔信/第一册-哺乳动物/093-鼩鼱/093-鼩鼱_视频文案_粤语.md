# 冷到你唔信 - 鼩鼱

鼩鼱，体型似手指咁大，但鼻哥就占咗个头嘅三分之一！鼻哥大到离谱，唔讲以为外星生物。样衰衰地似老鼠，但佢哋其实係食虫嘅，唔关老鼠亲戚！而且你知唔知？牙仔带毒液，被咬到痛到你原地爆炸！
