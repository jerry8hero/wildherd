# 冷到你唔信 - 鼩鼱

【开头】
停！你以为呢只似老鼠唔似老鼠嘅怪咖，净系普通害虫？但佢居然係地球上最原始嘅哺乳动物之一——存在咗整整6500万年，见过恐龙生恐龙死！而佢哋日日嘔紧自己体重三倍嘢，新陈代谢快到似台发动机——佢，就係鼩鼱！

【正文】
鼩鼱，体型似手指咁大，但鼻哥就占咗个头嘅三分之一！鼻哥大到离谱，唔讲以为外星生物。样衰衰地似老鼠，但佢哋其实係食虫嘅，唔关老鼠亲戚！而且你知唔知？牙仔带毒液，被咬到痛到你原地爆炸！

再爆你知，鼩鼱视力劲差，几乎係盲嘅，日常全靠声波定位——唔讲以为佢係蝙蝠亲戚！為咗维持超高体温同爆肝般嘅新陈代谢，佢哋每日要食自身体重80%到200%嘅嘢，一日唔食嘢分分钟瓜柴，補習都補唔切！

【结尾】
离谱嘅冷知识，你中意吗？中意嘅记得俾个掂同埋关注我，下期仲有更加离谱嘅动物冷知识等住你！有边只动物你想知嘅，评论区踢我，下期劲嘢预告！拜拜~

### 结尾优化亮点

| 原有问题 | 优化方案 |

### 顺口溜设计说明

```
四句短韵 · 押「啊」韵 · 便于传播
```

| 句 | 设计意图 |
|---|----------|
| 毒蛇绕道，我最大 | 呼应开场悬念，形成首尾闭环 |
| 亿年狠活，鼩鼱家 | 强调进化地位 + 点题收束 |
| 记住了就点赞 | 自然引导点赞，不生硬 |
| 咱们下期见 | 悬念预告，引导关注 |
