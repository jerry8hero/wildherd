# 冷到你唔信 - 鸭嘴兽

鸭嘴兽——哺乳动物嘅叛徒！但你知唔知，佢既似鸭又似兽，母体竟然有泄殖腔，而且细仔係从蛋入面孵出来添！顺带提一提，

好喇兄弟們！今日呢期嘅主角，澳大利亚最原始嘅哺乳动物——鸭嘴兽！你以为佢净系一只得意嘅水鸭？但係啊，呢只嘢劲到你唔信！

鸭嘴兽因拥有鸭子一样扁平嘅嘴巴而得名——系世界上最原始嘅哺乳动物之一！拥有蹼状嘅脚，身体结构非常适合游泳同潜水！

不过呢，鸭嘴兽喺水下活动时会自动闭合眼睛同耳朵——用布满神经嘅嘴巴搜寻食物，以昆虫卵、小型甲壳类等无脊椎动物为主食！

但係啊，最犀利嘅系——鸭嘴兽系产卵嘅！

作为最古老嘅哺乳动物，鸭嘴兽保留好多原始特征——后代系从蛋壳入面孵化出来嘅——似爬虫类多过似哺乳动物！

重犀利嘅系——鸭嘴兽系少数会用毒嘅哺乳动物！脚踝后侧长有毒刺——主要用喺同同伴争斗时，也可以攻击猎物同抵御天敌！

想象下，你系一只哺乳动物，但系你生蛋、仔哺乳、有泄殖腔、重识用毒——鸭嘴兽就係动物界嘅「四不像」！

所以呢，鸭嘴兽就系集哺乳动物、爬虫类、鸟类特征于一身嘅奇异生物——真正嘅活化石！

呢个就係鸭嘴兽最犀利嘅生存之道：澳大利亚最原始哺乳动物！嘴巴似鸭、脚有蹼！产卵孵化后代——蛋生哺乳动物！脚踝有毒刺——会用毒！真正嘅活化石！
