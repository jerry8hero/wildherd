# 冷到你唔信 - 西猯

【西猯】呢個名你聽過未？如果話比你知，佢似豬但完全唔係豬，你信唔信！？更估你唔到嘅係——佢哋嘅撩牙竟然向下生長！呢個冷知識，99%嘅人都唔知！！

好啦兄弟姊妹萌！今日嘅主角，外表似豬但實際上唔係豬——就係西猯！呢個得意嘅小可愛，真係勁到你唔信！

喺美洲大陆上，生活住4種「似野豬」嘅動物，佢哋統稱為「西猯」，歸屬於西猯科——係豬類動物家族嘅遠房親戚，但唔係豬喔！聽清楚未！？

🔥 西猯同普通豬最大嘅區別就係：

**第一，隻牙！**
豬豬嘅撩牙向上或者向兩側生長，而西猯嘅上犬齒竟然向下生長！完全相反！呢個設計估計連老天爺都覺得有創意😂

**第二，身形！**
西猯身形細細隻，最大都唔過50公斤，睇上去完全唔似野豬咁猙獰，反而有啲可愛

**第三，花名——臭鼬豬！**
點解叫臭鼬豬？因為西猯身上有腺體，會分泌類似麝香嘅刺鼻氣味！但呢種味道唔係用嚟防禦天敵，而係用嚟：
👉 標記領地——「我喺呢度，你唔好行埋嚟」
👉 同類交流——互相確認身份

**第四，好團結！**
西猯係群居動物，一齊搵食、一齊湊BB，集體意識拉滿！佢哋靠氣味分辨自己人——唔同群體聞到陌生氣味就會自覺保持距離，減少衝突！就係咁smart！

想象下：你喺美洲森林入面，周围充满陌生氣味，但只要碌起身周围走，就知道你係邊個群體、邊個係自己人——呢個就係西猯專屬嘅社交密碼！
