# 冷到你唔信 - 西猯

西猯——似猪但唔系猪？但你知唔知，佢哋嘅獠牙係向下生长，而唔系向上添！

好喇兄弟們！今日呢期嘅主角，生活喺美洲大陆，外表似猪但系又唔系猪——就系西猯！你以为佢净系似猪？但係啊，呢只嘢劲到你唔信！

喺美洲大陆上，生活着4种似野猪一样嘅动物，佢哋被统称为「西猯」，归属于西猯科——系猪类动物家族嘅成员，但唔系猪喔！

西猯同我哋熟悉嘅猪有乜唔同？

最大区别就系只牙！

虽然都有长獠牙，但系西猯嘅上獠牙系向下生长嘅——唔似我哋熟悉嘅野猪咁样向上或者向两侧生长！而且西猯身形几细只——最重都不过50千克，所以睇上去冇野猪咁狰狞恐佈！

不过呢，西猯有个花名叫「臭鼬猪」！

点解？因为西猯身体表面嘅腺体会散发出类似麝香嘅刺鼻气味！但系呢种气味唔系为咗抵御天敌，而系用作同类间划定领地、相互交流嘅工具！简单啲讲，就系「我喺呢度，你唔好行埋嚟」呢种意思！

但係啊，西猯虽然细只，但系都几团结！

佢哋通常会成群生活，一齐搵嘢食、一齐照顾BB，群体意识几强！有研究显示，西猯群之间会通过气味嚟分辨彼此，唔同群体之间就会保持距离，减少冲突！

想象下，你喺美洲森林入面，周围都系陌生气味，但系只要你碌起身周围走，就知道你系边个群体、边个係自己人——呢个就係西猯嘅社交方式！

呢个就係西猯最犀利嘅生存之道：美洲「似猪非猪」嘅动物！上犬齿向下长，同野猪完全相反！身面有腺体分泌气味，味道似麝香——叫「臭鼬猪」！气味用来标记领地，唔系防御！身形细只，50千克就封顶！但系好团结，群居生活通过气味分辨同伴！

下期我哋讲黑麂——食肉的鹿！记得三连，我哋下期见！
