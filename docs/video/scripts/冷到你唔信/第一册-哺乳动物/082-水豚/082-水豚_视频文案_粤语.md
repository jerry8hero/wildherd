# 冷到你唔信 - 水豚

水豚——最大嘅啮齿动物！但你知唔知，佢哋个名『水豚』源自印第安语，意思系『四只牙嘅主人』添！顺带提一提，

好喇兄弟們！今日呢期嘅主角，世界最大嘅啮齿动物——水豚！你以为佢净系一只大只嘅老鼠？但係啊，呢只嘢劲到你唔信！

水豚系世界上体形最大嘅啮齿动物——成年后体长可达1.3米，体重可达50千克！系南美洲嘅特有物种！

水豚个样几得意——方头方脑，身体粗壮，四肢较短，尾巴严重退化，几乎睇唔见！总体睇上去就像一只冇尾巴嘅大胖老鼠——几可爱！

不过呢，水豚以家族为单位生活——群体规模喺10到30只！钟意喺靠近水嘅地方活动——河流、湖泊、沼泽、湿地都係佢哋嘅地盘！

水豚有几样得意嘢：第一，唔擅长挖洞——累了就搵现成嘅土坑休息！第二，受惊时会跳进水中——迅速游到水下植物密集地方躲避！

但係啊，有个得意point——水豚排便时，好多小动物会喺不远处等待——佢哋会饮水豚嘅粪便，从中摄取粗蛋白！所以水豚又被叫「动物界嘅自助餐」——真正嘅共享经济！

想象下，你喺河边睇水豚，其他动物走过来等水豚排便——然后一齐食......呢个就係水豚嘅共享经济！

呢个就係水豚最犀利嘅生存之道：世界最大啮齿动物，50千克、1.3米！南美特有！方头方脑似大胖鼠！家族10-30只生活！其他动物食佢粪便——共享经济！真正嘅和平使者！
