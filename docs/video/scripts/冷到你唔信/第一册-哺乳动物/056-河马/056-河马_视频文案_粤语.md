# 冷到你唔信 - 河马

河马——嘴最大嘅哺乳动物！但呢只肥宅，虽然睇落肥笨，跑起身可以快过人类添！

好喇兄弟們！今日呢期嘅主角，唔好睇佢肥仔拖拖，但系识得「流血」——河马！你以为佢净系一只钟意浸水嘅肥嘢？但係啊，呢只嘢劲到你唔信！

河马系非洲特有动物，平均体重可达2吨——系仅次于大象同白犀嘅非洲第三大陆生动物！顺带提一提，虽然系食草动物，但系河马口中有巨大而锋利嘅犬齿——可以用于自卫同格斗！所以唔好惹呢只肥嘢！

不过呢，河马识「流血」？几恐怖？

其实呢个系为咗生存——河马生活在酷热嘅非洲，防晒系非常重要！但系河马没有汗腺，毛发都寥寥无几——点算？

河马嘅黏液腺会分泌一种黏稠液体——呢种液体中含有红色色素！排出体外后一段时间会呈现红色——就似流血咁样！但系呢个唔系血，而系天然防晒霜！河马用呢种液体有效避免紫外线对皮肤嘅伤害——就系佢哋嘅生存智慧！

但係啊，天然防晒霜同人类嘅一样，保护都系暂时性嘅——所以河马大多数时间都中意待喺水里面——避太阳啰！

想象下，你系一只河马，喺非洲大太阳下行嚟行去——但系你有天然防晒霜，黏黏稠稠叹出去，红色液体保护你皮肤！几醒！

呢个就係河马最犀利嘅生存之道：识「流血」嘅肥嘢！但系红色液体系天然防晒霜，并非血液！黏液腺分泌、含红色色素、可以防晒！虽然体重2吨、牙又利，但系主要系为咗自卫！大多数时间浸水——防晒妙招！

下期我哋讲一角鲸——牙似角嘅独角鲸！你唔俾个赞唔紧要，但系记得留意下期嘅精彩内容喔！
