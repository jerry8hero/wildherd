# 冷到你唔信 - 貘

貘——长得像猪像象，但居然是马的亲戚？这冷知识我能吹一辈子！

家人们！今天这个主角，丑到离谱但又萌到心坎——鼻子像大象、身子像猪，但你敢信？它居然是马！的！亲！戚！

对，就是那个在草原上狂奔的马，居然跟这货是一家人，惊不惊喜意不意外？

来，隆重介绍一下这位“丑萌界”的天花板——貘（mò）！

这货主要混迹在东南亚和中南美洲的热带雨林，长得确实抽象，但人家可是正儿八经奇蹄目动物，跟马一个目！

全球共5种貘：

亚洲的马来貘，体型最大，成年后那黑白配直接绝了——浑身漆黑，就脸、耳朵、腿是白的，活像穿着白色连体袜摔了个屁股蹲！这撞色审美，直接让设计师怀疑人生。

另外4种都在美洲，毛色就朴素多了，毕竟走的是低调路线。

但重点来了！别看这货圆滚滚的，游泳可是 yyds！

它们超爱泡在水里，水生植物当饭吃。遇到美洲豹追杀？直接一个猛子跳河里游走！美洲豹：我真的会谢！

这波陆地拉胯、水里开挂的操作，直接把天敌气得原地爆炸！

而且貘虽然是个独行侠，但会用口哨般的叫声互相联系——堪称雨林版加密通讯！

来，给你盘一盘：

似猪似象，却是马的亲戚——离谱吧？
黑白配的马来貘，动物界穿搭天花板——绝了吧？
独居社恐，但有专属口哨暗号——神秘吧？
游泳满级，水中躲避天敌——离谱吧？

这反差感，这隐藏技能，不关注说不过去吧？
