# 冷到你唔信 - 貘

貘——似猪又似象？但你知唔知，貘係亚洲野生哺乳动物入面最古老嘅科之一添！

好喇兄弟們！今日呢期嘅主角，样貌似猪又似象，但系——竟然系马嘅亲戚！就系貘！你以为佢净系一只奇怪嘅猪？但係啊，呢只嘢劲到你唔信！

貘主要生活喺东南亚同中南美洲嘅热带雨林及山地。样貌似猪，鼻子大而突出——但系呢两种动物完全唔同！

不过呢，虽然长得似猪同象，貘却同马系近亲——两者都属于奇蹄目！

貘有5种：

亚洲有马来貘——系体型最大嘅貘，成年后拥有黑白两种毛色！顺带提一提，其余4种都生活喺美洲——毛色都只系一种颜色！

但係啊，马来貘最得意——身体黑色，但系面部、耳朵、腿部都系白色！就好似着住一件白色外套咁样，睇落去几得意！

想象下，你喺雨林度行，突然有一只黑白配嘅「猪」走过嚟——但系佢唔系猪，而系马嘅亲戚，仲识游水！几得意！

而且貘通常独居生活，同伴之间除咗靠气味重会用类似吹口哨嘅叫声相互联系——几得意！

但係啊，貘有样嘢劲犀利——游水劲！

呢个技能令佢哋能够以水生植物为主食——重能够下水躲避天敌！主要天敌系美洲豹——但系一跳入水，美洲豹就追唔上啰！

呢个就係貘最犀利嘅生存之道：似猪似象但系马嘅亲戚，属奇蹄目！5种唔同类型，马来貘黑白配色最特别！独居但会用口哨声联系！游水好劲，水中逃避天敌！真正嘅雨林奇兽！
