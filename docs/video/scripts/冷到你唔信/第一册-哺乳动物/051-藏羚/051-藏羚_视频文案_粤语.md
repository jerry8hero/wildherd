# 冷到你唔信 - 藏羚

藏羚——高原精灵！但你知唔知，公嘅藏羚角係笔直，而角嘅形状似吉他添！

好喇兄弟們！今日呢期嘅主角，来自「世界屋脊」青藏高原——藏羚！你以为佢净系一只普通嘅羚羊？但係啊，呢只嘢劲到你唔信！

2008年北京奥运会个吉祥物「迎迎」，就系以藏羚为原型设计嘅——你可能唔知，但系一定见过！

虽然藏羚外形似羚羊，但系从基因上讲——佢哋同山羊、绵羊嘅关系更密切！成年藏羚体重可达42千克，雄性头顶有一对笔直朝天嘅角——系用来争夺配偶嘅武器！

不过呢，藏羚最特别嘅就系迁徙！

每年春夏时期，藏羚都会迁徙——但系有个好得意嘅Point：

迁徙时，藏羚队伍会一分为二！

所有雄性聚成一组，做短距离迁徙；
雌性则会长途跋涉，一路北上到达可可西里腹地嘅卓乃湖、太阳湖一带——喺嗰度生下小宝宝！

点解要分开？

为咗避免喺一个区域内过于集中，导致食物匮乏——分开就大家都有嘢食啰！

但係啊，藏羚现时生存状况唔太乐观。

因为非法猎杀、栖息地丧失等原因，藏羚数量曾经大幅下降——好在而家受保护，数量有恢复，但系依然属于濒危物种！

讲多个冷知识——藏羚个名有个「藏」字，但系其实佢哋唔单只西藏有，青海、四川、甘肃等高原地区都有——不过最密集就系可可西里！

呢个就係藏羚最犀利嘅生存之道：青藏高原骄傲！基因上同山羊绵羊更近！雄性有笔直朝天角用来争配偶！迁徙时男女分开——雄性短距离，雌性长途跋涉去可可西里生BB！北京奥运吉祥物「迎迎」原型！濒危物种，要保护！

下期我哋讲麝牛——北极生存高手！记得三连，我哋下期见！
