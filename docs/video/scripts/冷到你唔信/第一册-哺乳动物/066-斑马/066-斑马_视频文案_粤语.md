# 冷到你唔信 - 斑马

斑马——黑白条纹就似走进咗酒吧！但科学家发现，每只斑马嘅条纹都唔同添！

好喇兄弟們！今日呢期嘅主角，非洲草原上最sharp嘅穿着——斑马！你以为佢净系一只着住条纹衫嘅马？但係啊，呢只嘢劲到你唔信！

斑马曾系非洲特有动物，而家已引入北美并野化放归——共有3种！你知唔知？斑马嘅皮肤系黑色嘅——黑白花纹系被毛颜色唔同，底层皮肤其实系黑色！

不过呢，同马比起来，斑马更强壮——体重超过300千克！因为长期同狮子、豹、斑鬣狗呢啲猛兽打交道，佢哋嘅性格也更凶悍！

但係啊，斑马有两招绝活——踢同咬！

第一招：踢！两条后腿同时向后猛踹——呢一脚嘅力量，同中型卡车以每小时100千米速度撞击嘅力度相差无几！足以重创任何对手！

第二招：咬！斑马中意用锋利嘅犬齿进行撕咬——所以佢哋有个花名叫「老虎马」！

想象下，你系一只狮子，想追斑马——但系斑马一脚踢过来，好似被卡车撞到......算喇，唔好惹！

关于斑马条纹嘅作用，有好多研究——过去认为系保护色；后来认为系同伴间互相辨别嘅依据；最新研究显示——黑白条纹有助防止蚊虫叮咬！

呢个就係斑马最犀利嘅生存之道：皮肤黑色，花纹系被毛！比马更强壮，超过300千克！两招绝活：踢——后腿猛踹如卡车撞；咬——犬齿锋利叫老虎马！条纹最新研究防蚊虫！真正嘅草原狠角色！
