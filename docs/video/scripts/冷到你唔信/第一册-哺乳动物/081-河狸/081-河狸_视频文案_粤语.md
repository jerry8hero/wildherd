# 冷到你唔信 - 河狸

河狸——建筑师！但呢只加拿大大使，个尾可以拍水面发出警报添！

今日呢期嘅主角，尾巴似船桨嘅建筑大师——河狸！

河狸又叫海狸，系大型啮齿动物，体重可达30千克！

全世界共有2种：

一种系欧亚河狸——主要生活喺欧洲同亚洲部分地区；

另一种系美洲河狸——原产于北美地区，系加拿大嘅国兽！

河狸系半水栖动物——钟意喺湖泊、河流、池塘、沼泽等靠近水嘅地方安家活动！

河狸有几样法宝：

第一，尾巴似船桨般扁平——帮助游泳！

第二，两只长着蹼嘅后脚——划水劲！

第三，流线型身体——潜水灵活！

第四，两只强悍有力嘅前爪——挖掘能力强！

但系！河狸虽然水下灵活，喺岸上就行动迟缓——好容易成为食肉动物目标！

所以呢，河狸会将巢穴入口建喺水下——敌人就难进入啰！

河狸重系出色嘅建筑者——识得用树枝、泥土筑坝！

可以改变水流方向，保护自己个家——真正嘅水利工程师！

总结今日重点：

河狸——半水栖大型啮齿动物！尾巴扁平似船桨、后脚有蹼！巢穴入口喺水下——防天敌！识筑坝改变水流——水利工程师！加拿大国兽！
