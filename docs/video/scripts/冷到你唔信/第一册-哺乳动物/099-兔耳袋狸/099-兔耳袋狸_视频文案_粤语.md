# 冷到你唔信 - 兔耳袋狸

兔耳袋狸——耳仔似兔又似袋狸？但呢只澳洲嘅可爱小兽，尾巴可以储存脂肪添！

好喇兄弟們！今日呢期嘅主角，澳洲长耳仔——兔耳袋狸！你以为佢净系一只普通嘅兔仔？但係啊，呢只嘢劲到你唔信！

不过呢，兔耳袋狸体重可达2.5千克——系袋狸家族中体型最大嘅！同大象一样，兔耳袋狸嘅长耳朵都有散热作用——帮助喺酷热环境生存！

但係啊，为咗抵御酷热，兔耳袋狸中意昼伏夜出——日间就躲在洞穴里，避免阳光！兔耳袋狸识挖出2到3米深、拥有12个洞口嘅「地下迷宫」——为咗防备野狗或蟒蛇等食肉动物！

兔耳袋狸胃口几好——植物种子、根茎、各种昆虫、甚至更小嘅有袋类动物都食！通过从食物中摄取水分，兔耳袋狸几乎唔需要额外饮水——真正嘅沙漠节水达人！

想象下，你喺沙漠入面，日头好热，但你唔洗点水——因为你可以从食物度摄取水分！兔耳袋狸就係咁醒！

呢个就係兔耳袋狸最犀利嘅生存之道：澳洲西部特有！似兔又似鼠，长耳得名！体重2.5千克，袋狸家族最大！长耳散热、昼伏夜出！挖12洞口地下迷宫！几乎唔使饮水——沙漠节水专家！
