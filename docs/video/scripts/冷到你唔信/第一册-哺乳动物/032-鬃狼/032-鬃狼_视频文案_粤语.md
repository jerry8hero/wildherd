# 冷到你唔信 - 鬃狼

鬃狼——腿长就以为系超模？但呢只南美嘅狐狸，个名竟然就叫『牵着鬃毛嘅狼』！

好喇兄弟們！今日呢期嘅主角，个名入面有个狼字，但系性格完全唔似狼——就系鬃狼！讲多少少，你以为佢係狼？但係啊，呢只嘢完全唔係你想象中咁！

鬃狼并唔系狼，而系属于食肉目犬科鬃狼属嘅动物。

平均体长约1米，体重约24千克，嘴巴比较细长，耳朵大，毛色以红色同黄褐色为主——成只放大版嘅狐狸咁！

但系鬃狼最特别嘅地方，就系4条又细又长嘅腿——大概有1米长，係犬科动物入面最長嘅！

不过呢！你以为鬃狼系狼就一定会吃肉？错喇！

鬃狼虽然属食肉目，但系饮食以素为主——钟意食巴西狼果、番荔枝、番石榴、仙人掌、香蕉呢啲植物。

偶尔先会抓只兔仔、老鼠、昆虫呢啲小型动物开下荤，但系占比真系好少！

所以呢，鬃狼绝对系狼中嘅异类——名字入面有狼，但系系一个钟意食素嘅乖宝宝！

但係啊，鬃狼仲有一个好得意嘅特点——就系走路姿势！

唔似其他犬科动物咁样身体一侧两条腿交叉迈步，鬃狼行路係同时向前郁——就似人哋咁样「顺拐」！

呢个画面有几得意？你想象一下，成只红毛狐狸身形，但系行起路嚟呢个款，呢你就明啰！

因为实在太得意，有啲地方开始将鬃狼当宠物咁养......

但系我哋唔建议喔！鬃狼始终系野生动物，有自己嘅习性，唔适合当宠物ga！

呢个就係鬃狼最犀利嘅生存之道：唔系普通狼，而系犬科鬃狼属！身体似狐狸、四腿超长1米！最估唔到就系——属食肉目但系钟意食素！行路姿势顺拐，係狼中嘅异类！
