# 冷到你唔信 - 鬃狼

鬃狼——南美最离谱的“狼”，名字带狼但吃素，长腿逆天还会顺拐走路？？？

兄弟们！今天给你们介绍一位南美狠活嘉宾，明明叫狼但它根本不是狼！名字和长相都蹭狼的热度，结果吃斋念佛活成了“狐狸精”——没错，它就是鬃狼(zōng)！

先给你们敲个重点！鬃狼虽然名字带狼，但它既不是狼，也不在传统犬科阵营里混，人家是鬃狼属的独苗苗，学名Chrysocyon brachyurus，整个属就它一个VIP，孤独到没朋友！

外观方面，鬃狼就是一只红毛大狐狸plus版——体长1米左右，体重23-25公斤，耳朵大得像天线，毛色红褐发亮。但最离谱的是它的身材比例！那四条逆天长腿占了身高的一半还多，肩高0.8-1米，直接让鬃狼秒变犬科界的“超模”！站起来比你家衣柜还高，回头率拉满！

但是！重点来了！！！

你以为名字带狼就是肉食猛男？大错特错！！！

鬃狼虽然是食肉目出身，但吃素吃到飞起——植物性食物占比50%-70%！它们的主食是巴西狼果、番荔枝、番石榴、仙人掌、香蕉……简直就是行走的南美水果拼盘！偶尔抓个兔子老鼠昆虫打牙祭？那纯属开小灶，主要养生还是靠吃草吃果子！

所以鬃狼就是个典型的“挂羊头卖狗肉”——名字蹭狼的热度，本质是个吃素的乖宝宝！

除了吃素，鬃狼还有个名场面——走路姿势！

别的犬科动物都是身体两侧腿交替迈步，正常又稳重。但鬃狼偏不！它走路时同侧两条腿同时迈出——没错，就是传说中的“顺拐”！

想象一下这个画面：一只红毛大高个，身条模特级别，但走路咔咔咔同手同脚，像个行走的机器人……这反差感是不是已经在你脑子里播了好几遍？因为走路太有喜感，有些老外居然动起了歪心思想当宠物养……

达咩达咩达咩！！！鬃狼可是正儿八经的野生动物，有自己的生存习性和活动需求，城市鸽笼根本关不住它！喜欢就让它在野外当个快乐的“素食狼”吧，咱们远远看着就好！

好了来敲黑板总结：鬃狼不是狼，是鬃狼属的独行侠；长得像狐狸但腿超长；食肉目里最爱吃素的“叛徒”；走路顺拐的搞笑担当！
