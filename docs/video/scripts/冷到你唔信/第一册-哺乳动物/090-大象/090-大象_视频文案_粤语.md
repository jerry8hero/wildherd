# 冷到你唔信 - 大象

大象——地球上最大嘅陆地动物！但呢只记忆力惊人嘅巨兽，唔单只识哀悼死者，仲会拜访死者的骨头添！顺带提一提，

好喇兄弟們！今日呢期嘅主角，地球最大嘅陆生哺乳动物——大象！你以为佢净系一只大只嘅食草动物？但係啊，呢只嘢劲到你唔信！

大象系现今地球上体形最大嘅陆生哺乳动物！最大嘅雄性非洲草原象体重高达6300千克，最细只嘅非洲森林象都有1200千克！

但系！大就有大嘅问题——大象系唯一唔会跳跃嘅哺乳动物！

点解大象唔识跳？

第一，体重太大，跳唔起；

第二，脚部骨骼结构问题，大象脚上骨头之间缝隙好小，缺乏起跳时所需嘅弹性结构！

所以大象跳唔起几正常，而且从生活需求角度，大象身材高大，又有灵活嘅长鼻子，根本用唔着跳就能够到高处嘢！

不过呢，你知唔知？大象虽然跳唔起，但系可以跑得几快，每小时20到25千米！而且大象重识游水，天生识得浮！

想象下，你系一只大象，成日觉得自己大只唔灵活，但系你跑得几快，仲识游水！真正嘅全能运动员！

呢个就係大象最犀利嘅生存之道：地球最大陆生哺乳动物！6300千克！唯一唔会跳嘅哺乳动物——脚骨结构问题！跳唔起但跑得快、识游水！用鼻子代替跳！真正嘅巨无霸！
