# 冷到你唔信 - 狮子

99%嘅人都搞错！万兽之王竟然係“群居猫科”？！

兄弟萌！今日818呢只劲到离谱嘅百兽之王——狮子！

係讲之前我问大家一个问题：你以为狮子係森林之王？老虎？错！狮子係地球唯一群居生活嘅猫科动物！其他老虎、豹再犀利都係独来独往，唯独狮子钟意搞社团！

你以为狮子只配喺非洲草原威风？大错特错！历史上狮子足迹遍布成个非洲、亚洲、甚至欧洲部分地区！宜家除咗印度吉尔国家公园，全球野生狮子基本都集中喺撒哈拉以南——东非同南非为主。想睇野生狮子？记得去呢两个地方打卡！

最离谱嘅係，狮子嘅适应能力劲到爆！除咗草原，佢哋重可以生存喺丛林、荒漠、热带雨林等多种环境！呢啲地方其他大猫根本生存唔到，但狮子照样活得掂！

讲下体型——雄狮体重通常200-250公斤，雌狮120-180公斤！约等于一只大熊猫加一只大狗噶重量！想象下被一只咁嘅大猫扑过来几刺激？

不过最颠覆认知嘅係——好多人以为雄狮净系负责食嘢睇门口，但从近年嘅观察研究发现，捕捉水牛、河马、犀牛、长颈鹿呢啲大型猎物时，雄狮其实会落场！所谓“雄狮不捕猎”其实系误区——佢哋主要负责对付大猎物，小猎物先由雌狮负责！

狮群协作就係狮子最犀利嘅地方！正因为成员之间配合默契，狮子先至可以称霸各种环境。雌狮负责日常狩猎同湊BB，雄狮负责地盘保护同对付大猎物——分工明确，强强联合！

仲有一个冷知识——狮子分两大支系：南方狮同北方狮。南方狮就系常见嘅非洲狮，北方狮主要包括亚洲狮。唔同地区狮子体型有差异，但全部靠群居同合作生存！

想象下：你系百兽之王，但你唔系独行侠，而系靠团队打天下！有成群队友帮你捕猎、保护地盘、湊BB——就似你开咗间公司，有成个团队帮你打工，劲唔劲？

呢个就係狮子最犀利嘅生存之道：唯一群居猫科、适应多种环境、雄狮参与大猎物捕猎！
