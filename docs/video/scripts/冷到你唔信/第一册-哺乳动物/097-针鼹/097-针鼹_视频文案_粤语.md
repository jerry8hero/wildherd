# 冷到你唔信 - 针鼹

针鼹——身上长刺但唔系刺猬！但呢只澳洲嘅原始哺乳动物，細仔係从蛋生出来之后放入育儿袋添！

好喇兄弟們！今日呢期嘅主角，似刺猬但系唔系刺猬——针鼹！你以为佢净系一只普通嘅刺猬？但係啊，呢只嘢劲到你唔信！

针鼹系除咗鸭嘴兽之外，另一类产卵嘅哺乳动物——全世界共有4种，分布于大洋洲同亚洲印度尼西亚！顺带提一提，

针鼹外形同刺猬非常似——周身长满尖刺！但系唔同嘅系——刺猬嘅刺系骨头一部分，叫「骨质刺」；针鼹嘅刺系皮肤硬化产物，成分同人类指甲一样嘅角蛋白——可以脱落同再生！而且末端带有倒钩——如有「冒失鬼」敢来攻击针鼹，刺就会牢牢钩入身体！

不过呢，针鼹同食蚁兽一样中意食白蚁同蚂蚁——而且都系冇牙！但系针鼹有一样犀利嘢——舌头似管子一样细长且具有黏性，可以牢牢噃住猎物！

但係啊，想象下，你身上长满刺，但系你唔系刺猬，你系哺乳动物！你会生蛋！你有倒钩刺！几得意！

所以呢，针鼹就系靠呢条黏黏哋长舌头喺蚁穴入面摄食——真正嘅清蚁专家！

呢个就係针鼹最犀利嘅生存之道：似刺猬但唔系刺猬！刺系角蛋白、可以脱落再生！末端有倒钩——攻击者中招！细长黏性舌头——食白蚁蚂蚁！真正嘅蚁穴清道夫！

下期我哋讲袋食蚁兽——有52只牙嘅有袋类！记得三连，我哋下期见！
