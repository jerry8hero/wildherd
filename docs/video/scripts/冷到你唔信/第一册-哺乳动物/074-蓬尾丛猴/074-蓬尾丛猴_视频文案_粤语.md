# 冷到你唔信 - 蓬尾丛猴

蓬尾丛猴——条尾似松鼠？但你知唔知，佢哋可以用超声波互相倾计添！

好喇兄弟們！今日呢期嘅主角，来自非洲嘅小精灵——蓬尾丛猴！你以为佢净系一只细细只嘅松鼠？但係啊，呢只嘢劲到你唔信！

蓬尾丛猴又叫婴猴，生活喺非洲中部同南部——样貌有啲似狐猴，但系体形细好多！顺带提一提，蓬尾丛猴得15到17厘米——真系小小只！名字来源于身后那条毛发蓬松嘅长尾巴——尾巴比身体重好多！

不过呢，蓬尾丛猴系典型嘅夜行性动物——大多数时间喺树上生活！以各种昆虫为主食——系捕捉飞行昆虫嘅高手！

蓬尾丛猴靠嘅就系超强跳跃能力！可以跳出超过自己体长10倍嘅距离——两棵距离几米嘅树之间来回跳！

但係啊，蓬尾丛猴有个超级特别嘅习性——会喺四只脚上涂抹尿液！

你知唔知点解？因为咁样可以增强抓握力——便于高速跳跃后能稳稳咁降落喺树干上！

所以呢，蓬尾丛猴嘅脚除咗行路，重识用尿液增加摩擦——真正嘅天然防滑剂！

想象下，你喺树之间跳嚟跳去，但系成日滑——但系如果你用尿液涂脚，哇，抓得劲稳！蓬尾丛猴就係咁样嘅生存专家！

呢个就係蓬尾丛猴最犀利嘅生存之道：非洲中小型灵长类！体形细只15-17厘米！夜行性、以昆虫为食！跳跃能力劲——跳超体长10倍距离！会用脚涂尿液增加抓握！真正嘅跳跃大师！
