# 冷到你唔信 - 抹香鲸

抹香鲸——潜水最劲嘅哺乳动物！但呢只深海猎手，可以屏住呼吸两小时添！

好喇兄弟們！今日呢期嘅主角，系世界上体型最大嘅齿鲸——抹香鲸！你以为佢净系一只大只嘅鲸鱼？但係啊，呢只嘢劲到你唔信！

抹香鲸较大个体体长超过18米，体重可达70吨——真正嘅海洋巨无霸！但係啊，「抹香鲸」呢个名点解嚟？因为佢哋肠道内可以形成「龙涎香」——一种消化道残余物，非常珍贵，用来做香水嘅！

不过呢，抹香鲸最犀利就系潜水能力！

最深可下潜到2250米，最长可潜水80分钟——人类嘅潜艇就系仿照抹香鲸嘅身体原理制造㗎！

点解可以潜咁深？

第一，肺活量极大！

第二，身体留存氧气嘅能力强——潜水前喺水面上吸足氧气，大约一半会储存到肌肉同血液中，为身体提供能量！

但係啊，抹香鲸因为潜水能力劲，所以食物都系深海软体动物——乌贼、鱿鱼、章鱼等！其中最大嘅猎物系大王酸浆鱿——生活喺南极海域，号称可以同抹香鲸打喊路嘅巨型鱿鱼！

想象下，你喺深海2250米，周围黑蚊蚊，但系你照样可以搵到乌贼大王酸浆鱿——呢个就係抹香鲸嘅深海猎手本领！

呢个就係抹香鲸最犀利嘅生存之道：世界最大齿鲸，18米、70吨！潜水高手，最深2250米、最长80分钟！人类潜艇仿佢身体原理！食物系深海软体动物，包括巨型大王酸浆鱿！肠内有珍贵龙涎香！真正嘅深海猎手！

下期我哋讲貘——似猪又似象，但系马嘅亲戚！记得三连支持喔！
