# 冷到你唔信 - 羚牛

羚牛——中国四大国宝之一！但你知唔知，佢既不属牛也不属羊，係独特嘅存在添！

好喇兄弟們！今日呢期嘅主角，个名就叫「六不像」——羚牛！你以为佢净系一只普通嘅牛？但係啊，呢只嘢劲到你唔信！

羚牛系亚洲特有动物，共有4种，主要分布喺中国西南及喜马拉雅山脉附近嘅印度、不丹、缅甸等国境内。

点解叫六不像？

因为羚牛身体睇落似6种动物嘅合成品——

1. 角嘅形状酷似角马嘅角
2. 脸形同驼鹿接近
3. 背部有似棕熊一样凸起嘅肌肉
4. 四肢粗壮如牛
5. 站立时后腿弯曲似斑鬣狗
6. 身后有条似山羊尾巴般扁宽嘅尾巴

一句话：似呢似路，就唔知似边个——所以叫六不像！

不过呢，羚牛有个别名——扭角羚。现存动物中，佢哋同绵羊关系最近——但系脾气就比绵羊暴躁得多！曾有过顶伤大熊猫嘅记录——大熊猫都敢郁，够几恶！

但係啊，羚牛对大熊猫嘅威胁竟然大过食肉动物！

点解？

因为羚牛中意喺树上蹭角——呢个动作会导致好多树因掉皮而死亡，变成「倒木」！大熊猫唔钟意倒木，如果倒木太多，可供栖息嘅地方就少咗——间接威胁大熊猫！

想象下，你系一只大熊猫，点解最近成日冇地方住？原来系羚牛喺树上蹭角，搞到啲树死晒......呢个就係羚牛嘅间接杀伤力！

呢个就係羚牛最犀利嘅生存之道：六不像！似6种动物合成品！同绵羊关系最近但脾气更暴躁！顶伤大熊猫记录！中意树上蹭角，令树死变倒木，间接威胁大熊猫！真正嘅熊猫克星！

下期我哋讲叉角羚——擅跑不擅跳！记得三连，我哋下期见！
