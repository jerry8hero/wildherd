# 冷到你唔信 - 普氏原羚

普氏原羚——中国特有嘅精灵！但呢只青海湖畔嘅娇客，数量比大熊猫仲少添！

好喇兄弟們！今日呢期嘅主角，名字入面有个「普」字——普氏原羚！你以为佢净系一只普通嘅羚羊？但係啊，呢只嘢劲到你唔信！

普氏原羚系中国特有物种，学名嚟自最早将其标本呈现给科学界嘅沙俄探险家普尔热瓦尔斯基。如今只栖息喺青海省——真系好珍贵！

普氏原羚有几个好得意嘅特征：

平均体长约130厘米，体重25千克左右，属于中小型羚羊。最大特点就系——臀上有心形白圈！但系毛色以黄色为主，所以当地牧民俗称「黄羊」！

不过呢，普氏原羚雄性头顶仲有一对向内弯曲、角尖相对嘅角——所以又叫「中华对角羚」！呢对角唔系为咗防御，而系用来争夺配偶！

但係啊，普氏原羚有样嘢劲犀利——就系生存能力！

佢哋可以喺干旱贫瘠地区存活，至少有50种植物都喺佢哋食谱入面——其中重包括一啲有毒嘅植物！有毒都食得？唔使惊——普氏原羚已经进化出解毒能力，唔系边个都可以喺沙漠生存啰！

想象下，你喺青海高原上，周围都系干旱荒漠，但系普氏原羚照样搵到嘢食——50种植物都唔在话下，毒草都食得！真正嘅生存专家！

呢个就係普氏原羚最犀利嘅生存之道：中国青海特有！臀上有心形白圈，又叫中华对角羚！50种植物都喺食谱，甚至包括毒植物！生存能力劲到爆！濒危物种，要保护！
