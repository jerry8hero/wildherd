# 冷到你唔信 - 普氏原羚

有一种羚羊，数量比大熊猫还少！还是中国特有物种！但好多人连它名字都没听过——普氏原羚！

兄弟们！今天这位主角真的牛到离谱！名字带个“普”字——普氏原羚！你以为它就是只普通羚羊？错！它是中国独有的限定款，全世界只有青海有！

这货有啥特别？先说外观——体长130厘米，体重25公斤，中小型羚羊。但最大的亮点在于屁股上有个心形白圈！毛色偏黄，当地牧民就叫它“黄羊”。更绝的是雄性头顶那对向内弯曲的角，角尖相对，所以也叫“中华对角羚”！这对角不是武器，是求偶时的炫耀资本！

但普氏原羚最牛的地方，是它的生存能力！

它们能在干旱贫瘠的青海高原存活，食谱包含至少50种植物——其中不乏有毒植物！普通人吃了分分钟躺板板，但这家伙进化出了解毒能力，毒草照样干饭！真正的沙漠求生王者！

总结一下：普氏原羚——青海限定精灵，心形白屁股，中华对角羚，50种植物随便吃，毒草都不在话下！就问还有谁！

然而这么猛的物种，目前数量比大熊猫还少，已被列为濒危。再不保护，可能真的要凉凉。
