# 冷到你唔信 - 熊狸

熊狸——名字有熊又有狸，但系！你绝对估唔到，呢只嘢其实系灵猫科嘅！而且最离谱嘅系，佢条尾劲到离谱，竟然可以当手用？！

好喇兄弟们！今日呢期嘅主角，绝对系动物界嘅隐藏大佬——熊狸！讲真，呢只嘢劲到超出你想象！

首先科普时间！虽然名字入面有熊又有狸，但熊狸其实属于灵猫科，同熊和狸猫关系唔大。就好似你叫我做“陈龙”，但我其实唔系李小龙咁——就系个名叫得似！

熊狸主要分布在东亚、南亚、东南亚——印度、尼泊尔、不丹、缅甸、印尼等地，绝对系亚洲特有物种！

咁熊狸有几大？成年熊狸体重可以到9到20公斤，係灵猫科中体型最大嘅物种之一！不过，熊狸最犀利嘅地方唔系体重，而系佢条超级无敌嘅尾巴！

你知唔知熊狸条尾有几劲？

熊狸嘅尾巴几乎同身体一样长，粗壮有力，而且超级灵活！唔似其他动物嘅尾巴净系用来平衡，熊狸条尾可以直接抓握，起到类似人手、甚至象鼻嘅作用！

具体系点？熊狸可以用条尾缠绕树枝，将自己固定喺树上，腾出四肢做其他嘢——例如摘水果、捉昆虫、甚至拎嘢！

有动物学家做过实验，熊狸条尾可以承受成半身嘅重量，缠绕喺树枝上完全唔会滑落！呢个就叫做“天然安全带”！

熊狸主要栖息喺茂密嘅热带雨林，喜欢喺高大嘅榕树上生活，绝对系实打实嘅“树栖动物”！大部分时间都喺树上度过，真系落地都唔会行路咁滞！

食嘢方面，熊狸系杂食性——水果为主，但系都会捉啲小动物嚟补充蛋白。水果、嫩叶、昆虫、小型脊椎动物......样样都食，绝对不挑食！

但係啊，熊狸有一样嘢真系令人又爱又恨——就系佢哋会用一种特别嘅气味嚟标记地盘。研究话呢种气味似花香味夹住尿味，有人就话似“臭腺味”......总之就系好难形容，但係！呢种气味竟然系熊狸嘅“秘密武器”，佢哋用嚟话其他熊狸：“呢个系我嘅地方，你唔好行埋嚟！”

最后总结时间！熊狸虽然名字有熊有狸，但其实系灵猫科大佬！体重可以到20公斤，係灵猫科最大！而最犀利嘅就系佢条尾——可以当手用，缠绕树枝、固定身体、抓握物件，绝对系动物界嘅“六指神功”！
