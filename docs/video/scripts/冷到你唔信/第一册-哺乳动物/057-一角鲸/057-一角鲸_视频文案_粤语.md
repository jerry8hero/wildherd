# 冷到你唔信 - 一角鲸

冷到你震！呢个世界上真系有独角兽！

唔系你谂紧嗰只飞天喷火嗰只——唔会飞、唔会喷火、唔会魔法，但系佢有支劲嘞劲嘅“魔法杖”长喺个头度！北极圈限定，皮肤仲要免費！

今日就嚟介绍呢位——北极圈最神秘嘅存在，一角鲸！注意睇，唔系一角硬币，系一角鲸！🦄

一角鲸，又叫独角鲸，係北极冰海中最神秘嘅居民之一。佢最标志嘅特点就系头顶支长牙，唔系角嚟嘎！呢支嘢实际上係一支特化咗嘅犬齿，可以长到3米——去到三米啊！即系成层楼咁高，光系支牙就已经有10公斤——就系等于一只成年熊嘅重量！想象下你头上顶住只熊跑嚟跑去🦣

但系呢支牙唔系斋摆喺度威噃！佢内部有上百万条神经通道，绝对係海洋版超sensored雷达！可以感知海水温度、压力变化，仲可以探测到附近猎物位置——简直就係一支天然版嘅“万能探测仪”。科学家估佢可以用来探测冰层、搵嘢食，仲可以喺同伴面前威威！一牙多用，性价比如高！

一角鲸体长可以去到6米，体重约800到1600公斤——就係经典科幻小说《海底两万里》中独角兽嘅原型！当年凡尔纳写嘅时候，可能真系参考咗呢只嘢！唔怪得小说入边咁样描写啦！

另外，佢哋仲有一个超搞笑嘅特点——成群出没嗰阵，成个社群都会集体将支牙摆喺同一个方向！科学家到依家都未搞清楚到底为乜，但係场面真系几震撼——想象下成班人举住支“魔法杖”排排企，场面似一个巨型麦克风阵列🎤😂

讲开又讲，一角鲸嘅牙仲有一个劲搞笑嘢——佢可以旋转！科学家发现佢哋可以独立控制支牙摆动方向，黐家可以360度无死角炫耀🦄✨
