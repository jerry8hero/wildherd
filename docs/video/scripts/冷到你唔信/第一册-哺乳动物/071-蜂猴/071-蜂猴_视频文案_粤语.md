# 冷到你唔信 - 蜂猴

蜂猴——最慢嘅灵长类！但呢只东南亚嘅夜行者，手臂有分泌毒液嘅腺体添！

好喇兄弟們！今日呢期嘅主角，灵长类中嘅用毒高手——蜂猴！你以为佢净系一只慢吞吞嘅得意嘢？但係啊，呢只嘢劲到你唔信！

蜂猴个头好细只——平均只有20厘米高，1.5千克重！但系佢哋有一对很大嘅眼睛——夜晚活动时劲有用！

蜂猴主要生活喺中国西南部，以及印度东北部、孟加拉国等地区嘅林地和灌丛中。顺带提一提，独居生活，系典型嘅杂食动物——植物枝叶、果实、各种小型动物都食得！

不过呢，蜂猴最特别嘅就系——识用毒！

蜂猴被归入懒猴科。蜂猴喺树上每爬行一步，需要大约12秒——行动迟缓程度同树懒有得fight！

但係啊，行动迟缓、体形又细——蜂猴嘅保命武器就净剩毒液！

蜂猴会将手肘处嘅毒液同唾液混合——佢哋自身有免疫力，唔怕自己只毒——然后涂抹喺皮肤上，产生难闻嘅气味，最大程度降低被捕食者攻击嘅概率！

所以呢，蜂猴系唯一会用毒嘅灵长类动物——正一「毒王」！

想象下，你系一只想食蜂猴嘅老鹰——但系你扑过去，蜂猴话：「你敢食我？我手肘有毒喔！」你一掂，立即就知错......呢个就係蜂猴嘅必杀技！

呢个就係蜂猴最犀利嘅生存之道：灵长类中唯一用毒！个头像老鼠，细只20厘米1.5千克！行动迟缓——每步12秒！但系手肘有毒腺，与唾液混合涂皮肤！自身免疫唔怕自己毒！真正嘅毒王！

下期我哋讲指猴——用中指捕虫嘅奇异猴！记得三连，我哋下期见！
