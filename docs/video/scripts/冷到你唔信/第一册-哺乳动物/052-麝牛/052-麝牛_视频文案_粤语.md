# 冷到你唔信 - 麝牛

麝牛——似牛又似羊？呢只北极圈嘅毛茸茸坦克，一嘢冲向狼群可以吓到佢哋走都走唔切！

好！今日呢期嘅主角，来自北极苔原——麝牛！你以为佢系一只普通大只牛？但啊，呢只嘢劲到你估都估唔到！

麝牛系大型有蹄类动物，生活喺北美同北欧嘅北极苔原——系住得最北嘅牛科动物之一！体型最大嘅可以去到400公斤，接近一只成年白肢野牛咁大；正常都有200到350公斤——着埋嗰身毛茸茸外衣，睇上去就一个字：壮！

但系！麝牛虽然属于牛科，同真正嘅牛关系其实比较远，点解？

第一，角嘅位置唔同——麝牛嘅角长喺头顶正上方，贴近埋一齐；而牛角就偏向两侧，弯住向上！

第二，尾巴唔同——麝牛有条超短嘅尾巴，缩埋喺毛里面，睇都睇唔到；而牛有条长到尾巴！

第三，肠胃结构唔同——麝牛食草嘅方式更接近羊，消化系统更加适应粗纤维！

不过呢，麝牛个名有个「麝」字——来源于眼睛周围能分泌香味嘅腺体！呢个香味本来系用来吸引异性，但系分泌太多就会变臭——所以麝牛其实系「少少就系香水，多咗就变臭丸」！

讲防御，麝牛群遇到狼群时会围成铜墙铁壁——

最外面系体型更大更强壮嘅公麝牛，企喺前排负责硬食冲击；两边仲有啲侧翼保护，防止被包抄！

中间系其他成年麝牛，形成第二道防线！

最入面就系被护住嘅BB同老弱麝牛！

必要时，公麝牛会以每小时40公里冲向对方——呢个冲击力加上400公斤体重，几百磅都顶唔顺！

想象下，你系一只狼，想偷袭麝牛BB，但系面前围成一圈——你冲过去，一只400公斤嘅毛茸茸坦克以40公里时速向你撞嚟......

算喇，我走先！你真系想送人头都唔使咁送法！

呢个就係麝牛最犀利嘅生存之道：北极苔原最硬核嘅有蹄类动物，角似牛但唔系牛，眼眶腺体少少香多咗变臭，遇到敌人围成三层铁桶阵护BB，时速40公里冲击敌人！真正嘅北极毛茸茸守护神！
