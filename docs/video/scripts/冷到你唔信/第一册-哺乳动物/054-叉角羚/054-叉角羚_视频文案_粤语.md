# 冷到你唔信 - 叉角羚

叉角羚系北美地区特有动物，分布喺加拿大、美国、墨西哥境内。

叉角羚所生活嘅地方主要系开阔草原——一望无际，唔利于隐蔽！

但系！有一嘢就系叉角羚唔擅长——跳跃！
