
# 冷到你唔信 - 猞猁

你见过耳朵“插天线”的猫吗？30公斤的“小猫咪”了解一下！

今日吹下水，讲只“喵星人”入面最离谱嘅存在——猞猁！零下30度都生存到，劲到你怀疑人生！

你以为佢係普通猫咪？呢只嘢大到你唔信，重到离谱！

你可能話，猞猁？唔就普通猫咪咩？但系呢只“小猫”体重可以达30千克！係猫科动物入面“小猫”类最大只嘅，夸张到离谱！

但更离谱嘅係，猞猁耳仔上面有两簇挺立嘅毛发！呢两簇毛起着类似雷达天线嘅作用，配合长而尖嘅耳仔，可以帮助猞猁听到更远、更小嘅声音。就似你戴咗对超级耳机，别人听唔到嘅声你都听到，爽唔爽？

你知唔知猞猁有几劲？佢哋分布非常广泛，亚洲、欧洲、北美大陆都有！森林、苔原、荒漠、树丛、山地都系佢哋嘅宜居场所。呢个适应能力劲到你唔信。

讲到猞猁最犀利嘅地方，就梗係捕猎能力——不论系老鼠、兔子呢类细只嘢，定系羚羊、马鹿呢类大只嘢，全部都系佢哋嘅捕猎名单！就似你乜都食得，大只细只都走得甩。

想象下，你系“小猫”里面最大只嘅，有30千克重。你耳仔有两簇毛发天线，可以听到好远好细嘅声音。你乜嘢都食得，老鼠、兔子、羚羊、马鹿，统统都系你嘅菜单。分布仲要遍及三大洲，森林、苔原、荒漠、山地都住得！

呢个就係猞猁最犀利嘅生存之道：唔单止大只，仲要耳仔天线接收犀利，捕猎范围从老鼠到马鹿，分布遍及三大洲！
