# 冷到你唔信 - 白灵熊

兄弟们！今日继续冷到你唔信，但系今期呢只嘢，我敢打赌你肯定估错！

白灵熊——你以为系北极熊嘅远房亲戚？但真相可能比你諗嘅仲离谱！

你可能話，白灵熊唔就白色熊咯？漏！呢只白毛熊其实係基因突变嘅美洲黑熊！想象下白雪公主误入熊圈——佢唔係外来物种，而係家族里突然杀出嘅限定款！

重点嚟了——白灵熊学名叫柯莫德熊，以科学家William Kermode命名。但当地原住民叫佢"Spirit Bear"，觉得系神灵化身，从来唔会猎杀。所以呢，佢係受到文化同法律双重保护，几犀利下！

你知唔知白灵熊住喺边？加拿大BC省嘅温带雨林！呢个地方本来应该清一色黑色熊，但因为基因突变，突然出现白色版本。就似你喺一堆黑衫入面，发现有几件白色——但呢个机率有啲地方高达20%！

讲到呢度可能有人问：点解白色毛皮咁显眼，佢哋生存唔受威胁？好问题！

答案就系——白灵熊嘅白色毛皮其实係隐性基因造成嘅！呢种基因突变令佢哋皮毛呈现奶油白/白色，但系佢哋仍然係美洲黑熊嘅一员。就似你家族入面所有人都着黑衫，但系你突然着白衫——唔系另一个物种，只系颜色唔同。

更神奇嘅係——白色毛皮虽然喺陆地显眼，但反而帮咗佢哋大忙！因为雨林入面好多鱼同三文鱼，佢哋白色反而喺水面反光入面更难被发现，捕猎成功率更高！所以话，命运关上一道门，可能开翻个窗比你——就睇你识唔识用！

你知唔知？白灵熊喺当地原住民文化入面有特殊地位！

印第安人觉得白灵熊系神灵象征，唔单止唔会猎杀，仲觉得系幸运嘅化身！呢个就似你屋企人觉得你着白衫特别靓，所以唔准你换——但呢个系出于尊敬，唔系限制。

想象下，你喺温带雨林行行下，突然见到一只白色熊喺树影闪过。呢个画面几梦幻！但你知唔知，佢其实係黑熊家族嘅一员，只係着咗件白色外套。

呢个就係大自然最得意嘅“撞衫”——黑色家族出咗件白色限定版！
