# 冷到你唔信 - 红白鼯鼠

红白鼯鼠——滑翔界嘅纸飞机！但呢只中国特有嘅夜行者，可以用前后肢之间嘅翼膜滑翔200米添！顺带提一提，

好喇兄弟們！今日呢期嘅主角，高山森林里嘅滑翔高手——红白鼯鼠！你以为佢净系一只普通嘅松鼠？但係啊，呢只嘢劲到你唔信！

红白鼯鼠生活喺中国中部、西南，南亚印度，东南亚缅甸、泰国——分布几广！皮毛系红白两色，所以叫红白鼯鼠——系来自啮齿目松鼠科！

红白鼯鼠头身加起来超过50厘米——系比较大嘅啮齿动物！生活喺海拔1000到3000米嘅高山森林中——钟意食板栗！

不过呢，红白鼯鼠会喺距离地面至少5米高嘅树洞里安家——选择较高树木系为咗遇到危险时能快速逃跑！因为滑翔通常只能向下——如果本身位置太低，就会好快被迫「着陆」！

但係啊，红白鼯鼠滑翔靠两样嘢：第一，宽大嘅皮膜获取升力；第二，几乎同身体长度相等嘅尾巴稳定方向！

有个冷知识你知唔知？人类嘅「无动力翼装」就系受红白鼯鼠启发而发明嘅！运动员着住翼装从高处跳下滑翔——就系跟红白鼯鼠学的！

想象下，你着住翼装从飞机度跳落去，滑翔喺天空——但系红白鼯鼠早就喺几千万年前就识得呢招啰！

呢个就係红白鼯鼠最犀利嘅生存之道：红白皮毛嘅滑翔高手！头身50厘米，较大啮齿动物！住5米高树洞，滑翔逃跑！皮膜升力、尾巴稳定方向！人类翼装受佢启发！真正嘅滑翔先驱！

下期我哋讲仓鼠——用颊囊搬运粮食！记得三连，我哋下期见！
