# 冷到你唔信 - 水羚

水羚——识游水嘅羚羊？但你知唔知，佢哋个毛皮油脂丰厚，落水都唔会湿添！

好喇兄弟們！今日呢期嘅主角，个名已经话你知佢住边——就系中意临水而居嘅水羚！顺带提一提，你以为佢净系一只钟意游水嘅羚羊？但係啊，呢只嘢劲到你唔信！

水羚系生活在非洲嘅羚羊，体重可达300千克——喺羚羊家族入面属于中等偏大体型！从名就知，佢哋中意生活在湖泊、沼泽、湿地等靠近水嘅地方——临水羚唔知系几惬意啰！

但系水羚有个得意嘅特点——就系臀上有一个由白色绒毛组成嘅圈！

喺草原上奔跑嗰时非常醒目——就似一个白色嘅信号灯，帮你辨认同伴！不过呢，水羚有样嘢劲得意，就系臭味！

水羚嘅汗腺可以分泌一种油脂，会散发出刺鼻嘅臭味！

呢种臭味会随年龄增大、奔跑速度下降而愈发强烈——越老越臭！

正因为咁，狮子、花豹呢啲大型猛兽，除非饿到顶唔顺，否则都唔会对水羚下手——实在太臭，唔想搞！

所以呢，臭味就成为水羚嘅护身符——年纪大嘅水羚虽然跑得慢咗，但系靠住身身臭味，依然可以呃走天敌！正一「老人臭都唔系冇用」！

想象下，你系一只狮子，饿到晕，但系面前有只老水羚——你走过去，一阵臭味吹过来......算喇，唔好搞，搵第二只啦！

呢个就係水羚最犀利嘅生存之道：非洲临水而居嘅羚羊！300千克，体型几大！臀上有白色绒毛圈，奔跑时好抢眼！汗腺分泌臭味——越老越臭！靠臭保命，狮子花豹都怕！又叫臭鼬羚，真系名不虚传！

下期我哋讲普氏原羚——臀上有心形标记嘅羚羊！你哋知唔知还有边种动物靠臭味自卫？评论区告诉我！

钟意呢期记得转发，比朋友也开下眼界！
