# 冷到你唔信 - 水羚

水羚——你以为佢净系一只钟意游水嘅羚羊？太天真了兄弟！呢只嘢简直係非洲草原嘅“生化武器”，臭到狮子都要掉头走！

好喇兄弟们！今日嘅主角，名字已经话你知——就係中意临水而居嘅水羚！

非洲嘅湿地、沼泽、湖泊附近，你好容易遇到呢位“水中贵族”。体重高达250-300公斤，係羚羊家族入面嘅大块头！住埋啲一线湖景房，几叹世界～

但係！水羚最抢眼嘅唔係体型，而係——屁股！

佢哋臀部有个白色绒毛组成嘅圈圈，喺草原狂奔时超级显眼——简直就係天然嘅“白色信号灯”，方便同伴喺茫茫草原上认人！

不过！呢个“信号灯”主人，有个超级“突出”嘅特点——臭！

水羚嘅汗腺可以分泌一种油腻物质，一遇水就散发刺鼻臭味！而且越老越臭——岁月唔单止催人老，仲催人臭！年轻水羚靠速度生存，老水羚就只能靠……“老人臭”嚟保命！

正因如此，狮子、花豹呢啲草原霸主，除非饿到顶唔顺，否则绝对唔会对水羚下手——实在太臭，下唔到口！🤣

所以啊，臭味就成为水羚嘅护身符！年纪大嘅水羚虽然跑得慢，但靠住一身“老人臭”，依然可以呃走天敌！真正嘅“臭到没朋友，却活得最久”！

想象下：你係一只饿到打冷震嘅狮子，前面有只老水羚，你行过去，一阵臭味飘嚟……算喇，宁愿食只斑马都好啊！

呢个就係水羚嘅生存之道——靠“臭”出名嘅羚羊！

最后科普一下：佢哋又叫“臭鼬羚”，名副其实！成年雄性体重可达300公斤，喺非洲係体型最大嘅羚羊之一。下期讲普氏原羚——臀上有心形标记嘅羚羊，超浪漫！
