# 冷到你唔信 - 薮猫

兄弟们！今日介紹呢只「勁」到離譜嘅大貓——薮猫！

長腿配大耳，簡直係一隻迷你獵豹！呢個來自非洲嘅捕鼠專家，抓老鼠快到似閃電⚡

好啦先介紹一下——呢個系列就係講哺乳動物入面嗰啲超冷門知識，保證你聽完眼都碌埋，造物主真係乜都諗得出！

你可能覺得：薮猫？唔就係大隻嘅貓咩？

漏！薮猫可唔係普通貓！佢竟然可以原地向上跳起2米高！

2米高係乜嘢概念？就係相當於一個成年人跳到差不多成年人身高咁樣！你試下跳2米？唔得閒咩？犀利！

但係，重點嚟啦——薮猫仲可以唔轉頭唔轉身就向後蹦！呢個技能真係勁到你唔信——就似你向後跳嗰陣，頭同身體都唔洗郁，淨係用腳尖發力就得。

當遇到更大嘅獵食者，佢可以瞬間向後跳走，呢招「旱地拔蔥」真係絕！

你知唔知薮猫有幾大隻？

身形比家貓大一倍有多！修長四肢加大耳朵，就似一隻著咗大頭佛嘅貓。

但係佢重有更加勁嘅技能——空中捕獵！可以捉飛緊嘅小鳥！呢個難度有幾高？就似你伸手去夾緊飛緊嘅蒼蠅，真係勁！

你知唔知？薮猫係非洲特有動物！

唔單止跑得快、跳得高，仲識游水——可以捉水入面游嘅小型獵物。就似水陸兩棲嘅存在，地上走嘅、水入面游嘅、空中飛嘅，統統都走得甩！

想象下，你喺非洲大草原上面，遇到一隻大長腿大耳朵嘅「貓」。

佢唔單止大隻，仲可以跳2米高、向後蹦、捉小鳥、游水捉魚——呢隻唔係普通貓咪，而係全能運動員！

其實呢，就係薮猫嘅生存之道！
