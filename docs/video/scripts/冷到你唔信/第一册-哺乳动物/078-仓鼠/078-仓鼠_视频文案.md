# 冷到你唔信 - 仓鼠

兄弟们！刷到一个视频直接给我整破防了——

仓鼠那个腮帮子，居然能装比它脑袋还大的东西？！

好啦不卖关子！今天的主角，嘴巴比哆啦A梦口袋还能装的——仓鼠！点进来你就知道，这小东西到底有多离谱！

首先来点硬核科普：仓鼠科是哺乳动物中种类最多的科之一，全球有600多种，广泛分布在亚洲、欧洲、非洲的各个角落！

那么问题来了——仓鼠这个名字咋来的？

你仔细看它的身形，超级袖珍！最小的罗伯罗夫斯基仓鼠（就是那种公婆鼠）体重大概30到40克，跟一枚鸡蛋差不多轻！最大也就1千克上下，差不多两个鸡蛋的重量！

因为身体太小了，体温特别容易散失，一到冬天就扛不住啊！

所以仓鼠进化出了一个超级技能——颊囊！这两坨长在腮帮子里的口袋，弹性惊人，能装下超过体重20%的食物！想象一下，100斤的人能往嘴里塞20斤东西，是不是有点离谱？

而且这玩意儿不止能装，还不容易变质——仓鼠的颊囊堪称生物保鲜袋，食物放进去多久都不会坏！所以找到食物就往里塞，没找到也往里塞，反正就是塞塞塞塞塞……

所以仓鼠名字里的“仓”，指的就是它能往腮帮子里疯狂囤货的特性！

好啦来给大家总结一下今天的知识点：

颊囊能装体重20%的食物！
身形袖珍30克到1千克！
冬天窝在地下洞穴不出门！
哺乳动物最大科之一！
