# 冷到你唔信 - 貉

兄弟们！今天要聊的这个家伙，你一定在成语里见过它——没错，就是“一丘之貉”的那个“貉”！

先别划走！我赌你肯定以为它是只狗对吧？或者觉得它就是个谐音梗里的工具人？NONONO！大错特错！虽然这家伙天天混在狗圈里，但你知道吗——貉可是犬科家族里唯一一个会爬树的狠角色！

什么？还不够离谱？它不仅会爬树，还会冬眠、会游泳，更绝的是居然还会装死骗过天敌！这演技，不去拿个影帝都屈才了！

最让人蚌埠住的是——明明长着一张标准的“毛茸茸狗脸”，结果人家是浣熊的近亲！就问你惊不惊喜？意不意外？怀疑人生了没？

好了各位，今天的冷知识就到这里！这波信息量够不够顶？觉得涨姿势的兄弟们，把三连打在评论区！

我是小冷，你们还想看什么奇奇怪怪的动物冷知识？评论区告诉我，安排！
