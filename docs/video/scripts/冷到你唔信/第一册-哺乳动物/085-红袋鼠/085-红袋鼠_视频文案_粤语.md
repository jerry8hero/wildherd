# 冷到你唔信 - 红袋鼠

12.8米一跳——超越人类极限！呢只澳洲大汉，唔只跳得劲，打架先至系真正劲到离谱！

好嘞兄弟们！今日主角——草原上嘅拳击手红袋鼠！你以为佢净系一只会跳嘅大只老鼠？错晒！呢只嘢劲到你想象唔到！

红袋鼠系世界上最大嘅袋鼠——亦都系现存最大嘅有袋类动物！雄性体重最高可以去到90公斤，雌性都有成35公斤！主要栖息喺澳大利亚热带稀树草原。

但重点嚟啦——红袋鼠个名其实系来自雄性红棕色嘅毛，雌性其实系蓝灰色！所以唔系所有袋鼠都红嗰，认错就大镬喇！

红袋鼠全身肌肉发达，两条后腿粗壮有力，支撑整个身体重量完全冇问题。最离谱嘅系，佢可以直起身，用前肢搂抱同击打——活生生一个袋鼠版泰臣！所以先叫「草原拳击手」！

如果觉得拳头唔够劲？红袋鼠会用粗大尾巴撑地做支点，后腿飞踢！呢一脚劲到可以打断人哋肋骨！脚上第四根脚趾嘅爪子又长又锋利——可以撕裂肌肉——真系劲狠！

想象下，你喺草原度行，突然后面一只袋鼠跳起身一脚踢埋嚟——断肋骨嗰种！唔系讲笑，呢个就系佢哋嘅日常操作！难怪叫「草原狠角色」！

好，咁多位观众老细，係时候总结一下：世界最大袋鼠，90公斤！草原拳击手，直立用前肢击打！可用尾巴支撑、后腿飞踢！踢力大到可以断肋骨！真系草原一哥！
