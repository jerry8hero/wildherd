# 冷到你唔信 - 红袋鼠

红袋鼠——跳得劲过奥运会选手！但呢只澳洲嘅代表，跳远纪录系12.8米添！

好喇兄弟們！今日呢期嘅主角，澳大利亚草原上嘅拳击手——红袋鼠！你以为佢净系一只识跳嘅大只老鼠？但係啊，呢只嘢劲到你唔信！

红袋鼠系世界上最大嘅袋鼠——都系现存最大嘅有袋类动物！雄性体重可达90千克，雌性可达70千克！主要栖息喺澳大利亚热带稀树草原上！

不过呢，红袋鼠个名来源于雄性红色或棕红色嘅被毛——雌性就系蓝灰色！所以唔系所有袋鼠都红色㗎！

但係啊，红袋鼠拥有发达肌肉——两条后腿粗壮有力，足以支撑成个身体重量！顺带提一提，可以直起上身，腾出前肢做搂抱、击打等动作——所以红袋鼠又被叫「草原上嘅拳击手」！

如果觉得前肢击打力度唔够——红袋鼠重可以用粗大嘅尾巴当第三条腿支撑地面，用一双后腿给对方来个飞踢！呢种飞踢力量很大——足以弄断人哋肋骨！而且第四根脚趾上嘅爪子可以撕裂肌肉——劲狠！

想象下，你喺草原度行，突然一只袋鼠跳起身向你踢过来——呢一脚可以断肋骨喔！难怪袋鼠叫「草原狠角色」！

所以见到袋鼠跳起身踢脚——记得走远啲！

呢个就係红袋鼠最犀利嘅生存之道：世界最大袋鼠，90千克！草原拳击手，直立用前肢击打！可用尾巴支撑、后腿飞踢！踢力大到可断肋骨！真正嘅草原狠角色！

下期我哋讲树袋鼠——前肢比后肢长！记得三连，我哋下期见！
