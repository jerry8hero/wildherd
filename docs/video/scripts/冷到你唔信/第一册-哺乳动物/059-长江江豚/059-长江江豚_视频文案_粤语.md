# 冷到你唔信 - 长江江豚

长江江豚——微笑天使！但你知唔知，佢哋已经功能性灭绝，仅存约1000只添！

好喇兄弟們！今日呢期嘅主角，来自中国长江嘅可爱精灵——长江江豚！你以为佢净系一只识笑嘅靓仔？但係啊，呢只嘢劲到你唔信，但系同时又令人心痛！

长江江豚系我国特有嘅水生哺乳动物，栖息于长江中下游及部分支流中——系唯一只生活在淡水中的江豚！顺带提一提，属海豚科，同海洋霸主虎鲸系亲戚——系一种小型齿鲸！

但系长江江豚性格温顺，体长约1.5米——头部圆圆，嘴仔张开时样子似笑——被誉为「微笑天使」！

不过呢，长江江豚都有声呐系统！

额头隆起嘅地方叫「额隆」——具有增强所发射嘅声波信号强度并调整方向嘅作用！有助于更快咁寻找鱼群——就系佢哋嘅GPS导航系统！

但係啊，长江江豚有个弱点——佢哋嘅潜水能力并唔强！

平均每分钟要浮出水面呼吸两三次——持续蔽气嘅时长甚至不如专业潜水员！所以——长江江豚BB刚出世嗰时系会溺死㗎！

呢个时候江豚妈妈就会用身体将小江豚托出水面——帮BB呼吸！母爱真伟大！

想象下，你喺长江度游水，但系你每分钟都要上水面抖气——如果阿妈唔托你，你随时会浸死......呢个就係长江江豚嘅困境！

但系而家最惨嘅系，长江江豚数量已经非常稀少——被列为濒危物种！仅存约1000只，已经功能性灭绝......希望大家都关注呢种可爱又可怜嘅「微笑天使」！

呢个就係长江江豚最犀利嘅生存之道：中国长江特有！属海豚科，虎鲸嘅亲戚！额头有额隆，用声波导航！性格温顺，被称微笑天使！但系潜水能力弱，每分钟要上水面呼吸！BB会溺水，妈妈要托住！濒危物种，要保护！

下期我哋讲抹香鲸——潜水高手！记得三连，我哋下期见！
