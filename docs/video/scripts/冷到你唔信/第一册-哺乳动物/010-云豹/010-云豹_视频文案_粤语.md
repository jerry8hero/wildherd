# 云豹 - 森林里最神秘的「小剑齿虎」

兄弟们！今天给你们介绍一位狠角色——云豹！

等等，你说云豹就是身上有云朵花纹的豹子？那你格局可就太小了！

这家伙有个炸裂的外号——“小剑齿虎”！

啥意思？它的上犬齿最长能到9厘米！在现存所有猫科里，犬齿占头骨比例它是第一名！这就好比你嘴里随时挂着两把大宝剑，谁看了不迷糊？

但最离谱的来了——云豹嘴巴张开角度能超过100度！想象一下，你也得张这么大嘴，还长着一对快10厘米的獠牙，这配置简直是猛兽界的天花板！

你以为它只是长得凶？不好意思，人家真的是捕猎高手！野猪、鹿类、猴子都在它的菜单上！这就好比看着是个青铜，其实是隐藏王者，猛不猛？

说到这，必须提一个冷知识——云豹其实是两只！

没错，世界上有两位云豹成员：一位住在亚洲大陆，叫云豹；另一位住在苏门答腊岛和加里曼丹岛，叫巽他云豹。就像两兄弟，一个住陆地，一个住海岛。

重点来了——这俩货分开长达130万年，科学家一直以为它们是同一物种！直到2006年基因研究出来才发现，哇，原来这是两个完全不同的物种！巽他云豹直接喜提“新物种”认证，这波属于是科学家被啪啪打脸了。

总结一下：9厘米的“小剑齿虎”、100度的大嘴巴、130万年才分家的兄弟——云豹，猛就一个字！
