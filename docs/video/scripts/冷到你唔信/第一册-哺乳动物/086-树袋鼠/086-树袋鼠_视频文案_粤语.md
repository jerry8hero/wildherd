# 冷到你唔信 - 树袋鼠

树袋鼠——树上生活嘅袋鼠！但你知唔知，佢哋可以喺树与树之间跳8米添！

好喇兄弟們！今日呢期嘅主角，袋鼠家族嘅异类——树袋鼠！你以为佢净系一只识爬树嘅普通袋鼠？但係啊，呢只嘢劲到你唔信！

我哋熟悉嘅袋鼠——红袋鼠、灰袋鼠——都系前肢短、后肢长！但系树袋鼠就完全相反——前肢比后肢长！

树袋鼠生活喺新几内亚岛同澳大利亚昆士兰地区！除咗活动场所在树上，样貌同地上亲戚都大相径庭——树袋鼠拥有一张近似熊嘅脸，躯干似树懒，尾巴似猴子！睇上去比普通袋鼠更萌——真正嘅袋鼠界颜值担当！

不过呢，树袋鼠前肢唔单止长，上面重有锋利嘅钩状爪子——可以牢牢抓握住树干！

但係啊，因为会爬树，树袋鼠嘅饮食结构都唔同——唔系纯粹素食者！除咗摘取果子和枝叶，重会掏鸟蛋，甚至捕捉雏鸟——食埋肉！

想象下，你以为袋鼠净系食草——但系树袋鼠就话你知：「我唔同喔，我识捉鸟㗎！」几犀利！

所以呢，树袋鼠就系袋鼠家族嘅异类——前肢长、识爬树、重识食肉！

呢个就係树袋鼠最犀利嘅生存之道：袋鼠家族异类！前肢比后肢长！新几内亚、澳洲昆士兰！样貌似熊似树懒似猴子——更萌！前肢有钩爪识爬树！唔纯粹素食——识捉鸟食！真正嘅袋鼠界叛徒！

下期我哋讲犰狳——骨骼似双层汽车！记得三连喔！
