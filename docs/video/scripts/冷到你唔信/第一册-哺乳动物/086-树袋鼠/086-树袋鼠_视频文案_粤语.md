# 冷到你唔信 - 树袋鼠

你知唔知？有一只袋鼠，唔喺草原上跳嚟跳去，而系日日挂喺树上！而且重识捉雀嚟食！没错，袋鼠界嘅终极叛徒——树袋鼠！

好喇兄弟姊妹们！今日呢期嘅主角，袋鼠家族嘅另类天花板——树袋鼠！你以为袋鼠净系喺草原上用后腿弹弹跳？呢只嘢偏要话你知：“我偏唔！”

我哋熟悉嘅袋鼠——红袋鼠、灰袋鼠——个个都系前肢短、后肢长！但系树袋鼠就完全反转，前肢比后肢仲长！点解？因为佢要爬树啊！

树袋鼠住喺新几内亚岛，同埋澳洲昆士兰北部嘅热带雨林。除咗成日喺树上荡嚟荡去，样衰都同地上亲戚差天共地——树袋鼠拥有一张似熊嘅衰，驱干似树懒，尾仔似猴子！重有最关键嘅——睇上去比普通袋鼠更萌更得意，简直就系袋鼠界嘅颜值担当！

但系！树袋鼠前肢唔单止长，上面重有超犀利嘅钩状爪子——一爪插入树干，稳稳甩甩！

因为长期喺树上生活，树袋鼠嘅饮食习惯都唔同晒——唔系纯素食者！除咗摘水果同树叶，佢重会掏雀蛋、捉雏雀，甚至会捕捉小型昆虫！系咩话？袋鼠食肉？没错，就系咁离谱！

想象下，你以为袋鼠净系食草动物——但树袋鼠就话你知：“我唔同喔，我系捕雀达鼠！”几爆笑！

所以话，树袋鼠就系袋鼠家族嘅终极叛徒——前肢长、识爬树、重系杂食者！

重点笔记时间！树袋鼠生存之道：前肢比后肢长！分布新几内亚、澳洲昆士兰！样貌似熊似树懒似猴子——更萌！前肢有钩爪识爬树！唔纯素食——识捉雀！真正嘅袋鼠界叛徒！
