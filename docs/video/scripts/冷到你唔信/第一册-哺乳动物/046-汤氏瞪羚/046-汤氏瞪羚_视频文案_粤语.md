# 冷到你唔信 - 汤氏瞪羚

汤氏瞪羚——跳跃冠军！但呢只东非嘅短跑健将，一跃而起可以话似飞翔添！

好喇兄弟們！今日呢期嘅主角，生活喺非洲东部稀树草原上——汤氏瞪羚！你以为佢净系一只普通嘅羚羊？但係啊，呢只嘢劲到你唔信！

汤氏瞪羚系非洲特有嘅牛科动物。体型几细只——体长0.65到1.1米，体重25千克——因为身形较小，佢哋通常集群生活，一齐应对食肉动物嘅偷袭！顺带提一提，

不过呢，汤氏瞪羚嘅食物90%系草。

但系！为咗保障一年四季都能吃到鲜嫩嘅青草，佢哋每年都要跨越马拉河，喺塞伦盖蒂同马赛马拉两个草原之间迁徙！呢个就系自然界最壮观嘅场景之一——百万角马大迁徙，而汤氏瞪羚就系参与者！

但係啊，迁徙时有个得意嘢——汤氏瞪羚会跟喺斑马群同角马群后面！

点解？因为佢哋钟意食靠近土壤根部嘅嫩草，而斑马钟意食草嘅上部，角马钟意食草嘅中部——各有各食，唔会争！

而且汤氏瞪羚除咗食草，重识得用眼眶下面的腺体做标记！

呢个腺体可以分泌出一种液体，然后涂抹喺草上——用气味嚟联络同伴！就系话，「呢度有我嘅地盘，我同伴快点埋嚟」呢个意思！

所以下次你睇动物纪录片，见到一班羚羊跟喺斑马角马后面跑，呢只就好可能系汤氏瞪羚啰！

呢个就係汤氏瞪羚最犀利嘅生存之道：非洲东部小型羚羊！身形细只所以集群生活！跟斑马角马一齐迁徙，各食各位置！90%食草，但系会用眼眶下腺体分泌液体标记！又食草又识做记号，聪明！

下期我哋讲水羚——用臭味自保嘅羚羊！记得三连喔！
