# 吓坏海员的巨鳄——湾鳄

兄弟們，一般人都會以為鱷魚係喺河流、湖泊、沼澤入面游水。但係你知唔知，鱷魚都可以喺大海入面游水㗎！呢樣真係估你唔到吓！

灣鱷棲息喺河流出海口嘅三角洲、紅樹林等地方，係世界上已知咬合力最強嘅動物——佢哋嘅咬合力比大白鯊、東北虎仲要大！呢樣真係和利害！巨大嘅身形配上恐怖嘅嘴巴，令到佢哋成為頂級獵手，夠膽捉幼象、犀牛、水牛呢啲大型動物，甚至會喺海入面捉鯊魚！呢個真係巴閉到一個點！

灣鱷到底可以長到幾大呢？2011年，人們喺菲律賓捉到咗一隻超大嘅雄性灣鱷——佢身長6.17米，體重超過1噸，係人類目前捕獲圈養過嘅最大灣鱷！人們仲幫佢改咗個霸氣嘅名叫「洛龍」——呢個名改得幾有霸氣吓！

順帶提一提，關於灣鱷，清朝嘅《李準巡海記》有記載：1907年，廣東水師嘅官員乘坐「伏波」「琛航」兩艘軍艦，由海南島開往西沙群島一帶，途中竟然有灣鱷從海入面爬上咗軍艦——嗰吓真係嚇親晒全船人，係個真正的海上驚魂！
