# 大鳞鳞虎：我把鳞片都扔了

大鳞鳞虎：我把鳞片都扔了 大鳞鳞虎：我把鳞片都扔了 兄弟們，今日我哋嚟吹下水，講一種犀利到震嘅壁虎——大鳞鳞虎！

喺馬達加斯加北部安卡拉納國家公園嘅石灰岩山度，科學家撞到一種嘔心到暈嘅壁虎。佢哋一遇到危險，身上的鱗片就話甩就甩，如同「大珠小珠落玉盤」咁款，成身赤裸裸爆出嚟，跟手就梗係走為上著！

不過呢，其實呢種自虐逃生大法，並唔係大鳞鳞虎創先河嘅。早喺1867年，科學家就已經發現咗「鱗虎」家族呢班老嘢，而大鳞鳞虎就係當中最巴閉嘅一位——鱗片最大嗰個！

兄弟們你哋一定會問啦，呢啲鱗片究竟係點解甩落嚟嘅呢？原來喺佢哋皮膚底部，有一班神奇嘅打工仔——叫作「肌纖維母細胞」。只要呢班嘢收工，鱗片就會自動剝落，所以呢個位有個相當爆嘅名——「撕裂區」！

嘩，你話幾過癮？甩鱗片嚟走佬，呢招真係冇得頂！兄弟們你哋又點睇？歡迎喺下面吹水區留言吹兩句！如果覺得呢條片OK的話，记得撳個三連支持下喔！
