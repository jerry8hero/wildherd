# 令人恐惧的黑曼巴蛇

兄弟們，今晚又係我哋動物科普時間！講到蛇，你哋第一印象係啲咩啊？係咪一睇見就想掉頭走嗰隻？哈哈哈！但係啊，對於非洲撒哈拉沙漠以南啲老友記嚟講，佢哋嘅噩夢就只剩低一個名字——黑曼巴蛇！

好喇，咁呢條蛇究竟有幾巴閉？話俾你知，佢可以長到4米以上喎！犀利喎，係世界上第二長嘅毒蛇咋！佢哋細細隻嗰陣係白鼆鼆，大個咗就變成灰色或者褐色。但係啊，當佢準備攻擊你嘅時候，一張開把口——入面係黑蚊蚊一片，哇，真係幾得人驚囉！

不過最得人驚嘅仲喺後面——佢追人嗰陣速度！每個鐘頭可以跑到16公里，即係踩單車都未必追得上佢！如果俾佢追到你，咁就大獲囉，毒液會喺10分鐘之內發作，分分鐘攞你命！

順帶提一提，喺1957年到1979年期間，南非記錄咗2500幾單毒蛇咬人案，當中有70幾單就係黑曼巴蛇搞出嚟，呢70幾單入面重有21單係死人嘅！嗱，不過呢，唔使咁驚青嘅，其實呢條蛇係唔會主動埋嚟搞你嘅——只要你唔去撩佢，佢都懶得理你。
