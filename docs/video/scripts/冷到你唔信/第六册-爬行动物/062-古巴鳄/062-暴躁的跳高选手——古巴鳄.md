# 暴躁的跳高选手——古巴鳄

兄弟們，嘩！睇睇呢條鱷魚幾萌啊！佢身上黑色同黃色嘅鱗片閃令令嘅就好像漂亮嘅鵝卵石咁款。但係啊，唔好被佢外表呃親——佢係鱷魚界嘅一霸嚟㗎！就係生活在古巴沼澤同濕地嘅古巴鱷，幾巴閉吓！

古巴鱷嘅性格有啲暴躁吓，為咗可以捉到好嘢食，佢哋甚至會成個身跳出水面，絕對係一名出色嘅「跳高選手」！呢樣真係睇到我哂一一跳，佢哋竟然可以跳得咁高！佢哋個口入面靠後嘅牙又粗又結實，可以輕鬆噉壓碎貝類甚至係海龜殼，係個真正的硬啃王！

不過呢，雖然古巴鱷咁巴閉，但係喺古巴，人類曾經對佢哋進行瘋狂嘅捕殺，令到佢哋差啲滅絕——呢樣真係好慘吓！好彩嘅係，古巴政府喺1959～1960年喺野外一口气捉咗幾百條古巴鱷嚟保育，先至令到佢哋嘅種群得到發展，係個好彩結局吓！

兄弟們，見到呢種暴躁但係差啲被人類搞到滅絕嘅鱷魚，你有乜嘢諗法？留言吹下水！如果覺得今條片幾有趣，記得整個三連俾我！你嘅支持先係我繼續分享嘅最大動力！下次我哋再一齊探索更多得意動物！
