# 龟中海王——棱皮龟

兄弟們！今日我哋嚟介绍一种劲巴閉嘅海龟——棱皮龟！呢个名听起来怪怪噶？但係里面有个故仔噶。

一般嘅龟都有坚硬嘅龟甲（龟壳），呢个龟甲其实系由龟嘅特殊「皮肤」形成嘅「骨骼」，硬到不得了。但係棱皮龟就唔同嘞，佢哋嘅皮肤软变成皮革噉样，而且呢层皮革背脊上面有7条纵向排列嘅「棱」，所以就改名叫「棱皮龟」喇。几得意吖嘛！正一世界之大，无奇不有啊！

不过你唔好以为佢哋得个「软」字喎。棱皮龟呢身「皮」唔软嘅喔！佢哋背脊层皮革下面有骨片保护住，喺危机四伏嘅大海入便，呢个就系佢哋劲犀利嘅「护甲」嘞。

而且啊，棱皮龟长大之后系真正嘅「龟中巨无霸」，可以长到2米咁大！呢个体形已经可以吓走好多动物嘞。但係啊，佢哋最巴閉嘅地方唔止呢个——佢哋游水劲犀利，可以下潜到1000米深噶，係真系潜水大师！你话几犀利？不过呢，虽然棱皮龟咁巴閉，佢哋依家都系濒危物种添。由于海洋污染同埋人类捕捞，棱皮龟嘅数量不断减少，大家都要注意保护海洋环境啊！
