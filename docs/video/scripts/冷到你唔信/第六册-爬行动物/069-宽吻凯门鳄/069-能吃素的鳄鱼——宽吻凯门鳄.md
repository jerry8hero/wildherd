# 能吃素的鳄鱼——宽吻凯门鳄

能吃素的鳄鱼——宽吻凯门鳄 能吃素的鳄鱼——宽吻凯门鳄 在你的印象中，鳄鱼应该是个地道的肉食主义者吧？生活在南美洲东部、中部淡水沼泽和红树林里的宽吻凯门鳄，却做出了一些在鳄鱼家族看来“大逆不道”的怪异行为。 宽吻凯门鳄身体最大的特征是有宽宽的嘴巴，这种嘴巴特别适合用来撕扯沼泽中茂密的植物，而它们居然有时真的会在寻找食物的过程中吞下一些植物！更令人惊讶的是，有些人拍摄的照片中显示，宽吻凯门鳄还会在没有任何外部刺激的情况下，吞下南美洲当地一种叫作“羽裂蔓绿绒”的果实！有科学家研究认为，宽吻凯门鳄这种行为能够帮助植物传播种子，而宽吻凯门鳄很可能是能吃点素的杂食性动物。
