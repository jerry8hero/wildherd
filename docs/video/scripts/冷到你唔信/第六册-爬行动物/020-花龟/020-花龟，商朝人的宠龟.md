# 花龟，商朝人的宠龟

花龟，商朝人的"宠龟" 花龟，商朝人的"宠龟" 喺3000幾年前嘅中国商朝，噉时啲人好鍾意占卜，而占卜嘅其中一个重要程序就系喺龟甲或者兽骨上面刻字，将要占嘅事情刻上去，呢啲龟甲同兽骨上面嘅文字就系鼎鼎大名嘅甲骨文啦。但係龟有咁多种，究竟商朝人用得最多嘅龟甲系边种龟呢？根据考古学家嘅研究，商朝人最鍾意用嘅有一种系上文提到嘅草龟，另一种就系花龟。花龟依家主要分布喺我哋华南地区，但系喺中原商朝遗址——即系离华南以北千里之外——居然发现咗大量花龟甲！呢个说明乜？当时花龟嘅分布范围比依家阔得多喔！其实呢，花龟唔单止受商朝人鍾意，连秦汉时期嘅人都like得爆喎。喺广州发掘到嘅秦汉时期南越王墓入面，就发现咗唔少花龟甲碎片，劲唔劲？
