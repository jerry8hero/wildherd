# 冷到你唔信 - 花龟

商朝人就已经养嘅龟，花龟——3000年前嘅宠物，身份证书比好多国家都老！



你可能話，花龟？系未就系花纹好睇嘅龟？你如果係咁諗，就太简单啰。花龟唔单止花纹靓，佢重係3000几年前的商朝人用来写甲骨文嘅"御用龟种"！啲咩石破天惊？等你慢慢听落去。

你知唔知？商朝人占卜最中意用花龟！

喺3000几年前的商朝，王室成日都要搞占卜，问天问地问收成。占卜嘅方式就系喺龟甲上面刻字，然后烧睇裂纹断吉凶。考古学家研究完发现，商朝人最中意用嘅龟甲有两种——草龟，同埋花龟！但係问题嚟啰：花龟宜家主要分布喺华南地区，但係河南安阳呢啲中原地方，啱啱就系商朝首都遗址，喺呢度发现咗大量花龟甲！即系话，3000几年前，花龟嘅分布范围比依家阔好多——大概气候变暖或者其他原因，花龟就慢慢退守到南方去。呢个就叫做：沧海桑田，龟都识走位。

你知唔知？花龟都系"南越王"嘅爱！

唔单止商朝人锺意花龟，秦汉时期嘅南越王都一样！喺广州发掘嘅南越王墓入面，考古学家发现咗唔少花龟甲碎片——你就知花龟有几巴闭喇。古代宫廷唔系你话想养就养，要有身份先得。花龟可以话係龟入面嘅"贵族"，见证住成段中国历史添！

讲到花龟最得意嘅嘢，就梗係佢哋嘅花纹——天然嘅水墨画就係咁！

花龟个样就好似大自然随手画嘅艺术品，壳上面有唔规则嘅黄色同黑色条纹，好似山水画咁样添！所以好多爬友都好中意养花龟，唔单止因为佢性格温顺，仲因为佢真系几有睇头——就似养紧一幅识游动嘅水墨画。

你知唔知？花龟係淡水龟入面比较常见嘅一种，但係脾气好到不得了！

唔少龟仔咬人咬到飞起，花龟就相对温和好多，唔会话无缘无故就向你示威。就似龟入面嘅"文青"，性格安静，锺意慢慢游水的——但係你唔好以为佢好求其，佢游起水嚟灵活到你唔信，水面转身、水底漫步，样样都得！

各位，跟住落嚟讲嘅，就係花龟最犀利嘅生存技能。

花龟虽然性格温柔，但係佢识得适应唔同嘅环境！华南地区嘅池塘、沼泽、河流佢都可以生存，冬天冻嘅时候佢会钻入泥度冬眠，等春天暖返就先至出嚟。就似一个识得执生嘅"淡水精灵"，天冷就瞂，天热就活动——活得几醒神啰！
