# 分辨靠歌谣——东部珊瑚蛇

兄弟們，你哋有冇發覺，有些蛇真係樣衰衰差啲樣？就好像下面呢條，睇落去係咪幾似之前我哋講過嗰隻無毒加州山王蛇？不過呢，我勸你唔好靠估，因為呢條係美國東南部嘅劇毒蛇——東部珊瑚蛇！

好喇，問題嚟喇，點樣先至分辨到佢哋呢？當地啲老街坊就為呢件事頭痕咗好耐，最後編咗首得意歌謠幫手記：

「紅配黑，唔使走；
　紅配黃，殺人狂；
　紅配黑，好朋友；
　紅配黃，把命丟！」

其實呢道理好簡單——你唔使理佢身上有幾多種顏色嘅「環」，淨係睇紅環有冇黐住黃環就得。如果紅環黐住黃環，嗱，麻甩喇，梗係要走夊！因為嗰隻就係劇毒嘅東部珊瑚蛇（適用於北美珊瑚蛇）。

你睇下面張圖，呢條蛇身上嘅紅環係咪黐住黃環呢？紅配黃，殺人狂！快啲走啦！你再翻上去睇下加州山王蛇，佢身上係咪根本冇黃環，而且紅環係黐住黑環嘅？紅配黑，唔使走！佢係無毒嘅王蛇類嚟嘅！

所以啊兄弟們，遇到紅黑相間嘅蛇唔使慌，但係一见到紅黃黐埋一齊，就梗係要走夊，記住呢首歌謠，分分鐘救你一命！如果你鐘意呢類求生冷知識，三連支持我，comment話俾我知你想聽咩，下期再見！
