# 北草蜥：外号四脚蛇的蜥蜴

哎呀，兄弟們，呢個係乜嘢嚟嘅？睇落似蛇，但係再仔細睇多一眼——哇，有4隻腳喎！呢個就係啲人成日講嘅「四腳蛇」喇？唔怪得你話估錯佢係蛇啦！

佢係正正式式嘅蜥蜴——北草蜥，係我哋中國特有嘅一種蜥蜴嚟㗎！由北邊嘅吉林，到西邊嘅甘肅，再加埋華南地區大大片地方，都見到佢哋嗰「蛇一般嘅魅影」，真係分布好廣吓。

其實呢，唔怪得你會認錯佢，因為北草蜥擁有蜥蜴界超長嘅尾巴——佢條尾嘅長度可以達到頭體長度嘅2～3倍！同屬嘅南草蜥更巴閉吓，條尾可以係頭體長度嘅3倍以上，簡直就係尾巴界的長跑冠軍！

北草蜥有「風一般的速度」，無論係捉獵物定係逃跑，佢哋都能夠敏捷噉移動。但係啊，就算速度幾快，北草蜥都逃唔過被人類捕捉嘅命運。有些地方認為「四腳蛇」可以整成藥酒，於是就去捉佢哋——呢樣真係好可惜囉。

兄弟們要注意喇！北草蜥早喺2000年就已經被列入《三有保護野生動物名錄》，我哋要好好保護佢哋！你哋身邊有冇人見過北草蜥？快啲喺下面留言分享下！覺得有幫助嘅記得整個三連，等更多人睇到！我哋一齊為保護動物出一分力！
