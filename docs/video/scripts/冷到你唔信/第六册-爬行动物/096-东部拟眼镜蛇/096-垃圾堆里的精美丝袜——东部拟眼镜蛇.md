# 垃圾堆里的精美丝袜——东部拟眼镜蛇

兄弟們，如果你喺澳洲東部、中部或者新畿內亞南部嘅桉樹林入面，或者喺沿海地區嘅荒地、熱帶草原嘅林地、乾旱嘅灌木叢，甚至係農田度走過，咁你運氣好嘅話，可能會見到呢條——東部擬眼鏡蛇！

唔好以為佢淨係生存喺荒山野嶺，佢可以話係無孔不入——郊區、農場、建筑物後面、大石頭底，甚至垃圾堆裡面都可能搵到佢！巴閉啦，呢條蛇真係邊度都有！

不過呢，你又唔使太擔心，佢哋雖然周圍躲，但係唔會主動伏擊人嘅——淨係受到驚擾之後先至會變得危險。但係啊，佢依然係澳洲第二毒嘅蛇（淨係輸俾內陸太攀蛇），而且傷人數目就係所有毒蛇入面最多嘅！

其實呢，東部擬眼鏡蛇最犀利嘅地方在於佢哋身體表面嘅網紋圖案——靚到呢！動物學家喺1846年收集標本嗰陣，被呢啲美麗嘅圖案震撼到，感嘆話：「哇，真係似精美嘅絲襪！」所以啊，「垃圾堆裡嘅精美絲襪」呢個朵就係咁嚟嘅！
