# 冷到你唔信 - 太平洋丽龟

橄榄球咁大嘅龟你见过未？太平洋丽龟——集体产卵名场面，成千上万只同时爬上岸！



你可能話，丽龟？听个名几得意噃。佢又叫橄榄龟，你知唔知点解？因为佢只背甲似个橄榄球咁上下，心形噉样，灰黑色，滑滑哋——唔讲以为边个唔知喺边度执到个橄榄球。

更劲嘅係，太平洋丽龟曾经係世界上数量最多嘅海龟！1968年，单係墨西哥海岸一年就被商业捕捞咗100多万只！100万只喎，睇见个数字我都打冷震——呢个唔係龟，係"海鲜嘅士"啰！

但係，事情就喺呢度反转。

因为当年科学家太高估计佢哋嘅数量，以为资源充足，结果过度捕捞一路狂飙。等大家反应过来，太平洋丽龟已经跌咗三成有多——就似你以为只虾有好多，结果眨下眼就唔见咗三成，仲得唔得？

你知唔知？太平洋丽龟其实分布劲广嘅——太平洋、印度洋、大西洋、非洲西海岸、南美洲部分沿海，全部都见到佢哋嘅踪影。一只龟游匀成个地球，呢个先至係真正嘅"世界公民"！

讲到太平洋丽龟最犀利嘅，就係佢哋钟意群聚——成日几千只一齐上岸产卵，场面勥过大片！但係正因为咁，当过度捕捞出现，佢哋特别容易中招。

呢个就係典型嘅"你以为好多个，但係一跌就跌三成"嘅故仔。

各位，跟住落嚟讲嘅，就係太平洋丽龟最唔讲得之嘅悲情命运。

曾经佢係海洋入面最多嘅海龟，1968年单年捕获量超过100万只。但係好景不常，因为数量被过度高估，商业捕捞一发不可收拾。到今时今日，全球总数比高峰期下跌超过30%——而家仲係受保护嘅濒危物种。

所以下次见到太平洋丽龟，记得——呢唔系普通嘅龟，而係曾经嘅"海龟一哥"，而家只能低调生存。
