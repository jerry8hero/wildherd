# 长胡子的鬃狮蜥

兄弟們，你話下面呢張圖片隻蜥蜴下巴上面嘅「鬍子」係唔係P上去嘅？放心，絕對冇整蠱！呢隻被人叫做「鬍子龍」嘅就係鬃狮蜥喇。佢哋下巴上面所謂嘅「鬍子」，其實係「顎鬚」嚟嘅，因為呢啲顎鬚令到鬃狮蜥睇落似生咗鬃毛嘅獅子，所以就叫「鬃狮蜥」。

當佢哋感覺到危險嘅時候，鬃狮蜥會脹起個身同埋下巴上面嘅「鬍子」，遠遠睇上去真係似一架威風八面嘅雄獅！偷偷話你知，此刻鬃狮蜥嘅鬍子重會變成黑色，佢哋可能在警告你：唔好埋我，我唔係好惹嘅！

不過呢，並唔係淨係雄性鬃狮蜥先有「鬍子」，雌性都有嘅，只不過雄性更加鍾意「顯擺」自己嘅鬍子啫。
