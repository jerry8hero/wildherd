# 喙头蜥：不是蜥蜴，更胜蜥蜴

喙头蜥：不是蜥蜴，更胜蜥蜴 喙头蜥：不是蜥蜴，更胜蜥蜴 如果说蛇蜥是冒牌的蛇，那么接下来这位重量级“蜥蜴”可就是冒牌的蜥蜴了！它们是世界上最珍稀的爬行动物，仅生活在新西兰柯克海峡的几个小岛上，是新西兰的特有爬行动物，被奉为“国之骄傲”的喙头蜥。 为什么说它们是冒牌蜥蜴？因为它们根本就不是蜥蜴呀！在爬行动物中，有个被称为“喙头类”的家族，喙头蜥就属于喙头类，虽然它们长得十分像蜥蜴，但人家在爬行界可是“独立门户”。 喙头类早在大约2.4亿年前的三叠纪就出现了，在恐龙全盛的侏罗纪，喙头类也达到了家族最兴盛的时期，而且它们很幸运地躲过了白垩纪晚期的生物大灭绝。喙头蜥的牙齿在现生爬行动物中独一无二——下颌的一排牙齿紧紧咬在上颌的两排牙齿之间。据科学家估计，喙头蜥的寿命可达百岁。
