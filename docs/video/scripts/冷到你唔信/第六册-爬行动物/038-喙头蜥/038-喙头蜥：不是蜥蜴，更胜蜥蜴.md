# 喙头蜥：不是蜥蜴，更胜蜥蜴

兄弟們，如果話蛇蜥係冒牌嘅蛇，咁下面呢位重量級「蜥蜴」就係冒牌蜥蜴喇！佢哋係世界上最珍稀嘅爬行動物，凈係生活喺新西蘭柯克海峽嘅幾個小島上，係新西蘭嘅特有爬行動物，被奉為「國之驕傲」嘅喙頭蜥。

但係啊，點解話佢係冒牌蜥蜴呢？因為佢根本唔係蜥蜴！喺爬行動物入面，有個叫「喙頭類」嘅家族，喙頭蜥就屬於喙頭類，雖然佢生得十分似蜥蜴，但係人哋喺爬行界係「獨立門戶」嘅。

其實呢，順帶提一提，喙頭類早喺大約2.4億年前嘅三疊紀就出現咗，喺恐龍全盛嘅侏羅紀，喙頭類亦都達到咗家族最興盛嘅時期，而且佢哋好彩地躲過咗白堊紀晚期嘅生物大滅絕。喙頭蜥嘅牙齒喺現生爬行動物入面係獨一無二嘅——下顎嘅一排牙齒紧紧咬喺上顎嘅兩排牙齒之間。據科學家估計，喙頭蜥嘅壽命可達百歲！你話幾巴閉！
