# 周身為家既玫瑰蚺

兄弟們，你以為靚蛇就好脆弱？今日介紹既玫瑰蚺，完全顛覆你想像！唔好睇佢生得靚，其實呢條蛇堅巴閉！

玫瑰蚺呢，佢哋的的分佈範圍超級廣——由美國加利福尼亞州聖迭戈的沿海山脈，向北到莫哈維沙漠，向東到索諾蘭沙漠，到處都見到佢哋的身影。佢哋係真正的「周身為家」，乜嘢環境都難唔倒佢：干旱的灌木叢、沙質的平原、岩石斜坡、灌木覆蓋的山麓——全部都係佢哋的家！呢種適應能力，真係唔話得。

有水的地方？更加興奮喇！溪流旁邊、峽谷底部呢啲地方，玫瑰蚺見到實開心到彈起。夏天解暑一流！

佢哋其實幾細隻——出世時間先得20～30厘米，就算大個咗都係得90厘米以下，仲未有人哋老爺爺嗌拐杖咁長。不過佢哋既鱗片就好滑溜，條尾仲要好似工人叔叔用的錐子咁——幾得意下。
