# 白唇树蜥：白嘴唇，蓝战袍

白唇树蜥：白嘴唇，蓝战袍 白唇树蜥：白嘴唇，蓝战袍 兄弟們，如果我話俾你知有一種蜥蜴著住藍色「踢死兔」，嘴就係白色嘅，你信唔信？就係我哋今日要講的——白唇樹蜥！

白唇樹蜥如其名，從上嘴唇一路延伸到身體側面，都有白色斑紋。但係最搶眼的就係佢哋身藍色的體色——遠望過去，簡直就似一幅描繪蔚藍天空的油畫囉！

不過呢，其實平時白唇樹蜥係冇咁靚的顏色，而係呈現比較沉悶的棕色或者橄欖色。只有到咗交配季節，佢哋先至會著上呢件藍色「戰袍」（主要係身體前部），然後就不停上下點頭——就係想靠「高顏值」嚟引發異性注意囉！

兄弟們，動物界的「時裝表演」都幾犀利添！你話係唔係？覺得意猶未盡的話，记得撳個三連支持下喔！你重想知道咩新奇動物？留言話我知啦！
