# 背着一块饼干的饼干龟

背着一块“饼干”的饼干龟 背着一块“饼干”的饼干龟 咦？这只龟是背着一块薄饼干吗？开玩笑，这就是人家的壳。它是生活在非洲坦桑尼亚、肯尼亚等地的饼干龟，因其扁平的壳而得名。一些成年的饼干龟的壳，长度居然超过17厘米（反正我没吃过这么大的饼干）。 不光是外形像饼干，饼干龟的壳在重量上也不负“饼干”之名。要知道，龟壳其实是由龟的脊椎、肋骨、腹肋、胸骨和皮内成骨等愈合形成的，相对于大多数陆龟沉重的壳，饼干龟的外骨骼有很多孔洞，因此壳更加轻盈。这样它们就不用背着沉重的“包袱”，行动起来也比一般的陆龟更敏捷。但也由于饼干龟这种稀奇古怪的样子，引来人类收藏的兴致，因此它们被大量捕捉。
