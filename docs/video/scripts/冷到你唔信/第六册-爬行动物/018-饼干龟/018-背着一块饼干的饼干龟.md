# 背着一块饼干的饼干龟

背着一块"饼干"的饼干龟 背着一块"饼干"的饼干龟 兄弟们，睇嘞！呢只龟真系似到出奇——以为自己背住塊薄饼咁，唔讲唔知呢就係人家嘅壳嚟嘅！呢只嘢叫饼干龟，住喺非洲坦桑尼亚、肯尼亚呢啲地方，个名就係咁嚟咖。有啲大只啲嘅成年龟，壳长可以去到17厘米幾！（反正我未食过咁大隻嘅饼干咯）唔单止似饼，重量都系呢——轻到你想象唔到！你知唔知龟壳系点形成嘅？係由脊椎、肋骨、腹肋、胸骨同皮内成骨等等痴埋一齐整成嘅。相比其他陆龟嗰啲死沉死沉嘅壳，饼干龟嘅外骨骼有好多窿窿，所以成副壳轻好多！就係咁，佢哋唔使背住咁重嘅包袱，郁起身嚟都比一般陆龟灵活几倍。但係，正正因为佢生得咁古怪得意，引嚟好多人去捕捉嚟收藏，结果呢......你懂的。
