# 冷到你唔信 - 饼干龟

兄弟們，背脊似块饼干，扁平到可以滑下楼梯——饼干龟，龟界最得意嘅『压缩包』！

好喇，我哋从头讲起。今日吹下水，讲只名字劲得意嘅陆龟——饼干龟！

你可能話，饼干龟？係唔係味道嘅饼干啊？唔通只龟入面系空心㗎？如果你咁样諗，就晒你啦。饼干龟可唔係用嚟食嘅，而係因为佢个壳勆到似一块薄饼干！

你知唔知，饼干龟个壳可以有几夸张？勆平平咁，直头似你食开嘅消化饼！成年嘅可以超过17厘米——你就係当紧真系一块17厘米大嘅饼干啰。不过呢块「饼干」就唔系几好食住，味道应该一般。

但係啊，讲到个壳，就唔可以唔提佢最劲嘅地方——勆到呢，勆到个壳充满住孔洞！

佢个壳唔似其他龟咁实心，係由脊椎、肋骨、腹肋、胸骨埋一齐愈合形成嘅外骨骼，但因为有好多孔洞，所以勆轻盈。普通陆龟背住个壳就好似背住个石头，饼干龟呢？就好似背住块威化饼，轻飘飘，行起身都比其他陆龟敏捷好多。就似你背个书包同人哋背个铁箱咁，点追呀！

不过呢，勆得意嘅外形反而害咗佢。

因为个样太得意，饼干龟被人睇中，大量捕捉去做宠物。想象下，你天生勆样呃到好多粉丝，本来生活得好地地，结果被人话「你咁得意，我要带你走」——饼干龟表示：我只系想安静做块饼干姐！

其实呢，讲到食方面，饼干龟就唔系素食者啰。

佢哋喺野外乜都食——草、树叶、花、水果，甚至动物尸体都得。就似个素食者突然话「我今日想试下川菜」，完全唔似你想象中嗰种斋坐唔郁嘅乖龟。

顺带提一提，各位，跟住落嚟讲嘅，就係饼干龟最得意嘅生存绝技。

除咗勆轻盈，饼干龟重有一个劲犀利嘢——佢哋可以爬得劲高！虽然唔系好大只，但係四肢有力，攀石爬树都OK。就似一个灵活嘅小胖子，唔好睇少佢！

不过呢个技能都有代价——因为太容易被人捉走，加上栖息地破坏，而家野生饼干龟数量已经开始减少。下次见到咁得意嘅「饼干」，记得远观唔好亵玩啦！
