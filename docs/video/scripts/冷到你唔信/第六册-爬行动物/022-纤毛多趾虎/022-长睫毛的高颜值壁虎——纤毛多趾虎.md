# 长睫毛的高颜值壁虎——纤毛多趾虎

长“睫毛”的高颜值壁虎——纤毛多趾虎 长“睫毛”的高颜值壁虎——纤毛多趾虎 这只壁虎长了睫毛吗？其实，那不是睫毛，是纤毛多趾虎眼睛周围皮肤形成的毛发状突起。不过，它们的“睫毛”可不只长在眼睛周围，而是从脑袋一直延伸到尾巴下面，形成两排。虽然“睫毛”给纤毛多趾虎的颜值加分不少，但这也让它们少了很多壁虎的眼睛都有的一层“防护罩”——眼睑。那它们的大眼睛不会被风吹干吗？进了脏东西又怎么办？别担心，纤毛多趾虎的眼睛覆盖着透明的薄膜，这层薄膜能让它们的眼睛保持湿润。眼睛里有脏东西也不要紧，它们的舌头可是“清洁刷”，能把掉进眼睛里的灰尘等舔干净！ 和很多壁虎不同，纤毛多趾虎的尾巴一旦断了，就不能再长出新的了。所以，你在野外经常会见到没有尾巴的纤毛多趾虎。
