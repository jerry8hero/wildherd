# 冷到你唔信 - 绿海龟

绿海龟竟然唔係绿色？内脏先至係！佢重可以潜到100米深、一辈子净係食素——呢个『绿色』名唔係随便改㗎！



你可能話，海龟我见得多啦，唔就绿色嘅咩？你如果係咁諗就晒你啦。绿海龟可唔係你想象中嗰种普通绿色海龟，佢竟然可以落到几十米甚至上百米深！而且用肺呼吸噶，每隔一阵就要浮上嚟换气——嘢，呢个几攰啰，唔知以為紧做紧乜大工程。

更离谱嘅係，绿海龟一辈子几乎净係食素！餐餐都係海草同大型海藻，食到个身都变晒绿色——但呢个绿唔係外壳绿色，而係里面嘅脂肪同组织都染成淡绿色，呢个先係"绿海龟"嘅真正由嚟。呢个大概就係最健康嘅饮食方式啰，食吓食吓就将自己变成"绿色食品"。

你知唔知绿海龟可以潜几深？几十米甚至上百米！普通潜水员落到三四十米就劲啰，佢呢？两三倍啰。虽然潜得深，但係爬行动物嚟嘅嘛，始终要上水面换气㗎。就似你打机打到兴起，突然要比人呉走——绿海龟心想：我就想做条安静嘅深海王者，點解要咁对我！

你知唔知？绿海龟係国家一级保护动物！

讲到食素，绿海龟就係代表。严格素食主义者，食得多绿嘢，叶绿素慢慢渗入脂肪同组织入面。但唔好以为佢斋听唔打得喎，喺海里游起身快过好多鱼！仲可以活到60到80岁——呢个就係养生之道喇：清淡饮食，保持心境开朗，想唔长寿都几难。

你知唔知？绿海龟大个咗之后，里面变晒样！

出世嗰时黑蚊蚊似块炭，等大个咗因为食多咗海草，内脏同脂肪就慢慢染成淡绿色——但外壳都係棕绿色为主，唔系完全变绿㗎。

所以下次见到黑色小海龟，千祈唔好认错，佢係绿海龟嘅"童年黑历史"！

讲到绿海龟最犀利嘅能力，就梗係导航——动物界嘅GPS就係咁劲！

每年繁殖季节，绿海龟就会从茫茫大海入面，精准噉游返自己出世嘅嗰片沙滩度产卵。係几劲？就似你喺广东，突然间唔洗导航就知边间係你楼下嘅奶茶店——而且係知到门牌号咁！
