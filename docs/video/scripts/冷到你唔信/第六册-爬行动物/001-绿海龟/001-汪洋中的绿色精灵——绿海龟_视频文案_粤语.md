# 冷到你唔信 - 绿海龟

兄弟們，你知唔知綠海龜其實唔係绿色？內臟先至係！佢重可以潛到100米深、一世人都淨係食素——呢個『綠色』名唔係隨便改㗎！

好喇，你可能話，海龜我見得多啦，唔就綠色嘅咩？你如果係咁諗就晒你啦。綠海龜可唔係你想象中嗰種普通綠色海龜，佢竟然可以落到幾十米甚至上百米深！而且用肺呼吸噉，每隔一陣就要浮上嚟换气——嘢，呢個幾攰啰，唔知以為緊做緊乜大工程。

但係啊，更離譜嘅係，綠海龜一世人幾乎淨係食素！餐餐都係海草同大型海藻，食到個身都變晒綠色——但呢個綠唔係外殼綠色，而係裡面嘅脂肪同組織都染成淡綠色，呢個先係"綠海龜"嘅真正由嚟。呢個大概就係最健康嘅飲食方式啰，食吓食吓就将自己變成「綠色食品」。

你知唔知綠海龜可以潛幾深？幾十米甚至上百米！普通潛水員落到三四十米就勁啰，佢呢？兩三倍啰。雖然潛得深，但係爬行動物嚟嘅嘛，始終要上水面换气㗎。就似你打機打到興起，突然要比人呉走——綠海龜心想：我就想做條安静嘅深海王者，點解要咁對我！

不過呢，你知唔知？綠海龜係國家一級保護動物！

讲到食素，綠海龜就係代表。嚴格素食主義者，食得多綠嘢，葉綠素慢慢渗入脂肪同組織入面。但唔好以為佢齋聽唔打得喎，喺海裡游起身快過好多魚！仲可以活到60到80歲——呢個就係養生之道喇：清淡飲食，保持心境開朗，想唔長壽都幾難。

其實呢，你知唔知？綠海龜大個咗之後，裡面變晒樣！

出世嗰時黑蚊蚊似塊炭，等大個咗因為食多咗海草，內臟同脂肪就慢慢染成淡綠色——但外殼都係棕綠色為主，唔系完全變綠㗎。

所以下次見到黑色小海龜，千祈唔好認錯，佢係綠海龜嘅「童年黑歷史」！

順帶提一提，講到綠海龜最犀利嘅能力，就梗係導航——動物界嘅GPS就係咁勁！

每年繁殖季節，綠海龜就會從茫茫大海入面，精準噉游返自己出世嘅嗰片沙灘度產卵。係幾勁？就似你喺廣東，突然間唔洗導航就知邊間係你樓下嘅奶茶店——而且係知到門牌號咁！

科學家覺得佢主要靠地球磁場嚟導航，兼夾會靠埋嗅覺同視覺地標，但呢種與生俱來嚟嘅能力的确勁。

想象下，你喺海裡游咗二三十年，有一日凭住記憶入面某個神秘信號，你開始往某個方向游，歷經千辛萬苦，終於揾到你出世嘅沙灘——然後掘個窿，落完蛋，又游走。幾浪漫。

各位，跟住落嚟講嘅，就係綠海龜最刺激嘅生存故仔。
