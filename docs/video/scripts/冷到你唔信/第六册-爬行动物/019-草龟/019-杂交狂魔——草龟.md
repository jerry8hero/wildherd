# 杂交狂魔——草龟

"杂交狂魔"——草龟 "杂交狂魔"——草龟 讲到龟，好多人会第一时间噉諗：乌龟呀！，但其实呢，乌龟係代表唔到成个龟类家族咖——佢哋只系其中一个细分支，正式名叫"草龟"。如果你觉得"草龟"呢个名比起之前讲嘅嗰啲龟名太普通，咁你就大错特错喇！佢被誉为自然界的"杂交狂魔"，巴閉到爆！根据过往嘅统计，草龟可以同雌性马来闭壳龟、雄性四眼斑水龟、甚至人工饲养嘅日本石龟一齐杂交繁育，犀利吧？2006年嗰阵，科学家发现咗一种当时以为是新物种嘅"腊戍拟水龟"，但研究完之后，有啲科学家就觉得其实係草龟同黄喉拟水龟嘅杂交品种，仲要係早在史前时期就已经杂交埋，勐唔勐？正一证明咗草龟就系名副其实嘅杂交狂魔！顺带提一提，草龟唔单止系杂交狂魔，仲系3000幾年前中国商朝人最like嘅两种龟之一！呢个又系点解呢？下文揭晓！记得comment话俾我知你估啱唔啱！
