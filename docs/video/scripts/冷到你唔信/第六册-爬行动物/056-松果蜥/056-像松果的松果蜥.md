# 像松果的松果蜥

兄弟們！如果我話俾你知，有種蜥蜴係藍色舌頭嘅，你會諗起邊個？冇錯，就係斜紋藍舌蜥！但係呢，今日我要介紹嘅係佢哋家族入面更加奇葩、更巴閉的一位——松果蜥！

睇吓呢個位，佢生得真係幾另類吓，背脊上面嘅鱗片就好似鋪咗層碎石咁硬，隔遠睇埋去簡直就係一隻松果——松果蜥呢個名就係咁樣嚟嘅，幾得意吖！

除咗用藍色舌頭嚟虛張聲勢警告入侵者之外，松果蜥仲有一項技能——頭尾迷惑法！佢條尾巴短短粗粗嘅，形狀同佢個頭特別似，所以令到入侵者會有一種「頭尾分唔清」嘅感覺，從而達到迷惑嘅效果。正因為咁，松果蜥仲有個花名叫「雙頭石龍子」——呢個名幾過癮吖！

其實呢，松果蜥係一個幾得意嘅生物，佢哋用自己嘅方式嚟保護自己，真係好機智吓！兄弟們，你哋覺得呢種「頭尾迷惑」嘅策略幾犀利呢？留言分享一下你嘅看法啦！如果片片啱你口味，記得俾個三連支持下！你嘅支持係我繼續創作嘅最大動力！
