# 海中蜥蜴——海鬣蜥

兄弟們，估下圖入面呢個家伙係咩來頭？冇錯，佢都係鬣蜥嚟嘅，不過係可以喺大海度暢游嘅海鬣蜥！海鬣蜥主要生活喺加拉帕戈斯群島嘅海岸邊，成年雄性海鬣蜥喺肚餓到嘰嘰叫嗰時就會落海，因為海入面有佢鍾意食嘅美食——海藻同埋生喺海底岩石上面嘅其他植物。

但係啊，呢度嘅海水真係凍到飛起！海鬣蜥係爬行動物，佢哋需要成日曬太陽嚟保持體溫，咁凍嘅海水，佢可以順利食到海里嘅植物咩？

其實呢，海鬣蜥犀利得很！佢哋會先趴在海邊嘅岩石上曬夠太陽，暖暖身，然後一頭扎進海里，游到海底嘅岩石上，争分奪秒，大口啃食植物。佢哋重會用個鈎一樣嘅爪子紧紧抓住岩石，防止被海底強大嘅水流冲走。食飽之後，海鬣蜥會迅速游上岸曬太陽，以免身體完全冷卻。

好喇，今日嘅介紹就到呢度。如果你鐘意呢類有趣嘅動物冷知識，記得點讚、投幣、收藏，我哋下次再傾過！
