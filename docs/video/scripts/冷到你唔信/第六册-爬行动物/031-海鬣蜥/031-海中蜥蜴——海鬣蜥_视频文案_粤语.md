# 冷到你唔信 - 海鬣蜥

各位，今日继续冷到你唔信！

今日吹下水，讲只勭到你唔信嘅爬行动物——海鬣蜥！

你冇听错，係蜥蜴，但係會游水嘅蜥蜴！一只喺大海里边游边食生猛海藻嘅蜥蜴！唔係呃你，係真係存在，佢就係——海鬣蜥！

你可能話，爬行动物唔係净係会喺陆地爬嘅咩？几时轮到佢哋下海撈嘢食啈？唔到你唔信，海鬣蜥就係爬行动物界嘅异类，係造物主，食饱饭太无聊整出嚟嘅奇葩作品。

海鬣蜥主要栖息喺加拉帕戈斯群岛——就係赤道上面嗰个神奇群岛，成日都会有啲稀奇古怪嘅生物喺度出现。呢度就係海鬣蜥嘅主场。

你知唔知海鬣蜥有几犀利？

作为爬行动物，佢竟然可以喺海水里边游水揾食！唔好以为好简单——海水好冻㗎！海鬣蜥係冷血动物，体温会跟随环境变，落到海里边冻到震啰。但係呢个唔係问题——因为佢哋有自己嘅生存智慧。

讲到海鬣蜥最犀利嘅能力，就梗係——未落水，先热身！

你知唔知，海鬣蜥落海之前，会先搵块岩石喥起身，利用太阳嘅能量将身子晒暖——呢个就似充电咁，晒够咗先至落水。就似你去冻水池游水之前，要先喺岸边整热身操咁，不过佢係靠晒太阳嚟热身，係纯天然嘅太阳能发热。

呢个就係劲：身为爬行动物，用太阳嘅热量嚟做活动能量，晒到暖晒暖再落水揾食——就似你去 buffet 之前先在家食饱咁，先warm up再冲！

你知唔知海鬣蜥点解唔会被海水冲走？

海鬣蜥对爪似钩咁样，密密麻麻噉抓紧海底岩石！呢个就係勭嘅嘞——一嘢爪落去，想被冲走都几难。佢哋就似攀岩高手咁样，喺海底嘅岩石上面稳稳阵阵噉揾食，任凭海浪点冲都唔怕。

而且佢哋游水嘅姿势劲有型——S形噉摆动个尾，就似条鱼咁样，流畅到呢！

你知唔知海鬣蜥食啲乜？

成年雄性海鬣蜥主要食海藻同海底岩石上面嘅植物——係素食者嚟嘅！不过细嘅时候唔同，佢哋会食昆虫同海中小动物，慢慢大个咗先至转素。

但係重点唔喺呢度——重点係：海水好冻，而爬行动物需要热量嚟消化食物。佢哋就发明咗一个劲嘅操作：吃饱之后迅速游上岸晒太阳，继续充电！就似你食完饭唔立即坐下，而係去做运动帮助消化——不过佢係靠晒太阳嚟消化，係真正嘅太阳能消化系统。

讲到海鬣蜥最特别嘅嘢，就梗係——佢哋係唯一会游水嘅鬣蜥！

整个蜥蜴家族入面，得佢一个可以喺大海里边自由噉游。呢个就係勭到呢——几千万年前，佢哋嘅祖先决定落海搵嘢食，结果竟然适应咗海嘅生活，演化成咗而家呢个样。

科学家发现，海鬣蜥嘅祖先可能喺南美洲大陆生活，后来飘流到加拉帕戈斯群岛，发现陆地上冇乜竞争对手，食物又丰富——然后就慢慢适应咗海边同海洋嘅生活。所以呢个就係典型嘅「搵到蓝海就转型」例子。

各位，跟住落嚟讲嘅，就係海鬣蜥最唔讲武德嘅生存技能。

海鬣蜥有个劲犀利嘅技能——佢可以喺水下边个停车！

你冇听错，真係停车——就咁样悬浮喺海中间，一嘢都唔游，靠嘅就係佢嘅密度低、骨骼轻、仲有肺部空气多。所以佢可以静静鸡噉喺度休息，等太阳继续帮佢热身。
