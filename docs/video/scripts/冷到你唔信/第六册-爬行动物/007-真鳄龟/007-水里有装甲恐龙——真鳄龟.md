# 水裡有裝甲恐龍——真鱷龜

兄弟們，今日我哋嚟聊聊爬行動物寵物界嘅一對明星——大鱷同小鱷！唔好衰咗心思以為我係講鱷魚啊，呢度嘅「大鱷」係來自北美洲大名鼎鼎嘅真鱷龜！

真鱷龜係北美洲淡水「龜王」，成年龜嘅殼可以長到80厘米，體重可以達到80千克！有記錄話1937年喺美國堪薩斯州發現咗一隻180幾千克嘅巨型真鱷真鱷龜，嘩，唔好話龜龜，就係人都冇咁重！
