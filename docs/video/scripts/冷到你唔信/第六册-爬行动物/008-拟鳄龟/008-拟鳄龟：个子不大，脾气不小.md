# 拟鳄龟：个子不大，脾气不小

兄弟们，如果你钟意爬宠，咁你一定听过「小鳄」呢个名。但係啊，你知唔知「小鳄」系边个？佢就系同样嚟自北美洲淡水水域嘅——拟鳄龟！呢个就系今日我哋嘅主角嘞！

好喇，虽然话佢叫「小鳄」，但係佢哋成年之后，个样同真鳄龟比较起身，体型同咬合力都系细好多。不过呢，呢度就有个得意嘅地方嘞——虽然身体细，但係脾气就认真巴閉！拟鳄龟会主动追猎物，对於入侵者更加系一点都不客气，「一言不合」就上嘴咬！

而且啊，佢哋条颈又长又钟灵活，可以向后伸到身体后便，攻击速度超快，理论上可以轻松咬断手指添！自20世纪60年代被日本作为外来宠物引进之后，拟鳄龟就被记录为日本严重咬伤来源之一——你就知佢几咁「恶死」嘞！不过呢，虽然性情咁凶猛，但系岩岩出世嘅拟鳄龟宝宝就有几得意噶：佢哋喺离开巢穴之前可以发出「咔嗒」「咯吱」噉嘅声音；有些重可以发出似手指喺梳仔度摩擦噉样嘅奇异声音。呢样嘢，你知唔知？
