# 夏天的舞蹈家——铲鼻蜥

兄弟們，夏天熱到飛起！點降溫好呢？冷氣、冷飲、西瓜梗係要安排上！但你知唔知，生活喺撒哈拉沙漠裡面嘅鏟鼻蜥，夏天降溫方法就犀利喇——天氣太熱嗰時，佢哋會跳起奇怪嘅「舞蹈」！

其實呢，鏟鼻蜥並唔係真係喺跳舞。天氣炎熱嘅時候，沙漠嘅沙子就會變得非常燙，鏟鼻蜥冇辦法好好地站在沙子上。佢哋會靠尾巴保持身體平衡，然後一次抬起兩隻腳，比如先抬起左前腳同右後腳，然後大約間隔10秒，再抬起右前腳同左後腳，這樣交替抬腳，睇上去就好像喺跳奇怪嘅舞蹈！

唔好笑住先！呢個動作其實幫到佢哋冷卻腳掌，否則腳掌一路貼住咁燙嘅沙子，分分鐘燙熟！所以話，鏟鼻蜥為食都冇咁易為食嘅。
