# 牵扯进乌龙案的萨氏巨蜥

兄弟們，而家世界上公認最大嘅蜥蜴係科莫多巨蜥，最重又最長，可以超過3米，係真正的蜥蜴之王！但係啊，很久以前，世界最長蜥蜴呢個稱號曾經授予咗新幾內亞嘅薩氏巨蜥，佢都曾經風光過一時吓！

有啲報道話，薩氏巨蜥可以長到3米以上；有些消息仲話佢哋可以長到3.5米；甚至有報告話世界上存在超過6米嘅薩氏巨蜥——但係呢，呢啲說法都未經證實，大約係啲人以訛傳訛。目前已知最長嘅薩氏巨蜥大概2.5米左右，比起最大嘅科莫多巨蜥，佢就遜色唔少喇。

雖然薩氏巨蜥生得都幾大隻，但係佢哋嘅菜單上面全部都係啲弱小動物，佢哋都係走溫和路線嘅。佢哋鐘意食鳥類同埋鳥蛋，亦都會食啲腐肉，有時竟然會捉狗嚟食——呢樣就真係幾估你唔到囉，佢哋竟然連狗都敢捉！
