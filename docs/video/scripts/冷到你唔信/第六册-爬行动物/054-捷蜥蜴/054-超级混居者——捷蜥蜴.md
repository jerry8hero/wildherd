# 超级混居者——捷蜥蜴

超级“混居者”——捷蜥蜴 超级“混居者”——捷蜥蜴 兄弟們，我哋今集嚟介紹一位足迹遍天下的小子——捷蜥蜴！

由草原到森林，果園到灌木叢再到谷地，甚至海拔2000米以上的高山——總之冇邊度係去唔到嘅，你話幾屈機呢？

但係啊，捷蜥蜴唔單止環境適應力爆燈，仲係難得一见的「交際花」！喺中國新疆，科學家就發現捷蜥蜴會同其他動物「同埋住」——

佢哋可以同樣係蜥蜴家族成員的敏麻蜥和平共處，又可以同某類蛙類乜乜呵呵，重巴閉的係——佢哋甚至同含有劇毒的中介蝮一齊紮營！你話幾離譜！

不過呢，你唔好以為佢哋係冇脾氣的。在交配季節，雄性捷蜥蜴就會一百八十度轉變，同競爭對手大打出手——有的只係鬥幾秒，有的就可能嚟個超過15分鐘的馬拉松大戰！
