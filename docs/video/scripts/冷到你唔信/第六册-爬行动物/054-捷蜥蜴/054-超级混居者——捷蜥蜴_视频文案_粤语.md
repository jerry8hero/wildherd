# 冷到你唔信 - 捷蜥蜴

东欧到西亚，城市到乡村——捷蜥蜴，边个话蜥蜴就一定恶样？



你可能話，蜥蜴我见得多啦，唔就到处爬嗰啲？你如果係咁諗就晒你啦。捷蜥蜴可唔係你想象中嗰种普通蜥蜴，佢竟然可以喺草原、森林、果园、灌木丛、谷地乜都揾到食！最离谱係，海拔2000米以上嘅高山都有佢哋嘅足迹——呢个咪就係真正嘅"浪子"啰，周游列国，边度都系屋企。

你知唔知捷蜥蜴有几巴闭？佢係爬行动物界嘅"交际花"！

科学家喺新疆发现，捷蜥蜴可以同好多其他动物"混居"。同敏麻蜥和谐共处，乜嘢事？两个唔同物种竟然可以一齐生活，唔洗打唔洗闹，就似两个老友记围埋一齐倾偈。

更劲嘅係，捷蜥蜴竟然可以同某些蛙类亲密接触——蜥蜴同青蛙？呢个组合本身就够晒骑呢，但係佢哋竟然可以埋埋一齐食嘢，唔知以为佢哋系亲戚。

但最离谱嘅仲喺后面——捷蜥蜴竟然敢同含有剧毒嘅中介蝮待喺埋一齐！中介蝮係乜嘢？毒蛇嚟嘅，一口可以攞你命。但係捷蜥蜴就偏偏可以喺呢种毒蛇身边走来走去，仲要乜嘢事都冇。呢个就似你喺老虎笼入面散步，仲要帮老虎刷牙——唔知你劲在边度。

讲到点解捷蜥蜴咁叻，可以同唔同物种相处，科学家就觉得，可能系因为佢性格比较随和，唔会点样主动攻击人。而且佢哋嘅生存能力劲，适应力强，喺边个环境都可以揾到方法生存——就似一个语言天才，英文得，普通话得，粤语都得，喺边个国家都沟通无障碍。

你知唔知？捷蜥蜴喺交配季节，就会变成"打仔"！

每年到咗繁殖季节，雄性捷蜥蜴就会为咗争女仔大打出手。佢哋会互相追逐、撕咬、压制，就似两个后生仔喺篮球场上面片一样。但係有几刺激？就系呢啲打斗可以好快搞掂——有啲只係几秒钟，一嘢就分出胜负；但有啲就劲啰，可以打超过15分钟，就似一场持久战咁样。

想象下，两个男仔为咗女仔喺度打交，打到一半突然停低大家休息一阵，然后继续打——呢个就係捷蜥蜴嘅爱情故事，充满住戏剧性。

各位，跟住落嚟讲嘅，就係捷蜥蜴最犀利嘅生存之道。

佢嘅足迹遍布各地，从草原到高山，从森林到果园，乜嘢环境都可以生存。而且佢哋可以同唔同物种和平共处，呢种能力就叫做"超级适应者"——无论环境点变，佢都可以揾到方法活落去。
