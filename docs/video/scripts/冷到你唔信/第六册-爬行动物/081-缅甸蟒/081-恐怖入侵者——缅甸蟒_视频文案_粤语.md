# 冷到你唔信 - 缅甸蟒

美国搞到生态灾难——缅甸蟒，恐怖入侵者，叻到识开电视！



你可能話，缅甸蟒？唔就东南亚嗰边好常见嘅大蟒蛇？你估啱了！但你知唔知，呢只嘢入侵美国佛罗里达州之后，搞到当地鸡飞狗走！

你知唔知缅甸蟒有几大只？

成年缅甸蟒长约5米，有些甚至可以达到6米长！满嘴都系锋利嘅牙齿，肌肉劲到可以紧紧缠住猎物勒死——然后成条吞落肚！呢个就叫做"终极猎食者"！

你知唔知缅甸蟒入侵美国之后搞到几糟糕？

佛罗里达州嘅兔子、浣熊、狐狸等动物数量大大减少，甚至连郊狼、美洲狮呢啲"狠角色"都受到威胁！而且缅甸蟒重会同美国短吻鳄打起来——有時系短吻鳄食缅甸蟒，有时系缅甸蟒反杀短吻鳄！

但係！你知唔知我哋中国对缅甸蟒有唔同睇法？

对美国人嚟讲，缅甸蟒系入侵物种；但对我哋中国嚟讲，缅甸蟒系本土原生物种，被列为国家二级保护动物——食唔着，摸唔得！

各位，跟住落嚟讲嘅，就係缅甸蟒最犀利之处。

6米长，可以勒死短吻鳄——但系呢种威力只可以喺野外睇到。如果入侵其他地方，就会变成生态灾难——呢个就叫做"强者做错地方"！
