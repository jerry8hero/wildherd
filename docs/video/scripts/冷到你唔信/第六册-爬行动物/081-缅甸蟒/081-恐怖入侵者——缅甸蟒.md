# 恐怖入侵者——緬甸蟒

兄弟們，今日講呢條蛇，真係聽見都覺得巴閉——緬甸蟒！呢條嘢，真係大隻到不得了！

緬甸蟒本來係東南亞叢林、沼澤、山腳同河谷嗰度生活，係世界上最大既蛇之一。佢哋發育完成之後，可以去到5米長，有啲更加離晒大譜到6米咁長！滿口都係犀利既牙，肌肉又發達到嚇死人，一旦捉住獵物就會即刻缠上去，將對方勒到斷氣，然後一啖嘢吞嗮落肚。作為頂級捕食者，緬甸蟒喺大自然入面基本上係冇得輸，係食物鏈最頂層的存在。

但係！問題嚟喇——當緬甸蟒入侵去其他國家，情況就大件事喇！例如佢哋去到美國之後，佛羅里達州嘅兔子、浣熊、狐狸呢類動物數量急跌，就算係郊狼同美洲獅呢啲「狠角色」都受到威脅，仲要靠政府出手治理。緬甸蟒重會同美洲短吻鱷打大交，兩個纏鬥起來，輸贏係好難話——有時係短吻鱷食咗緬甸蟒，有時又會被緬甸蟒反殺！呢場龍虎鬥，真係幾誇張。

不過呢，順帶提一提，喺我國呢，緬甸蟒並唔係入侵物種，而係本地原生物種，已經被列為國家二級保護動物，受法律保護㗎！所以話，係要睇地方既，唔可以一概而論。
