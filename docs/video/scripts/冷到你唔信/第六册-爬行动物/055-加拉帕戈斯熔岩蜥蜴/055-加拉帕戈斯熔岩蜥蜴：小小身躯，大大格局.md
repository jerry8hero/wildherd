# 加拉帕戈斯熔岩蜥蜴：小小身躯，大大格局

兄弟們，今日我哋嚟睇吓一種超級犀利嘅蜥蜴！讲到加拉帕戈斯群岛，好多人都會記得嗰種身肥體壯嘅海鬣蜥——冇錯，佢就係被譽為「地球上最似哥斯拉嘅動物」嘅那位大哥。但係啊，我哋今日唔係要講呢位大哥，而係要講佢頭頂上面嗰隻小小嘅蜥蜴！你發現咗未？就係我哋今日嘅主角——加拉帕戈斯熔岩蜥蜴！

別睇佢哋通常得十幾厘米長，呢班細佬真係好巴閉，一啲都唔驚人啊！你近距離去睇佢，佢都唔會走嘅，幾大方吓。雖然身形嬌小，但係呢班友仔係島上嘅「除害標兵」嚟㗎——例如佢哋可以幫手滅好多好多加拉帕戈斯群島上面嘅蝗蟲。不過呢，當氣候乾燥嘅時候，熔岩蜥蜴都可以接受食下素嘅，佢哋都幾識嘆世界吓。

至於啲雄性熔岩蜥蜴呢，佢哋就好緊張自己嘅地盤㗎！佢哋會喺一舊大石頭上面標記自己嘅領土，然後不停噉擺動個頭，來表達「呢個係我嘅地方！」——呢個一系列動作睇上去就好似做掌上壓咁款，真係好笑！

好喇，講咗咁多，兄弟們有冇見過呢種熔岩蜥蜴呢？歡迎喺下面留言吹下水！如果覺得今條片幾過癮嘅，記得整個三連俾我！你嘅支持先係我繼續分享趣事嘅最大動力！下次我哋再一齊探索更多得意動物！
