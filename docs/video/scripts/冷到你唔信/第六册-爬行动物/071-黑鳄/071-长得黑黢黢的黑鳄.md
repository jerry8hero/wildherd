# 长得黑黢黢的黑鳄

长得黑黢黢的黑鳄 长得黑黢黢的黑鳄 兄弟們，之前我哋讲过，扬子鳄喺中国古代被叫作"鼍龙"，但係喺大洋彼岸嘅美洲大陆，仲有好多唔一样嘅"鼍"嘅呢！

其中呢就有一样嘢被誉為"战五渣"嘅凯门鳄，佢哋构成咗鳄鱼大家族入便嘅一个小家族——鼍类。不如我哋顺势讲下，唔理系扬子鳄定系凯门鳄，似乎身型都相对娇小，呢个时候你可能会问：兄弟們，鼍类入便就冇一个"扛把子"啊？

好喇，唔洗惊，等我介绍下呢位大哥——黑鳄！佢哋又叫"亚马孙鼍""黑凯门鳄"，系鼍类入便块头最大嘅嗰位！平均体长可以去到3～4米，最大嘅个体纪录仲话超过5米添！劲唔劲啊？佢甚至可以将亚巴孙雨林入便令人闻风丧胆嘅绿水蚺撕成碎片！你话有几巴閉咯！

其实呢，黑鳄喺鳄类家族入便，实力净系排在湾鳄、尼罗鳄、美洲鳄之后㗎咋，同奥利诺科鳄差唔多。不过呢，佢哋虽然个名入便都有个"凯门"，但系喺分类上並唔属于凯门鳄，而系独立嘅黑凯门鳄嚄！
