# 冷到你唔信 - 黑鳄

龟仔其实唔係用肺呼吸？变色龙可以同时睇两个方向？呢啲唔係玩笑，系爬行动物界嘅日常操作。冷到你唔信，爬行动物篇，带你拆解呢班冷血家伙嘅诡异生存技能！


你可能話，黑鳄？係唔係黑色嗰只？你如果係咁諗，就答啱啰！但你知唔知，呢只嘢竟然系鼍类中最大只嘅——平均体长3到4米，最大个体纪录超过5米！就係几咁巨型！

你知唔知黑鳄有几劲？

佢可以将亚马孙雨林中令人闻风丧胆嘅绿水蚺撕成碎片！绿水蚺係世界上最劲嘅蛇之一，但系黑鳄就可以制服佢——呢个就叫做"以大欺小"！

你知唔知？黑鳄虽然名入面有"凯门"，但其实唔属于凯门鳄，而系独立嘅"黑凯门鳄"！

讲到最犀利，就係黑鳄可以捉绿水蚺——

想象下，6米长嘅绿水蚺对上5米长嘅黑鳄，边个赢？科学家观察到，黑鳄可以撕裂绿水蚺——呢个就係自然界最疯狂嘅"鱿鱼游戏"！

各位，跟住落嚟讲嘅，就係黑鳄最黑实嘅外号。

"黑凯门鳄"——黑就梗概，但系点解叫"凯门"？因为佢曾经被归类为凯门鳄，但系后来发现基因唔同，就分开咗。呢个就叫做"分家"事件，黑鳄表示：我唔想再做"凯门"啦！
