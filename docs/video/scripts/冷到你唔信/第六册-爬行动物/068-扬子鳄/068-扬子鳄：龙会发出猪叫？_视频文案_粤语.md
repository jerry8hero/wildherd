# 冷到你唔信 - 扬子鳄

龟仔其实唔係用肺呼吸？变色龙可以同时睇两个方向？呢啲唔係玩笑，系爬行动物界嘅日常操作。冷到你唔信，爬行动物篇，带你拆解呢班冷血家伙嘅诡异生存技能！


你可能話，扬子鳄？哦，就嗰种好稀有嘅鳄鱼啰？你如果係咁諗，就晒你啦。扬子鳄竟然有个劲犀利嘅名——鼍龙、猪婆龙！而且佢会发出似猪叫嘅声音——龙叫猪叫傻傻分不清！

你知唔知扬子鳄点解被古人称为"龙"？

先秦古籍《山海经》记载，有一种长住龙头嘅"雷神"，当鼓起肚仔时能够发出雷鸣般嘅吼叫——呢个雷神嘅原型就可能系扬子鳄！古人见到佢发声，以为系神兽降世——就係几犀利，我哋中国本土都有"龙"！

但係你知唔知？扬子鳄其实冇声带！

冇错，佢根本冇声带——但係佢可以利用腭帆、喉褶、同埋鼻瓣等器官发出两种唔同嘅叫声。听起来似敲鼓、似雷鸣、更有点似猪叫——呢个就叫做"口技"！

你知唔知古人点解叫佢"猪婆龙"？

因为有人觉得佢嘅叫声似猪叫，所以又叫佢"猪婆龙"。但係你唔好以为佢系猪喎，佢系正牌恐龙嘅亲戚——鼍类嘅代表！就係几得意，历史记载同现实混合，名字就变成咗咁样。

讲到扬子鳄最犀利之处，就係佢系我哋中国特有嘅极危物种！

数量少到呢，野生扬子鳄大概只有几百只。但係国家好重视，已经列为一级保护动物——你敢搞佢？唔好彩啰！

想象下，你喺安徽、浙江一带见到佢——呢个"龙"慢慢游过去，但系叫声似猪——几得意！

各位，跟住落嚟讲嘅，就係扬子鳄最得意嘅身份。

佢系"鼍龙"，系"猪婆龙"，系中国特有嘅极危物种。叫佢"龙"又好，叫佢"猪"又好，佢依然系我哋国家嘅宝贝——而且係活化石，几千万年前已经存在！
