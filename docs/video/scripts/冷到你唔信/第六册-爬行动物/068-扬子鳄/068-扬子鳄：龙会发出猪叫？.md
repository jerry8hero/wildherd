# 揚子鱷：「龍」會發出豬叫？

兄弟們，之前我哋講過五爪金龍——圓鼻巨蜥——似中國傳說中的龍。其實，有啲鱷魚都有「龍之姿」㗎，例如生活喺中國長江流域嘅揚子鱷！

揚子鱷喺中國歷史上有個威威嘅名——鼉龍、豬婆龍。點解會被當成龍呢？其中一個原因就係因為佢哋嘅叫聲！

古書《山海經》記載了一種長着龍頭的「雷神」，鼓起個肚時可以發出雷鳴般嘅吼叫。呢個雷神嘅原型好可能就係揚子鱷。後尾因為有人覺得揚子鱷叫聲有啲似豬叫，又叫佢做「豬婆龍」。

但係！其實揚子鱷係冇聲帶㗎！佢哋靠腭帆、喉褶仲有鼻瓣等器官發聲，可以發出兩種唔同嘅叫聲，聽落似敲鼓、又似雷鳴、再加上少少似豬叫！一個身體幾種聲音，認真犀利！

兄弟們，下次有人問你中國有冇鱷魚，你要好大聲答有——就係揚子鱷！但記住啊，揚子鱷係國家一級保護野生動物，唔好傷害佢哋！
