# 中华鳖：尿尿……用嘴？

中华鳖：尿尿……用嘴？ 中华鳖：尿尿……用嘴？ 兄弟们，有冇咁古怪嘢嗱？新加坡啲研究人员发现，当中华鳖生活嘅沼泽干涸之后，佢哋居然仲会一头扎进啲细水坑度。奇就奇喺呢度——中华鳖系爬行动物，唔系用肺呼吸咩？咁点解佢成日鍾意将个头浸入水里面呢？究竟系口渴定系做紧乜嘢呢？研究人员后嚟得出一个勐到痹嘅结论：中华鳖鍾意将个头扎入水里面，唔系为咗饮大嘢，而系——尿尿！ 就好似有啲鱼会将体内嘅氮以尿素形式经过鱼鳃排到体外咁，中华鳖把声都有类似嘅鳃状结构！当佢哋将头浸入水嗰阵，就会用把声入面嘅鳃状结构将尿素通过口腔掟出嚟。天啊！呢只鳖真系用嘴尿尿咖！为食咁样嘅事情忍到，但用嘴尿尿......真系接受唔到！
