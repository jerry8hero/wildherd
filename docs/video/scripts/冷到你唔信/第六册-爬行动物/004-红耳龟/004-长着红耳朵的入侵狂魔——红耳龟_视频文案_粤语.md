# 冷到你唔信 - 红耳龟

兄弟們，红耳龟？巴西龟？傻傻分唔清楚？但系红耳龟已经入侵全球，危害程度超出你想象！

好喇，你可能話，红耳龟？哦，就嗰種寵物店成日見到嘅巴西龟啰！你如果係咁諗就晒你啦。呢只嘢，虽然名叫「巴西龟」，但係唔係来自巴西，而係美国同墨西哥！而且佢有个劲爆嘅名——被列入《全球100种最具威胁嘅外来物种》，几得人惊！

你知唔知红耳龟个名係点嚟嘅？呢个就劲搞笑——佢对耳仔同附近位置，有红色嘅条纹！就似挂住对红耳环咁样，係佢嘅标志性特征。但係你以为佢净系靠样呃人？错错咗！佢哋适应能力劲到呢，本土龟根本唔够打！

讲到红耳龟有几能顶，就係一个「劲」字可以形容。

佢哋係半水栖龟类，咩环境都掂——池塘、湖泊、河流、甚至沼泽地，统统都係佢哋嘅主场。适应能力强到呢，新环境？湿湿碎啦！正因为佢咁打得，我哋中国嘅本土龟喺生存竞争入面，完全输蚀。你话阴功唔阴功。

但係啊，更阴公嘅係，佢繁殖快到呢！一只红耳龟一年可以生几窝蛋，每窝十零只，而且佢哋带菌，可以传播沙门氏菌等多只病。本土龟一感染，成个族群都麻烦撈。你话係唔係劲？

不过呢，你知唔知？红耳龟喺好多地方係犯法嘅！

澳大利亚、欧洲部分国家，贩卖红耳龟係犯法嘅！日本更加喺2015年宣布禁止进口！点解？因为佢哋实在太劲喇——适应能力强、繁殖快、带菌，係外来物种入面嘅「战斗机」。一旦进入新环境，本土物种就好难企理。

所以下次见到红耳龟，记住——佢唔係普通嘅宠物龟，係个劲犀利嘅入侵者！

顺带提一提，讲到红耳龟最犀利之处，就梗係佢嘅「全球扩张」——宠物市场嘅超级明星！

话说当年美国宠物市场兴起红耳龟热，商人见到有钱赚，就将大量红耳龟当作货品，卖到全世界各个国家。结果呢？红耳龟凭住超强嘅适应力，喺好多地方扎稳阵脚，慢慢取代本土龟种。

呢个就係「宠物贸易」嘅代价——你以为买只龟好小事，但係一旦佢走甩或者被放生，佢就可以摧毁成个生态系统。你话几阴功啰。

想象下，你喺宠物店买咗只红耳龟，觉得得意就养大咗，后来觉得麻烦就放生。唔知就放生一只「生态炸弹」！佢会抢本土龟嘅嘢食同埋地方，传播病症，繁殖又多——呢个就係好心做坏事啰。

各位，跟住落嚟讲嘅，就係红耳龟最犀利嘅入侵能力。

红耳龟之所以成为入侵物种，就係因为佢有三个必杀技：第一，适应能力强，咩环境都掂；第二，繁殖快，一年几窝蛋；第三，带菌，传播病俾本土物种。

有研究显示，喺引入红耳龟嘅地方，本土龟类数量普遍下跌三成以上，有啲地方甚至跌到得番个位数。呢个就係「外来物种入侵」嘅威力——睇唔到，但係好阴公。
