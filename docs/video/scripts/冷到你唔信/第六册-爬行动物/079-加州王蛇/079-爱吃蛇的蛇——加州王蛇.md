# 爱吃蛇的蛇——加州王蛇

爱吃蛇的蛇——加州王蛇 爱吃蛇的蛇——加州王蛇 兄弟們，加州王蛇系冇毒嘅，但系佢哋就敢称"王"！你知唔知点解啊？

好喇，等我话比你知！啮齿动物、鸟类、两栖动物、其他种类嘅蛇……甚至像响尾蛇呢种毒蛇，都喺加州王蛇嘅"食谱"上面！佢哋基本系见咩食咩！你话有几巴閉咯！

其实呢，加州王蛇钟意食蛇嘅习惯，令到一啲养佢哋嘅人好头疼，于是呢，人哋就总系将佢哋单独关喺一个爬箱入便，免得佢哋走咗去食隔离嘅蛇！你话有几得意咯！
