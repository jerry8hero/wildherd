# 水面狂奔者——双冠蜥

兄弟們，當你漫步喺墨西哥南部或者哥倫比亞北部嘅雨林中，見到一隻正在水面狂奔嘅蜥蜴，你千祈唔好以為自己眼花花——呢個就係最令人眼前一亮嘅双冠蜥！

聽到個名，你可能覺得双冠蜥頭上生兩個「冠」，但係啊，其實淨係雄性先有兩個「冠」，雌性得一個「冠」啫。不過無論係一個冠定兩個冠，双冠蜥嘅形象都好容易令人聯想到歐洲傳說入面嘅蛇怪巴西利斯克，所以1768年著名生物學家林奈就用傳說中嘅蛇怪來稱呼双冠蜥（拉丁學名：Basilicus）。

更巴閉嘅係，為咗擺脫捕食者嘅追擊，双冠蜥真係會喺水面狂奔，速度快到可以唔沉落水！研究顯示，有些双冠蜥水面奔跑嘅最大速度可達每秒1.6米！你話幾犀利！
