# 可怕的食人鱼——巴拉圭凯门鳄

可怕的"食人鱼"——巴拉圭凯门鳄 可怕的"食人鱼"——巴拉圭凯门鳄 兄弟們，同样生活喺南美洲嘅巴拉圭凯门鳄，呢只嘢有一样嘢真系巴閉犀利——就系佢哋嘅生长速度简直系劲到飞起！

唔到你唔信，巴拉圭凯门鳄嘅幼体啊，唔计尾巴平均长度得12厘米左右姐！《美国国家地理》都评价话：“佢哋睇上去不过系漂浮喺巴西偏远内陆地区湖边灯芯草丛入便嘅，被风吹动嘅种子……”你话有几细粒咯！

但係啊，好戏喺后头噶！当巴拉圭凯门鳄到5岁噶时候，唔计尾巴就已经可以长到50厘米左右嘞！再大个10年，到15岁生长基本完成嘅时候，哇，可以长到80厘米至1米几添！兄弟們，你试下想象下，你由5岁到15岁，身高能增长一倍吗？梗系唔得啦！但係呢只嘢就做到啰，你话几犀利咯！

其实呢，巴拉圭凯门鳄另外一个花名叫“食人鱼凯门鳄”。佢哋下颚有啲牙齿可以穿过上颚上面嘅洞，令到佢哋嘅牙齿十分突出，正系鳄鱼入便嘅“食人鱼”嚄！你话得唔得闲惊啊？
