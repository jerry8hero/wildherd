# 可怕的食人鱼——巴拉圭凯门鳄

可怕的“食人鱼”——巴拉圭凯门鳄 可怕的“食人鱼”——巴拉圭凯门鳄 同样生活在南美洲的巴拉圭凯门鳄，拥有惊人的生长速度。巴拉圭凯门鳄的幼体，不算尾巴的平均长度只有12厘米左右，《美国国家地理》评价“它们看上去只不过像漂浮在巴西偏远内陆地区湖边灯芯草丛中的被风吹动的种子……” 巴拉圭凯门鳄到了5岁的时候，不算尾巴就已经能长到50厘米左右，而到了15岁生长基本完成的时候，可以长到80厘米至1米多。想象一下，你从5岁到15岁，身高能增长一倍吗？ 巴拉圭凯门鳄还有一个绰号叫“食人鱼凯门鳄”。它们下颚的一些牙齿可以穿过上颚上的洞，使得它们的牙齿十分突出，被誉为鳄鱼中的“食人鱼”。
