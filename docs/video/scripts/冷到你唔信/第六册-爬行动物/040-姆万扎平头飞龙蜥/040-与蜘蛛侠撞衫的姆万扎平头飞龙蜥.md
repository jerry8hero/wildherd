# 与蜘蛛侠撞衫的姆万扎平头飞龙蜥

兄弟們，咦？蜘蛛俠點解變成一隻蜥蜴啊？唔好以為自己眼花花，係呢隻蜥蜴正好同蜘蛛俠「撞衫」！

生活喺非洲坦桑尼亞、盧旺達同肯尼亞等地嘅姆萬扎平頭飛龍蜥係有名嘅「蜘蛛俠蜥蜴」，佢哋真係好像酷炫嘅蜘蛛俠——上半身紅色或紫色，下半身深藍色。不過呢，只有雄性姆萬扎平頭飛龍蜥先會同蜘蛛俠「撞衫」，雌性姆萬扎平頭飛龍蜥全身呈棕色，好為食，普通得好似我哋屋企隻貓咁。

雄性姆萬扎平頭飛龍蜥不單止「著」咗件蜘蛛俠「外套」，佢哋嗰飛簷走壁嘅技能亦似足蜘蛛俠——遇到天敵或其他危險險嗰時，佢哋可以用兩條後腿狂奔，又可以爬上棲息地嘅石塊，幾巴閉！

但係啊，順帶提一提，喺肯尼亞，姆萬扎平頭飛龍蜥嘅家園正在被人類侵占，但係不可思議嘅係，佢哋已經開始嘗試同人類共同生活喺一個屋簷下喇。
