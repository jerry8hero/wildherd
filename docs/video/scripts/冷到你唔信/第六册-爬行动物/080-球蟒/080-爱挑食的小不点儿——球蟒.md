# 爱挑食的小不点儿——球蟒

爱挑食的“小不点儿”——球蟒 爱挑食的“小不点儿”——球蟒 球蟒生活在非洲的大草原、大森林里。你印象中的蟒蛇是不是特别大？但球蟒是蟒蛇中的“小不点儿”，最大的身长也只有1米多。它们为什么叫球蟒呢？是喜欢玩球吗？其实，在遇到危险的时候，球蟒会把身体蜷缩成一个球的形状，然后把自己的脑袋、脖子等容易受到攻击的部位藏在这个“球”的中心，这就是它们名字的由来。 球蟒有点儿懒，不喜欢自己挖洞，总是躲在别的动物挖好的洞里。夏天的时候，球蟒可能会呼呼大睡。它们是爱挑食的主儿，喜欢吃大点儿的猎物，要是碰到的猎物不满意，球蟒一般就不会吃。被人工圈养的球蟒有时还会选择“绝食”，甚至会把自己饿死！ 球蟒在尼日利亚的东南部特别受当地人尊敬，当地人认为它们是地球的象征。遇到球蟒时，人们会把它们送回远离人群的安全地带；遇到死了的球蟒，人们还会为它们举行葬礼。
