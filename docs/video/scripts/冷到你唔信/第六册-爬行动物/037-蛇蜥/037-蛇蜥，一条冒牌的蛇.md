# 蛇蜥，一条冒牌的蛇

兄弟們，哎呀！呢條係唔係蛇啊？嘿嘿，唔好俾佢呃親，人哋係貨真價實嘅蜥蜴——蛇蜥！

蛇蜥身體幼長，又冇腳，一眼睇上去真係會令人誤以為係蛇。不過呢，蛇蜥有唔折不扣嘅蜥蜴屬性——似大多數蜥蜴咁，當蛇蜥遇到捕食者或受到驚嚇嚇嗰時，佢哋會舍卒保車，自斷尾巴逃走。只不過，蛇蜥斷尾之後，尾巴就唔可以長到之前咁長喇。

别睇蛇蜥生得像蛇，佢哋可冇好多蛇嗰種超快嘅移動速度，也冇好多蛇咁鬼死咁凶悍嘅戰鬥力。喺歐洲，家貓係出名嘅「蛇蜥殺手」，佢哋對貓嘅進攻完全冇招架之力。

但係啊，蛇蜥都有值得驕傲嘅地方——佢哋係出名嘅長壽蜥蜴。有些野生蛇蜥嘅壽命可達30年，有些被人類飼養嘅蛇蜥重可以活過50歲！你話幾犀利！
