# 打伞的狂奔者——褶伞蜥

兄弟們，呢個家伙算係最誇張嘅蜥蜴之一喇！感覺到危險嗰時，脖子上嘅薄膜會「撐」起身，遠遠睇上去就好似一把大傘，同時張大嘴巴嚇走敵人，逃跑嗰時重可以用兩條後腿喺乾燥嘅草原灌木地帶「狂奔」——呢個就係褶傘蜥！

說起澳洲嘅動物明星，好多人會諗起袋鼠、樹熊，但係褶傘蜥都係！佢哋因為生得靚同埋行為太「奇葩」，成日被影視作品拿去「戲弄」。比如，大導演史皮爾伯格1993年拍嘅荷里活大片《侏羅紀公園》入面，有啲恐龍被設計成脖子上有「薄膜」嘅形象，發動攻擊嗰時薄膜能似傘一樣撐起身，呢個靈感就來源於褶傘蜥！

其實呢，順帶提一提，褶傘蜥嘅形象重曾經出現喺澳洲嘅硬幣上面，你話幾巴閉！
