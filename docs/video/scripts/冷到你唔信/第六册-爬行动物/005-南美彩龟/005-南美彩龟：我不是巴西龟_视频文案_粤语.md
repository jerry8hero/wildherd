# 冷到你唔信 - 南美彩龟

今日继续冷到你唔信！

好，我哋从头讲起。今日吹下水，讲只超级冤嘅龟——南美彩龟！

你可能会问，南美彩龟？唔熟喈！但我讲个名你一定知——"巴西龟"！花鸟市场几文钱一只，入门级水龟，几乎每个爬友都养过。但係你知唔知，你养嗰只"巴西龟"，可能连巴西国旗都没见过。

呢个就係今日要讲嘅冷知识——"巴西龟"呢个名，根本就係张冠李戴！

你知唔知？"巴西龟"其实唔係来自巴西，而係来自南美洲！

点解会搞成这样？事缘在几十年前，台湾最早引入呢只龟嗰阵，因为佢来自南美洲，就噏做"巴西龟"。后来大陆也从台湾引进，因为运费等问题，改用北美洲的红耳龟——但係个名已经叫开咗，"巴西龟"呢个名就咁样继续用落去。

所以呢，你买嗰只"巴西龟"，可能係坐长途机从美国飞过嚟嘅，但係被叫咗几十年"巴西龟"，佢自己都一头雾水。

讲到分辨南美彩龟同红耳龟，就係睇耳仔！红耳龟如其名，耳仔附近有两条红色条纹；南美彩龟呢？耳仔附近干干净净，乜嘢红色条纹都冇！就似你睇人，红色围巾就梗係红耳龟，冇围巾就係南美彩龟——几简单。

不过，而家好多杂交种，真系要专业眼光先分得清。所以如果你喺宠物店买龟，唔好单靠个名，要自己睇清楚先。

你知唔知？南美彩龟虽然样似红耳龟，但係性格就好安静好多。

红耳龟乜名？"扫地僧"——走位风骚，攻击性强，成日追住人手转。南美彩龟呢？就比较淡定，唔会主动攻击，就算你行近近佢，佢都系一副"你玩你嘅，我游我嘅"噉样。

但係，唔好以为佢好求其！南美彩龟其实几钟意晒太阳嘅——每日都会喺石头上晒一阵，等体温升高，帮助消化。所以养佢记得整块高出水面嘅晒台，等佢可以做个安静嘅日光浴。

讲到最犀利嘅，就係南美彩龟嘅适应能力。

佢原产于南美洲，气候温暖，但係被引进到各地之后，几劲？冻天气都顶得顺！北方冬天室外虽然唔得，但係室内常温就得；而且佢哋对水质要求唔高，唔洗成日换水——呢个就非常适合新手。

不过，佢哋繁殖能力就好劲，一只母龟一年可以生20到30只蛋！所以呢，如果你系养嚟玩嘅，唔好混养太多，免得一年后突然多咗几十只龟仔，唔知点算好。

各位，跟住落嚟讲嘅，就係南美彩龟最冤嘅地方——名唔副实。

佢明明唔係来自巴西，但係被叫咗几十年"巴西龟"。佢明明几安静好养，但係因为个名呃咗好多人，以为佢系红耳龟，结果养完先发现完全系两个品种。

呢个就係造物主嘅骑呢逻辑：搞错咗你嘅产地，但係改唔到你嘅户口。
