# 加拉帕戈斯象龜：長得大，活得長

兄弟們，今日我哋嚟聊聊一種可以叫做「老壽星」的龜——加拉帕戈斯象龜！

之前有科幻節目《狂想曲：未來世界物種大揭秘》話，一億年以之後的地球上會演化出一種比大象重20幾倍、比一般人多4倍的巨龜。雖然我哋依家都未見過咁大隻的龜龜，但係地球上都有令人「大」開眼界的龜，就係現存體形最大的陸龜——加拉帕戈斯象龜！

加拉帕戈斯象龜可以話係龜龜界嘅巨無霸。成年象龜身長可達1.5米，平均體重175千克，呢個體重等於一個超級大肥仔的人類！唔好以為佢只係大隻，佢重有的係超長壽命——野生加拉帕戈斯象龜的平均壽命超過100歲！

但係最爆的係，有隻名叫哈麗的加拉帕戈斯象龜，2006年6月23日喺澳洲動物園因心臟衰竭去世，享年170幾歲！當時發現佢係世界上最長壽的動物。170幾歲啊，兄弟們！由清朝活到依家，見證埋幾代人的歷史轉變，佢都仲係龜龜一隻！

加拉帕戈斯象龜之所以叫象龜，就係因為佢哋的腿仔粗壯如同象腿，可以支撑晒咁大隻的身體。佢哋主要生活喺厄瓜多爾加拉帕戈斯群島，呢個群島又叫「巨龜之家」，就係因為有成群的大象龜住喺度。

兄弟們，如果你想見識下咁大隻又長壽的龜龜，加拉帕戈斯群島係一定一定要去的地方！但係記住啊，保護野生動物，唔好打擾佢哋的生活。
