# 被古人玩坏的恒河鳄

兄弟們，生活喺南亞淡水河流入面嘅恆河鱷，佢哋嘅樣生得好怪——身體又細又長，嘴巴亦都又細又長，就好似早點攤上賣油條嘅叔叔夾油條嘅夾子——呢個比喻幾到會吓！

早在4000多年前，南亞地區嘅人就已經認識呢種樣生得奇怪嘅鱷魚。考古學家發現嘅印章、石碑上面刻住呢個畫面——恆河鱷周圍全部都係魚，佢哋口入面仲叼住魚添！呢個說明幾千年前，人們就已經知道呢個位最鐘意食魚，係個真正嘅為食精！

有些古人將恆河鱷睇成河神、風神嘅化身。喺恆河鱷生活嘅河流附近，當地人認為恆河鱷可以治好多疾病，直接將恆河鱷當作藥物——呢樣真係幾離譜吓！喺尼泊爾，有些人認為雄性恆河鱷擁有神秘力量，人們經常殺死恆河鱷，然後收集佢哋嗰個奇怪嘅鼻子！塔魯人認為，將恆河鱷嘅屍體放喺田野度燃燒，能驅除害蟲，幫農民帶來豐收——呢啲古人真係乜都想得出吓！

其實呢，恆河鱷並冇可證實嘅攻擊人致死嘅記錄，亦都缺乏捕食大型動物嘅能力。呢個位同我哋中國嘅揚子鱷一樣，係極危物種，一度差啲喺野外滅絕——呢樣真係好慘吓！
