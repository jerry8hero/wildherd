# 冷到你唔信 - 恒河鳄

嘴巴幼长似夹子——恒河鳄，古代就被玩坏，当乐器又当神器！

你可能話，恒河鳄？印度嗰边嘅？唔就普通鳄鱼咩？如果你係咁諗就晒你啦。恒河鳄可唔係你想象中嗰种普通鳄鱼，佢竟然有只似油条夹嘢噉样嘅超长嘴巴——细到咁上下，但係劲犀利！

兄弟們你知唔知恒河鳄个样有几骑呢？

佢哋身体又细又长，嘴巴又细又长，就似早点摊卖紧油条个叔叔夹油条用嘅夹子！就係咁样款，唔好以为佢系衰嘢，呢个就係造物主嘅创意——细唔代表唔好睇！

兄弟們你知唔知？恒河鳄早在4000几年前已经被人认识！

考古学家发现咗好多印章、石碑，上面刻住恒河鳄周围都系鱼，佢嘴里还叼住鱼——呢个就话俾我哋知，几千年前人类已经知道恒河鳄系最爱食鱼嘅家伙。就係几犀利，佢系印度嘅"鱼王"！

讲到恒河鳄最惨嘅，就係被古人玩坏咗！

古时啲人将恒河鳄睇成河神、风神嘅化身。有人认为雄性恒河鳄拥有神秘力量，成日杀佢攞个鼻哥——呢个就真系几阴公啰。而且塔鲁人认为将恒河鳄尸体放喺田野度燃烧，可以驱除害虫——你就知古人有几钟意搞呢啲嘢。

但係兄弟們你知唔知？恒河鳄其实完全唔系杀人狂！

恒河鳄完全没有可证实攻击人致死嘅记录，佢亦都缺乏捕食大型动物嘅能力。就似个温柔巨人，明明大只得滞，但系就系唔会乱嚟——呢个就叫做"大只但系温驯"！

兄弟們你知唔知？恒河鳄同我哋中国嘅扬子鳄一样，都系极危物种！

一度险些喺野外绝种——呢个就係最惨嘅事实。因为人类捕捉、栖息地破坏，佢哋数量越来越少。所以呢，下次见到恒河鳄，记得——呢个唔系普通鳄鱼，而系差点被人类玩坏嘅"印度河神"！

各位兄弟，跟住落嚟讲嘅，就係恒河鳄最骑呢嘅长相。

你仔细睇下，佢个嘴仔似油条夹——细长细长。但係呢个唔系缺陷，而系为食鱼而生！佢可以快速噉将鱼夹住，然后吞落肚——呢个就係真正嘅"印度鱼王"本事。

所以下次见到恒河鳄，唔好以为佢系普通鳄鱼——呢个系4000年历史嘅"河神"，差点被古人玩坏，但依然坚持喺印度河度默默生存！
