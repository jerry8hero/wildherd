# 赤龟：红色精灵，望寒兴叹

兄弟們好啊！今日我哋嚟介绍一种好犀利嘅海龟——赤蠵龟！呢个名有几巴閉？等我话俾你知。

赤蠵龟呢个「赤」系代表红色，「蠵」喺我哋中国古代就系「大龟」噶意思。咁赤蠵龟担唔担得起呢个名？梗系担得起啦！赤蠵龟嘅壳一般系红棕色嘅，完全符合「赤」噶意思。而且佢哋仲系世界上最大嘅硬壳海龟添！有报告显示，有些赤蠵龟可以长到2米几，500几公斤重！话佢哋系「蠵」（大龟）真系一点都不夸张，几巴閉啊！

但係啊，就这么大只嘅「红色精灵」，「身世」就有几惨噶。原来呢，大约300万年前嘅上新世，今日嘅中美洲浮出海面，将大西洋同印度洋—太平洋之间嘅海流隔断咗。地球进入冰川循环，海流因为气候变化产生新流向，冷水上升到好望角附近，将好望角海水温度大降低。

呢个对於需要迁徙嘅赤蠵龟就系恶梦嘞——冷水令到佢哋「望寒兴叹」，完全冇计。最终，大西洋嘅赤蠵龟同太平洋嘅赤蠵龟就被完全分开嘞。其实呢，赤蠵龟嘅故仔话俾我哋知，即使系几巴閉嘅生物，喺大自然变化面前都显得几渺小。你又有咩感想啊？
