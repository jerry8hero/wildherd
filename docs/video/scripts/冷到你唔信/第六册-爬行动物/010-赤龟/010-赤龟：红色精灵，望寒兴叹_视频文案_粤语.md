# 冷到你唔信 - 赤龟（赤蠵龟）

兄弟們，红色精灵现身！但系佢最劲嘅技能竟然係——耐寒？赤龟，带你睇睇冰海入面嘅生存专家！

好喇，你可能話，赤龟？係咪因为佢红噉噉咁？恭喜你，答中一半！呢个「赤」的确係指红棕色，但「蠵」字先犀利——喺古代汉语入面，「蠵」就係大龟噉解。所以「赤蠵龟」三名合埋就係：红色大龟。起名嗰个人真系懒醒，但又几贴切。

你知唔知赤蠵龟有几大？成2米几长、500几斤重！係硬壳海龟入面最大嗰只。想象下，係海里游紧游紧，突然撞到个餐台咁大嘅龟——唔驶问，肯定係赤蠵龟。科学家仲发现过接近600斤嘅超级巨龟，真系龟中巨无霸。

但係啊，讲到赤蠵龟，最惨嘅係佢嘅身世。

大约300万年前，上新世纪嗰阵，中美洲突然从海里升上来——就似有个神秘力量攞横额阻住你哋咁，佢将大西洋同印度洋-太平洋之间嘅海流彻底截断。赤蠵龟本来游得好好哋，突然间条路就唔见咗。

跟住落嚟，地球进入冰川循环，冷水慢慢上升到好望角附近，海水温度大跌。赤蠵龟係要迁徙噉嘛，但係呢浸冻水就似条冰墙，佢哋只可以「望寒兴叹」——想游过去但係真系游唔动。

最终，大西洋同太平洋嘅赤蠵龟被完全隔开，各自发展。就係咁，赤蠵龟一家被逼散，经历几百万年进化，变成两个唔同嘅群体。

所以下次见到赤蠵龟，记得——呢只龟唔单止大，仲要经历过沧海桑田噉全球地理大变动，係历史见证者嚟噤。

顺带提一提，各位，跟住落嚟讲嘅，就係赤蠵龟最犀利嘅迁徙本能。

虽然被冰水阻隔，但赤蠵龟依然年年跨越大洋进行长距离迁徙。佢哋可以游成几千公里，从搵食区游到繁殖区，导航能力劲到呢。

科学家发现佢哋主要靠地球磁场嚟定位，就似内置GPS一样，精准噉知道自己喺边、要去边。呢种能力係天生嘅，唔需要学，係几百万年入面刻入DNA入面嘅。
