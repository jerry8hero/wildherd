# 会两种走路姿势的美国短吻鳄

兄弟們，美國短吻鱷主要分佈喺美國東南部嘅淡水濕地，佢哋係西半球鱷魚家族入面嘅大家伙嚟㗎！最大嘅雄性美國短吻鱷大個之後，身長可以達到4米以上——咁長都趕得上一架小汽車喇，幾和利害吓！

美國短吻鱷行路嘅姿勢特別有趣，分為「低走」同埋「高走」兩種。你成日見到嗰啲鱷魚行路嘅樣——美國短吻鱷會用肚皮貼住地面行，就叫「低走」；但係佢哋仲會通過伸展四肢將肚子抬離地面，姿勢就好似人做掌上壓時用四肢撐起身體咁樣，呢種行路方式就叫「高走」——呢個名改得幾貼切吓！

美國短吻鱷把口入面有七八十隻可怕嘅牙齒，係個真正的牙利王！佢哋係「為食精」，幾乎乜都食——魚類、鳥類、蛇類、龜類，佢哋都食，有時甚至會食鹿、野豬呢啲大型哺乳動物，係個乜都想試嘅為食精！
