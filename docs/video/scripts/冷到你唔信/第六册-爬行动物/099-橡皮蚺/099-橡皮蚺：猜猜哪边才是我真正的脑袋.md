# 橡皮蚺：猜猜哪边才是我真正的脑袋

兄弟們，一講到蚺類，你哋係咪即刻諗起啲巨大無比、能夠絞殺獵物嘅超級大蛇？但係，如果我話俾你知，下面圖片入面呢條都係一種蚺，你會唔會覺得出奇？

有鐵頭梗係要展示下「鐵頭功」啦！橡皮蚺特别擅長用堅硬嘅頭部挖掘地洞，仲會闖入啲鼠類嘅地洞度搵獵物。

不過呢，可能有兄弟會問：究竟邊個係頭邊個係尾啊？點解睇落兩邊都一樣？其實呢，橡皮蚺嘅頭同尾都係圓鈍形嘅，非常容易混淆敵人。有時佢哋重會把尾抬高嚟模仿緊郁動緊嘅頭！如果你想知道邊個係真正個頭？最簡單就係搵眼——有棕紅色小圓眼嘅一邊，就係佢哋嘅正牌腦袋！
