# 尼罗河里有没有尼罗鳄

兄弟們，尼羅鱷係非洲最大嘅爬行動物，佢哋嘅個頭淨係比灣鱷小少少——但係都已經好大吓！雄性尼羅鱷可以長到5米長，體重超過1噸，仲要係鱷魚家族入面嘅「長壽之星」——平均可以活到85歲，係個真正嘅長壽冠軍！

尼羅鱷主要棲息喺非洲東部、中部、南部，以及馬達加斯加島嘅河流、湖泊之中。科學家發現佢哋竟然都可以似灣鱷咁樣生活喺鹹水入面——人們就曾在三角洲、鹹水湖發現過佢哋嘅蹤影，幾得意吓！

有趣嘅係，歷史上尼羅鱷嘅分佈範圍好廣泛，曾經向北延伸到成個尼羅河流域。但係啊，由1000多年前開始，人類睇中咗佢哋嗰件珍貴嘅「鎧甲」，瘋狂噉捕殺佢哋——呢樣真係好可惜囉！目前喺尼羅河入面，尼羅鱷已經比較罕見，我哋凈係可以喺尼羅河流域嘅支流中間偶然見到佢哋，係個真正的稀有動物吓！
