# 冷到你唔信 - 钝吻古鳄

细过一个人的手臂——钝吻古鳄，娇小玲珑但系胆生毛，危险！

你可能話，钝吻古鳄？名字几得意噃！佢几大只呀？如果你咁问，就错咗——呢只嘢成年得1.2到1.6米，最大都只有1.7米左右，係世界上最细嘅鳄鱼之一！体重得6到7千克——就相当于湾鳄体重嘅百分之一！

你知唔知钝吻古鳄有几巴闭？

虽然细，但系佢被评為"世界上最轻嘅鳄鱼"——百分之一嘅体重，够晒犀利！但係最巴闭嘅唔系呢个，而系佢虽然细，但系好惊嘢——背侧、腹侧都有厚厚嘅鳞甲，如同古代士兵嘅铠甲！

你知唔知？钝吻古鳄虽然细，但系好打得！

有铠甲护身，就算面对体型大嘅捕食者都唔驶怕！佢唔似得其他细路仔见到大只就走，佢会留下来顶硬上！呢个就叫做"细路仔有胆色"！

你知唔知钝吻古鳄同其他鼍类有咩唔同？

佢更能够忍受低温！干旱嗰阵时，唔系走去捱眼训，而系主动出击揾水——呢个就叫做"搵食态度认真"！唔坐以待毙，主动出击，呢个就係生存之道！

各位，跟住落嚟讲嘅，就係钝吻古鳄最得意嘅生存哲学。

细？就细啰！但系着定铠甲，主动出击，唔怕大只佬——呢个就叫做"浓缩即系精华"！

所以下次见到细细只嘅钝吻古鳄，记得——呢个系世界最轻鳄鱼，但系着住铠甲主动出击嘅"小钢炮"！
