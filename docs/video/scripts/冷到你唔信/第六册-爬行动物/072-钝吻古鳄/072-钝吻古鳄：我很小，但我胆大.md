# 钝吻古鳄：我很小，但我胆大

钝吻古鳄：我很小，但我胆大 钝吻古鳄：我很小，但我胆大 兄弟們，你哋知唔知，呢世界之大，真系乜都有啊！钝吻古鳄呢只嘢就是一种生活喺南美洲河滨森林、湖、河畔洪溢林入便嘅袖珍鳄鱼。

好喇，唔到你唔信，佢哋成年体长得1.2～1.6米，就算最大嘅钝吻古鳄都至得1.7米左右添！系世界上最细只嘅鳄鱼之一啊！成年人嘅体重得6～7千克，净系相当于湾鳄体重嘅百分之一，呢个真系惨到笑，紧系被评为"世界上最轻嘅鳄鱼"啰！

但係啊，唔好以为佢哋细只就好欺负喔！其实呢，钝吻古鳄可唔系省油的灯㗎！佢哋背侧、腹侧都有一层厚厚的鳞甲，就好似古代士兵嘅铠甲咁，可以提供好好嘅防护。所以呢，当遇到体形大嘅捕食者威胁时，佢哋並唔系咁惊嘅！

其实呢，相比于其他鼍类，钝吻古鳄更犀利嘅地方就系可以忍受低温！喺干旱噶时候，佢哋会主动出击，揾水源饮！兄弟們，一句到尾，就系细只但系够晒胆正！你钟意呢只小个子吗？
