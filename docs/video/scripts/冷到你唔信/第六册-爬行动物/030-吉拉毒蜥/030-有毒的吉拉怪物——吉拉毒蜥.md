# 有毒的吉拉怪物——吉拉毒蜥

兄弟們，今天帶大家睇一睇呢隻勁巴閉嘅家伙——吉拉毒蜥！佢係北美洲灌木叢、橡樹林或者沙漠裡面生活嘅。不過呢，喺蜥蜴大家庭入面，佢哋係少數有劇毒嘅蜥蜴嚟嘅。在美國西部，流傳住一個叫「吉拉怪物」嘅傳說，話佢身體散發住有劇毒嘅氣息，一滴毒液就可以毒死人！呢個「吉拉怪物」嘅原型很可能就係我哋今日嘅主角——吉拉毒蜥。

但係啊，好多人唔知道嘅係，吉拉毒蜥唔會衰咗去用毒液嘅，就算抓獵物嘅時候都唔使。只有當佢覺得危險，為咗保護自己嗰時先至會用毒液。好少有吉拉毒蜥咬死人嘅案例，但係聽說被咬過之後會非常痛癢——一位動物記者被咬之後話：「呢次係我一生中經歷過最嚴重嘅痛，就像熱熔岩喺血管入面流。」

其實呢，順帶提一提，別看吉拉毒蜥唔係好出名，佢哋係真正嘅「老前身」嚟嘅！古生物學家發現，吉拉毒蜥早喺1億多年前嘅白堊紀就出現咗，同恐龍生喺同一片天地！
