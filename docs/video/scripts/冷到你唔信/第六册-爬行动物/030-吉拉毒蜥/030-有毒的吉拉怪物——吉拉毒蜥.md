# 有毒的吉拉怪物——吉拉毒蜥

有毒的“吉拉怪物”——吉拉毒蜥 有毒的“吉拉怪物”——吉拉毒蜥 吉拉毒蜥生活在北美洲的灌木丛、橡树林或沙漠中。蜥蜴大家庭中，它们是少有的剧毒蜥蜴。在美国的西部，流传着一个叫“吉拉怪物”的传说，它们身体散发着有剧毒的气息，一滴毒液就能毒死人！这种“吉拉怪物”的原型可能就是吉拉毒蜥。 实际上，吉拉毒蜥不会随便使用毒液，就连抓猎物的时候都不用。只有在感觉到危险，为了保护自己时才会使用毒液。很少有吉拉毒蜥咬死人的案例，但据说被吉拉毒蜥咬过之后会非常疼，一位动物记者被咬之后说：“这是我一生中经历过的最严重的疼痛，就像热熔岩在血管中流淌。” 别看吉拉毒蜥不是很有名，它们可是“老前辈”啦，古生物学家发现，吉拉毒蜥早在1亿多年前的白垩纪就出现了，和恐龙生活在同一片天地！
