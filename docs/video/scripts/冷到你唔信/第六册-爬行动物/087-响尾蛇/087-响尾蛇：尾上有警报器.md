# 響尾蛇：尾上有警報器

兄弟們，你哋有冇聽過會發出響聲的蛇？今日就介紹呢位——響尾蛇！呢樣野真係創新科技都整唔出咁巴閉！

響尾蛇的尾點解會响呢？原來係因為尾末端有「響環」，當佢哋受到威脅，呢個響環就會振動起來發出聲音——就好似話你知：「喂！唔好埋嚟呀，再埋近我就咬你！」呢個就係大自然既報夢系統，幾先進！

點解會有咁得意既設計呢？原來響尾蛇每蛻一次皮，尾上就會留低一個空心既「尾節」。響尾蛇一年到頭要蛻好多次皮，所以個尾就越堆越多空心尾節，串埋一齊就變成左「響環」喇！係咪好神奇呢？就係因為咁，響尾蛇既尾就越來越長，變成一個內有乾坤的警報器！

呢條紅菱斑響尾蛇的響環振動緊——雖然佢既毒性喺響尾蛇家族入面算係比較弱，但係被咬一口都係好危險㗎！所以話，千祈唔好亂咁摸佢。

響尾蛇最鍾意食兔子呢類細隻動物，佢哋制服獵物的法寶就係毒液——毒素會令獵物好快麻痹或者死亡。如果獵物咬咗一口仲有氣力走佬，響尾蛇就會靜雞雞咁跟踪，等對方毒發身亡先再回去享受大餐——真係幾陰功。就係因為咁，佢哋先至可以生存到今日。
