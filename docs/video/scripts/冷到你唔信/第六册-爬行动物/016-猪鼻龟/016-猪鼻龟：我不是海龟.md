# 猪鼻龟：我不是海龟

兄弟们好啊！今日我哋嚟介绍一种勂得意但系被勂多人误解的龟——猪鼻龟！因为个鼻生得似猪鼻，所以人们叫佢哋「猪鼻龟」。但係啊，有勂多人误会咗，以为佢哋生活喺海里——呢个就大错特错嘞！

其实呢，猪鼻龟系唔折不扣的淡水龟！佢哋的活动范围非常有限，仅生活喺澳大利亚、新几内亚部分地区的淡水环境入便。不过啊，佢哋的四肢就真系似海龟——猪鼻龟的四肢演化成鳍状，和海龟勂似勂似！

有科学家就认为，猪鼻龟系地球上唯一完全生活喺淡水里的淡水龟。但系重有些科学家话，猪鼻龟只系大部分时间待喺水下，偶尔都会上岸，只过佢哋嗰似鱼鳍一样的四肢，会令到其喺陆地上的行动勂笨拙。就系咁，先至会被人误会系海龟啰！

另外啊，因为猪鼻龟的前后足仅有两个爪仔，佢哋重有个花名叫「两爪鳖」，几得意吖嘛！不过呢，虽然佢哋个名叫「鳖」，但系其实唔系鳖来着，係属於蛇颈龟科添！你知唔知？
