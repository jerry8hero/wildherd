# 冷到你唔信 - 猪鼻龟

兄弟們，叫猪鼻但係唔係猪，唔係海龟但係住水里——猪鼻龟，名字搞到你估佢係边个！

好喇，你可能話，猪鼻龟？听个名就知係海里游咯喔？猪鼻——唔就似猪鼻哥咁样响水度喷泡啰——如果你係咁諗，恭喜你，你中咗全世界最多人犯嘅错！猪鼻龟佢唔係海龟，而係不折不扣嘅淡水龟嚟嘅！就似你听人叫「海马」，就以为马仔识游水咁——呃你嘅！

你知唔知猪鼻龟几犀利？佢係地球上唯一完全生活在淡水入面嘅淡水龟——注意，係唯一！其他龟类要么海水龟，要么半淡水，但係猪鼻龟就净係钟意fresh water！呢个就係佢哋嘅核心竞争力：一门专。

但係啊，佢同海龟又真系几似样喔——因为佢哋四肢都演化成鳍状，就似两对细嘅桨咁样。呢个就係猪鼻龟最骑呢嘅地方：明明係淡水龟，四肢偏偏似足海龟。就似你明明係广东人，但係学咗北方人讲嘢——係又唔係，唔係又係，呃你嘅！

其实呢，你知唔知猪鼻龟个样有几得意？个鼻哥隆起嚟，似个猪鼻咁——所以就叫猪鼻龟。但係佢哋唔单止个样得意，仲有个好得意嘅习惯：佢哋可以似鱼咁样，用肺呼吸，但係又可以长时间潜入水底！科学家发现，佢哋可以响水底呆到几十分钟唔洗上嚟换气——呢个技能几紧要？就似你打游戏打到紧张关头，突然间唔洗去厕所咁爽！

顺带提一提，你知唔知猪鼻龟有几稀有限制？佢哋活动范围劲窄嘅——净係生活在澳大利亚北部同新几内亚部分地方嘅淡水环境。换句话讲，你想去睇野生猪鼻龟，净係可以去呢个地球上面好细嘅几个地方。呢个就叫做「地方限定」啰——好似啲潮牌咁，唔係你去嗰个地区就买唔到！

各位，跟住落嚟讲嘅，就係猪鼻龟最得意嘅生存方式。

虽然四肢似鱼鳍咁样，但係佢哋喺陆地上就好笨拙喔——爬下爬下，呢个鳍脚完全帮唔到手。就似你着住拖鞋去跑步咁——理论上得，但实际上呢？你明嘅。

不过佢哋响水入面就好掂啰——对鳍脚推进力劲足，游起身劲顺畅。但係上岸呢？就算係繁殖咁重要嘅事，佢都净係短暂噉爬上岸噃。科学家就发现，佢哋大部分时间都响水下度少就少，唔会去太远嘅地方——就似个死宅女，明明有快递要去攞，但係就係唔想落床！

呢个就係猪鼻龟嘅生存之道：陆地？唔该，我选择大海——喔，唔对，係淡水！
