# 头上冒泡的资深潜水员——安乐蜥

兄弟們，估下圖入面呢隻蜥蜴頭上做咩冒泡泡？冇錯，佢就係新西蘭特有嘅喙頭蜥！唔好以為得我哋人類先至識潛水，喙頭蜥係「兩棲運動健將」嚟嘅——佢哋可以喺水里和陸上活動，但係最巴閉嘅係，佢哋識得用肺呼吸之餘，重識得通過皮膚進行氣體交換，呢樣嘢喺爬行動物界真係少見！

但係啊，問題來喇——如果佢哋用肺呼吸，點解頭上會冒泡泡呢？原來呢，當喙頭蜥潛入水中水中嗰時，佢哋會將空氣儲存喺肺部同埋喉囊入面，當肺部嘅氧氣用完時，佢哋就會从喉囊入面提取氧氣，呢個時候就會睇到頭上冒出一啲細細嘅泡泡！研究顯示，喙頭蜥可以潛水長達一個鐘頭，呢個技能真係幾犀利！
