# 蚯蚓和蜥蜴的合体——五趾双足蚓蜥

兄弟們，嘩！呢隻嘢好得意喎！身體似蚯蚓咁又長又粉嫩，但係又長咗兩隻「腳」，每隻腳上有5隻萌萌嘅小腳趾——睇落又係有少少似蜥蜴，呢個到底係變異咗嘅蚯蚓定係變異咗嘅蜥蜴啊？真係估你唔到囉！

其實呢，呢個位唔係蜥蜴亦都唔係蚯蚓，而係屬於蚓蜥家族嘅爬行動物——五趾雙足蚓蜥。咁唔就係蜥蜴囉？之前我哋講喙頭蜥嗰時已經話過你哋——唔係個名叫「蜥」就係蜥蜴嚟嘅，呢樣真係要記住佢！

五趾雙足蚓蜥嘅兩條後腿已經退化咗，前面兩條腿仲喺度，所以叫做「雙足」。佢哋成身粉紅色，喺地下可以似蚯蚓咁蠕動身體前行，所以就叫「蚓」蜥。呢個名改得幾貼切吓！

有趣嘅係，五趾雙足蚓蜥嘅日常菜單入面，既有蚯蚓，又有小蜥蜴……就係咁得意！佢哋真係乜都食得落吓，係個為食精！

兄弟們，呢種「兩邊都沾」嘅動物你見過未？留言吹下水話俾我知！如果覺得今條片幾得意，記得整個三連支持一下！你嘅支持係我繼續分享趣事嘅動力！
