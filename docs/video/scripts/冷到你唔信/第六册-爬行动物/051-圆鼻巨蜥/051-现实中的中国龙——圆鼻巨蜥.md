# 现实中的中国龙——圆鼻巨蜥

现实中的中国“龙”——圆鼻巨蜥 现实中的中国“龙”——圆鼻巨蜥 兄弟們，我哋中國人成日話「龍的傳人」，龍係我哋中華民族的精神圖騰。但係你知唔知道，現實世界中都有動物被叫做「龍」呢？

就係生活喺我國華南、西南部分地區的圓鼻巨蜥！喺當地佢有個型的名——五爪金龍！

圓鼻巨蜥係一種大型水棲蜥蜴，最大只的超過3米——長長的尾巴、利利的牙同爪（每隻爪有5個腳趾），加上一張充滿爆發力的嘴，睇落真係威風凜凜的巨龍款！

不過呢，2023年有單嘢上咗熱搜——有人話五爪金龍的唾液有毒！但係科學家就未有一致結論，究竟係有毒係冇毒，到今日都係個謎。
