# 蓝灰扁尾海蛇：我要吃鳗鱼

但係啊，唔好以為所有海蛇都係能唔上岸就唔上岸嗰種，呢條嘢就特別為食——藍灰扁尾海蛇係半水生嘅，成年之後大概一半時間喺海洋，一半時間喺陸地。雖然佢哋喺陸地上嘅移動速度淨係得水入面嘅五分之一，但係正因為咁，人類遇到佢哋嘅機會反而多好多！
