# 恰住老實蛇，後果好嚴重——藍腹伊澳蛇

兄弟們，喺澳洲，紅腹伊澳蛇有位好兄弟——藍腹伊澳蛇！呢條蛇就好似隱世高手咁，一出手就知有沒有！

佢哋同紅腹伊澳蛇最明顯的區別就係肚的顏色——藍腹伊澳蛇的腹部係灰色或者藍灰色的。佢哋鍾意住喺開闘的草地、草原嗰度。就係因為咁，佢哋先至可以匿草叢裡面伏擊獵物。

藍腹伊澳蛇的性格有啲害羞，但係如果你激怒佢哋，例如踩到佢哋、用棍仔嘟佢哋——佢哋就唔會就咁放過你。佢哋會咬你，而且一咬就唔會輕易鬆口，仲會喺你傷口上面反覆嚼！呢招就係佢哋的必殺技，幾狼死！

不過呢，藍腹伊澳蛇的毒液比紅腹伊澳蛇勁好多，被咬傷的一定要即刻去看醫生——呢樣真係唔係讲笑的！所以話，千祈唔好亂咁恰善良的蛇，否則後果自負！
