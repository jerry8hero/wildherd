# 冷到你唔信 - 加拉帕戈斯陆鬣蜥

各位，今日继续冷到你唔信！

好，我哋从头讲起。今日吹下水，讲只被伟大科学家嫌弃嘅爬行动物——加拉帕戈斯陆鬣蜥！

你可能話，陆鬣蜥？又係鬣蜥？我见得多啦！慢住，呢只嘢当年被达尔文嫌弃到贴——佢係咁形容嘅："上半身橘黄色，下半身红褐色，长得又蠢又难看！" 我就问：达尔文叔叔，你係几犀利观察到呢只嘢咁样衰？

不过话说转来，我反而觉得呢只嘢几可爱——似只橙色加红色夹埋嘅大型变色龙，又似条着错晒色嘅大蜥蜴。但係呢种独特配色，唔知係咩潮流风格呢？

你知唔知加拉帕戈斯陆鬣蜥嘅生活几叹？

日头就癞喺火山岩上面叹世界——就咁样样摊开四肢，让太阳公公帮佢哋取暖。呢种生活态度，我只有两个字：羡慕！如果我哋人类都可以日日癞喺石头上晒太阳，唔知几爽。

但係一到晚上，就唔同讲法啰——佢哋会乖乖噉钻返入自己嘅洞穴度，冚实被仔冚到晒，噉就先可以维持体温。你知唔知，爬行动物冇自己嘅体温调节系统，佢哋係靠环境嚟控制体温噶！就似你冬天焗住被窦度，唔想出嚟——但係佢哋係日头焗太阳，晚上焗被窦，认真识叹。

讲到最犀利嘅，就係佢哋识合作——动物界嘅互利共生，就係咁样！

加拉帕戈斯岛上有啲得意嘅小鸟，呢啲小鸟专登跳上陆鬣蜥身上，帮佢啄走啲虱子同寄生虫——陆鬣蜥梗系梗係唔郁，等住小鸟帮佢免费 grooming。

呢个时候陆鬣蜥就梗係——"你就食你嘅，唔好搞我"。小鸟就噉食寄生虫食到饱，双方都满意，合作共赢！你话係几和谐？

呢种关系就似你去餐厅食饭，有人为你剥虾，有人帮你剔牙——你就坐喺度乜都唔使做，舒服！

你知唔知？呢种跨物种合作喺自然界其实几常见，但係每次见到都会觉得造物主真系鬼咩都谂得出！

各位，跟住落嚟讲嘅，就係加拉帕戈斯陆鬣蜥最得意嘅外貌特徵。

达尔文嫌佢丑？我就唔觉得。佢哋上半身橙黄色，下半身红褐色——呢种撞色搭配，如果係人类着衫，肯定被话唔会着嘢。但係放喺动物界，就变成独一无二嘅标志。

而且你知唔知，呢只嘢虽然样衰衰，但係有几百万年前就住喺加拉帕戈斯群岛度——係岛上嘅原住民嚟嘅！比达尔文嚟嘅时候早到几百万年。所以话，论资排辈，佢先係大佬！
