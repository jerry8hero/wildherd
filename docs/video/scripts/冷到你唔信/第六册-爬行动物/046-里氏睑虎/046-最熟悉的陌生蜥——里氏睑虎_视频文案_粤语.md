# 冷到你唔信 - 里氏睑虎

你屋企见到嘅那只壁虎，就係佢——里氏睑虎，最熟口面嘅陌生蜥！

你可能話，睑虎？睑虎係咩？食得嘅？唔使问，肯定有好多人连个名都未听过。但係我讲一样嘢，你可能即刻"哦"一声——佢哋一家都係夜行动物对眼超大似金刚狼，而且会爬墙！係呀，睑虎係壁虎嘅亲戚嚟嘅，不过个样再型少少。

里氏睑虎，又叫"利希登费尔兹睑虎"，呢个名夹唔夹先？夹到呢——呢个名係来自越南河内一个工程师，叫卡尔雷斯·利希登费尔兹。呢位猛人唔知係几时捉咗几只睑虎，寄咗去法国国家自然历史博物馆。1897年，科学家正式帮佢哋命名——里氏睑虎就此诞生！睑虎家族第一位成员，係祖宗嚟㗎！

你知唔知？里氏睑虎同海南睑虎似到入木三分！

两者身体底色都係黑色或灰色，身上都有4条黄色横纹——第一眼望过去，基本分唔清。你以为係同一只？唔好喇，虽然似，但係有微妙分别：里氏睑虎颈上第一条横纹，靠近耳孔同嘴角嗰阵，颜色会慢慢变淡直至消失；但海南睑虎呢？条横纹就好似揸住支荧光笔咁，穿过耳孔直达嘴角，颜色从头到尾都咁饱满！

你就咁记：里氏睑虎条横纹"甩色"，海南睑虎条横纹"唔甩色"——够晒简单啦下话？

讲到里氏睑虎最犀利嘅身份，就梗係睑虎家族嘅老祖宗！

1897年被正式命名，成为睑虎科第一位正式成员。简单嚟讲，就係"睑虎呢个家族，由我开山"。你话几巴闭？但係呢个几巴闭身份係要付出代价嘅——因为佢係最早被研究嘅睑虎，所以科学家成日攞佢嚟同其他睑虎比较，结果发现海南睑虎、广西睑虎等各种睑虎都係"后来者"，全部似足里氏睑虎但又各有唔同。就似家族入面个大哥，个个细佬都似你但又唔完全係你——压力有几大你知唔知？

各位，跟住落嚟讲嘅，就係里氏睑虎最骑呢嘅求生技能。

睑虎类嘅英文名叫"eyelid gecko"——即系有眼睑嘅守宫！普通壁虎只可以靠层膜眨眼，但睑虎就有眼皮，可以好似我哋一样眨眼！呢个设定有几luxury？养过守宫嘅都知道，呢个係几犀利嘅特征。

更劲嘅係，睑虎同其他守宫一样，受到威胁时会自动断尾走人——然后条尾会原地跳跳扎扎，吸引捕食者注意，佢就趁机走佬。呢个就係经典嘅"分身术"喇：牺牲一条尾，换成条命，抵！
