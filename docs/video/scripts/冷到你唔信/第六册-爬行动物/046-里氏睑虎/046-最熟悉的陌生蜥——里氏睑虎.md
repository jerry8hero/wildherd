# 最熟悉的陌生蜥——里氏睑虎

最熟悉的陌生蜥——里氏睑虎 最熟悉的陌生蜥——里氏睑虎 兄弟們好嘠，今次我哋嚟吹下水，講一種被稱為「最熟絡的陌生蜥」嘅里氏睑虎！

里氏睑虎又嗌「利希登費爾茲睑虎」，個朵係來自越南河內一位工程師——叫做卡爾雷斯·利希登費爾茲。因為呢位工程師係第一個將里氏睑虎標本寄去法國國家自然歷史博物館嘅！1897年正式被命名，成為睑虎家族的老大哥。

你可能會覺得里氏睑虎的樣貌唔知喺邊度見過。冇錯！我國的海南睑虎同里氏睑虎就好似兩兄弟咁款——兩者身底都係黑色或者灰色，身上都有4條標誌性的黃色横紋。

但係啊，佢哋的唔同就喺呢啲黃色横紋度——里氏睑虎頸上第一條横紋，喺耳孔同嘴角區域顏色會慢慢散直至消失；而海南睑虎呢，呢條横紋就穿耳孔而過一路去到嘴角，而且顏色一路都係咁飽滿。

其實呢，里氏睑虎真係可以話係最熟絡的陌生蜥囉！兄弟們，你哋以前有冇見過呢類睑虎呢？留言吹下水啦！鐘意嘅話三連俾多兩錢肉緊喔！
