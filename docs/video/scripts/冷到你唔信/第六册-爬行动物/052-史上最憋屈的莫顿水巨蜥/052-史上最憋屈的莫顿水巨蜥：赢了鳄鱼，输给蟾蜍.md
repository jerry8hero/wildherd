# 史上最憋屈的莫顿水巨蜥：赢了鳄鱼，输给蟾蜍

史上最憋屈的莫顿水巨蜥：赢了鳄鱼，输给蟾蜍 史上最憋屈的莫顿水巨蜥：赢了鳄鱼，输给蟾蜍 兄弟們，我哋今集講一隻可以話係史上最無奈、最陰功的巨蜥——莫頓水巨蜥！

喺亞洲隔洋相望的澳洲，都有水生巨蜥——就係生活喺澳洲北部的莫頓水巨蜥。佢哋比圓鼻巨蜥更靠水，連曝太陽都要搵塊石頭或者浮木浸喺水度歎世界。

但係啊，講起莫頓水巨蜥，真係令人喊笑不得。雖然佢哋一般唔夠1米長，比唔上圓鼻巨蜥，但係佢哋都係極度惡啃的角色——魚、兩棲動物、鳥類、哺乳動物，統統都係佢哋的「餸」，甚至有人影到佢哋啃死年幼的淡水鱷魚喔！

不過呢，最陰功的係——就係咁巴閉的友仔，竟然會「輸」俾一隻唔起眼的蟾蜍！自從1935年蔗蟾蜍被引入澳洲，莫頓水巨蜥的數量就大跌——被蔗蟾蜍毒死成為近百年來最常見的死因之一。
