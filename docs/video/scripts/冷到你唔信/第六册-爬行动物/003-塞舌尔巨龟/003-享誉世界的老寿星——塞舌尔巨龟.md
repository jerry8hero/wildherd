# 享誉世界的老寿星——塞舌尔巨龟

兄弟們，今日我俾你哋介绍一位劲犀利嘅老寿星——塞舌尔巨龟！呢只嘢几咁巴閉？佢哋可以话系地球上最长寿嘅动物之一，有些幸运嘅塞舌尔巨龟可以活到超过200岁！200岁啊，唔好话龟，就系人类都唔知几多人做到添！

其实呢，喺18世纪嘅时候，有个英国海员从塞舌尔群岛带咗4只塞舌尔巨龟翻嚟，送咗俾英国东印度公司。呢4只入面有一只叫「阿德威塔」，1875年被送去加尔各答动物园，一直住到2006年先走，佢嘅寿命达到255岁！255岁喎，犀利到不得了添！阿德威塔1750年就已经出世，去到动物园嗰阵已经120几岁，呢part真系离谱！

顺带提一提，阿德威塔走咗之后，喺南大西洋圣赫勒拿岛仲有一只塞舌尔巨龟叫「乔纳森」，佢被认为系地球上依家存活紧嘅最老陆地动物。1832年出世嘅佢，依家已经192岁有突，真系老而弥坚！不过呢，呢只巨龟就真系几凄凉嘞，一个人仔孤独咁生活喺个岛度，得个「长命」又点话呢？讲到呢度我都唔知应唔应该替佢开心好。
