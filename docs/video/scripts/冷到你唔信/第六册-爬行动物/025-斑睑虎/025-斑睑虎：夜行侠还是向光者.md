# 斑睑虎：夜行侠还是向光者

斑睑虎：夜行侠还是向光者 斑睑虎：夜行侠还是向光者 兄弟们，佢就系身浪就好像着住件"豹纹衫"咁嘅斑睑虎——皮肤上面有好多似豹纹嘅黑色斑点。有啲科学家就认为，斑睑虎系一个唔折不扣嘅"夜行侠"，因为喺2016年之前，尼泊尔从来未发现过斑睑虎，佢可能真系成日晏嗰阵先至出嚟。但係有啲人就唔同意嘞——佢哋根据常年饲养斑睑虎嘅经验，发现斑睑虎喺圈养环境下可以利用紫外线合成维生素D3！喺实际饲养入面，饲养人员的确成日用紫外线为斑睑虎照射。我哋都知道，太阳系自然界最主要嘅紫外线光源，呢个样讲，斑睑虎嘅生长系需要光嘅吗？斑睑虎其实系唔系"唔排斥"太阳呢？虽然暂未有定论，但係有一啲人声称，佢哋曾经喺白天日光强烈嘅沙漠中，见到斑睑虎几休闲咁喺度曬太阳，唔惊热个喔！
