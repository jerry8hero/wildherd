# 尾巴上长了片叶子的平额叶尾壁虎

尾巴上长了片"叶子"的平额叶尾壁虎 尾巴上长了片"叶子"的平额叶尾壁虎 兄弟们，够钟睇下呢只壁虎有几古怪！佢条尾上面生咗片"叶子"——唔使笑，呢就系人家嘅尾巴嚟嘅！呢个就系马达加斯加特产嘅平额叶尾壁虎啦。平额叶尾壁虎平时就好鍾意匿喺树浪，除非万不得已都唔会落树，比如话喺产卵嗰阵，佢哋先至会冒险落树。平额叶尾壁虎喺树上面生活得点様？嘿嘿，真系"如鱼得水"喔！佢哋有片好似叶子咁阔大扁平嘅尾巴，呢个唔系斋睇嘅——有大作用咖！可以让佢哋喺树枝之间好轻便咁跳嚟跳去，拣都唔使諗！但係正如好多壁虎咁，平额叶尾壁虎遇到威胁嗰阵都会将条尾甩走，然后迅速走人。哈哈，呢个就真正慨"落叶"啦——唔使风吹，自己走！呢招"金蝉脱壳"几掂啊！
