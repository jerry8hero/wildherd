# 腾云驾雾的金花蛇

兄弟們，一講到龍，你會諗起啲咩？騰雲駕霧、乘風破浪？現實中又有冇會「飛」嘅蛇呢？答案係：有！就係生活在我國南方地區，以及印度同東南亞等熱帶或亞熱帶地區嘅金花蛇！

不過呢，佢哋唔係真係識飛，而係識喺樹林之間滑翔——呢招就好似武俠小說入面嗰種高手先做到嘅輕功！

究竟佢哋點解做到呢？原來佢哋用好有力嘅肌肉將兩側肋骨向外張，咁一嚟，本嚟管道狀嘅身體就變扁平咗，喺空氣中下降嘅阻力就會大大增加，可以滑翔得好遠！

科學家重研究過佢哋「飛」嘅動作，發現佢哋會進行一種好獨特嘅扭動，呢種扭動可以令到佢哋喺降落嗰陣唔會繙筋斗！犀利啦，佢哋簡直就係蛇中嘅輕功高手！
