# 倒霉的蜡皮蜥：大壁虎的伪品

倒霉的蜡皮蜥：大壁虎的“伪品” 倒霉的蜡皮蜥：大壁虎的“伪品” 兄弟們，我哋今集講一隻好陰功的蜥蜴——蠟皮蜥！

喺中國廣東、海南一帶，有一種穴居性蜥蜴。廣東湛江嗌佢「沙神」，海南就叫佢「坡馬」——就係蠟皮蜥。

蠟皮蜥鍾意窩喺開揚的沙質土地度，雖然環境麻麻，但係佢生得靚拎喔——有一身花紋，所以又嗌「蝴蝶蜥」，相當賞心悅目！

但係啊，過去幾十農曆年間，蠟皮蜥被人類瘋狂捕捉，以大壁虎「贗品」的身份拿去煲藥，導致种群數量一落千丈。2021年2月被列為國家二級保護動物——不過已經太遲囉，佢哋已經差唔多被人掃清光囉。

不過順帶提一提，以前文獻話珠海的淇澳島係冇蠟皮蜥分布的，但係2023年初，人們就O嘴發現咗佢哋——呢個發現重創造咗珠海物種新紀錄添！

兄弟們，保育呢家嘢真係要快手搞，希望蠟皮蜥唔好消失埋！你點睇？留言吹下水！鐘意嘅話記得撳三連喔！
