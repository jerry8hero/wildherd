# 指甲盖儿大的变色龙：侏儒枯叶避役

指甲盖儿大的变色龙：侏儒枯叶避役 指甲盖儿大的变色龙：侏儒枯叶避役 2012年，德国和美国的科学家在非洲马达加斯加发现了4种新的变色龙。让世人震惊的是，这些变色龙成年以后，只能长到大约人类的指甲盖儿那么大，它们很可能是世界上最小的蜥蜴。这些身形短小的变色龙都属于侏儒枯叶避役家族。这几种侏儒枯叶避役生活在马达加斯加东北部哈拉岛的热带雨林中，经过科学家测量，它们从鼻尖到尾部末端的总长度不足30毫米。这些侏儒枯叶避役平时只在枯叶之间生活，身体呈棕色。虽然也是变色龙，但并不变色，而身体的棕色能让它们与枯叶很好地“融为一体”，起到伪装的作用。由于这几种侏儒枯叶避役长得实在太像了，科学家费了好大功夫，用分子遗传学分析才认定它们原来是4种不同的侏儒枯叶避役。
