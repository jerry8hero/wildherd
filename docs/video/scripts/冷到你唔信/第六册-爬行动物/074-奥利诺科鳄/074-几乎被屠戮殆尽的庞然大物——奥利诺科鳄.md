# 几乎被屠戮殆尽的庞然大物——奥利诺科鳄

几乎被屠戮殆尽的庞然大物——奥利诺科鳄 几乎被屠戮殆尽的庞然大物——奥利诺科鳄 兄弟們，锥吻古鳄可以幸免被捕杀，但系同样生活喺南美洲嘅奥利诺科鳄，就冇咁运气啰！

好喇，等我话比你知奥利诺科鳄有几巴閉！相比于锥吻古鳄同钝吻古鳄呢两个"小家伙"，奥利诺科鳄可算系庞然大物啰！唔到你唔信，从20世纪80年代中期至90年代初期，野外被观察到嘅雌性奥利诺科鳄平均体长就已经超过3米啰！而成年的雄性奥利诺科鳄甚至可以长到6米添！正系巴閉啊！

鉴于其庞大嘅体形，奥利诺科鳄被认为系可以与黑鳄并列嘅美洲鳄鱼之王啊！但係唔好以为大只就得噶，佢哋嘅寿命仲好长添，可以活到80岁！真系老寿星啊！

但係啊，呢只嘢有几巴閉就几巴閉，可惜呢，因为奥利诺科鳄嘅皮好受市场青睐，为咗得到佢哋嘅皮制作皮包、皮带、皮鞋等，人类对其就进行咗大规模嘅捕杀！20世纪60年代，分布在委内瑞拉、哥伦比亚等地嘅奥利诺科鳄几乎被屠戮殆尽！你话有几阴公咯！
