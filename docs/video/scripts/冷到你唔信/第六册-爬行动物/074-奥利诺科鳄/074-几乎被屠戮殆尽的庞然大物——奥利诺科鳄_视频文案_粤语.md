# 冷到你唔信 - 奥利诺科鳄

打到快绝种——奥利诺科鳄，人类疯狂屠杀，而家得返少量！

你可能話，奥利诺科鳄？几大只呀？呢只可以话係南美洲嘅"鳄鱼之王"！成年雄性可以长到6米，雌性平均体长超过3米——呢个就叫做"鳄鱼巨无霸"！

你知唔知奥利诺科鳄有几劲？

被认为可以同黑鳄并列嘅"美洲鳄鱼之王"！而且寿命劲长，可以达到80岁——80岁呀，够一个人由出生到退休再揾食两次！

但係！你知唔知奥利诺科鳄几阴公？

20世纪60年代，分布喺委内瑞拉、哥伦比亚等地嘅奥利诺科鳄几乎被屠戮殆尽！点解？因为佢哋嘅皮太好卖——皮包、皮带、皮鞋，全部都想要！为咗靓，人类可以有几尽？睇下奥利诺科鳄就知！

你知唔知？奥利诺科鳄从"鳄鱼之王"变成"濒危物种"——

就系因为一张皮！人类想要靓，佢就要用命换——呢个就叫做"美麗嘅代价"。80年代开始保护，但系已经太迟——数量已经恢复唔到去！

讲到最惨，就係奥利诺科鳄嘅皮——

一张皮可以整几多个皮包？几十个？几百个？为咗人类嘅时尚，唔知有几多奥利诺科鳄失去咗宝贵嘅生命——呢个就叫做"时尚受害者"！

各位，跟住落嚟讲嘅，就係奥利诺科鳄最令人心痛嘅事实。

从6米巨兽到几乎灭绝，只因为人类想要佢张皮！80年寿命，但系唔够80年，人类就将佢杀到差点绝种——呢个就叫做"人类几犀利"！

所以下次见到皮制品，记得——呢个可能曾经系一只80岁嘅"鳄鱼之王"！
