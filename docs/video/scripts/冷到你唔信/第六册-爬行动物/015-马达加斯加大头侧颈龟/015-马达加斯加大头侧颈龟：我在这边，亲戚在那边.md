# 马达加斯加大头侧颈龟：我在这边，亲戚在那边

兄弟们好啊！今日我哋嚟介绍一种身世劲惨情嘅龟——马达加斯加大头侧颈龟。2018年，佢被评为「世界上最濒危的龟类」，唔单止珍稀，身世重非常离奇！

呢只龟头大碌碌，有着黑褐色外壳，但係最诡异的地方就喺呢度——佢属于「南美侧颈龟科」呢个家族！但係等等，马达加斯加系非洲的一个岛屿，而南美洲同非洲隔住个大洋嘞，呢只龟的「亲戚」点解会喺大洋彼岸的南美洲呢？

其实呢，科学家就推测，原本马达加斯加岛同南美洲有大陆桥相连。大约喺7800万年前的白垩纪晚期，连接南美洲与南极洲同马达加斯加的大陆桥沉没咗，马达加斯加大头侧颈龟就成为咗南美洲以外唯一被发现的南美侧颈龟科家族成员。呢个真系沧海桑田的变化啊！一諗起就觉得有几浪漫有几凄美。

不过呢，虽然身世咁得意，但系依家佢哋已经系最濒危的龟类之一，得番好少好少。两个「亲戚」分隔两地，永远冇办法再聚首，呢个真系有几凄美吖嘛！你觉唔觉得？
