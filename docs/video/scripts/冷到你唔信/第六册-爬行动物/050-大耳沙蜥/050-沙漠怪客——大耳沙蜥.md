# 沙漠怪客——大耳沙蜥

沙漠怪客——大耳沙蜥 沙漠怪客——大耳沙蜥 兄弟們，我哋今集嚟介紹新疆荒漠的一位怪客——大耳沙蜥！

當大耳沙蜥覺得危險埋身，佢就會張開一對「大耳朵」啪啪聲扇唔停，仲會發出「呼呼」声嚟警告——幾有款！

但係其實呢，呢對所謂的「大耳朵」，並唔係佢哋真有的耳朵，而係分布在嘴角兩側的皮褶——因為生得太似耳仔，所以就先叫佢做大耳沙蜥囉。

大耳沙蜥仲唔只呢項絕招！佢哋有一身能融入周邊環境的「保護色」，重可以将自己剷入沙中。爬行動物觀察家就試過——當感覺到人類步步逼近，佢哋就話咁快狠準扭動身體，瞬間潛入沙海，唔留一點痕跡！

順帶提一提，大耳沙蜥係沙蜥家族中身形最巨的，扇動兩個「大耳朵」嗰時睇落好得意，但係其實佢性格火爆，係個唔好惹的主兒！
