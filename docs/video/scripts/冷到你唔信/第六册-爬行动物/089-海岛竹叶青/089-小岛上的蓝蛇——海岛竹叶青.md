# 小島上的「藍蛇」——海島竹葉青

兄弟們，稀奇啰！世界上竟然有藍色的蛇？今日就介紹呢位——海島竹葉青！呢條蛇真係靚到暈！

佢哋主要係東南亞一些海島上面生活。不過其實呢，藍色的海島竹葉青只係少數，常見的顏色其實係綠色，只有好少部分係藍色或者黃色。就係因為咁，藍色版本就好矜貴。

藍色的海島竹葉青可以分布到海拔1200米咁高，佢哋平時多數待在樹上，等到繁殖季節或者一場大雨之後先會落樹。就係因為咁，佢哋先至可以避開地面上的敵人。

有人會問，會唔會係藍色的其實係臨時變色的綠色海島竹葉青，好像變色龍咁？澳洲博物館的專家就解釋話：「藍色海島竹葉青係唔會變色的，佢哋天生就係藍色，一生都係藍色。」所以話，藍就係藍，唔關變色龍事！

不過有趣的是，生活喺澳洲北部的「綠樹蟒」呢，就會喺怀孕期間臨時變成藍色——完全係兩個世界！就係因為佢懷孕的時候荷爾蒙有變化。
