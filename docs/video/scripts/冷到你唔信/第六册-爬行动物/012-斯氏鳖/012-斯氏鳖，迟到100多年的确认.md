# 斯氏鳖，迟到100多年的确认

兄弟們好啊！今日我哋嚟介绍一种劲惨情嘅鳖——斯氏鳖。呢只嘢几巴閉？佢哋可以长到超过1米，重70至100千克添！但係呢，佢哋的故仔就有点惨嘞。

斯氏鳖世世代代喺中国南部、越南北部的淡水河流入便生活、繁殖。其实呢，佢哋早喺1873年就已经被西方学者科学命名，但系长期以来都被误认为系鼋或中华鳖。直到一百几多年后，喺21世纪先至被确认为有效物种。呢个「迟到100几年前的确认」，真系有点离谱！
