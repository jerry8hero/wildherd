# 刺山龟：防御升级，带刺的铠甲

刺山龟：防御升级，带刺的铠甲 刺山龟：防御升级，带刺的铠甲 也许你会幻想：如果我像龟类一样拥有一副坚硬的“铠甲”，那我就能抵御很多危险了。很少有人知道，在东南亚一些国家和地区海拔900米以下的森林河流附近，生活着一种防御升级的龟类——它们不仅拥有坚硬的龟甲，在龟甲的边缘还有刺一样的尖棘——它们就是刺山龟。棕色的外壳加上锋利的刺状边缘，远远望去好似一轮红日，因此人们也叫它们“太阳龟”。 刺山龟的壳边缘长着这些“刺”有什么用？一些科学家认为，这些像刺的尖棘可能是用来对付蛇等天敌的，让它们无从下口。不过刺山龟的尖棘会随着年龄增长而变钝甚至消失。尽管拥有这么厉害的防御技能，刺山龟目前的生存状况仍不容乐观，主要原因是人类破坏了它们生存的天然森林。
