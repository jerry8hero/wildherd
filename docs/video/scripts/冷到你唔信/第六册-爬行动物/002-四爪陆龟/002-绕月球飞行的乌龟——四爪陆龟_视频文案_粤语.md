# 冷到你唔信 - 四爪陆龟

你可能話，陆龟我见得多啦，唔就四只爪嘅咩？你如果係咁諗就晒你啦。四爪陆龟可唔係你想象中嗰种普通陆龟，佢竟然可以成日赖喺度训觉——一年到头有9个月喺度呼呼大睡！係呀，你冇听错，9个月！相当于成只熊仔咁款，成日都喺度冬眠状态——但佢唔係熊喎，係龟喎，呢个几离谱啰。

你知唔知？四爪陆龟分辨公母有几搞笑！

讲起四爪陆龟，就唔可以唔提佢哋嘅"辨别大法"。你想知你屋企只龟係仔係女？等我教教你：

第一睇尾巴——如果条尾又短又肥，咁就係妹妹；如果条尾较长，咁就係仔。

第二睇爪子——如果对爪好短，咁就係妹妹；如果对爪几长，咁就係仔。

第三就係睇尾巴尖个"口"——呢个係重点！如果个口似星星，唔洗问咁多，係妹妹；如果个口係一条缝，咁就係仔。

所以呢，四爪陆龟嘅性别确几难分，连个口都整埋唔同款，真系服咗佢。

讲到四爪陆龟最得意嘅地方，就梗係——鍾意赖在度唔郁！

你知唔知？四爪陆龟係动物界嘅"睡公主"！
