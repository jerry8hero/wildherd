# 绕月球飞行的乌龟——四爪陆龟

兄弟们好啊！今日我哋嚟倾下一种犀利到不得了嘅乌龟——四爪陆龟。你哋中意瞓懒觉？但係啊，四爪陆龟嘅睡功先至系真正的「犀利」，佢哋一年到头有9个月系度「大觉睡」，简直系龟中「瞓觉王」！呢样嘢真系为食又懒閷閷，你话笑唔笑死人？

好喇，咁点样分辨四爪陆龟係「仔」定「女」呢？其实好简单：如果佢哋条尾短短肥肥，爪亦都好短，尾尖个「口」似星星噉样，咁就梗系女仔咯；相反，如果条尾有几长，爪都有几长，尾尖个「口」似一条缝噉，咁就係仔仔啦。呢个几得意吖嘛！不过呢，公乸之分你就知几多？

但係啊，你知唔知？呢种四爪陆龟竟然已经实现过「奔月」梦想！事情系咁噉：1968年，苏联整咗个「探测器5号」，入面装咗两只四爪陆龟，绕月球飞咗一圈，完之后安全翻地球。呢两只四爪陆龟就系人类航天史上第一批真正「上过月球」嘅地球生物！巴閉唔巴閉？真系唔讲唔知，一讲呃晒！

其实呢，四爪陆龟唔单止系航天先驱，重系龟界入面几得意噉存在添。佢哋原产于中亚地区，因为爪仔只有四只所以就叫四爪陆龟，呢样嘢几特别吖嘛！不过呢，依家四爪陆龟都已经属于濒危物种，大家记得支持下我哋嘅频道，三连评论下啦！
