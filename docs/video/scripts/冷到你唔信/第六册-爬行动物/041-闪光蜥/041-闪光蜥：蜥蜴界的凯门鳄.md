# 闪光蜥：蜥蜴界的凯门鳄

兄弟們，每個人都有自己特別鍾意食嘅嘢，蜥蜴界都有呢種為食鬼——生活喺巴西、哥倫比亞、厄瓜多爾、秘魯同圭亞那等國家嘅閃光蜥，特別鍾意食帶殼嘅嘅嘢，咁螃蟹、小龍蝦、蛤蜊，甚至連蝸牛都不放過。可問題來喇——呢啲嘢咁硬，蜥蜴又唔似得我哋人類咁有用靈活嘅手或者合適嘅工具去剝殼，佢哋點樣食呢？
