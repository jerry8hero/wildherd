# 喷血的尖刺铠甲战士——角蜥

喷血的"尖刺铠甲"战士——角蜥 喷血的"尖刺铠甲"战士——角蜥 哎吔，呢只肥仔，满身都系刺，究竟系唔系蜥蜴嚟咖？冇错！呢个就系主要住喺美洲沙漠嘅角蜥！佢着住一身生满硬刺嘅"铠甲"，边个够胆嚟惹佢，佢一定送你去验验边个头壳硬！但係其实呢，角蜥都系爬行动物，佢哋呢啲硬刺就系爬行动物界一种几特别嘅鳞片。有啲角蜥仲劲——识喷血咖！佢哋可以将自己头部嘅血压升高，令到眼角破裂，然后喷出鲜血，呢种情况下敌人梗系就唔够胆埋嚟嘞！角蜥可以将血喷到1米以外，掂唔掂？但係点解捕食者一见到血就惊走咗呢？系晕血吗？又有啲科学家觉得，角蜥平时食好多收获蚁，收获蚁体内有啲化学元素堆喺角蜥身体入面，令到佢哋嘅血带住一股难闻嘅味，喷出嚟可以将嚟犯嘅犬类、猫科动物"熏走"，痹唔痹？
