# 以岛为家，圣埃斯特万岛黑叩壁蜥

兄弟們，你聽過一座島嘅名字同一種蜥蜴有關係未？呢個故仔就犀利喇！喺加利福尼亞灣，有座聖埃斯特萬島，呢座島最早被命名為「大蜥蜴的島嶼」——呢個名就係源於島上特有嘅一種大型蜥蜴，聖埃斯特萬島黑叩壁蜥，佢哋係呢個島嘅真正主人嚟㗎！

呢個島上面有好多好多黑叩壁蜥，但係一離開咗個島就完全見唔到佢哋嘅蹤影——呢班友真係死都唔肯離開自己嘅老家吓！

好耐好耐之前，曾經有人將聖埃斯特萬島黑叩壁蜥帶出島外養，但係呢，呢班蜥蜴離開咗自己嘅「故鄉」之後，根本就冇辦法成功存活落嚟，佢哋真係戀親呢個地方吓！

聖埃斯特萬島當地嘅居民將黑叩壁蜥視為重要嘅食物，佢哋仲驕傲噉稱自己係「大型蜥蜴的人民」——呢樣真係幾巴閉吓！由此可見，聖埃斯特萬島黑叩壁蜥同呢座島以及島上嘅居民，真係有斩不断嘅羁絆，係真正嘅命運共同體！
