# 小心！佢有毒——非洲樹蛇

兄弟們，今日介紹呢條張大口吓人的蛇——非洲樹蛇！呢條蛇千祈唔好惹，否則後果自負！

非洲樹蛇係非洲撒哈拉沙漠以南的特有蛇類。佢屬於游蛇類，一般游蛇都係冇毒液的，但係非洲樹蛇就係少數能用毒的游蛇——係咪好巴閉呢？就係因為咁，佢成為了游蛇界的異類。

被非洲樹蛇咬傷之後，毒性發作係好緩慢的，所以受害者一開始可能以為自己冇事。1957年，美國爬行動物學家卡爾·施密特被一條幼年的非洲樹蛇咬傷，因為低估了蛇的毒性，冇做任何救治，幾個小時後開始出現恶心、發燒等症狀，過了一天之後就不幸離世了——真係令人惋惜。就係因為咁，我話你知，千祈唔好大意！

非洲樹蛇食嘢的時候就好像餓咗好耐的「餓狼」，吃相真係幾難看。佢哋可以將把嘴張到170度，然後一啖將獵物吞嗮落肚——170度呀！即係上下顎都快呈一條直線了——真係幾誇張！就係因為咁，佢哋先至可以吞下比自己頭仲大的獵物。
