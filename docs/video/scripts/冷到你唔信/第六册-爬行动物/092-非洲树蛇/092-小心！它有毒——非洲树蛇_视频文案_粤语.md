# 冷到你唔信 - 非洲树蛇

纤细但有剧毒——非洲树蛇，靓仔样呃你，实则劲毒！

你可能話，非洲树蛇？树蛇？应该都系无毒挂？错！非洲树蛇虽然属游蛇类，一般无毒，但系佢系少数能用毒嘅游蛇——就系呃你！

你知唔知非洲树蛇有几阴毒？

被咬伤之后毒性发作好慢，"受害者"初时以为自己没事——点知后来就开始恶心、发烧，过埋一日就讲拜拜！1957年，美国爬行动物学家卡尔·施密特就系被一条幼年非洲树蛇咬，结果低估咗毒性，几个小时后出现症状，过埋一日就死咗！

你知唔知非洲树蛇有几能吃？

可以将嘴张到170度——就系上下颚快呈一条直线！然后一口吞甩猎物——呢个就叫做"超级大嘴"！

你知唔知非洲树蛇生活习惯？

属游蛇类，生活喺非洲撒哈拉沙漠以南——鍾意爬树，鍾意埋伏，等猎物过路然后一击即中！呢个叫做"伏击型选手"！

各位，跟住落嚟讲嘅，就係非洲树蛇最令人惊讶嘅事实。

明明系游蛇类，本来应该无毒，但系就进化出毒液——呢个就叫做"转型成功"！而且毒性发作慢，但系依然可以致命——呢个就叫做"笑里藏刀"！

所以下次见到非洲树蛇，记得——呢个系可以张嘴170度、毒性发作慢但系依然致命嘅"伪装者"！
