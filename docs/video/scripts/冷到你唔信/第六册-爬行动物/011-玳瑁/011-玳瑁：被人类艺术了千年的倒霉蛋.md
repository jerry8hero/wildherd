# 玳瑁：被人类艺术了千年的倒霉蛋

兄弟們，呢日我哋嚟倾下一种好惨嘅海龟——玳瑁。玳瑁喺海龟入便真系几特别嘅存在，佢哋上颚弯曲成钩状，所以有人叫佢哋「鹰嘴海龟」。而且啊，玳瑁系目前人类已知唯一以海绵为食嘅爬行动物，呢样嘢几犀利！

但係呢，佢哋最巴閉嘅地方就系个样——看佢哋嗰五颜六色嘅甲壳，几咁养眼啊！古人早就发现呢点，将玳瑁嘅甲壳采嚟加工，雕上花纹，整成首饰，象征祥瑞幸福、健康长寿，以「海金」呢个名堂享誉世界。喺中国，玳瑁工艺有上千年的历史，唐代阵已经炉火纯青，真系犀利！正一就系千年传承嘅工艺啊！

不过呢，好野就呢点为止嘞。古人发现，将玳瑁抓上嚟，剥离甲壳之后再放回大海，玳瑁有时重可以再长出龟甲。但係啊，其实呢个讲法系大错特错添！龟甲密布神经同血管，剥离角质会对龟甲造成极大连损害，大多数情况下会导致玳瑁感染死亡。

而家人类对玳瑁的过度需求，已经令到佢哋被列为极危物种同国家一级保护动物。佢哋真系被人类「艺术」咗千年嘅倒霉蛋！呢个真系令人感慨，人类为咗靓嘢可以几残忍。你哋点睇？
