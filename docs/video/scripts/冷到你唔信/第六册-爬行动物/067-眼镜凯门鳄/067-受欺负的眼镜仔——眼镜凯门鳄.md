# 受欺负的眼镜仔——眼镜凯门鳄

受欺负的"眼镜仔"——眼镜凯门鳄 受欺负的"眼镜仔"——眼镜凯门鳄 咦？兄弟們，呢条鳄鱼点解戴住一副眼镜噶？哈哈哈，唔系真嘅眼镜啊！原来呢条嘢叫眼镜凯门鳄，就系因为佢只眼球前面有一条横骨睇上去就好像眼镜架一样，所以先至有呢个咁风趣嘅名。

好喇，兄弟們你哋知唔知，眼镜凯门鳄喺鳄鱼界入便出名系几犀利嘅小不点嚄！唔到你唔信，佢哋成年噶时候至有1～2米长，最大嘅都唔超过2.5米。你哋试下对比下恒河鳄、尼罗鳄、湾鳄咁多位大佬，就知道眼镜仔有几弱鸡啰。

但係啊，正因为佢哋身型咁娇细，喺大自然入便就梗系受尽欺负啰！你以为得人类先至会恰佢哋啊？蜥蜴、老鹰、长鼻浣熊、猴子呢啲混蛋，一有机会就走去偷佢哋嘅蛋嚄！未大个嘅眼镜凯门鳄呢，就梗系成为鲶鱼、龟龟哋嘢噶食物啰，甚至连屋企猪都会走去食小小眼镜凯门鳄！真系惨无人道啊！

就算系啲大个咗嘅眼镜凯门鳄，佢哋都有排天敌㗎：美洲豹、森蚺、黑凯门鳄、奥利诺科鳄等等，一大堆都对其虎视眈眈㗎！你话几阴公犀利咯！

所以呢，兄弟們，如果下次见到呢只戴眼镜嘅小可伶，记得比啲爱心佢哋啊！记得一键三连支持我哋，下期片片见！
