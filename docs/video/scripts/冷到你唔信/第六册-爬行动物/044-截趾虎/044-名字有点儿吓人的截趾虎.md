# 名字有点儿吓人的截趾虎

名字有点儿吓人的截趾虎 名字有点儿吓人的截趾虎 兄弟們好嘠，今次我哋嚟吹下水，講一種名你聽落幾得人驚嘅壁虎——截趾虎！

一聽個名你可能會諗，吓，係咪被人斬晒腳趾嘅壁虎呀？其實唔係咁嘅！截趾虎之所以叫呢個朵，係因為佢哋前後腳內側的腳趾上幾乎冇爪，所以先至俾人改埋咁樣一個幾嚇人的名。

但係啊，雖然少咗幾隻爪，佢哋依然威風到不得了！森林、城市，總之冇邊度係去唔到嘅。唔止咁，佢哋的勢力範圍仲要勁到暈——東南亞、夏威夷、墨西哥，橫跨成個太平洋！

喺夏威夷沙灘度，佢哋就好常見，不過當地人就當佢哋係入侵貨色。但係呢，當地人又冇話點抗拒——可能因為佢哋幫手捉下蟲掛！

其实呢，截趾虎幾可愛嘅，佢哋淨係晚黑先至出沒，大多數時間都黐喺高高的牆身同天花板度。咁樣睇，佢哋又幾似係屋企的一份子添！
