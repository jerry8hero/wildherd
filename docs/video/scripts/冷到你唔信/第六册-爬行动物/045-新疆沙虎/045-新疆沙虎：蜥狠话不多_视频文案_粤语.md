# 冷到你唔信 - 新疆沙虎

沙漠入面最安静嘅杀手——新疆沙虎，一于到你听到沙声都已经走迟！



呢期我哋唔讲龟，讲条"蜥"——新疆沙虎！听个名你以为係老虎咩？唔係，但係一样狼死！

你知唔知，新疆沙虎虽然只係一条蜥蜴，但维吾尔族人就唔叫佢"沙虎"，而係叫佢"库莱伦"——翻译埋係"有手的蛇"！一条蜥蜴，点解叫做"有手的蛇"呢？等我就话你知，因为佢个样真係似蛇多啲，但係四肢又几短仔，爬起来似只手喺度舞——就係咁解叫"有手嘅蛇"。呢个命名，真系够晒贴同埋够晒骑呢。

讲到新疆沙虎，第一样嘢就係——估你唔到咁大只！佢身长70到80厘米，係戈壁入面已经係巨物级别。但係呢个外表就呃得下人——望落去似条肥肥哋嘅蜥蜴，细细粒好可爱咁款。你信佢就死，呢只嘢脾气劲过佢嘅体型！

你知唔知新疆沙虎对声音有几敏感？係沙漠入面，佢哋对每一粒声都当係生存警报！一有动静——火速走人！但係如果你继续嘈落去，慢慢接近佢呢？呢个时候佢就唔走啰，係原地等你埋嚟，然后——狠狠咬你一口！呢招叫咩？就好似武侠片入面，你追我，我跑；但你追多两步，我就回身反杀！呢个就係新疆沙虎嘅生存之道：你走我走，你嚟我就焗你！

讲到新疆沙虎最犀利嘅嘢，就梗係佢哋嘅地下宫殿。

佢哋啲洞穴劲复杂，有啲甚至有5条岔道！你以为佢哋係随便揾个窿就钻进去？唔係咁嘅，係正经嘅五房一厅！科学家推测，咁复杂嘅洞穴结构除咗防御捕食者，仲可以调节温度——热天时深处凉浸浸，冬天时又可以揾到暖啲嘅角落。呢个就係建筑工程学应用到极致，你以为得人类识起楼咩？爬行动物一样得！

你知唔知新疆沙虎分布有几广？从吐鲁番盆地——嗰度热到可以煮蛋——到海拔1700几米既高山，佢哋都可以活得好好哋。吐鲁番夏天成40度，高山冬天得几度，呢种温差，换着係我都顶唔顺，但係新疆沙虎就当作係免费嘅冷气暖气——热天钻窿底，冻天揾向阳位，呢个就係真正嘅随遇而安。

讲到新疆沙虎点解可以生存喺呢种咁撈绞既环境，就唔可以唔提佢既示警方式。

呢只嘢遇到威胁，唔会第一时间就走佬，而係发出低微的声音警告你——呢种声音似微微既嘶嘶声，就好似aje在你耳边话"你别过来啊"。如果你收声慢啲，佢就真系当你答应啰，冲埋嚟咬你一口！呢种"先礼后兵"既策略，真系教精你——同人一样，唔好虾我，要你知我唔系你好惹。

各位，跟住落嚟讲嘅，就係新疆沙虎最得意既地方。

呢只嘢外表似蛇似蜥蜴，细细只但系脾气劲；住喺沙漠但系最犀利既係地下工程——五岔隧道就係基本操作；听到声音就走，但系慢慢接近就反杀；分布从火焰山到高山都得。呢种"样样都唔掂但系样样都掂"既生存能力，就係新疆沙虎称霸戈壁既真正原因。
