# 新疆沙虎：蜥狠话不多

新疆沙虎：“蜥”狠话不多 新疆沙虎：“蜥”狠话不多 兄弟們，我哋今集嚟介紹一位好猛料的沙漠蜥蜴——新疆沙虎！

新疆沙虎主要紮根喺中國甘肅、內蒙古、新疆呢啲乾旱戈壁地帶。當地維吾爾族人就叫佢哋「庫萊倫」，即係「有手嘅蛇」——你話幾搞嘢！

順帶提一提，呢班友仔雖然得70～80厘米咁大隻，但係火爆脾氣就一觸即發！佢哋對聲音異常靈敏，一出洞對任何响動都即時話知——會极速走佬。

但係啊，如果你繼續追埋過去呢，佢哋就會迎面衝上來，「不分青紅皂白」咁款開口就啃，仲會發出低微聲音嚟警告你——相當有性格！

另外，科學家又發現，新疆沙虎的洞穴結構複雜到不得了，有的洞甚至有5條岔道！兄弟們，你哋話係唔係好堅離撐呢？
