# 世界上最大的蜥蜴——科莫多巨蜥

世界上最大的蜥蜴——科莫多巨蜥 世界上最大的蜥蜴——科莫多巨蜥 前面说了世界上最小的蜥蜴，仅有人类指甲盖儿那么大，那么世界上现存最大的蜥蜴是哪种呢？它就是生活在印度尼西亚几个小岛上的科莫多巨蜥。 成年的科莫多巨蜥，体长一般在2米以上，而最大的科莫多巨蜥能超过3米！在吃饱的状态下，科莫多巨蜥的体重可以超过160千克。不过，因为科莫多巨蜥比较能吃，有时候一次吃掉的食物重量相当于自身体重的60%，因此它们的体重是有“水分”的，但科学家推测，最重的科莫多巨蜥也能达到100千克。以前，人们推测科莫多巨蜥是依靠口腔里致命的细菌来杀死猎物的，但2005年和2009年的研究表明，科莫多巨蜥的口中有毒腺，也能分泌毒液，对猎物造成一定伤害。
