## 顺口溜设计说明

```
芝麻唔讲大话，勥到鱼仔都惊嗮！
```

- **押韵**："话"和"嗮"押韵
- **反转**：小芝麻→大勥劲，形成反差萌
- **传播性**：短句好记，适合弹幕互动
- **扣题**："勥"字贯穿全文，强化主角人设

---

## 三件套表达设计

```
如果你中意呢类勥嘢，记得点赞投币三连！
```

- **自然过渡**：用"如果你中意"承接上文情绪
- **人设延续**："呢类勥嘢"延续主角标签，强化账号调性
- **明确行动**：直接说明"点赞投币三连"，不模糊
