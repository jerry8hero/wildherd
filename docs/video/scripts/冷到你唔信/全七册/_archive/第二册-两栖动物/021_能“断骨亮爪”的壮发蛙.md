# 能“断骨亮爪”的壮发蛙

动漫角色“金刚狼”可以在需要时变出合金利爪，这并非完全天马行空的想象，现实里的壮发蛙就有类似的能力。

壮发蛙是无尾目节蛙科发蛙属的物种，分布于中非地区，雄蛙在繁殖期躯干和大腿上会长出大量的真皮乳突，看上去就像毛发一样，因此也叫“多毛蛙”。

身体“长毛”并不是壮发蛙最神奇的地方，“断骨亮爪”才是它们的绝活儿。就像除猎豹外的猫科动物平时会把爪子收缩在肉垫儿里一样，壮发蛙的爪子也隐藏在两条拥有大量结缔组织、可以最大限度膨胀和收缩的后腿中。平日里，壮发蛙的爪子就如同放在刀鞘里的利刃一样待在腿骨中，一旦遇到危险，它们就会用力收缩肌肉，把自己的骨头折断，让爪子从皮肤的破口处刺出，给对方突然一击。

和猫科动物的爪子不同，壮发蛙的爪子是纯粹的骨头，上面没有皮肤覆盖，完成攻击后也无法保持在体外，而是会自动缩回腿骨内，破损的身体组织也会恢复如初。
