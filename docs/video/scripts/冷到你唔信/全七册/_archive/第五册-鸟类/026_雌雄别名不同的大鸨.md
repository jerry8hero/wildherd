# 雌雄别名不同的大鸨

大鸨是生活在欧、亚、非三洲的大型鸟类，全世界只有一种，是匈牙利的国鸟。在中国、俄罗斯和东北亚其他地区，以及土耳其、欧洲东部、伊比利亚半岛和摩洛哥北部，大鸨活跃于平原、草原、稀树草原和半荒漠等开阔区域。

说大鸨是大型鸟，主要是由于其雄性的体形较大。成年雄性大鸨体长1米左右，体重不少于10千克。相比之下，雌性大鸨就袖珍了许多，体长大多不超过50厘米，体重通常不超过6千克。当一对大鸨夫妻站在一起的时候，相差几乎一倍的体形让它们看上去就像成鸟和幼鸟，堪称“最萌身高差”。

除了体形上的巨大差距，到了繁殖季节，雄性大鸨的下巴上会长出酷似山羊鬃毛的白色须羽，它们也因此被俗称为“羊须鸨”。雌性因为没有类似的须羽，被称为“石鸨”。大鸨也是罕见的两性别名不同的物种。

大鸨是杂食性鸟类。食物中素的包括嫩芽、叶子、草籽、种子、谷物；荤的则是以象鼻虫、蝗虫、油菜金花虫为主的各种昆虫，有时也捕捉蛙类。
