# 声音模仿家——华丽琴鸟

很多野生鸟类有模仿声音的本领，比如华丽琴鸟。

华丽琴鸟生活在澳大利亚东南部，体长约1米，是一种大型的鸣禽。但在分类上却和麻雀同属雀形目，具体为雀形目的琴鸟科。它们名字里的“华丽”二字，来源于雄鸟尾巴上16根长长的羽毛。这16根羽毛由大量丝状羽毛相连，展开时如同一把羽扇，非常漂亮。

既然是鸣禽，华丽琴鸟自然非常善于鸣叫。科学研究发现，华丽琴鸟可以模仿多种不同的声音，光是其他鸟类的叫声就有20多种。如果算上其他生物或非生物发出的声音，那可以达到将近100种，如婴儿的啼哭声、警车的警笛声、电钻的伐木声被它们模仿得惟妙惟肖。

华丽琴鸟通常独居生活，当预感到周围可能有天敌时，它们会一边伺机逃跑，一边模仿一大群鸟生活在一起才能发出的声响，以此来“警告”对方要面临一群对手。在求偶期，一些雄鸟还会耍心机，发出很可怕的声音吓唬雌鸟，让它们不敢离开自己。
