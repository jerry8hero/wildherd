# 吐口水的贼鸥

贼鸥是鸻形目贼鸥科贼鸥属的鸟，平均体长60厘米，比海鸥强壮，全世界共有7种，我国有4种，主要生活在广东、江苏等南方沿海地区，山西、黑龙江等北方内陆地区也有分布。贼鸥夏季在南北两极的苔原上繁殖，冬天则到温暖地区越冬。

贼鸥是肉食性鸟类，鱼虾、企鹅等鸟类的幼鸟、鸟蛋，以及兽类的尸体都在它们的食谱中，贼鸥有时还会去偷人类的食物，这也是它们得名的原因。

除了偷，贼鸥还喜欢明抢，经常凭借体形上的优势从其他鸟类口中夺食。而且它们从不筑巢，看到哪个体形比自己小的鸟的巢穴好，就直接占为己有。

贼鸥喜欢偷盗，却讨厌被偷盗。当其他动物或人类靠近它们的巢穴时，贼鸥首先会大声鸣叫示威，如果实在不行就使出撒手锏，要么吐口水，要么飞到对方头上排泄，直到把入侵者赶走才罢休。
