# 专吃蚂蚁的啄木鸟——蚁

按照通常理解，啄木鸟就是在树干上不停凿洞，寻找虫子的“森林卫士”（这一说法尚有争议，因为有研究发现啄木鸟凿洞对树的损害比虫害还大）。但如果把啄木鸟的范围扩大到科，情况就不完全是这样了，一种名叫蚁的鸟会在地面上找食物。

蚁体长15～20厘米，在啄木鸟家族中属于小个子，拥有非常灵活的头部，尾巴的长度在7厘米以上，在啄木鸟中属于比较长的。蚁在亚洲、非洲、欧洲都有分布，在我国南北方的众多省份也都有栖息。

名字中有个“蚁”字，蚁的食谱中自然少不了蚂蚁。它们超长且带有黏性的舌头可以深入蚂蚁洞深处，只需轻轻一舔，就可以粘住大量的蚂蚁。在大量进食蚂蚁的同时，蚁也不会放过蚂蚁的卵和蛹，对于它们来说，这些食物的营养价值更高，更容易消化吸收。

蚁大多数时间在灌丛等有遮蔽物的地面上活动，但它们的攀爬能力很强，可以像那些在树上取食的亲戚一样垂直站在树干上。
