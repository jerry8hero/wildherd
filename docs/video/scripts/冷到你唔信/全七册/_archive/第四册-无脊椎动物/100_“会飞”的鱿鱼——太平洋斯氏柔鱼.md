# “会飞”的鱿鱼——太平洋斯氏柔鱼

鸟类、蝙蝠、昆虫是现存的三大类会飞的动物。那些不会飞的动物中，有些也能在空中做短途移动，也就是“滑翔”。太平洋斯氏柔鱼就是一种会滑翔的鱿鱼。

太平洋斯氏柔鱼也叫日本飞鱿鱼，生活在太平洋北部海域，和所有的家族亲戚一样，都有10条腕足和箭头形尾部，身材细长。

由于体形较小，太平洋斯氏柔鱼成为其他海洋动物捕食的对象。当面临危险时，它们除了像其他鱿鱼那样“喷射墨汁”外，还能跃出水面逃跑。利用从头部腹面（下方的一面）的漏斗状开口喷射出的海水（事先吸入体腔中的）对身体的反作用力，太平洋斯氏柔鱼可以向后跃出水面（头足类动物倒着前行）；与此同时，它们会像鸟展开翅膀一样，展开尾部两侧的肉鳍，用来获得空气中的浮力并维持身体平衡。太平洋斯氏柔鱼的跳跃高度为10～20米，滑翔时间可持续10秒以上。
