# 农林卫士——红绒螨

生物学上的螨虫泛指节肢动物门蛛形纲（不是昆虫）蜱螨亚纲的动物，是蜘蛛和蝎子的亲戚；全世界共有大约5万种螨虫，喜欢在人体内寄生的只是其中的一部分；有些螨虫非但不祸害人类，还是守护农林的益虫，比如红绒螨。

红绒螨因血红色的体表而得名，世界上有300多种，大多数种类体长在0.5厘米左右，较大的有1厘米以上，是螨虫家族里的大个子。

红绒螨小时候像昆虫一样长6条腿，在出生后的前一两周内，这些小家伙只能吃流食，它们会附着在昆虫的触角、足节等骨骼的缝隙处，用口器的螯肢刺破寄主的皮膜，吸食体液。等身体稍微长大一些，小红绒螨会进入地下生活，直到成年后才会破土而出。此时它们最大的变化就是多了两条腿，并且可以吃固体食物了。成年红绒螨是肉食性动物，最爱吃蚜虫等危害林木和农作物的害虫。
