# “弑夫”的棒络新妇

和螳螂一样，很多种类的雌蜘蛛也有吃掉配偶的习惯，这其中，棒络新妇“谋杀亲夫”的概率当属最高。

棒络新妇是蛛形纲蜘蛛目肖蛛科络新妇属的物种；在东亚、南亚、东南亚都有分布，是我国最常见的蜘蛛之一。其属名来源于日本鬼神故事中以男性为食的女蜘蛛精络新妇。

现实中的雌性棒络新妇当然无法变成美女去诱杀人，其毒性对人的影响微乎其微，但其对雄性伴侣的威胁却是所有蜘蛛里最高的（有研究显示其弑夫概率为89%）。棒络新妇的弑夫率如此高，其体形是个重要因素，成年雌蛛体长为2～2.8厘米，相比之下，雄蛛只有伴侣的四分之一大小。

虽然有较高的被杀风险，但雄性棒络新妇也并非完全没有在婚配后活命的可能。有些个体会选择在雌蛛吃饭时完成交配，然后赶快撤离。
