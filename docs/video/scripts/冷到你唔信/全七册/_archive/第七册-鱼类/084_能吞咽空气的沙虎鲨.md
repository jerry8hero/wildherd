# 能吞咽空气的沙虎鲨

少数种类的鱼可以呼吸水面上的空气，沙虎鲨就是它们中的一员。

沙虎鲨的中文正名叫“后鳍锥齿鲨”，是软骨鱼纲鼠鲨目锥齿鲨科的成员，成年后体长为3～5米，雌性体长大于雄性。沙虎鲨广泛分布于除东太平洋外的温带海域的浅海海底，我国的东海、南海、台湾海峡都有它们的踪迹。它们锥子一样锋利的牙齿，是捕杀鱼类和头足类动物的工具。

沙虎鲨的活动范围很大，除休息时所待的水底外，也出现在中层和上层海水中。除了垂直方向的移动，沙虎鲨还会在不同季节进行迁徙，它们出色的活动能力离不开良好的呼吸系统。沙虎鲨除从水中获取氧气外，还能把头抬出水面直接呼吸空气并把吸入的空气储存在胃里，这让它们可以在水中悬浮。
