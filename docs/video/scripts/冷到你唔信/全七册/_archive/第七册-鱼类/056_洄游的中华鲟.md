# 洄游的中华鲟

很多种类的鱼都有洄游的习惯。从目的上说，鱼类的洄游有3种形式，其中1种叫“生殖洄游”（另外两种是寻饵洄游和越冬洄游），也就是为了繁衍后代而进行的洄游，中华鲟的洄游形式就属于此类。

中华鲟是硬骨鱼纲鲟形目鲟科鲟属的鱼，拥有纺锤形身材，体长1.7～3米，雌性体长大于雄性。在我国，中华鲟主要生活在长江和近海流域，属于底层、洄游或半洄游性鱼，以各种小型的底栖鱼为食。

中华鲟的寿命很长，可达40年。它们的青少年时期几乎全部在大海中度过，但出生地却是江河。现存的中华鲟几乎都是在我国的长江出生。每年秋天，性成熟的中华鲟都会成群结队地从海洋洄游到长江上游的金沙江产卵。新出生的幼鱼会在出生后的几个月到一年内，随着水流和自身活动，渐渐游向长江口。大约在第二年，当身体发育到一定程度时进入海洋生活；待有繁殖能力后（大约8～14年），开始每年像父母一样，在海、河间进行生殖洄游。
