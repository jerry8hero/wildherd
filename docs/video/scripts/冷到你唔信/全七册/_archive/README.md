# _archive/

本目录是"冷到你唔信"系列的**早期扁平化单文件版**归档，仅作历史参考。

## 与现行版本的差异

| 路径 | 形态 | 状态 |
|------|------|------|
| `全七册/_archive/第一册-哺乳动物/002_xxx.md` | 扁平单文件（早期单文案） | 历史版本 |
| `../第一册-哺乳动物/002-大熊猫/002-大熊猫_视频文案.md` | 子目录双文件（视频文案 + B 站发布内容） | **当前版本** |

当前以分册子目录版为 canonical（含 `_视频文案.md` + `_B站发布内容.md` 配对）。`_archive/` 中是早期没有分目录时的中间产物。

`粤语文案_backup/` 是某次批量重命名前的旧版本备份。
