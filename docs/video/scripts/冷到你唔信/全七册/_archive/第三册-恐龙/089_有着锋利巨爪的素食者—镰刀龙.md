# 有着锋利巨爪的素食者—镰刀龙

镰刀龙生活在大约7000万年前晚白垩世的蒙古国，体长在10米左右，体重3～6吨，名字来源于前肢末端的巨大爪子。镰刀龙的前肢上 长有3个指头，每个指头的末端都有一个大而弯曲、形状像镰刀的爪子，其中最大的爪子长度超过1米，可用来对付当时蒙古地区最强大的捕食者——特暴龙。

和大熊猫的祖先吃肉但现代大熊猫几乎只吃竹子一样，镰刀龙的祖先也是吃肉的，后来为适应环境变化才改吃素。它们的牙齿并不适合磨碎植物，只能简单地咀嚼一番就吞下去。为消化吸收这些粗糙的植物纤维，镰刀龙的肠胃很大，拥有复杂而庞大的消化系统，这使得它们的肚子看上去也很大，走路时总是挺着个大肚腩，慢悠悠的。
