# 汪洋中的绿色精灵——绿海龟

绿海龟在热带、亚热带的大海里畅游，它们能在水下100～200米的深度中潜游；但绿海龟是爬行动物，用肺呼吸，所以它们还是会到海面上来换气。绿海龟几乎顿顿饭都离不开海中的海草和大型海藻，这些植物中的绿色素在其体内脂肪中逐渐积累，导致它们的身体呈现淡淡的绿色，因此得名“绿海龟”。

绿海龟一生大部分时间都在海里生活，但每到繁殖季节，就会回到海岸的沙滩上产卵。刚破壳而出的小海龟，一般需3～7天的时间才能爬出窝，海岸上有很多可怕的捕食者，因此小海龟要奋力向大海爬去。在到达海边时，它们会循着海浪的声音，冲进浪花里，拼命地向外海游去。小海龟可以不停地游24小时。对它们而言，游得越远越安全。

# 绕月球飞行的乌龟——四爪陆龟

你喜欢睡懒觉？但你肯定没有四爪陆龟能睡，它们一年12个月中有长达9个月的时间在呼呼大睡！那么，如何分辨一只四爪陆龟是“男孩子”还是“女孩子”呢？如果它的尾巴又短又肥，爪子也很短，尾巴尖的“口”像一颗星星，那么它就是“女孩子”；如果它的尾巴有点儿长，爪子也有点儿长，尾巴尖的“口”是一条缝，那么它就是“男孩子”。

你曾经梦想飞向月球吗？四爪陆龟已经先于你实现了这个梦想。1968年，苏联发射了“探测器5号”，载着两只四爪陆龟绕月飞行，并在完成任务后安全返回地球。这两只四爪陆龟可是人类航天史上第一批飞往月球的地球生物！

# 享誉世界的“老寿星”——塞舌尔巨龟

塞舌尔巨龟是地球上最长寿的动物之一，有些塞舌尔巨龟可能活到超过200岁。据说，在18世纪的时候，一位英国海员从塞舌尔群岛带回了4只塞舌尔巨龟，并将它们送给了英国东印度公司的人。这4只塞舌尔巨龟中，有一只名叫阿德威塔的巨龟，于1875年被送至加尔各答动物园，一直到2006年，阿德威塔才去世，它的寿命达到255岁（阿德威塔生于1750年，来到加尔各答动物园的时候就已经120多岁了）！阿德威塔去世以后，生活在南大西洋圣赫勒拿岛的塞舌尔巨龟乔纳森，被认为是地球上现存最高龄的陆地动物，它出生于1832年，目前已经192岁了！

# 长着“红耳朵”的“入侵狂魔”——红耳龟

红耳龟，顾名思义，它们耳朵和耳朵附近的位置有红色的条纹。别看它们长得挺可爱，可它们却是“入侵狂魔”。这种原产于美国、墨西哥的半水栖龟类在宠物市场上特别受欢迎，被当作商品出售到世界各地。由于红耳龟有着超常的适应性，很多引入红耳龟的国家，其本土龟在生存竞争中都败下阵来。红耳龟繁殖能力强，能携带病毒并传播多种疾病，个头儿还比较大……因此，它被列入《全球100种最具威胁的外来物种》，成为不折不扣的“入侵狂魔”。

由于红耳龟超强的入侵性，许多深受其害的国家绞尽脑汁寻求应对之策。在澳大利亚及欧洲部分国家，贩卖红耳龟是犯法的；日本也在2015年宣布禁止进口这种“入侵狂魔”，以遏制其进一步扩散。

# 南美彩龟：我不是巴西龟

在宠物市场上，红耳龟有个更通俗的名字——巴西龟。咦？刚才不是说红耳龟原本生活在美国和墨西哥吗？美国和墨西哥都属于北美洲，巴西属于南美洲，为什么原产于北美洲的红耳龟会和南美洲的“巴西”扯上关系呢？原来，最先被引入我国台湾的宠物龟是被称作“巴西龟”的南美彩龟，南美彩龟原产于南美洲，长得和北美洲的红耳龟很像，只是南美彩龟的耳朵附近没有红色的条纹。后来，人们觉得从南美洲把南美彩龟运过来的费用太高了，于是改为进口北美洲的红耳龟，并且把“巴西龟”这个名字用在红耳龟上，就这样“张冠李戴”了。

# 龟中“海王”——棱皮龟

这个海龟宝宝叫棱皮龟。名字好奇特？这里隐藏着奥秘：一般的龟是覆盖着龟甲（龟壳）的，龟甲其实是由龟特殊的“皮肤”形成的“骨骼”，十分坚硬。而棱皮龟的皮肤软化成革质，在它们的背部的革质皮肤上，有7条纵向排列的“棱”，因此得名“棱皮龟”。

棱皮龟这身“皮”可不软，其背部革质皮肤下面有骨片，在危机四伏的海洋里，这是它们有力的“护甲”。别看它们一副老实的样子，海洋里可没有多少生物能欺负它们，而且它们长大之后可是龟中巨无霸——可以长到2米，庞大的体形能帮它们吓退不少动物。而且，它们非常善于潜泳，能下潜到1000米深的水里，是泳技高超的潜水大师！

# 水里有“装甲恐龙”——真鳄龟

在爬行动物宠物界，有一对明星：大鳄和小鳄，它们并非鳄鱼，这里的“大鳄”就是来自北美洲大名鼎鼎的真鳄龟。它们那大大的脑袋、长有3条大鳞脊的壳，颇有已灭绝的装甲恐龙之遗风。

真鳄龟是北美洲淡水“龟王”，成年龟的壳长度可达80厘米，体重能达到80千克（据说1937年，在美国堪萨斯州发现了一只180多千克的巨型真鳄龟）。它们那如雄鹰的大嘴拥有强劲的咬合力，能将人的手指干脆利落地咬断！但它轻易不咬人，除非感觉到了威胁。由于过度捕捞，真鳄龟已经被世界自然保护联盟列为易危物种。

# 拟鳄龟：个子不大，脾气不小

那么爬宠界说的“小鳄”是谁呢？它就是同样来自北美洲淡水水域的拟鳄龟。成年拟鳄龟虽说个头儿、咬合力都比真鳄龟小，但“脾气”可比真鳄龟大。拟鳄龟会主动追逐猎物，对待入侵者也毫不客气，“一言不合”就上嘴。拟鳄龟的脖子又长又灵活，可以向后伸到身体后部，攻击神速，理论上也能轻松咬断人的手指。自从20世纪60年代被日本作为外来宠物引进，拟鳄龟被记录为该国严重的咬伤来源之一！虽然性情凶猛，但刚出生的拟鳄龟宝宝还有一点儿萌：它们在离开巢穴之前可以发出“咔嗒”“咯吱”的声音；有的还可以发出像手指在梳子上摩擦一般的奇异声音。

# 用吸管“喘气”的枯叶龟

脑袋看起来像一片枯黄的树叶，这正是枯叶龟名字的由来。所以，当你在南美洲流动缓慢的河流或者静谧的湖泊、沼泽中看到一堆飘落在水上的枯叶时，说不定这家伙就藏匿其中。枯叶龟超级爱吃鱼，科学家曾对20只野生的枯叶龟进行解剖，发现它们的胃里竟然全部是鱼！枯叶龟长了一双“火眼金睛”，就算在晚上昏暗的水里，照样可以捕捉到猎物。

它们是真不爱离开这片水域，看到它们那像吸管一样的鼻子了吗？枯叶龟把鼻孔一端的鼻尖伸出水面，都不用露头，就能在水下呼吸。

# 赤龟：红色精灵，望寒兴叹

赤蠵龟的“赤”是指红色，“蠵”在我国古代有“大龟”的意思。赤蠵龟担得起这个名字吗？赤蠵龟的壳一般是红棕色的，符合“赤”意。它们还是世界上最大的硬壳海龟——有报告显示，有的赤蠵龟能有2米多长，500多千克重！说它们是“蠵”（大龟）一点儿也不夸张。

就是这么巨大的“红色精灵”，“身世”还有点儿可怜呢！在大约300万年前的上新世，我们今天的中美洲浮出海面，把大西洋与印度洋—太平洋之间的海流阻隔了。地球进入冰川循环，海流因为气候变化产生新的流向，冷水上升到好望角附近，把好望角的海水温度大大降低。这对需要迁徙的赤蠵龟是个坏消息，冷水让它们“望寒兴叹”，最终，大西洋的赤蠵龟和太平洋的赤蠵龟被完全隔开。

# 玳瑁：被人类“艺术”了千年的“倒霉蛋”

玳瑁在海龟里真的很另类。它们的上颚弯曲成钩状。所以人们也常叫它们“鹰嘴海龟”。它们也是目前人类已知的唯一以海绵为食的爬行动物。再看它们那色彩斑斓的甲壳，不仅你看着养眼，古人也是——玳瑁的甲壳被古人采来进行加工，雕上花纹，做成首饰，象征祥瑞幸福、健康长寿，以“海金”的名声享誉世界。在中国，玳瑁工艺有上千年的历史，唐代时已经炉火纯青。

古人还发现，把玳瑁抓上来，剥离甲壳后再放回大海，玳瑁有时还能再长出龟甲。实际上，龟甲密布神经和血管，剥离角质会对龟甲造成极大的伤害，大多数情况下会导致玳瑁感染死亡。人类对玳瑁的过度需求，已经让它们被列为极危物种和国家一级保护动物了。

# 斯氏鳖，迟到100多年的确认

这类长度可超过1米、重量可轻松达到70～100千克的“庞然大鳖”——斯氏鳖，世世代代在中国南部、越南北部的淡水河流里栖息、繁衍。

早在1873年，西方学者便已为它们科学命名，但长期以来，它们一直被误认为是鼋或中华鳖，直到一百多年后，在21世纪，它们才被确认为有效物种。所以，斯氏鳖错失了20世纪80年代被列入中国《国家重点保护野生动物名录》的机会。2021年，斯氏鳖终于位列中国《国家重点保护野生动物名录》并被定为一级保护动物。可是，目前活着的斯氏鳖可能只有两只了，而且都为雄鳖！中国苏州动物园有一只雄鳖，这是目前已确认的中国最后一只斯氏鳖；另一只雄鳖可能生活在越南河内。

# 刺山龟：防御升级，带刺的铠甲

也许你会幻想：如果我像龟类一样拥有一副坚硬的“铠甲”，那我就能抵御很多危险了。很少有人知道，在东南亚一些国家和地区海拔900米以下的森林河流附近，生活着一种防御升级的龟类——它们不仅拥有坚硬的龟甲，在龟甲的边缘还有刺一样的尖棘——它们就是刺山龟。棕色的外壳加上锋利的刺状边缘，远远望去好似一轮红日，因此人们也叫它们“太阳龟”。

刺山龟的壳边缘长着这些“刺”有什么用？一些科学家认为，这些像刺的尖棘可能是用来对付蛇等天敌的，让它们无从下口。不过刺山龟的尖棘会随着年龄增长而变钝甚至消失。尽管拥有这么厉害的防御技能，刺山龟目前的生存状况仍不容乐观，主要原因是人类破坏了它们生存的天然森林。

# 风靡世界的超大“橄榄球”——太平洋丽龟

哇！这里有一颗超大的橄榄球！看我的，本垒打！你才是橄榄球呢！人家是长着灰黑色心形背甲的海龟——太平洋丽龟，只是看着像橄榄球而已。

就像橄榄球风靡世界一样，太平洋丽龟也在全球广阔的海域畅游，从太平洋、印度洋的温暖水域，到大西洋、非洲西海岸和南美洲部分沿海地区，都有它们的身影。太平洋丽龟曾经是世界上数量最多的海龟。数据显示，仅1968年一年，墨西哥海岸被商业捕捞的太平洋丽龟就有100多万只。不过，由于人们过高估算太平洋丽龟的数量，所以它们被过度捕捞。目前在全球范围内，太平洋丽龟的数量已经比最高峰时下降了30%以上！

# 马达加斯加大头侧颈龟：我在这边，“亲戚”在那边

这个长着大脑袋，有着黑褐色外壳的家伙，是马达加斯加大头侧颈龟，2018年，它被评为“世界上最濒危的龟类”。它不仅珍稀，身世更是离奇，因为它属于南美侧颈龟科这个家族。咦？马达加斯加是非洲的一个岛屿，而南美洲和非洲隔着大洋呢，马达加斯加大头侧颈龟的“亲戚”怎么会在大洋彼岸的南美大陆呢？一些科学家推测，原本马达加斯加岛和南美洲有大陆桥相连，大约在7800万年前的白垩纪晚期，连接南美洲与南极洲和马达加斯加的大陆桥沉没了，马达加斯加大头侧颈龟就成了南美洲以外唯一被发现的南美侧颈龟科家族成员，真是沧海桑田的变化呀！

# 猪鼻龟：我不是海龟

因为鼻子长得像猪的鼻子，人们叫它们“猪鼻龟”。很多人误以为猪鼻龟生活在海里，而实际上，它们是不折不扣的淡水龟。猪鼻龟的活动范围非常有限，仅生活在澳大利亚、新几内亚部分地区的淡水环境中。可是它们的四肢，明明就是海龟的模样！猪鼻龟的四肢演化成鳍状，和海龟很像。一些科学家认为，猪鼻龟是地球上唯一完全生活在淡水里的淡水龟，但有的科学家也说猪鼻龟只是大部分时间待在水下，偶尔也会上岸，只不过它们那像鱼鳍一样的四肢，会让其在陆地上的行动很笨拙。

因为猪鼻龟的前后足仅有两个爪子，它们还有个外号叫“两爪鳖”。

# 加拉帕戈斯象龟：长得大，活得长

科幻节目《狂想曲：未来世界物种大揭秘》描述了一亿年以后的地球上会演化出一种比大象要重20多倍、比一般人高4倍的巨龟。

虽然目前地球的龟类家族中没有这么大的家伙，但也有让人“大”开眼界的龟，它们就是现存体形最大的陆龟——加拉帕戈斯象龟。

成年加拉帕戈斯象龟身长可达1.5米，平均体重175千克，相当于一个人类的超级大胖子的体重。不光是体形大得吓人，加拉帕戈斯象龟的寿命也长得吓人。据统计，野生加拉帕戈斯象龟的平均寿命超过100岁，而2006年6月23日在澳大利亚动物园因心脏衰竭去世的一只名叫哈丽的加拉帕戈斯象龟，其寿命居然有170多岁，是当时发现的世界上最长寿的动物。

# 背着一块“饼干”的饼干龟

咦？这只龟是背着一块薄饼干吗？开玩笑，这就是人家的壳。它是生活在非洲坦桑尼亚、肯尼亚等地的饼干龟，因其扁平的壳而得名。一些成年的饼干龟的壳，长度居然超过17厘米（反正我没吃过这么大的饼干）。

不光是外形像饼干，饼干龟的壳在重量上也不负“饼干”之名。要知道，龟壳其实是由龟的脊椎、肋骨、腹肋、胸骨和皮内成骨等愈合形成的，相对于大多数陆龟沉重的壳，饼干龟的外骨骼有很多孔洞，因此壳更加轻盈。这样它们就不用背着沉重的“包袱”，行动起来也比一般的陆龟更敏捷。但也由于饼干龟这种稀奇古怪的样子，引来人类收藏的兴致，因此它们被大量捕捉。

# “杂交狂魔”——草龟

说到龟，很多人会脱口而出：乌龟呀！实际上乌龟可代表不了整个龟类家族，它们只是龟类家族中的一个小家族，也叫“草龟”。如果你觉得“草龟”这个名字比起前面那些龟的名字显得太普通，那你可小看草龟了——它们被誉为自然界的“杂交狂魔”。根据以往的统计，它们能与雌性马来闭壳龟、雄性四眼斑水龟、人工饲养的日本石龟杂交并繁育。2006年，科学家发现了一种当时被认为是新物种的“腊戍拟水龟”，但经过研究，有些科学家认为它们是草龟和黄喉拟水龟的杂交品种，而它们的杂交早在史前时期就发生了，这验证了草龟确实是“杂交狂魔”。不仅如此，草龟还是3000多年前的中国商朝人最青睐的两种龟之一！这又是为什么呢？请看下文。

# 花龟，商朝人的“宠龟”

在3000多年前的中国商朝，人们非常喜欢占卜，当时占卜的一个重要程序就是在龟甲或兽骨上刻字，把要占卜的事情刻在上面，这些龟甲和兽骨上的文字就是著名的甲骨文。那么，龟有很多种，商朝人用得最多的龟甲是哪种龟的呢？根据考古学家的研究发现，商朝人最喜欢用的，一种是上文提到的草龟，另一种就是花龟。花龟目前分布在我国的华南地区，但在这些地方以北千里之外的中原商朝遗址中居然发现了大量的花龟甲，说明在商朝的时候，花龟的分布范围比现在广得多！花龟不仅受商朝人喜爱，也受秦汉时期的人所喜爱。在广州发掘的秦汉时期的南越王墓中，就发现了不少花龟甲的碎片。

# 中华鳖：尿尿……用嘴？

新加坡一些研究人员发现，在中华鳖生活的沼泽干涸之后，它们居然还会一头扎进小水坑里。真是奇怪。中华鳖作为爬行动物，不是用肺呼吸的吗？为什么它们总喜欢把脑袋扎进水里呢？难道它们很容易口渴，需要经常喝水？

后来，研究人员得出了一个惊人的结论：中华鳖喜欢一头扎进水里，不是在咕咚咕咚喝水，而是在——尿尿！就像有些鱼类会将自己体内的氮以尿素的形式，经过鱼鳃排到体外那样，中华鳖的嘴里也有类似的鳃状结构，当它们把头扎进水里时，它们会利用嘴里的鳃状结构，将尿素通过口腔排到体外。天哪！这鳖真的在用嘴尿尿！

# 长“睫毛”的高颜值壁虎——纤毛多趾虎

这只壁虎长了睫毛吗？其实，那不是睫毛，是纤毛多趾虎眼睛周围皮肤形成的毛发状突起。不过，它们的“睫毛”可不只长在眼睛周围，而是从脑袋一直延伸到尾巴下面，形成两排。虽然“睫毛”给纤毛多趾虎的颜值加分不少，但这也让它们少了很多壁虎的眼睛都有的一层“防护罩”——眼睑。那它们的大眼睛不会被风吹干吗？进了脏东西又怎么办？别担心，纤毛多趾虎的眼睛覆盖着透明的薄膜，这层薄膜能让它们的眼睛保持湿润。眼睛里有脏东西也不要紧，它们的舌头可是“清洁刷”，能把掉进眼睛里的灰尘等舔干净！

和很多壁虎不同，纤毛多趾虎的尾巴一旦断了，就不能再长出新的了。所以，你在野外经常会见到没有尾巴的纤毛多趾虎。

# 豹变色龙：地点不同，颜色各异

要评选世界上外表最炫丽的爬行动物，马达加斯加岛的豹变色龙绝对可以入选。有意思的是，分布在不同地区的豹变色龙，颜色也不完全一样。比如生活在贝岛、安基菲和安班扎的豹变色龙一般是耀眼的蓝色，而在安比卢贝、安齐拉纳纳、桑巴瓦地区的豹变色龙则是红色、绿色或橙色的，马鲁安采特拉、塔马塔夫等地的豹变色龙主要是红色的。

豹变色龙喜欢生活在干燥的森林或草原地区，它们活动范围内原有的植物大多被人类破坏了，但人们惊奇地发现，豹变色龙仍能在这种恶劣的环境中生存，还经常出没于道路、房屋、种植园等人类生活的场所附近。

# 指甲盖儿大的变色龙：侏儒枯叶避役

2012年，德国和美国的科学家在非洲马达加斯加发现了4种新的变色龙。让世人震惊的是，这些变色龙成年以后，只能长到大约人类的指甲盖儿那么大，它们很可能是世界上最小的蜥蜴。这些身形短小的变色龙都属于侏儒枯叶避役家族。这几种侏儒枯叶避役生活在马达加斯加东北部哈拉岛的热带雨林中，经过科学家测量，它们从鼻尖到尾部末端的总长度不足30毫米。这些侏儒枯叶避役平时只在枯叶之间生活，身体呈棕色。虽然也是变色龙，但并不变色，而身体的棕色能让它们与枯叶很好地“融为一体”，起到伪装的作用。由于这几种侏儒枯叶避役长得实在太像了，科学家费了好大功夫，用分子遗传学分析才认定它们原来是4种不同的侏儒枯叶避役。

# 尾巴上长了片“叶子”的平额叶尾壁虎

平额叶尾壁虎在树上的生活怎么样？嘿，那可谓“如鱼得水”。它们那叶子般宽大而扁平的尾巴有大作用，能让它们在树枝间轻松地跳来跳去。和很多壁虎一样，平额叶尾壁虎在遇到威胁时，也会将尾巴脱落，然后迅速逃走。哈哈，这就是名副其实的“落叶”嘛。

# 喷血的“尖刺铠甲”战士——角蜥

哎哟，这浑身长刺、胖乎乎的家伙竟然是一只蜥蜴？没错，这就是主要生活在美洲沙漠的角蜥！它披着一身长满硬刺的“铠甲”，谁要敢来惹它，它一定让来犯者头破血流！

很多爬行动物都有鳞片，角蜥也是爬行动物，它们的这些硬刺就是爬行动物界一种十分特别的鳞片。

一些角蜥还有一项技能：喷血！它们能让自己头部的血压增高，眼角破裂，喷出鲜血，这样敌人就吓得不敢靠近了。角蜥可以把血喷到1米以外！为什么捕食者一看到血就不敢靠近呢？是晕血吗？有些科学家认为，角蜥平时吃了很多收获蚁，收获蚁体内的一些化学元素在角蜥身体里堆积，让它们的血液带着一股难闻的气味，喷出来会把来犯的犬类、猫科动物“熏走”。

# 喉咙下长着“肉垂”的绿鬣蜥

你知道“鬣”字是什么意思吗？“鬣”指动物头上或脖子上的毛。欸？这么说，绿鬣蜥头上或脖子上也有毛吗？其实，绿鬣蜥从头到脖子、背，再到尾巴，有一排特殊的鳞片，这排鳞片又细又长，看起来就像根根立着的毛一样。

绿鬣蜥是个脾气很好的家伙，平常爱吃点儿植物的叶子、花、果子等。遇到危险时，绿鬣蜥有不少办法应对。你看到它们喉咙下面那块肉嘟嘟的东西了吗？如果有天敌来攻击，它们就会把这块肉嘟嘟的东西展开，吓对方一大跳！要是吓不住对方，绿鬣蜥还会用像鞭子一样的尾巴抽打对方！

达尔文到加拉帕戈斯群岛探险时，第一次看到这里的陆鬣蜥，感叹道：“这家伙上半身是橘黄色的，下半身是红褐色的，长得又蠢又难看！”

你们也觉得它又蠢又难看吗？我觉得它很萌呀！白天的时候，加拉帕戈斯陆鬣蜥会趴在火山岩上，悠闲地晒太阳。到了晚上，就回到自己的洞里呼呼大睡，这样才能维持体温。

别看加拉帕戈斯陆鬣蜥有点儿呆萌，它们还会与别的动物合作呢！加拉帕戈斯岛上的一些鸟类会把陆鬣蜥身上的虱子等讨厌的寄生虫吃掉，这样，陆鬣蜥的身上干净了，鸟类也吃得饱饱的，它们都得到了好处，这就是合作的力量。

# 有毒的“吉拉怪物”——吉拉毒蜥

吉拉毒蜥生活在北美洲的灌木丛、橡树林或沙漠中。蜥蜴大家庭中，它们是少有的剧毒蜥蜴。在美国的西部，流传着一个叫“吉拉怪物”的传说，它们身体散发着有剧毒的气息，一滴毒液就能毒死人！这种“吉拉怪物”的原型可能就是吉拉毒蜥。

实际上，吉拉毒蜥不会随便使用毒液，就连抓猎物的时候都不用。只有在感觉到危险，为了保护自己时才会使用毒液。很少有吉拉毒蜥咬死人的案例，但据说被吉拉毒蜥咬过之后会非常疼，一位动物记者被咬之后说：“这是我一生中经历过的最严重的疼痛，就像热熔岩在血管中流淌。”

别看吉拉毒蜥不是很有名，它们可是“老前辈”啦，古生物学家发现，吉拉毒蜥早在1亿多年前的白垩纪就出现了，和恐龙生活在同一片天地！

# 海中蜥蜴——海鬣蜥

图中这个家伙也是鬣蜥，是能在大海里畅游的海鬣蜥！海鬣蜥主要生活在加拉帕戈斯群岛的海岸边，成年雄性海鬣蜥在肚子饿得咕咕叫时会下海。因为海里有它们爱吃的美食——海藻和长在海底岩石上的其他植物。

但是，这里的海水可冷了，海鬣蜥是爬行动物，需要经常晒太阳来保持体温，海水那么冷，它能顺利地吃到海里的植物吗？

海鬣蜥很聪明，它们先趴在海边的岩石上晒够太阳，暖和身子，然后一头扎进海里，游到海底的岩石上，争分夺秒，大口啃食植物。它们还会用钩子一样的爪子紧紧抓住岩石，防止被海底强大的水流冲走。吃饱后，海鬣蜥会迅速游上岸来晒太阳，以免身体完全冷掉。

# 长“胡子”的鬃狮蜥

你说这只蜥蜴下巴上的“胡子”是修图修上去的？绝对没有！这是被人戏称为“胡子龙”的鬃狮蜥。它们下巴上所谓的“胡子”，其实是“颚须”，因为这些“颚须”让鬃狮蜥看起来像长着鬃毛的狮子，所以它们有了“鬃狮蜥”的名字。

当感觉到有危险的时候，鬃狮蜥会鼓胀起身体和下巴上的“胡子”，远远看上去真的像一头威风八面的雄狮！偷偷告诉你，此时鬃狮蜥的胡子还会变成黑色，它们可能是在警告你：别靠近我，我可不是好惹的！

并不是雄性鬃狮蜥才有“胡子”，雌性鬃狮蜥也是有“胡子”的，只不过雄性鬃狮蜥更爱“显摆”自己的“胡子”。

# 夏天的“舞蹈家”——铲鼻蜥

夏天太热啦！怎么降温呢？空调、冷饮、西瓜安排上！但你知道吗？生活在撒哈拉沙漠中的铲鼻蜥在夏天的降温方法很特别，一旦天气变得太热，它们就会跳起奇怪的“舞蹈”。

其实，铲鼻蜥并不是真的在跳舞。天气炎热的时候，沙漠的沙子就会变得很烫，铲鼻蜥就没办法好好地站在沙子上。它们会依靠尾巴保持身体的平衡，然后一次抬起两只脚，比如先抬起左前脚和右后脚，然后大约间隔10秒，再抬起右前脚和左后脚，这样交替抬脚，看起来就像在跳奇怪的舞蹈！别笑，这样的动作可以帮它们冷却脚掌，否则脚掌一直贴着那么烫的沙子，还不得烫熟了啊！

# 世界上最大的蜥蜴——科莫多巨蜥

前面说了世界上最小的蜥蜴，仅有人类指甲盖儿那么大，那么世界上现存最大的蜥蜴是哪种呢？它就是生活在印度尼西亚几个小岛上的科莫多巨蜥。

成年的科莫多巨蜥，体长一般在2米以上，而最大的科莫多巨蜥能超过3米！在吃饱的状态下，科莫多巨蜥的体重可以超过160千克。不过，因为科莫多巨蜥比较能吃，有时候一次吃掉的食物重量相当于自身体重的60%，因此它们的体重是有“水分”的，但科学家推测，最重的科莫多巨蜥也能达到100千克。以前，人们推测科莫多巨蜥是依靠口腔里致命的细菌来杀死猎物的，但2005年和2009年的研究表明，科莫多巨蜥的口中有毒腺，也能分泌毒液，对猎物造成一定伤害。

# 水面狂奔者——双冠蜥

当你漫步在墨西哥南部或哥伦比亚北部的雨林中，看到一只正在水面狂奔的蜥蜴时，你千万不要以为自己眼睛花了——它们是最令人眼前一亮的双冠蜥。听到名字，你可能觉得双冠蜥的头上都长着两个“冠”，其实，只是雄性双冠蜥的头上有两个“冠”，而雌性双冠蜥的头上只有一个“冠”。不过不管是一个冠还是两个冠，双冠蜥的形象都很容易让人联想到欧洲传说中的蛇怪巴西利斯克，以至于1768年，著名生物学家林奈用传说中的蛇怪来称呼双冠蜥（拉丁学名：Basilicus）。更怪异的是，为了逃脱捕食者的追击，双冠蜥真的会在水面狂奔，速度快到可以不让身体沉入水中。这是有多快呢？科学家研究认为，一些双冠蜥在水面奔跑的最大速度可达每秒1.6米！

# 犰狳环尾蜥：我是带刺的“车轮”

这家伙是在模仿犰狳穿了一身带刺的铠甲吗？它怎么做到自己咬自己的尾巴，首尾相连的呢？

这是生活在非洲南部沙漠中的犰狳环尾蜥，它们的名字正说明了它们身上的“铠甲”就像大洋彼岸美洲的奇特哺乳动物犰狳那样无坚不摧！不过犰狳环尾蜥的“铠甲”要比犰狳的更胜一筹，因为它们的“铠甲”带刺——突出鳞骨的鳞甲硬刺，让捕食者“望而却口”，这样它们就能躲过一劫。

那它们为什么要咬住自己的尾巴，做成“刺球”的样子呢？这是因为犰狳环尾蜥虽然有一身带刺的“铠甲”，但肚子比较柔软，万一暴露给敌人，那可就不得了。所以在遇到敌人或受到惊吓的时候，犰狳环尾蜥会像犰狳那样咬住尾巴不放，变成一个车轮的样子，护住自己的软肚。

# 蛇蜥，一条冒牌的蛇

妈呀，这是一条蛇吗？嘿嘿，人家是货真价实的蜥蜴——蛇蜥！

蛇蜥身体细长，又没有脚，一眼看上去真的会让人误认为是蛇。不过，蛇蜥有着不折不扣的蜥蜴属性——像大多数蜥蜴一样，当蛇蜥遇到捕食者或受到惊吓的时候，它们会舍卒保车，自断尾巴逃走。只是，蛇蜥断尾之后，尾巴就不能长到原来那么长了。

别看蛇蜥长得像蛇，它们可没有很多蛇那种超快的移动速度，也没有很多蛇那样凶悍的战斗力。在欧洲，家猫是出了名的“蛇蜥杀手”，它们对家猫的进攻毫无招架之力。

不过，蛇蜥也有值得骄傲的地方，它们是著名的长寿蜥蜴。一些野生蛇蜥的寿命可达30年，有些人工饲养的蛇蜥还能活过50岁呢！

# 喙头蜥：不是蜥蜴，更胜蜥蜴

如果说蛇蜥是冒牌的蛇，那么接下来这位重量级“蜥蜴”可就是冒牌的蜥蜴了！它们是世界上最珍稀的爬行动物，仅生活在新西兰柯克海峡的几个小岛上，是新西兰的特有爬行动物，被奉为“国之骄傲”的喙头蜥。

为什么说它们是冒牌蜥蜴？因为它们根本就不是蜥蜴呀！在爬行动物中，有个被称为“喙头类”的家族，喙头蜥就属于喙头类，虽然它们长得十分像蜥蜴，但人家在爬行界可是“独立门户”。

喙头类早在大约2.4亿年前的三叠纪就出现了，在恐龙全盛的侏罗纪，喙头类也达到了家族最兴盛的时期，而且它们很幸运地躲过了白垩纪晚期的生物大灭绝。喙头蜥的牙齿在现生爬行动物中独一无二——下颌的一排牙齿紧紧咬在上颌的两排牙齿之间。据科学家估计，喙头蜥的寿命可达百岁。

# 头上冒泡的资深“潜水员”——安乐蜥

水生安乐蜥是生活在美洲哥斯达黎加西海岸的蜥蜴，它们是资深的“潜水员”，掌握着“水肺潜水”的技能。2021年，研究人员将6种不同的水生安乐蜥放入装满水的水箱中，发现这些水生安乐蜥在入水前会先深深地吸一口气，让自己的鼻子上膨胀出一个大大的气泡。水生安乐蜥需要在水下用氧时，就会从这个气泡中吸氧，气泡中的氧气少了，气泡也会往鼻子里缩小。然后，研究人员用氧气传感器对气泡里的氧气饱和度进行测量，发现这些家伙在水下待的时间越长，气泡里的含氧量就越低。这个实验充分说明，水生安乐蜥消耗了气泡里的氧气。它们先把氧气存在气泡里，用的时候再从气泡里吸取，这就是水生安乐蜥惊天的“水肺潜水”技能。

# 与蜘蛛侠“撞衫”的姆万扎平头飞龙蜥

咦？蜘蛛侠怎么变成了一只蜥蜴？不，是这只蜥蜴正好和蜘蛛侠“撞衫”啦！

生活在非洲坦桑尼亚、卢旺达和肯尼亚等地的姆万扎平头飞龙蜥是有名的“蜘蛛侠蜥蜴”，它们还真像酷炫的蜘蛛侠：上半身是红色或紫色的，下半身是深蓝色的。不过只有雄性姆万扎平头飞龙蜥才会和蜘蛛侠“撞衫”，雌性姆万扎平头飞龙蜥全身呈棕色，很是普通。雄性姆万扎平头飞龙蜥不仅“穿”了件蜘蛛侠“外套”，它们那飞檐走壁的技能也像极了帅气的蜘蛛侠，遇到天敌或其他危险的时候，它们既能用两条后腿狂奔，又能爬上栖息地的石块。

在肯尼亚，姆万扎平头飞龙蜥的家园正在被人类侵占，但不可思议的是，它们已经开始尝试与人类共同生活在一个屋檐下了。

# 闪光蜥：蜥蜴界的“凯门鳄”

每个人都有自己特别爱吃的东西，蜥蜴界也有这样的吃货——生活在巴西、哥伦比亚、厄瓜多尔、秘鲁和圭亚那等国家的闪光蜥，特别爱吃带壳的东西，什么螃蟹、小龙虾、蛤蜊，甚至是蜗牛也不放过。可问题来了，这些东西那么硬，蜥蜴又不可能像人类一样用灵动的手或合适的工具去剥壳，它们怎么吃呢？看看闪光蜥的“配置”就知道了，它们的头大而粗壮，“下巴”的肌肉十分发达，还长了一嘴又圆又钝的牙齿。与穿透或撕裂皮肉的尖牙不同，这种牙齿可是压碎那些食物外壳的“神器”！

这些带壳食物里，闪光蜥尤其爱吃蜗牛，它们吃蜗牛的时候，先咬住蜗牛，然后把脑袋一仰，用那粗壮的牙齿把蜗牛“咔”地压碎，吃掉里面的肉，最后吐出蜗牛壳。有些粗暴的闪光蜥甚至连壳都不吐，直接连壳带肉一起吞进肚里！

# “打伞”的狂奔者——褶伞蜥

这家伙算是最夸张的蜥蜴之一了：感觉到危险时，脖子上的薄膜会“撑”起来，远远望去就像一把大伞，同时张大嘴巴吓走敌人，逃跑时还能用两条后腿在干燥的草原灌木地带“狂奔”，它就是褶伞蜥。

说起澳大利亚的动物明星，很多人会想到袋鼠、考拉，褶伞蜥也是！它们由于的长相和行为太“奇葩”，常常被影视作品拿去“戏弄”。比如，在大导演斯皮尔伯格1993年拍摄的好莱坞大片《侏罗纪公园》里，有的恐龙被设计成脖子上有“薄膜”的形象，在发动攻击时，薄膜能像伞一样撑起来，这个灵感就来源于褶伞蜥。此外，褶伞蜥的形象还曾经出现在澳大利亚的硬币上呢！

# 大鳞鳞虎：我把鳞片都扔了

在马达加斯加北部的安卡拉纳国家公园的喀斯特地貌的石灰岩山中，科学家看到一种奇怪的壁虎——大鳞鳞虎遇到危险的时候，它们身上的鳞片居然会像“大珠小珠落玉盘”那样从身上掉下来，“裸露”出身体，然后趁机逃跑。

其实，这并不是科学家第一次发现这么奇葩的壁虎。早在1867年，科学家就发现了会使用这种自虐逃命法的“鳞虎”家族成员，而大鳞鳞虎是这些成员里鳞片最大的。那么，大鳞鳞虎的鳞片究竟是怎么能从它们身上掉下来的呢？原来，在它们的皮肤底部，有一些神奇的“肌纤维母细胞”，通过收缩这些细胞，鳞片就能从这层细胞上分离，因此这个区域被叫作“撕裂区”。

# 名字有点儿吓人的截趾虎

这个名字听起来有点儿吓人，难道它们是被截掉脚趾的壁虎？其实，截趾虎名字的来源是，它们的前后脚的内侧脚趾上几乎没有爪，所以就被人叫了这么个略显惊悚的名字。截趾虎虽然少了一些爪，却能在各种环境中茁壮成长，森林、城市都可以见到它们的身影。不仅如此，它们的分布地还十分广泛，从东南亚到夏威夷、墨西哥，横跨太平洋。截趾虎在夏威夷的沙滩上很常见，但被那里的人视为入侵物种。不过在当地，人们不介意与它们一起生活。在人类的家中，它们只在晚上出来活动，而且大部分时间都待在高高的墙壁和天花板上，还能帮人类抓昆虫，好像它们也是家里的一分子。

# 新疆沙虎：“蜥”狠话不多

新疆沙虎生活在中国甘肃、内蒙古、新疆等地干旱的戈壁，一些维吾尔族人叫它们“库莱伦”，意思是“有手的蛇”。别看这些小家伙体长才70～80厘米，从吐鲁番盆地到海拔1700多米的地区，都有它们的身影。

它们个头儿不大，脾气可不小。它们的耳朵异常灵敏，出洞以后对一点点响动都能迅速做出反应——火速远离声音发出的地方。不过，一旦这个声音逐渐接近新疆沙虎的脑袋，它们就会迎面冲上去，“不分青红皂白”地张开嘴巴狠狠地咬，并发出低微的声音以示警告。说起它们的洞穴，科学家在野外考察时发现，新疆沙虎的洞穴结构十分复杂，有的洞里甚至有5条岔道！

# 最熟悉的陌生蜥——里氏睑虎

里氏睑虎又名“利希登费尔兹睑虎”，其名字来源于越南河内的工程师卡尔雷斯·利希登费尔兹，因为是这位工程师最先把里氏睑虎的标本寄给法国国家自然历史博物馆的；1897年被正式命名，成为睑虎家族的首位成员。

你可能会觉得里氏睑虎的长相好像在哪里见过。没错，我国的海南睑虎与里氏睑虎非常相似——两者的身体的底色均为黑色或灰色，身上均有4条标志性的黄色横纹。但里氏睑虎和海南睑虎的明显区别，也正在这黄色横纹上——里氏睑虎脖子上的第一条横纹，在耳孔和嘴角区域，其颜色会慢慢变淡直至消失；而海南睑虎的这条横纹则穿耳孔而过直至嘴角，并且颜色始终饱满。里氏睑虎真可以称得上是最熟悉的陌生蜥呀！

# 白唇树蜥：白嘴唇，蓝战袍

这个穿着蓝色“衣服”的家伙，嘴巴怎么白了？莫非是用白色粉笔涂的吗？这是白唇树蜥，顾名思义，从它的上嘴唇一直延伸到身体侧面，有着白色的斑纹。不过更让我们眼前一亮的是，白唇树蜥有一身炫亮的蓝色“外衣”，那是它们蓝色的体色，远远望去，是不是很像一幅描绘蔚蓝天空的油画？其实，平时白唇树蜥并不是这个颜色，而是呈比较单调的棕色或橄榄色，只有当它们迎来交配季时，才会换上这身好看的蓝色“战袍”（主要是身体的前部），并频繁地上下摇晃自己的小脑袋，以“高颜值”的面貌和引人注目的动作，来吸引异性的眼球。

# 斜纹蓝舌蜥：看我的舌头

正当你夸赞白唇树蜥那身漂亮的蓝色战袍时，生活在澳大利亚的斜纹蓝舌蜥可不服气了：谁还没有点儿蓝色，我的舌头也是蓝色的！当蓝舌蜥吐出舌头的那一刻，人们震惊了，它还真有一条蓝色的大舌头！

在人们看来，蓝舌头很新奇，但对于蓝舌蜥的天敌而言，蓝舌头代表着蓝舌蜥向天敌发出虚张声势的警告。一些科学家发现，蓝舌蜥会根据天敌的危险程度来调节舌头蓝色的深浅。

你如果以为这条蓝舌头只能虚张声势，就大错特错了！蓝舌蜥的舌头能分泌黏稠的液体，用来捕捉猎物；人们还发现，蓝舌蜥能像猫狗一样用舌头“舔”水喝，只不过没有向后“舀”的动作。

# 倒霉的蜡皮蜥：大壁虎的“伪品”

在中国广东、海南一带有这样一种穴居性蜥蜴，广东湛江叫它“沙神”，海南则叫它“坡马”，这就是蜡皮蜥。

蜡皮蜥非常喜欢在开阔的沙质土地中生活，虽然环境没那么优美，但它们长得很让人“赏心悦目”——蜡皮蜥拥有一身漂亮的花纹，又被叫作“蝴蝶蜥”。

但在过去的几十年里，蜡皮蜥被人类大量捕捉，以大壁虎“伪品”的身份入药，导致其种群数量急速减少，2021年2月被列为国家二级保护动物。

在以前的文献中，珠海的淇澳岛是没有蜡皮蜥分布的，但在2023年初，人类惊奇地在珠海淇澳岛发现了它们，这次发现创造了珠海物种的新纪录！

# 沙漠怪客——大耳沙蜥

在我国新疆的荒漠中，有这样的“怪客”，它们叫大耳沙蜥，当感觉到危险临近的时候，它们就会张开一对大耳朵不停地扇动，还发出“呼呼”声来示警。

其实，所谓的大耳朵，并不是大耳沙蜥真正的耳朵，而是分布在它们嘴角两侧的皮褶，这皮褶形似耳朵，所以人们就形象地叫它们大耳沙蜥了。

大耳沙蜥还不只有这一项绝技，它们有一身能融入周边环境的“外衣”，还能将自己埋入沙中。一些爬行动物观察家就曾看到过大耳沙蜥这项绝技——感觉到人类靠近，它们果断快速扭动身体，钻进沙海，不留一点儿踪迹！

大耳沙蜥是沙蜥家族中体形最大的，它们扇动两个“大耳朵”时看着很萌，其实生性凶猛，是个不好惹的家伙！

# 现实中的中国“龙”——圆鼻巨蜥

中国人自称是龙的传人，龙作为一种神话动物，是我们中华民族的精神图腾。在现实世界中，也有一些动物被我们叫作“龙”，比如生活在我国华南、西南部分地区的圆鼻巨蜥，在当地有个霸气的名字——五爪金龙！

圆鼻巨蜥是一种大型水栖蜥蜴，最大的个体超过3米，长长的尾巴，锋利的牙齿和爪子（每个爪子上有5个脚趾），再加上一张强有力的嘴巴，看上去真像威风凛凛的巨龙。

2023年，曾有一条新闻上了热搜，有人说：五爪金龙的唾液有毒。专家对此表示：作为巨蜥家族的一员，圆鼻巨蜥的唾液是否有毒，科学家们意见不一致，目前尚无定论。

# 史上最憋屈的莫顿水巨蜥：赢了鳄鱼，输给蟾蜍

在与五爪金龙生活的亚洲隔洋相望的澳大利亚，也有一种水生巨蜥，它们就是生活在澳大利亚北部的莫顿水巨蜥。它们比圆鼻巨蜥更依赖水，晒日光浴都要在水中找块石头或浮木来当床。

说起莫顿水巨蜥，可真是让人哭笑不得，虽然它们的体长一般不超过1米，比不过圆鼻巨蜥，但它们也是极其凶悍的角色，不少鱼类、两栖动物、鸟类、哺乳动物都会成为它们的“盘中餐”，甚至有人拍到它们捕杀年幼的淡水鳄鱼的画面！

可就是这样的家伙，谁也没想到它们会“输”给不起眼的蟾蜍。根据科学家的调查，自从1935年蔗蟾蜍引入澳大利亚，莫顿水巨蜥的数量大幅减少，被蔗蟾蜍毒死成为莫顿水巨蜥近百年来最常见的死因之一。

# 胎生蜥蜴：地球上耐寒的蜥蜴

如果你问我分布范围最广的蜥蜴是谁？我会先给你介绍一种蜥蜴的英文俗名：Common Lizard，意思是“普通的蜥蜴”或“普遍的蜥蜴”，可见它们的分布范围有多广。Common Lizard的足迹遍布欧亚大陆，甚至有报道称，天寒地冻的北极圈内都有它们活动的身影！它们也被誉为“地球上最耐寒的蜥蜴”。

好了，现在我们回到最开始的问题，揭晓答案！Common Lizard的中文正名是“胎生蜥蜴”。顾名思义，它们并非像很多爬行动物那样先下蛋，然后宝宝从蛋里孵化出来；它们是直接产下蜥蜴宝宝，和包括我们人类在内的绝大多数哺乳动物一样。

# 超级“混居者”——捷蜥蜴

捷蜥蜴是足迹遍布各地的“浪子”，从草原到森林，从果园到灌木丛再到谷地，甚至在海拔2000米以上的高山上，你都能看到它们。捷蜥蜴不仅能适应各种生存环境，还是不可多得的“交际高手”。在中国的新疆，科学家就发现捷蜥蜴有和其他动物“混居”的情况。比如，它们能和与它们同为蜥蜴家族成员的敏麻蜥和谐相处，也能和某些蛙类亲密接触，甚至还和含有剧毒的中介蝮待在一起！

但是，你不要以为它们是好脾气的，在交配季节，雄性捷蜥蜴会一反常态，与竞争对手大打出手，有的雄性之间会争斗几秒钟，有的雄性之间可能展开超过15分钟的持久战！

# 加拉帕戈斯熔岩蜥蜴：小小身躯，大大格局

这个身躯庞大的是被誉为“地球上最像哥斯拉的动物”——生活在加拉帕戈斯群岛的海鬣蜥。没错，但今天我们要说的不是它，而是它头上那只小小的蜥蜴！你发现它了吗？它是加拉帕戈斯熔岩蜥蜴。

别看加拉帕戈斯熔岩蜥蜴通常只有十几厘米长，它们却不怎么害怕人类，一般情况下，你近距离观察，它们也不会逃走。而且，它们虽然身躯小，却是岛上的“除害标兵”，比如，它们能消灭很多加拉帕戈斯群岛上的蝗虫。不过，在气候干燥的时候，加拉帕戈斯熔岩蜥蜴也可以接受吃素。雄性熔岩蜥蜴非常在意自己的领地，它们会在一块大石头上标记自己的领地，不停地摆动脑袋，来表达这是它们的地盘，这一系列动作看上去就像做俯卧撑！

# 像松果的松果蜥

拥有蓝舌头的蜥蜴可不止斜纹蓝舌蜥，蓝舌蜥是一个大家族，在这个大家族里，还有一个更奇葩的存在，那就是松果蜥。

看，这家伙长得真的很另类，背部的鳞片如铺着一层砾石一样坚硬，远远看上去就像松鼠喜爱的松果，松果蜥也因此而得名。

除了用蓝舌头虚张声势地警告来犯者，松果蜥还有一项技能——头尾迷惑法。松果蜥的尾巴短粗，形状与它们的脑袋特别像，因此也会给来犯者一种头尾分不清的感觉，从而达到迷惑来犯者的效果。也正因为这样，松果蜥还有个好玩的俗名，叫“双头石龙子”。

# 北草蜥：外号“四脚蛇”的蜥蜴

哎呀，它是谁？长得像蛇，但再仔细看看，有4条腿，这就是人们常说的“四脚蛇”吗？

它是名副其实的蜥蜴——北草蜥，是我国特有的一种蜥蜴，北到吉林，西到甘肃，以及广大的华南地区，都有它们那“蛇一般的魅影”。

其实不怪你把它错认成蛇，因为北草蜥拥有蜥蜴界超长的尾巴——其尾巴的长度可以达到头体长度的2～3倍！同属的南草蜥尾巴更长，可达头体长度的3倍以上！

北草蜥有“风一般的速度”，不论是抓猎物还是逃跑，它们都能敏捷地移动。不过，即使速度快，北草蜥也免不了被人类捕捉的命运。有些地方认为“四脚蛇”可以制成药酒，便去捕捉它们。大家要注意，北草蜥早在2000年就被列入《国家保护的有益的或者有重要经济、科学研究价值的陆生野生动物名录》，我们要好好保护它们哟！

---

# 蚯蚓和蜥蜴的“合体”——五趾双足蚓蜥

哇，这个小东西好可爱，身体像蚯蚓一样细长而粉嫩，可是又长着两只“脚”，每只“脚”上有5个萌萌的小脚趾，看起来又有点儿像蜥蜴，这到底是变异的蚯蚓还是变异的蜥蜴啊？

其实，这家伙既不是蜥蜴也不是蚯蚓，它是一类属于蚓蜥家族的爬行动物——五趾双足蚓蜥。那不还是蜥蜴？之前咱们讲喙头蜥的时候讲过，不是名字里有“蜥”的就是蜥蜴。五趾双足蚓蜥的两条后腿已经退化了，前面两条腿还在，所以称“双足”。它们通体为粉红色，在地下能像蚯蚓那样蠕动身体来前行，所以称 “蚓”蜥。有意思的是，在五趾双足蚓蜥的日常食谱中，既有蚯蚓，又有小蜥蜴……

---

# 没耳朵的“龙”——拟毒蜥

棕色的身体、长长的脖子、短小的四肢、长而锋利的爪子，再加上背上那6列纵向分布的龙骨状鳞片，如果我不说它是蜥蜴，你真的会以为神话传说中的龙走进了现实——这就是生活在东南亚婆罗洲的拟毒蜥，也叫“无耳巨蜥”。尽管说拟毒蜥“没耳朵”——没有耳孔等可见的耳朵外部构件，但拟毒蜥仍然可以听到声音。有趣的是，威风凛凛的外表下，拟毒蜥却长了一双“拉胯”的眼睛——眼睛小而且是半透明状。

不仅长得像龙，拟毒蜥还能学“龙吟”。它们有时候会像青蛙那样振动喉咙，有时候像蛇那样轻弹分叉的舌头，从而发出刺耳的叫声。像传说中的龙一样，拟毒蜥也喜欢有水的地方，一项研究显示，科学家曾观察19只拟毒蜥在晚上的生活状况，发现它们之中有一半待在水中，另一半也在靠近水的陆地上。

---

# 牵扯进“乌龙”案的萨氏巨蜥

目前，世界公认最大的蜥蜴是科莫多巨蜥，最重也最长（可超过3米）。然而在很早以前，世界最长蜥蜴的称号曾被授予新几内亚的萨氏巨蜥。有关报道称，一些萨氏巨蜥体长也能长到3米以上；还有一些消息称，萨氏巨蜥能够长到3.5米，甚至还有些报告称世界上存在超过6米的萨氏巨蜥，然而这些说法并没有经过证实。目前已知的最长的萨氏巨蜥在2.5米左右，比起最大的科莫多巨蜥还是逊色不少的。虽然个子长得不小，但萨氏巨蜥的食谱上都是些弱小动物，它们喜欢吃鸟类和鸟蛋，也会吃一些腐肉，有的时候居然会捕食狗。

---

# 以岛为家，圣埃斯特万岛黑叩壁蜥

你听说过一座岛的名字与一种蜥蜴有关系吗？在加利福尼亚湾，有座圣埃斯特万岛，这座岛最初被命名为“大蜥蜴的岛屿”，这个名字源于岛上特有的一种大型蜥蜴——圣埃斯特万岛黑叩壁蜥。该岛有很多圣埃斯特万岛黑叩壁蜥，在岛外却丝毫没有这种蜥蜴的踪迹。

很久以前，有人曾把圣埃斯特万岛黑叩壁蜥带到岛外饲养，可这些蜥蜴在离开自己的“故乡”后没能成功地存活下来。圣埃斯特万岛当地的居民，将黑叩壁蜥视作重要的食物，还骄傲地称自己是“大型蜥蜴的人民”。由此可见，圣埃斯特万岛黑叩壁蜥和这座岛屿以及岛上的居民，真的是有斩不断的羁绊。

---

# 暴躁的“跳高选手”——古巴鳄

乍一看这条鳄鱼是不是还挺萌的呢？它身上黑色和黄色的鳞片亮闪闪的，就像漂亮的鹅卵石。可千万不要被它的外表欺骗了，它可是鳄鱼界的一霸——生活在古巴沼泽、湿地的古巴鳄！

古巴鳄的性格有点儿暴躁，为了能抓到好吃的，它们甚至会整个身体跃出水面，称得上是一名优秀的“跳高选手”。它们嘴里靠后的牙又粗又结实，可以轻松地压碎贝类甚至是海龟的壳。古巴鳄虽然厉害，但在古巴，人类曾经对它们进行疯狂的捕杀，让它们差点儿灭绝！幸亏古巴政府1959～1960年在野外一口气抓了好几百条古巴鳄进行保育，才使它们的种群得到发展。

---

# 会两种走路姿势的美国短吻鳄

美国短吻鳄主要分布在美国东南部的淡水湿地里，它们可是西半球鳄鱼家族中的大家伙。最大的雄性美国短吻鳄长大以后，身长可达 4米以上，都赶上一辆小汽车的长度了！美国短吻鳄走路的姿势特别有意思，分为“低走”和“高走”。像你经常见到的鳄鱼走路的样子，美国短吻鳄会用肚皮贴着地面走，就叫“低走”；它们还会通过伸展四肢把肚子抬离地面，姿势就像人做俯卧撑时用四肢把身体撑起来一样，这样行走就叫“高走”。

美国短吻鳄的嘴里有七八十颗可怕的牙。它们是“吃货”，几乎不挑食，鱼类、鸟类、蛇类、龟类，它们都吃，有时甚至还会吃鹿、野猪这样的大型哺乳动物。

---

# 吓坏海员的巨鳄——湾鳄

人们通常认为，鳄鱼在河流、湖泊、沼泽里游泳。但你知道吗，鳄鱼也能在大海里游泳。

湾鳄栖息在河流入海口的三角洲、红树林等地，是世界上已知咬合力最强的动物，它们的咬合力比大白鲨、东北虎还大！巨大的个头儿和恐怖的嘴巴让它们成为顶级猎手，能够捕食幼象、犀牛、水牛等大型动物，甚至还会在海里捕食鲨鱼！

湾鳄到底能长多大呢？2011年，人们在菲律宾抓到了一只超大的雄性湾鳄，它身长6.17米，体重超过1吨，是人类目前捕获圈养过的最大湾鳄，人们还给它起了个霸气的名字——“洛龙”。

关于湾鳄，清朝的《李准巡海记》中有过记载：1907年，广东水师的官员乘坐“伏波”“琛航”两艘军舰，从海南岛开往西沙群岛一带，途中居然有湾鳄从海里爬上了军舰，可把军舰上的人吓坏了！

---

# 尼罗河里有没有尼罗鳄

尼罗鳄是非洲最大的爬行动物，它们的个头儿只比湾鳄小。雄性尼罗鳄可以长到5米长，体重超过1吨，而且它们还是鳄鱼家族中的“长寿之星”，平均可以活到85岁。

尼罗鳄主要栖息在非洲东部、中部、南部，以及马达加斯加岛的河流、湖泊之中。科学家发现它们居然也能像湾鳄一样生活在咸水中，人们就曾在三角洲、咸水湖发现过它们的踪迹。有趣的是，在历史上，尼罗鳄的分布范围很广，曾经向北延伸到整个尼罗河流域。但从1000多年前开始，人类看中了它们那珍贵的“铠甲”，疯狂地捕杀它们。目前在尼罗河中，尼罗鳄比较罕见了，我们只能在尼罗河流域的支流中偶见它们。

---

# 被古人“玩坏”的恒河鳄

生活在南亚淡水河流中的恒河鳄的长相很奇怪，身体又细又长。它们的嘴巴也又细又长，就像早点摊上卖油条的叔叔夹油条的夹子！

早在4000多年前，南亚地区的人们就认识了这种长相奇怪的鳄鱼。考古学家发现的印章、石碑上刻着这样的画面：恒河鳄的周围全都是鱼，它们嘴里还叼着鱼。这说明几千年前，人们就知道这家伙最爱吃鱼了！

有些人把恒河鳄看成河神、风神的化身。在恒河鳄生活的河流附近，当地人认为恒河鳄可以治很多疾病，直接把恒河鳄作为药物。在尼泊尔，有些人认为雄性恒河鳄拥有神秘的力量，人们经常杀死恒河鳄，然后收集它们那奇怪的鼻子！塔鲁人认为，把恒河鳄的尸体放在田野里燃烧，能驱除害虫，给农民带来丰收。恒河鳄没有可证实的攻击人致死的记录，也缺乏捕食大型动物的能力。这家伙和我们中国的扬子鳄一样是极危物种，一度险些在野外灭绝！

---

# 受欺负的“眼镜仔”——眼镜凯门鳄

咦？这条鳄鱼怎么戴着一副眼镜？哈哈，那不是眼镜！这条鳄鱼叫眼镜凯门鳄，因为它的眼球前有一条横骨，看上去就像眼镜架一样，所以才有了这个有趣的名字。

眼镜凯门鳄是鳄鱼界出了名的小不点儿，成年的眼镜凯门鳄有1～2米长，最大的不超过2.5米，跟恒河鳄、尼罗鳄、湾鳄那些大哥没法相比。因为个头儿较小，它们在大自然里受尽了欺负！蜥蜴、鹰、长鼻浣熊、猴子等会偷它们的蛋吃！没长大的眼镜凯门鳄会成为鲶鱼、龟类的食物，甚至连家猪都会吃小眼镜凯门鳄！即使是长大的眼镜凯门鳄，也有美洲豹、森蚺、黑凯门鳄、奥利诺科鳄等一大堆天敌对它们垂涎！

---

# 扬子鳄：“龙”会发出猪叫？

之前我们说五爪金龙（圆鼻巨蜥）像中国传说中的龙。其实，一些鳄鱼也有“龙之姿”，比如生活在中国长江流域的珍稀鳄类——扬子鳄。

在中国历史上，古人曾称扬子鳄为鼍龙、猪婆龙。扬子鳄被人们当成龙的一个重要原因是它们的叫声。先秦古籍《山海经》，就记载了一种长着龙头的“雷神”，当鼓起肚子时，它们能发出雷鸣般的吼叫，这雷神的原型很可能就是扬子鳄。后来，有些人觉得扬子鳄的叫声有点儿像猪叫，又戏称其为“猪婆龙”。其实，扬子鳄是没有声带的，但它们可以利用腭帆和喉褶，以及鼻瓣等器官发出两种不同的叫声，听起来既像敲鼓，又像雷鸣，更有点儿像猪叫！

---

# 能吃素的鳄鱼——宽吻凯门鳄

在你的印象中，鳄鱼应该是个地道的肉食主义者吧？生活在南美洲东部、中部淡水沼泽和红树林里的宽吻凯门鳄，却做出了一些在鳄鱼家族看来“大逆不道”的怪异行为。

宽吻凯门鳄身体最大的特征是有宽宽的嘴巴，这种嘴巴特别适合用来撕扯沼泽中茂密的植物，而它们居然有时真的会在寻找食物的过程中吞下一些植物！更令人惊讶的是，有些人拍摄的照片中显示，宽吻凯门鳄还会在没有任何外部刺激的情况下，吞下南美洲当地一种叫作“羽裂蔓绿绒”的果实！有科学家研究认为，宽吻凯门鳄这种行为能够帮助植物传播种子，而宽吻凯门鳄很可能是能吃点素的杂食性动物。

---

# 可怕的“食人鱼”——巴拉圭凯门鳄

同样生活在南美洲的巴拉圭凯门鳄，拥有惊人的生长速度。巴拉圭凯门鳄的幼体，不算尾巴的平均长度只有12厘米左右，《美国国家地理》评价“它们看上去只不过像漂浮在巴西偏远内陆地区湖边灯芯草丛中的被风吹动的种子……”

巴拉圭凯门鳄到了5岁的时候，不算尾巴就已经能长到50厘米左右，而到了15岁生长基本完成的时候，可以长到80厘米至1米多。想象一下，你从5岁到15岁，身高能增长一倍吗？

巴拉圭凯门鳄还有一个绰号叫“食人鱼凯门鳄”。它们下颚的一些牙齿可以穿过上颚上的洞，使得它们的牙齿十分突出，被誉为鳄鱼中的“食人鱼”。

---

# 长得黑黢黢的黑鳄

前面我们说，扬子鳄在中国古代被称为“鼍龙”，而在大洋彼岸的美洲大陆，还有很多不一样的“鼍”，这其中就包括被誉为“战五渣”的凯门鳄，它们构成了鳄鱼大家族中的小家族——鼍类。

不管是扬子鳄还是凯门鳄，似乎身材都相对娇小，那么，鼍类里就没有一个“扛把子”吗？见识一下黑鳄吧，它们又叫“亚马孙鼍”“黑凯门鳄”，是鼍类中个头儿最大的，平均体长可达3～4米，最大的个体纪录超过5米，甚至可以把亚马孙雨林中让人闻风丧胆的绿水蚺撕成碎片！在鳄类家族中，黑鳄的实力是仅排在湾鳄、尼罗鳄、美洲鳄之后的，和奥利诺科鳄在伯仲之间。不过，它们虽然名字里也有“凯门”，但在分类上并不属于凯门鳄，而是独立的黑凯门鳄。

---

# 钝吻古鳄：我很小，但我胆大

钝吻古鳄是一种生活在南美洲河滨森林和湖、河畔的洪溢林中的袖珍鳄鱼，成年体长在1.2～1.6米，甚至最大的钝吻古鳄也只有1.7米左右，是世界上最小的鳄鱼之一。成年钝吻古鳄的体重在6～7千克，只相当于湾鳄体重的百分之一，因此也被评为“世界上最轻的鳄鱼”。

别看钝吻古鳄的身形娇小，它们可不怕事，它们的背侧、腹侧都有一层厚厚的鳞甲，如同古代士兵的铠甲能提供很好的防护，因此，当面对体形大的捕食者的威胁时，它们也并不那么惧怕。

相比于其他鼍类，钝吻古鳄更能忍受低温，在干旱的时候，它们会主动出击，寻找水源。

---

# 头戴“王冠”的世界第二——锥吻古鳄

小小的钝吻古鳄，在巨物横行的鳄鱼王国显得很可怜？别担心，它们还有同病相怜的好朋友，这就是世界上第二小的鳄鱼——锥吻古鳄。

成年锥吻古鳄体长1.4米～2.3米，比钝吻古鳄只大一点儿。

“锥”和“钝”？这对“难兄难弟”有标志性的区别：钝吻古鳄的脑袋比较短，下巴很平，整体看起来很“敦厚”；锥吻古鳄的脑袋则很修长，给人一种锋芒毕露的感觉。

和钝吻古鳄一样，锥吻古鳄的皮肤硬厚得出奇，很不适合被人类“利用”，因此幸免于人类的大肆捕杀。

有趣的是，锥吻古鳄在刚出生的时候，脑袋上会有一个金色的斑块，看上去就像王冠一样，因此也被叫作“加冕鳄鱼”，不过随着它们渐渐长大，这个“王冠”会消失。

---

# 几乎被屠戮殆尽的庞然大物——奥利诺科鳄

锥吻古鳄免遭捕杀，同样生活在南美洲的奥利诺科鳄，可就没那么幸运了。

相比于锥吻古鳄和钝吻古鳄两个“小家伙”，奥利诺科鳄可算是庞然大物，从20世纪80年代中期至90年代初期，野外被观察到的雌性奥利诺科鳄的平均体长就超过了3米，成年的雄性奥利诺科鳄甚至可以长到6米。鉴于其庞大的体形，奥利诺科鳄被认为是能与黑鳄并列的美洲鳄鱼之王。不仅体形大，奥利诺科鳄的寿命也很长，可以达到80岁。不过，由于奥利诺科鳄的皮备受市场青睐，为了得到它们的皮制作皮包、皮带、皮鞋等，人类对其进行了大规模的捕杀。20世纪60年代，分布在委内瑞拉、哥伦比亚等地的奥利诺科鳄几乎被屠戮殆尽。

---

# 澳洲淡水鳄，快就一个字

说起鳄鱼，很多人会瑟瑟发抖，觉得它们生性凶猛，还常听说它们会吃人。但这有些误会鳄鱼了，在全球20多种鳄鱼中，有些是没杀人和吃人的本事的。比如，生活在澳大利亚的澳洲淡水鳄。和同样在澳大利亚有分布的湾鳄相比，澳洲淡水鳄小得可怜，一般个体不到2米，在个头儿上也就只能欺负一下前面讲过的锥吻古鳄和钝吻古鳄。而且，要吃人，除了体形要足够大，还要拥有强有力的咬合力，你看澳洲淡水鳄，像极了远在大洋彼岸印度的恒河鳄，细长的嘴巴只能做渔民了。

不过，你也不要以为澳洲淡水鳄完全纯良无害，因为在澳大利亚当地有不少澳洲淡水鳄咬人致伤的记录。澳洲淡水鳄体态轻盈、速度奇快，在水里捕锯鳐、硬杠巨蜥，蹿出水面抓空中飞行的狐蝠，都是它们的拿手之举。

---

# 跟美洲豹“有仇”的危地马拉鳄

危地马拉鳄又称墨西哥鳄，生活在中美洲地区，是当地的一霸。然而，在这片地区，还有一霸就是大名鼎鼎的美洲豹。

俗话说“一山不容二虎”，危地马拉鳄和美洲豹时不时地发生冲突。有人曾目击，危地马拉鳄和美洲豹争夺中美貘的尸体，危地马拉鳄率先发难，试图将中美貘的尸体拖入河中，美洲豹毫无作为。也有人观察到，在墨西哥的一个湖泊中，危地马拉鳄成功杀死了一头美洲豹。

但也有人发现，在离危地马拉鳄的巢穴不到一百米的地方，有一具危地马拉鳄的尸体，而在危地马拉鳄巢穴附近，发现了属于美洲豹拖拽猎物的痕迹，而这个痕迹正好延伸到危地马拉鳄的尸体下！因此，目击者推测，这是美洲豹对危地马拉鳄发动的一次致命袭击！

# 人鳄表演里的暹罗鳄

估计你曾在一些视频里看到，在一些东南亚国家，有些不要命的表演者将自己的手或脑袋伸进巨大的鳄鱼嘴里，看得人胆战心惊，这就是不被提倡的人鳄表演。人鳄表演不仅使表演的人处于高危中，也是对鳄鱼的禁锢。

在这种人鳄表演中被迫充当演员的鳄鱼，主要是分布在东南亚地区的暹罗鳄，也叫“泰国鳄”。这家伙可不是什么善类，暹罗鳄是一种大型鳄鱼，雄性体长可达3～4米，性情凶猛。当感觉自己受到威胁的时候，暹罗鳄会主动攻击人。当人在暹罗鳄生活的水体旁边活动的时候，暹罗鳄发动攻击的可能性也极高。所以，这种人鳄表演行为无异于玩火，翻车的惨剧也并不鲜见。

# 假装自己是毒蛇——加州山王蛇

提到蛇，一般人们对它的印象是：会咬人、很凶、很可怕。但你不知道吧，世界上也有温顺的蛇，这就是生活在美国加利福尼亚州的加州山王蛇。它们在蛇类大家族里不仅算温顺的，而且长得很漂亮。它们的皮肤由红、白、黑等颜色组成，远远看上去像极了用好几种颜色的彩线编织成的艺术品。

加州山王蛇没有毒，但是，有一种和它颜色很近似的家伙叫“珊瑚蛇”，可是凶猛的毒蛇！和毒蛇珊瑚蛇长得像，反而成了加州山王蛇的“护身符”。你想呀，它们又温顺，又没毒，在大自然里很容易被欺负，但是想欺负它们的动物一看：咦，这家伙是不是那种剧毒的珊瑚蛇？就吓得不敢轻易靠近它们了！

# 爱吃蛇的蛇——加州王蛇

加州王蛇没有毒，为什么敢称“王”？啮齿动物、鸟类、两栖动物、其他种类的蛇……甚至像响尾蛇这样的毒蛇，都在加州王蛇的“食谱”上，它们基本见什么吃什么。加州王蛇怎么敢吃有毒的响尾蛇？它们不怕被毒死吗？那是因为加州王蛇对响尾蛇的毒液有天生的免疫力！厉害吧？可能也是因为这个，它们的名字中带一个“王”字。

加州王蛇爱吃蛇的习惯，让一些养它们的人们很头疼，于是人们就总把它们单独关在一个爬箱里，免得它们吃别的蛇。加州王蛇猎食时，会在缠住猎物后，通过收缩自己的身体把猎物绞死，它们缠住猎物时的压力比人的血压还高！所以，很难有猎物能在它们的“绞杀”下活下来。

# 爱挑食的“小不点儿”——球蟒

球蟒生活在非洲的大草原、大森林里。你印象中的蟒蛇是不是特别大？但球蟒是蟒蛇中的“小不点儿”，最大的身长也只有1米多。它们为什么叫球蟒呢？是喜欢玩球吗？其实，在遇到危险的时候，球蟒会把身体蜷缩成一个球的形状，然后把自己的脑袋、脖子等容易受到攻击的部位藏在这个“球”的中心，这就是它们名字的由来。

球蟒有点儿懒，不喜欢自己挖洞，总是躲在别的动物挖好的洞里。夏天的时候，球蟒可能会呼呼大睡。它们是爱挑食的主儿，喜欢吃大点儿的猎物，要是碰到的猎物不满意，球蟒一般就不会吃。被人工圈养的球蟒有时还会选择“绝食”，甚至会把自己饿死！

球蟒在尼日利亚的东南部特别受当地人尊敬，当地人认为它们是地球的象征。遇到球蟒时，人们会把它们送回远离人群的安全地带；遇到死了的球蟒，人们还会为它们举行葬礼。

# 恐怖入侵者——缅甸蟒

缅甸蟒原本生活在东南亚的丛林、沼泽、山麓及河谷地带，是世界上最大的蛇类之一。成年缅甸蟒长约5米，有些甚至可以达到6米长。缅甸蟒满嘴都是锋利的牙齿，身上的肌肉特别发达，可以紧紧缠住猎物，将其勒死，然后整个吞下。作为顶级捕食者，缅甸蟒在自然界中鲜有对手。

然而，当缅甸蟒入侵其他国家后，它们会给当地的生态系统带来严重影响，给动物们带来灭顶之灾。例如：在缅甸蟒入侵美国后，佛罗里达州的兔子、浣熊、狐狸等动物的数量大大减少，甚至连郊狼和美洲狮这样的“狠角色”也受到了缅甸蟒的威胁。缅甸蟒还会和美国短吻鳄打起来，二者之间的胜负难以预料，有时是短吻鳄捕食缅甸蟒，有时则是缅甸蟒反杀短吻鳄。

不过对于我国来说，缅甸蟒并非入侵物种，而是本土原生物种，并且已被列为国家二级保护动物，受到法律的保护。

# 华丽的地毯——丛林地毯蟒

丛林地毯蟒是生活在澳大利亚昆士兰热带雨林中的蟒蛇。看，它们长得多么华丽：鳞片是黑天鹅绒毛一样的黑色，间以鲜艳的黄色，看上去就像精美的地毯！但它们的华丽外表并不是“摆设”，这种黑黄搭配的颜色，有利于它们在斑驳的阳光下更好地伪装自己，而且，较深的颜色让它们待在阳光下时能有效地吸收热量，更快地让身体暖和起来。

丛林地毯蟒是蟒蛇家族中的“小个子”，通常体长不会超过2米，而且很细，这种身形在大自然中很容易被当成猎物。因此，丛林地毯蟒从小就有很强的防御意识，一旦感受到威胁就会发动攻击！丛林地毯蟒的“饭量”不大，有时一年只进食10次，真是“一顿管饱”啊！

# “四处为家”的玫瑰蚺

你可能觉得，这么好看的蛇，一定很娇气吧？不！从美国加利福尼亚州圣迭戈的沿海山脉，向北到莫哈维沙漠，向东到索诺兰沙漠，都可以看见美丽的玫瑰蚺。它们“四处为家”，可以在各类不同的环境中生活，比如在干旱的灌木丛中，在沙质的平原中，在岩石斜坡上，在灌木丛覆盖的山麓中，都有它们的身影。

如果一个地方有水，比如溪流中、峡谷的底部等，那玫瑰蚺更高兴了！玫瑰蚺小时候长度只有20～30厘米，即使长大以后，一般长度也不会超过90厘米，还不如老爷爷拄着的拐杖长。玫瑰蚺的鳞片滑溜溜的，尾巴的形状就像工人叔叔用的锥子。

# 阳光下的彩虹仙子——哥伦比亚虹蚺

人们所知的哥伦比亚虹蚺明明是棕色的，为什么叫“虹蚺”呢？它们身上有彩虹吗？其实如果你在明媚的阳光下观察哥伦比亚虹蚺，就会发现它们的鳞片会反射出迷人的彩虹般的光泽。

哥伦比亚虹蚺是游泳高手，在水里游得飞快；但同时它们还是爬树高手，在地面上遇到可怕的敌人时，它们就会快速地爬到树上，躲避敌人。哥伦比亚虹蚺年轻的时候待在树上的时间比较多，随着年龄的增长，它们待在陆地上的时间会越来越长。

哥伦比亚虹蚺有个秘密，它们到了晚上才出来活动，而且在晚上还会变色呢。在黑黢黢的夜幕下，它们的身体侧面和肚子会变成皎洁的银白色。

# 身上有“马鞍”的红尾蚺

红尾蚺的个头儿大小差别很大，图片里的红尾蚺不算大的。有两条生活在南美洲苏里南的红尾蚺，身长都超过了4米，相当于2～3头成年猪连起来那么长！

红尾蚺的尾就像一只灵活、有力的“大手”，能帮它们抓紧东西。

有些地方的红尾蚺脾气特别不好，一言不合就生气，发出“咝咝”声，然后猛咬对方，但有些地方的红尾蚺性格就很温柔。

红尾蚺最爱吃老鼠，但偶尔会换换口味，抓一些大蜥蜴，甚至是猫科动物来吃。

红尾蚺身上有红棕色的“马鞍”图案，这种图案在它们的尾端更明显，所以才有了“红尾蚺”这个名字。

# 陆地毒蛇之最——内陆太攀蛇

内陆太攀蛇只生活在澳大利亚中东部的干旱地区，它们被誉为陆地毒蛇中的“冠军”。

它们的毒性到底有多强呢？内陆太攀蛇一次释放的毒液，就足以让至少100个成年人丧命！如果被内陆太攀蛇咬伤之后不及时去治疗，那么很可能在30～45分钟内死亡！更可怕的是，内陆太攀蛇攻击敌人的速度极快，而且极精准。

内陆太攀蛇在19世纪末首次被人类发现，可是接下来的数十年间，人们几乎在野外碰不到它们，以至于人们一度认为它们只是一种传说中的邪恶毒蛇。直到90多年后，人类才再次发现了这种令人害怕的毒蛇。事实上，内陆太攀蛇性情羞怯，喜欢躲藏在洞穴里，一般不会对人发起攻击。

# 响尾蛇：尾上有“警报器”

响尾蛇的尾之所以会响，是因为它们尾的末端有“响环”，当它们受到威胁时，“响环”就会振动起来，发出声音，好像在说：“不要靠近啦，再靠近我会要你好看！”为什么响尾蛇的尾上有这种有趣的“警报器”呢？原来，响尾蛇每蜕皮一次，尾上就会留下一个空心的“尾节”，响尾蛇一年到头要蜕皮好多次，所以在尾上的空心“尾节”越来越多，就串成了“响环”。看，这条红菱斑响尾蛇的“响环”振动起来了，虽然它的毒性在响尾蛇家族里算比较弱的，但被咬上一口也很危险。

响尾蛇爱吃兔子之类的小动物，它们制服猎物的法宝就是毒液，它们的毒素会让猎物很快麻痹或死亡。如果猎物被响尾蛇咬了一口还有力气逃走的话，响尾蛇就会悄悄地跟踪猎物，等猎物毒发身亡，再去享用大餐！

# 夸张的“表情包”——食鱼蝮

如果要在蛇界评选“遇到危险时样子最夸张”的蛇，那食鱼蝮肯定是有力的竞争者。

食鱼蝮感到威胁时，会把身体盘绕起来，把大脑袋往后一仰，然后张大嘴巴，惨白的嘴里露出可怕的毒牙！它们有时还会振动自己的尾，发出“咝咝”的警报声，脖子和身体前部扭成一个大大的“S”形，仿佛预示着死亡之神的降临！最搞笑的是，它们甚至还会“放屁”，就是从肛门附近的腺体中喷出有强烈刺激性的分泌物！

科学家测试过食鱼蝮咬人的概率，实验结果表明，45条食鱼蝮中，有将近一半在遇到危险时会想要逃跑，更多的食鱼腹只有在人用手把它们捡起来时，它们才会咬人！

食鱼蝮是世界上少有的半水生毒蛇。正如它们的名字，食鱼蝮最爱吃鱼，另外也会吃青蛙之类的小动物。不过它们有时会“没出息”地被鱼的鳍棘卡到嗓子。

# 小岛上的“蓝蛇”——海岛竹叶青

稀奇！世界上居然有蓝色的蛇！它们就是生活在东南亚一些海岛上的海岛竹叶青。

其实海岛竹叶青常见的颜色是绿色，只有很少的一部分是蓝色或者黄色。蓝色的海岛竹叶青分布的最高海拔可达1200米，它们平时待在树上，到了交配的季节，或是一场哗啦啦的大雨之后才会下树。

有人问，该不会这个蓝色的海岛竹叶青是临时变了颜色的绿色海岛竹叶青，就像变色龙那样？澳大利亚博物馆的专家解释说：“蓝色海岛竹叶青不变色，它们本来就是蓝色的，它们一生都是蓝色的。”有趣的是，生活在澳大利亚北部的“绿树蟒”会在怀孕的时候临时变成蓝色！

# 红肚兜，黑披风——红腹伊澳蛇

这条蛇真有趣，上半身是有光泽的黑色，下半身是暗暗的红色，就像里面穿着红肚兜，外面披着黑披风的奇怪家伙。哈哈，它是澳大利亚东部最常见的蛇类之一——红腹伊澳蛇。

有的红腹伊澳蛇的肚子还是粉红色的呢！虽然它们也是毒蛇，但是在“毒王横行”的澳大利亚，它们的毒性简直不值一提！不过可别招惹它们，它们的毒液可以让人丧失嗅觉，你就闻不到它们身上的味道了。你没听错，红腹伊澳蛇身上有强烈的气味，一些科学家正是靠着闻气味找到它们的！

红腹伊澳蛇喜欢待在哺乳动物挖的洞里、草丛里、水里。它们在水里的姿势太搞笑啦——笔直地漂浮在水面，一动不动，就像一根根棍子……

# 欺负老实“蛇”，后果很严重——蓝腹伊澳蛇

在澳大利亚，红腹伊澳蛇还有个亲密的朋友——蓝腹伊澳蛇。它们和红腹伊澳蛇的一个很明显区别就是肚子的颜色，蓝腹伊澳蛇的腹部是灰色或蓝灰色的。蓝腹伊澳蛇喜欢在开阔的草地、草原生活。它们的性格有些害羞，但如果你把它们惹怒，比如踩到了它们、用棍子戳它们，它们可不会任你欺负。它们会咬你，而且一旦咬上你，就不会轻易松口，还会在你的伤口上反复嚼！蓝腹伊澳蛇的毒液可比红腹伊澳蛇厉害，被咬伤的人要马上去看医生！

# 小心！它有毒——非洲树蛇

这条张着大嘴吓人的蛇叫“非洲树蛇”，它是非洲撒哈拉沙漠以南的特有蛇类。非洲树蛇属于游蛇类，这类蛇一般没有毒液，但非洲树蛇是少数能用毒的游蛇。被非洲树蛇咬伤之后，毒性发作是很缓慢的，所以“受害者”刚中毒时，还以为自己没事呢。1957年，美国爬行动物学家卡尔·施密特被一条幼年的非洲树蛇咬伤了，因为低估了蛇的毒性，没有做任何救治，几个小时后，开始出现恶心、发烧等症状，过了一天后，他不幸身亡。

非洲树蛇进食的时候就像好多天没吃东西的“饿狼”，吃相太难看了。它们可以把嘴张到170度，然后一口把猎物吞掉。170度是什么概念？相当于上下颌快呈一条直线了！

# 令人恐惧的黑曼巴蛇

你如果问一个生活在撒哈拉沙漠以南的人，他的噩梦是什么，也许你会得到一个答案：黑曼巴蛇！

黑曼巴蛇身长可达4米以上，是仅次于眼镜王蛇的世界第二长毒蛇。它们小时候白白的，长大后身体就变成灰色或褐色了，攻击敌人时张开嘴巴露出黑黢黢的口腔，让人不寒而栗。它们能以每小时16千米的速度追击敌人，几乎可以赶得上自行车快速骑行的速度！如果哪个倒霉蛋被追上了，等着他的就是黑曼巴蛇可怕的毒液。

黑曼巴蛇的毒液可以让人在10分钟以内毒性发作！1957年至1979年，南非记录过2500多例毒蛇咬人事件，其中有70多例是黑曼巴蛇造成的！在这70多例咬人事件中，有高达21例致人死亡！不过，黑曼巴蛇本身是不会主动攻击人的。

# 分辨靠歌谣——东部珊瑚蛇

这家伙好像在哪里见过？很像前面讲的无毒加州山王蛇？小心！它可是生活在美国东南部的剧毒蛇——东部珊瑚蛇！仔细看看，它和加州山王蛇还真不一样！

生活在当地的居民很头疼，为了分辨它们，编了一首歌谣：红配黑，不慌忙；红配黄，杀人狂；红配黑，好朋友；红配黄，把命丢！也就是说，你别管这蛇到底有几种颜色的“环”，你就看它们身上的红环有没有紧挨着黄环。如果有，赶紧跑，那就是剧毒的东部珊瑚蛇（适用于北美珊瑚蛇）。你看下面的图，这条蛇身上的红环和黄环是不是挨着啦？红配黄，杀人狂！赶紧跑！你再往前翻翻，看看加州山王蛇，身上是不是根本没有黄环，而且红环挨着黑环？红配黑，不慌忙！它就是无毒的王蛇类！

# 训练有素的“狙击手”——莫桑比克喷毒眼镜蛇

莫桑比克喷毒眼镜蛇的脾气暴躁，当准备攻击你的时候，它们会先调整姿势，把三分之二的身体从地面上仰起来，然后张开嘴巴，“噗”的一下，毒液就从嘴里喷出来，可以喷到2～3米远的地方！而且说喷哪里就喷哪里，“枪法”就像训练有素的狙击手一样准！

每当需要用毒液的时候，莫桑比克喷毒眼镜蛇就会收缩毒囊附近的肌肉，用挤压的力量把毒液从毒囊中挤出，毒液就会流进空心的毒牙里。当毒液离开毒牙的一瞬间，莫桑比克喷毒眼镜蛇的肺里就会冲出一股气流，把毒液变成很小的毒液滴，然后射向敌人。

# 垃圾堆里的“精美丝袜”——东部拟眼镜蛇

在澳大利亚东部、中部和新几内亚南部干燥的桉树林中，在沿海地区的荒地里，在热带草原的林地、干旱的灌木丛，甚至是农田和偶尔洪水泛滥的地方，你都可以看到东部拟眼镜蛇。别惊讶！你甚至能在郊区、农场看到它们，或许它们正躲在建筑物后面、大石头下面，甚至还可能在垃圾堆里……

别看它们“无孔不入”，但别担心，它们不会主动伏击人，受到惊扰后才会变得危险，是澳大利亚第二毒的蛇（仅次于内陆太攀蛇）和伤人最多的毒蛇。

东部拟眼镜蛇最引人注目的是它们身体表面网纹般的图案，动物学家在1846年收集它们的标本时被这些美丽的图案震撼到了，感叹道：“哇，这真像精美的丝袜！”

# 蓝灰扁尾海蛇：我要吃鳗鱼

蓝灰扁尾海蛇生活在广阔的太平洋、印度洋中。和许多“能不上岸就不上岸”的海蛇不一样，蓝灰扁尾海蛇是半水生的，成年的蓝灰扁尾海蛇大概一半时间在海洋里，一半时间在陆地上，虽然它们在陆地上的移动速度只有在水里的五分之一。因此，人类遇到它们的机会很多。

蓝灰扁尾海蛇的毒液有很强的神经毒素，它们用毒液捕杀最爱的美味之一——海鳗。由于总被蓝灰扁尾海蛇捕食，为了活下去，一些海鳗甚至对蓝灰扁尾海蛇的毒液产生了抵抗力。比如，新几内亚附近海域的裸胸鳝，在被注射了一定剂量的蓝灰扁尾海蛇毒液之后，居然毫发无伤。

# 会装死的“小猪”——猪鼻蛇

猪鼻蛇的鼻子上翘，就像装了个猪鼻子，这种长相看起来是挺搞笑的。但“猪鼻子”并不是摆设，有的科学家认为，这样的鼻子能帮助猪鼻蛇在沙土里高效地挖掘。

不仅长得搞笑，它们的应敌策略也搞笑。猪鼻蛇会试图吓跑敌人，它们的脖子能像眼镜蛇那样膨胀，变得扁平，并发出嘶嘶声来警告敌人。猪鼻蛇平时特别不愿意“动口”，但必要时它们也会“假装”攻击敌人。如果这都不能吓跑对方，猪鼻蛇就会装死——翻身仰卧，张着大嘴，吐出舌头，还从泄殖腔里散发出难闻的气味。

如果这时候敌人还不罢休，把它们摆弄到正常的姿态，它们就会再次翻身仰卧，让敌人坚定地相信它们已经死了……它们可真是有一百个心眼子啊！

# 橡皮蚺：猜猜哪边才是我真正的脑袋

说到蚺类，你的脑海里首先会浮现的是那些体形巨大、拥有强绞杀能力的超级大蛇吧？但如果我告诉你右图中的蛇也是种蚺，你会不会吃惊？

生活在非洲中部地区的橡皮蚺可以说是蚺类中的袖珍蚺，一般长度不会超过1米。而且，这家伙虽然缺乏一般蚺类可怕的上颚齿，却拥有“铁头”——坚固的前额叶头骨。有“铁头”自然要施展“铁头功”了，橡皮蚺特别善于用坚硬的头部挖掘地洞，还会闯进一些鼠类的地洞里找寻猎物。也许你有点儿蒙——橡皮蚺的头在哪儿呀，怎么看着两头儿都一样？橡皮蚺的脑袋和尾都是圆钝形的，特别容易迷惑敌人，有时候它们还会把尾抬起来模仿移动的脑袋。实际上，有棕红色的小圆眼睛的一头儿，就是它们真正的脑袋！

# “腾云驾雾”的金花蛇

有很多人把蛇看作传说中的龙，传说中的龙能腾云驾雾、乘风破浪，现实中有许多蛇能在水里游泳，那么有没有可以“腾云驾雾”（会飞）的蛇呢？世界上还真有会“飞”的蛇，它们就是生活在我国南方地区，以及包括印度在内的东南亚等热带或亚热带地区的金花蛇。

这些地区的森林危机四伏，为了让自己更加安全，金花蛇演化出在树木间滑翔的惊世绝技！

它们是如何做到的？原来它们用强有力的肌肉将两侧肋骨外张，这样一来，管道状的身体就变扁了，从而让它们在空中下降的阻力大大提升。此外，科学家还研究了金花蛇“飞行”时的动作，发现它们会进行一种独特的扭动，这种扭动能让它们在降落时不会翻跟头。
