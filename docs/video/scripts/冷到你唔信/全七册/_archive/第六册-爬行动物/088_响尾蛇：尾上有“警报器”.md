# 响尾蛇：尾上有“警报器”

响尾蛇的尾之所以会响，是因为它们尾的末端有“响环”，当它们受到威胁时，“响环”就会振动起来，发出声音，好像在说：“不要靠近啦，再靠近我会要你好看！”为什么响尾蛇的尾上有这种有趣的“警报器”呢？原来，响尾蛇每蜕皮一次，尾上就会留下一个空心的“尾节”，响尾蛇一年到头要蜕皮好多次，所以在尾上的空心“尾节”越来越多，就串成了“响环”。看，这条红菱斑响尾蛇的“响环”振动起来了，虽然它的毒性在响尾蛇家族里算比较弱的，但被咬上一口也很危险。

响尾蛇爱吃兔子之类的小动物，它们制服猎物的法宝就是毒液，它们的毒素会让猎物很快麻痹或死亡。如果猎物被响尾蛇咬了一口还有力气逃走的话，响尾蛇就会悄悄地跟踪猎物，等猎物毒发身亡，再去享用大餐！
