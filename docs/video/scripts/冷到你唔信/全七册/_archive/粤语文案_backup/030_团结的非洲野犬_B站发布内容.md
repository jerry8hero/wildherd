1. 名字由来：因为身体呈黄、白、黑三色，所以又叫三色犬

3. 团结协作：追逐猎物时群体成员通力合作，不断变换"领跑者"，这样每个个体都能节省体力
4. 捕猎成功率：高于80%，比狮子的高将近3倍
