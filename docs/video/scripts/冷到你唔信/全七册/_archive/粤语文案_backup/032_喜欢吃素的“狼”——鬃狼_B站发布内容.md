## 【置顶评论】

🐺 鬃狼实战经验总结

呢期从真正身份到外形特征、从吃素习惯到走路姿势全部讲清楚啰，重点都喺视频度！

大家最关心嘅问题统一回复：

1. 真正身份：唔係狼，而係食肉目犬科鬃狼属动物，平均体长1米，体重约24千克
2. 外形特征：嘴巴比较细长，耳朵很大，毛色以红色和黄褐色为主，长得有点像放大版狐狸
3. 长腿秘密：4条腿大约有1米长，是犬科动物中最长的
4. 吃素习惯：虽然属食肉目，但饮食以素为主，喜欢巴西狼果、番荔枝、番石榴、仙人掌、香蕉等
5. 走路姿势：不像其他犬科动物咁交叉迈步，而係同时向前迈出，看上去似顺拐
