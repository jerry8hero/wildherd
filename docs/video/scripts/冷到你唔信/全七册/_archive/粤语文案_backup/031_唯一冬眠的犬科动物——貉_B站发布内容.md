## 【置顶评论】

🦝 貉实战经验总结

呢期从成语由来到长相特征、从冬眠秘密到杂食习惯全部讲清楚啰，重点都喺视频度！

大家最关心嘅问题统一回复：

1. 成语由来："一丘之貉"指嘅就係呢只犬科动物，分布于欧亚大陆，中国、俄罗斯、日本、蒙古国、朝鲜、越南都有
2. 长相特征：有点像浣熊和狗的结合体，拥有像狗一样的长嘴巴和黑鼻头，眼睛下方的两块黑斑又和浣熊相近
3. 古老分支：貉是犬科动物中较为古老的一个分支
4. 冬眠习惯：生活在俄罗斯等寒冷地区的貉，进入冬季后有冬眠的习惯，是唯一冬眠的犬科动物
5. 杂食习惯：能适应森林、灌丛、甚至农田和城镇等多种环境，谷物、果蔬、肉类来者不拒
