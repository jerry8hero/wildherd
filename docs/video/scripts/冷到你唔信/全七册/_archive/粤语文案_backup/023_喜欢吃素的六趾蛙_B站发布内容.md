## 【置顶评论】

六趾蛙实战经验总结

呢期从食性到第六趾、从偷袭到分布全部讲清楚啰，重点都喺视频度！

大家最关心的问题统一回复：

1. 食性：以金鱼藻为主食，素食比例很高，系少有以素为主嘅蛙类
2. 第六趾真相：脚掌末端一个形状似指头嘅延长，其实唔系真正第六趾，有研究者猜测可能喺水下挖洞穴时起到辅助作用
3. 偶尔开荤：必要时会偷袭落水小型鸟类补充蛋白质，繁殖期雌蛙尤为明显
4. 蝌蚪期：婴儿时期比其他蛙类蝌蚪更能吃肉，昆虫类食物占90%
5. 分布：印度、孟加拉国、斯里兰卡
6. 栖息环境：海拔760米以下淡水环境，有时进入半咸水区域
