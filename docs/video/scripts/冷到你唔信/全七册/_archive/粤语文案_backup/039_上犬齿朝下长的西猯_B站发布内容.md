## 【置顶评论】

🐗 西猯实战经验总结

呢期从身份到獠牙特点、从臭鼬猪名由来到真正用途全部讲清楚啰，重点都喺视频度！

大家最关心嘅问题统一回复：

1. 身份：属西猯科，是猪类动物大家族的成员，美洲大陆有4种
2. 獠牙特点：上獠牙是朝下长的，不像熟悉的野猪那样向上及两侧生长
3. 外形：个头较小，最大的西猯也不过50千克，看上去不那么狰狞恐怖
4. 臭鼬猪名由来：身体表面的腺体中会散发出类似麝香的刺鼻气味
5. 真正用途：唔係用于抵御天敌，而係用作同类间划定领地、相互交流的工具
