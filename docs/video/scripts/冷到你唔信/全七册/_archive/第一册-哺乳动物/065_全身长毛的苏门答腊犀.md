# 全身长毛的苏门答腊犀

苏门答腊犀是世界上体形最小的犀牛，体重为500～1000千克，也是亚洲唯一长两只角的犀牛，因最早发现于苏门答腊岛而得名，该岛也是这个物种最主要的分布区域。

苏门答腊犀是现存犀牛中最古老的，祖先在大约2000万年前就已经出现。它们也是现存犀牛中毛发最多的，全身覆盖着红棕色被毛，和已经灭绝的披毛犀是近亲。

苏门答腊犀还是最善于“交际”的犀牛，它们喜欢用声音来联络同伴。有时，为了表达不同的情感，苏门答腊犀还会发出不同的声响：平日里打招呼会发出“咦”的短促声音；遇到危险时则会发出很尖的声音，类似吹哨；而在表示高兴时，它们的声音则有点儿像露脊鲸的叫声，有人因此形容它们会唱“鲸歌”。
