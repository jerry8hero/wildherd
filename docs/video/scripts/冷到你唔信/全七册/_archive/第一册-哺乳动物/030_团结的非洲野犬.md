# 团结的非洲野犬

非洲野犬是一种生活在撒哈拉沙漠以南地区的犬科动物，主要集中在非洲东部和南部的稀树草原、半荒漠、灌从等植被较为稀疏或低矮的环境中，因身体呈黄、白、黑三色，也叫三色犬。每一只三色犬身上的三种颜色比例和分布位置都不相同，这是它们之间个体识别的标记。

非洲野犬的体重只有17～36千克，爪子和牙齿也不够锋利，身体条件在非洲的食肉动物中并不占优势，但捕猎成功率高于80%，比狮子的高将近3倍。

非洲野犬能取得如此突出的成绩，除了具备犬科动物普遍拥有的良好耐力，最重要的就是团结，在追逐猎物时，群体成员会通力合作，不断变换“领跑者”，这样每个个体都能节省体力，以便最后有力气完成猎食。用餐的时候，非洲野犬会让幼犬先吃。
