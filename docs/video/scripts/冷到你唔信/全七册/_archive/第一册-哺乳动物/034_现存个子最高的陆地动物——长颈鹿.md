# 现存个子最高的陆地动物——长颈鹿

长颈鹿是现存个子最高的陆地动物，成年雄性长颈鹿身高接近6米，雌性长颈鹿也可达5米高。

能够在身高上压过大象，长颈鹿那条长脖子起了决定性作用。它们的脖子最长有2.4米，约是人类脖子长度的20倍。长颈鹿脖子如此长是由其颈椎的长度决定的，虽然和人类一样都有7节颈椎，但长颈鹿每一节的颈椎都很长，其中第二到第六的5节都长达30厘米。

虽然很长，但长颈鹿的脖子并不僵硬，因为它们每一节的颈椎之间都是由圆球形的球窝关节相连的，结构和人类的肩关节相同，非常灵活。长颈鹿之间打架，经常用脖子当鞭子，互相抽打。

长颈鹿的拉丁学名是Giraffa camelopardalis，其中，“camelopardalis”被译为“长着豹纹的骆驼”，也反映了长颈鹿的外在特征。
