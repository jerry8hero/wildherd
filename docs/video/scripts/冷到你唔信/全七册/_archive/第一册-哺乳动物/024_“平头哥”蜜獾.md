# “平头哥”蜜獾

如果评选非洲草原上最好斗的动物，那冠军非蜜獾莫属。因长了一个看上去扁平的头顶而获得“平头哥”绰号的蜜獾，虽然体重不足15千克，脾气却非常暴躁，网上隔三岔五就会出现它们捕猎毒蛇、打退狮子的视频和照片，以至于有了“非洲乱不乱，平头哥说了算”的说法。

实际上，蜜獾的生存范围远不止非洲，西亚的阿拉伯半岛、中亚的哈萨克斯坦、南亚的印度也是它们的生存范围，做到这点得益于它们强大的适应能力，密林、疏林、草原、湿地、半荒漠和荒漠等不同的生态环境，都是它们宜居的场所。

不过，适应能力强不等于战斗力强，更不等于战无不胜。捕猎毒蛇的情况的确有，因为蜜獾可自行分解蛇毒；至于打退狮子，那完全是因为狮子没想真的和它们对打。
