# “偷蛋贼”的二指亲戚——三头鹰龙

三头鹰龙生活在距今约7000万年前晚白垩世的蒙古国，约有2米长、1米高，头顶上长有冠饰，身上披着羽毛，看上去就像一只大号火鸡。三头鹰龙和著名的“偷蛋贼”窃蛋龙同属一科，名字来源于3块在一起的头骨化石，让研究人员想到了阿尔泰神话中的三头鹰。

和大多数窃蛋龙拥有3根“手指”不同，三头鹰龙的前肢末端只能看到两个指头，第三指则退化得只剩下一点点痕迹，这是由于生活习性改变而造成的。不同三头鹰龙个体的化石挨在一起，让古生物学家推测它们很可能群居生活，并且有着类似鸟类那样一起休息的习惯，因为这样不仅能相互取暖，还可以随时预警。

在窃蛋龙家族中，三头鹰龙和来自中国的河源龙亲缘关系最近。古生物学家由此推测，它们的祖先可能来自中国南方。
