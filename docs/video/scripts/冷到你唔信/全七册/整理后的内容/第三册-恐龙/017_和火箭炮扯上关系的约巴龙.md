# 和火箭炮扯上关系的约巴龙

有些武器装备会以动物名来命名，比如著名的“响尾蛇导弹”；有些甚至会以古生物名来命名，比如以“约巴龙”来命名的阿联酋的约巴龙火箭炮。

约巴龙生活在中侏罗世的非洲尼日尔共和国，体重约16吨，高度可达10米。从出土的化石看，约巴龙脖子长，尾巴短，前肢长于后肢，身体呈前高后低的姿态，是一种较为原始的蜥脚类恐龙，名字中 的“约巴”取自当地神话中的物种。

约巴龙的化石发现于撒哈拉沙漠内。古气候及古环境学研究显示，这片后来成为沙漠的地区在当时温暖湿润，拥有丰富的植被和水，为约巴龙的生存提供了充足的保障。约巴龙的化石经常和捕食恐龙非洲猎龙的化石一起出土，这表明它们很可能是后者的猎物。
