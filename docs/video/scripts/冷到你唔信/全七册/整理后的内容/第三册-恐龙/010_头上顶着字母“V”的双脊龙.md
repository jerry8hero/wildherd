# 头上顶着字母“V”的双脊龙

双脊龙是早侏罗世的兽脚类恐龙，生活在距今1.97亿～1.83亿年前的北美地区，拉丁文学名含义为“拥有一对头冠的蜥蜴”，因头顶上的两个冠饰而得名。双脊龙的头冠合在一起呈V字形，由很薄的类似脆骨的组织构成。

就像公鸡的冠子比母鸡的更大一样，雄性双脊龙的头冠也比雌性的更大、更鲜艳。有研究者认为，在求偶期，雄性双脊龙会通过比拼头冠大小来赢得异性的芳心。

成年后的双脊龙体长约6米，体重约500千克。两个前肢上共有6个锋利的弯曲爪子和满嘴锋利的牙齿，显示它们是不折不扣的捕食者。

为了捕获更多的猎物，双脊龙喜欢在水边待着，借着周边植物的掩护，偷袭前来喝水的植食恐龙，有时也会下水抓鱼。
