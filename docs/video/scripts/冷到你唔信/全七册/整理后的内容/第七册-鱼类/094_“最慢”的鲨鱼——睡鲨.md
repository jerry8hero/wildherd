# “最慢”的鲨鱼——睡鲨

很多人喜欢用“龟速”来形容事情进展缓慢。自然界中，比乌龟移动速度慢的动物其实有很多，睡鲨就是其一。

睡鲨是软骨鱼纲角鲨目睡鲨科睡鲨属物种的统称，目前共发现6种，南北两极和太平洋、大西洋、印度洋的冰海海域是它们的地盘。睡鲨不同种类间的体形差距很大，最小的不到1.5米，最大的则有将近8米。

生活在极寒海域，减少能量消耗、保持体温最好的方法就是降低活动速度，睡鲨也的确是这么做的。它们的游泳速度和婴儿爬行的速度差不多。除了减少消耗，及时补充热量也是抗寒的法宝。睡鲨是杂食性动物，科学家曾在它们的胃中发现过海豹、北极熊和驯鹿的尸骸，猜测是食腐的结果。

慢节奏的生活带来的另一个好处就是长寿。以小头睡鲨为例，生活在北极冰海深处的它们由于新陈代谢缓慢，寿命可以达到300多岁，是最长寿的脊椎动物之一。
