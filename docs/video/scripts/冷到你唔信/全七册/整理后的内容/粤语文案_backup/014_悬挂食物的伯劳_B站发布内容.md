## 【简介】

各位，今期讲一种虽然细只但系好打得嘅鸟——伯劳！

伯劳係雀形目伯劳科伯劳属鸟类嘅统称，全世界约30种，中国约13种。伯劳俗称"虎不拉"，一个"虎"字道出咗佢好勇斗狠嘅性格——虽然体长唔超过20厘米，但身体非常粗壮，嘴似猛禽带钩带锯齿，敢同乌鸦、喜鹊、小鸮呢啲大只对手对战，甚至敢搏野猫！伯劳食肉为生，但爪子力量唔够按住猎物拆毛，所以佢哋会搵尖锐树枝将猎物挂稳，然后再用嘴一点点啄食。今期一次过讲清伯劳有几劲、包你笑住听完！
