## 【简介】

各位，今期讲一种似河马嘅大型恐龙——沉龙！

沉龙系发现于非洲尼日尔嘅鸟臀目鸟脚类恐龙，属于禽龙家族，喺距今1.21亿～1.12亿年前早白垩世生活。拉丁文学名含义为"沉重嘅蜥蜴"，呢个名非常贴切——沉龙不单止有6吨重、9米长嘅庞大体型，仲因为腿短，腹部距离地面唔到1米，成个身体有往下沉嘅感觉！沉龙腿短跑唔快，但好处就系重心低唔容易摔倒。遇到捕食恐龙时，沉龙可以快速转动身体确保对方始终喺自己视线范围内，再趁机用前掌拇指上钉子般大爪子反击！同大多数禽龙相比，沉龙嘅脖子好长，大约1.6米。从化石分析，沉龙很可能像今日河马咁，大部分时间都泡喺水中。沉龙脚底长有厚厚肉垫，可以防滑，，令佢哋可以在满是淤泥嘅水底行走。今期一次过讲清沉龙有几得意、包你笑住听完！
