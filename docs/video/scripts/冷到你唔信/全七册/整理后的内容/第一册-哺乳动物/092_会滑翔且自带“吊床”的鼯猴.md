# 会滑翔且自带“吊床”的鼯猴

鼯猴又叫“飞狐猴”，是生活在东南亚森林中的小型哺乳动物，因长得和狐猴有几分相似，又能“飞”，而有了“飞狐猴”这个名字。

飞狐猴的飞并不是像鸟类那样的飞行，而应该叫滑翔。两者的区别是，飞可以从低到高且持续时间较长，而滑翔不能向上且距离较短。飞狐猴的体表有一层“滑翔膜”，从颈部一直延伸到尾巴和四肢，展开后呈正方形，看上去就像披上了一个巨大的斗篷。对于飞狐猴来说，滑翔膜就像一个降落伞，可以在气流的帮助下从空中把它们带到另一个地方，飞狐猴最远可滑翔136米。

对于刚出生的飞狐猴来说，妈妈的滑翔膜折叠起来时刚好和肚皮构成一个育儿袋。当母飞狐猴倒挂在树枝上时，育儿袋会自然撑大，变成吊床。
