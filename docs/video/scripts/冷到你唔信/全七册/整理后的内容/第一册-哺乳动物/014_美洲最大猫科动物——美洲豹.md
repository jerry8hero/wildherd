# 美洲最大猫科动物——美洲豹

美洲豹也叫美洲虎，从北美洲的墨西哥到南美洲的阿根廷都有分布，雄性体重可达158千克，雌性也有100千克，是美洲大陆体形最大的猫科动物，也是全世界仅次于老虎和狮子的第三大猫科动物。

和豹子一样，美洲豹的皮毛上同样布满圆形纹路，但里面多了几个黑点，这是两者的最显著区别。

另一个区别体现在生活习性上。豹子大部分时间都待在树上，美洲豹更喜欢泡在水里。这样做一是因为美洲豹栖息的热带或亚热带密林潮湿闷热，二是因为它们拥有高达1250磅的咬合力，足以对付像凯门鳄和绿水蚺这样的大型爬行动物。

除了身体条件给力，美洲豹也非常聪明。在巴西，曾经发生过美洲豹在浅水区域成功捕获正在捕鱼的水豚的事件。正所谓“螳螂捕蝉，黄雀在后”，水豚在捕鱼时往往会因为全神贯注而察觉不到潜在的危险，美洲豹利用这个时机来捕猎，足以说明其善于动脑。
