# 在臭水中产卵的刘氏泰诺蛙

大多数蛙都会在相对干净的水域产卵，刘氏泰诺蛙却偏偏给孩子选了个臭水坑。

刘氏泰诺蛙是无尾目叉舌蛙科大头蛙属的物种，分布于我国云南部分地区以及东南亚的泰国、缅甸、柬埔寨、老挝境内，热带区域海拔550～850米的山中溪流，以及水流湍急的水沟，是它们平日的栖身之所。刘氏泰诺蛙体长为3.2～3.9厘米，身体背面和腹面分别为棕黄和肉黄色，背部中央的“八”字形皮肤褶皱和四肢上的横纹都是黑色的，喉咙处分布着棕黑色斑点。

每年五六月，刘氏泰诺蛙进入繁殖期。雌蛙会来到河漫滩（洪水期淹没的河床以外的河谷谷底部分，由河流的横向迁移和洪水漫堤的沉积作用形成）上，选择腐殖质层较厚、水质浑浊的小水塘产卵。这些小水塘虽然散发着腐臭的气味，但却富含有机质，有助于卵的发育。
