# “打击乐高手”——蝉

“池塘边的榕树上，知了在声声叫着夏天”，这是歌曲《童年》中的歌词。里面提到的知了就是蝉。

蝉是半翅目蝉科昆虫的统称，全世界大约有2000种，分布在温带和热带地区，它们的别名 “知了”取自鸣叫时发出的声音；其所属分类中的“半翅”并不是真的只有半个翅膀，而是指翅膀基部加厚硬化，导致翅膀一半革质一半膜质的状态。

与人和其他脊椎动物不同，鸣虫（会鸣叫的昆虫）普遍不是用口器发声的，蝉自然也不例外。它们用来发声的器官和肌肉是长在肚子上的。蝉腹部左右两侧各有一个“发音膜”（相当于声带），包裹着能发声的肌肉（鸣肌）。当蝉想要发声时，就会收缩腹部的肌肉，通过振动致使“发音膜”形态改变，从而发出声响。蝉的两块鸣肌可以先后或同时使发音膜产生振动，每秒振动的次数多达224次。
