# 被古人玩坏的恒河鳄

被古人“玩坏”的恒河鳄 被古人“玩坏”的恒河鳄 生活在南亚淡水河流中的恒河鳄的长相很奇怪，身体又细又长。它们的嘴巴也又细又长，就像早点摊上卖油条的叔叔夹油条的夹子！ 早在4000多年前，南亚地区的人们就认识了这种长相奇怪的鳄鱼。考古学家发现的印章、石碑上刻着这样的画面：恒河鳄的周围全都是鱼，它们嘴里还叼着鱼。这说明几千年前，人们就知道这家伙最爱吃鱼了！ 有些人把恒河鳄看成河神、风神的化身。在恒河鳄生活的河流附近，当地人认为恒河鳄可以治很多疾病，直接把恒河鳄作为药物。在尼泊尔，有些人认为雄性恒河鳄拥有神秘的力量，人们经常杀死恒河鳄，然后收集它们那奇怪的鼻子！塔鲁人认为，把恒河鳄的尸体放在田野里燃烧，能驱除害虫，给农民带来丰收。恒河鳄没有可证实的攻击人致死的记录，也缺乏捕食大型动物的能力。这家伙和我们中国的扬子鳄一样是极危物种，一度险些在野外灭绝！
