# 蚯蚓和蜥蜴的合体——五趾双足蚓蜥

蚯蚓和蜥蜴的“合体”——五趾双足蚓蜥 蚯蚓和蜥蜴的“合体”——五趾双足蚓蜥 哇，这个小东西好可爱，身体像蚯蚓一样细长而粉嫩，可是又长着两只“脚”，每只“脚”上有5个萌萌的小脚趾，看起来又有点儿像蜥蜴，这到底是变异的蚯蚓还是变异的蜥蜴啊？ 其实，这家伙既不是蜥蜴也不是蚯蚓，它是一类属于蚓蜥家族的爬行动物——五趾双足蚓蜥。那不还是蜥蜴？之前咱们讲喙头蜥的时候讲过，不是名字里有“蜥”的就是蜥蜴。五趾双足蚓蜥的两条后腿已经退化了，前面两条腿还在，所以称“双足”。它们通体为粉红色，在地下能像蚯蚓那样蠕动身体来前行，所以称 “蚓”蜥。有意思的是，在五趾双足蚓蜥的日常食谱中，既有蚯蚓，又有小蜥蜴……
