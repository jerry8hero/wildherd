# 犰狳环尾蜥：我是带刺的车轮

犰狳环尾蜥：我是带刺的“车轮” 犰狳环尾蜥：我是带刺的“车轮” 这家伙是在模仿犰狳穿了一身带刺的铠甲吗？它怎么做到自己咬自己的尾巴，首尾相连的呢？ 这是生活在非洲南部沙漠中的犰狳环尾蜥，它们的名字正说明了它们身上的“铠甲”就像大洋彼岸美洲的奇特哺乳动物犰狳那样无坚不摧！不过犰狳环尾蜥的“铠甲”要比犰狳的更胜一筹，因为它们的“铠甲”带刺——突出鳞骨的鳞甲硬刺，让捕食者“望而却口”，这样它们就能躲过一劫。 那它们为什么要咬住自己的尾巴，做成“刺球”的样子呢？这是因为犰狳环尾蜥虽然有一身带刺的“铠甲”，但肚子比较柔软，万一暴露给敌人，那可就不得了。所以在遇到敌人或受到惊吓的时候，犰狳环尾蜥会像犰狳那样咬住尾巴不放，变成一个车轮的样子，护住自己的软肚。
