# 夸张的表情包——食鱼蝮

夸张的“表情包”——食鱼蝮 夸张的“表情包”——食鱼蝮 如果要在蛇界评选“遇到危险时样子最夸张”的蛇，那食鱼蝮肯定是有力的竞争者。 食鱼蝮感到威胁时，会把身体盘绕起来，把大脑袋往后一仰，然后张大嘴巴，惨白的嘴里露出可怕的毒牙！它们有时还会振动自己的尾，发出“咝咝”的警报声，脖子和身体前部扭成一个大大的“S”形，仿佛预示着死亡之神的降临！最搞笑的是，它们甚至还会“放屁”，就是从肛门附近的腺体中喷出有强烈刺激性的分泌物！ 科学家测试过食鱼蝮咬人的概率，实验结果表明，45条食鱼蝮中，有将近一半在遇到危险时会想要逃跑，更多的食鱼腹只有在人用手把它们捡起来时，它们才会咬人！ 食鱼蝮是世界上少有的半水生毒蛇。正如它们的名字，食鱼蝮最爱吃鱼，另外也会吃青蛙之类的小动物。不过它们有时会“没出息”地被鱼的鳍棘卡到嗓子。
