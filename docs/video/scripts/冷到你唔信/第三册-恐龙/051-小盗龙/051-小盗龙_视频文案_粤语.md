# 冷到你唔信 - 小盗龙

---

各位兄弟，你哋记唔记得《阿凡达》入面嗰只飞龙？

科学家话你知——1.25亿年前，地球上已经出现咗长4只翅膀嘅滑翔恐龙！佢就係小盗龙，来自中国辽宁嘅热河生物群，体型得噉大——0.45至1米长，体重约1千克。

呢只嘢有几巴閉？佢哋属于兽脚类恐龙中嘅驰龙家族，周身披着类似乌鸦嘅黑色羽毛。但最犀利嘅係——前肢同后肢都长有飞羽！

即係话，小盗龙有4只翼！你冇听错，4只翼喔！

问题嚟喇——既然有咗4只翼，咁小盗龙係唔係可以自由飞翔呢？

答案係：唔得！

点解？因为佢哋嘅翅膀同身体比例唔够大，胸部亦冇鸟类咁发达嘅龙骨突，唔可以从地面直接起飞。所以呢，佢哋必须先爬上树冠等高处，然后扇动翅膀向下俯冲。

有学者认为，呢种飞行模式处于滑翔同飞翔之间——就好似依家玩滑翔伞咁，先爬高再借气流滑翔。
