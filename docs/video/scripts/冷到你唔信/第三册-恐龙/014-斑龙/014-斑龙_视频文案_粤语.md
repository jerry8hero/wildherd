# 冷到你唔信 - 斑龙

斑龙——科学文献记载嘅第一条恐龙！1824年命名，比『恐龙呢个词』仲老！就係呢只龙揭开咗恐龙呢个神秘世界！

对于发现嘅恐龙化石，古生物学家要进行研究、归类、命名等工作——而世界上最早被命名嘅恐龙就系斑龙！

斑龙别名巨齿龙或巨龙——生活在中侏罗世嘅英国，身长7～9米，体重约1.5吨！

拉丁文学名含义为「巨大的蜥蜴」——由英国地质学家威廉·巴克兰于1824年命名！

其实早喺1678年，就已经有人喺无意中发现咗斑龙嘅化石——可惜当时现代生物学研究尚未发端！

呢啲化石就被当成传说中嘅巨人或龙——无人识货！就好似依家啲古董咁，唔知系宝就当破烂喔！（几巴閉！）

斑龙也因此被归入肉食恐龙所在嘅蜥臀目兽脚亚目！
