# 冷到你唔信 - 戟龙

---

你估头顶生满8支角、随时用个头撞过来嘅恐龙有几型？今日讲嘅呢只嘢，唔单止满头都系角，重似个披甲嘅移动城堡——佢就系戟龙，几巴閉！

戟龙又叫刺盾角龙，生活喺距今7650万～7500万年前嘅北美地区，属于角龙科尖角龙亚科。体长大概5米，体重有成6吨——大家可以想象一下，呢个体型就好似一辆放大版嘅SUV咁，企喺你面前压迫感爆棚！

讲到最犀利嘅地方，梗系佢个头上面嗰堆角啦！佢头盾上生咗4～6个角，夸张嘅重可以去到8个！除咗头盾，佢鼻子上面重有一条大概半米长嘅鼻角，睇落去有啲似犀牛——但系犀牛都唔够佢多角，佢简直就系恐龙界嘅刺猬，成身都系防御装备！

呢啲角唔系斋威嘅，长短仲有讲究。正中央嗰只最長，大概有60厘米，两边越往外就越短，最短嗰啲得返几厘米。呢种排列方式令到戟龙睇起身更加霸道——前面啲角可以顶住捕食者，后面啲角保护后颈，一嘢形成立体防御网，撞埋嚟都撞唔入！

而且呢啲角唔单止系武器，重系求偶神器！每年繁殖季节，角越霸气体型越劲嘅雄性就越容易吸引异性——讲真，恐龙都系睇样揾食㗎嘛！

所以呢，戟龙就系满头都系角嘅北美角龙——真正嘅多角战士，几巴閉！
