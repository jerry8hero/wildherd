# 冷到你唔信 - 奇翼龙

---

你知唔知？恐龙唔单止有羽毛，仲有翼膜！奇翼龙——唔用羽毛用翼膜，勐到离谱！（几巴閉！）

讲起会飞嘅脊椎动物，你可能会谂到长羽毛嘅雀仔，或者有翼膜嘅蝙蝠。但奇翼龙偏偏就係混合体——两者特征佢都有！就好似混合玩具咁，边样都要混合下喔！（几巴閉！）

呢只嘢生活在1.6亿年前嘅晚侏罗世，就喺中国辽宁省西部。身长得大约60厘米——虽然体型迷你，但样样嘢都劲得意！

佢满嘴小尖牙，明显系抓昆虫食嘅肉食恐龙。大部分时间都喺树上度嘥，属于擅攀鸟龙家族。身体同尾巴都覆盖住类似鸟类嘅羽毛，但最特别嘅系——前肢上有皮质翼膜，连接住身体两侧。平时收缩埋，但一打开就似披咗件斗篷！就好似依家啲翼装飞行咁酷喔！（几巴閉！）

呢个独特设计由中国古生物学家徐星命名，用拼音拉丁化写成「Yi qi」，意思就系「奇翼」！中国人命名，几犀利喔！（几巴閉！）

所以呢，奇翼龙就系恐龙世界入面嘅蝙蝠——羽毛加翼膜嘅跨界明星！

好喔，今日重点再强调一次：奇翼龙——恐龙中嘅蝙蝠！晚侏罗世，辽宁！身长60厘米！羽毛加翼膜，识滑翔！中国命名「Yi qi」！
