# 几乎被屠戮殆尽的庞然大物——奥利诺科鳄

几乎被屠戮殆尽的庞然大物——奥利诺科鳄 几乎被屠戮殆尽的庞然大物——奥利诺科鳄 锥吻古鳄免遭捕杀，同样生活在南美洲的奥利诺科鳄，可就没那么幸运了。 相比于锥吻古鳄和钝吻古鳄两个“小家伙”，奥利诺科鳄可算是庞然大物，从20世纪80年代中期至90年代初期，野外被观察到的雌性奥利诺科鳄的平均体长就超过了3米，成年的雄性奥利诺科鳄甚至可以长到6米。鉴于其庞大的体形，奥利诺科鳄被认为是能与黑鳄并列的美洲鳄鱼之王。不仅体形大，奥利诺科鳄的寿命也很长，可以达到80岁。不过，由于奥利诺科鳄的皮备受市场青睐，为了得到它们的皮制作皮包、皮带、皮鞋等，人类对其进行了大规模的捕杀。20世纪60年代，分布在委内瑞拉、哥伦比亚等地的奥利诺科鳄几乎被屠戮殆尽。
