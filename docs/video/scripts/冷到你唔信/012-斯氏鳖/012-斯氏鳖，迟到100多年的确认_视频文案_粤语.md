# 冷到你唔信 - 斯氏鳖

各位，今日继续冷到你唔信！

我今日讲只勆到你唔信嘅鳖——斯氏鳖！

你可能会问，斯氏鳖？未听过喔？唔紧要，我哋慢慢吹。

呢只嘢，几劲？身长成米几，重量成70到100千克——就係鳖界入面嘅"巨无霸"！你以为鳖都係细细只蹲喺度咩？呢只就係鳖入面嘅"大只佬"，唔讲以为係细个富士山。

但係呢只嘢劲唔止咁简单。

你知唔知，斯氏鳖喺中国南部、越南北部嘅淡水河度住咗几耐？成几代人啦！但係——1873年西方学者已经为佢科学命名，但係一直以来都被误认为係鼋或者中华鳖。就係因为你未认清我，我就唔存在？呢个就係生物学界嘅"隐形鳖"。

更离谱嘅係，等到21世纪先至确认佢係有效物种——即係迟到咗成100年！100年喔！佢喺度住咗成年，人类就当佢唔存在。呢种待遇，几阴公。

讲到斯氏鳖最惨嘅故仔，就係咁——

本来80年代国家出名单保护野生动物，佢应该上榜㗎！但係因为一直被误认为係鼋或者中华鳖，唔好意思，你唔属于你，你只属于"其他鳖类"。所以呢，就係呢100年间，斯氏鳖就係度静鸡鸡咁做佢嘅"隐形生物"。

但係！2021年终于平反！斯氏鳖正式列入中国《国家重点保护野生动物名录》，而且係一级保护！即係国家层面嘅最高级别㗎！但係呢——

各位，跟住落嚟讲嘅，就係斯氏鳖最催泪嘅事实。

目前已知活喺世上嘅斯氏鳖，可能净係得两只，而且——两只都係雄性。一只喺苏州动物园，另一只可能喺越南河内。两只都係公公嚟㗎！

你想下，100几年前，佢哋嘅祖先喺河度游紧游紧，而家得返两只公公，而且唔同地方，想撐撑对眼都未得。

呢个就係最残酷嘅现实：确认得太迟，唔记得得太耐，而家想保护，都已经太迟。

不过，科学家仲未放弃希望。如果搵到越南嗰只，或者有其他野生种群未被发现，咁就仲有转机嘅。所以苏州动物园嗰只公公，就係目前最重要嘅"独苗"。
