# 冷到你唔信 - 在树叶上产卵的溅水鱼

溅水鱼——树上生仔？但呢只中美洲嘅爱树鱼，就系咁奇葩！




鱼生活在水中，它们中的大多数把卵也产在水中。溅水鱼却“别出心裁”，把卵产在了树上。

溅水鱼在分类上属于脂鲤科，目前已发现50余种，野生种群分布于亚马孙河流域。

从外表看，体长不足10厘米的溅水鱼毫无特色，但它们的产卵方式却非常值得一提。为了防止后代被其他鱼甚至自己吃掉（有些溅水鱼饥饿时会吃掉自己产下的一部分卵），溅水鱼选择在具有黏附性的叶子上产卵。一旦选定目标，雄鱼和雌鱼就会身体紧挨着一起跳...

你知唔知在树叶上产卵的溅水鱼有几劲？

上去，并在跳跃过程中完成交配，将卵产到其他鱼不能到达的叶子上。孵化期间，雄鱼还要经常用尾鳍拍打水面，把水溅到叶子上，以此来防止鱼卵因缺水而干裂，这个工作要一直持续到幼鱼出生。
