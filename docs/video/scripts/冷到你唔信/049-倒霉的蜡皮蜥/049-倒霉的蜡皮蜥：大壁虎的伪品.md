# 倒霉的蜡皮蜥：大壁虎的伪品

倒霉的蜡皮蜥：大壁虎的“伪品” 倒霉的蜡皮蜥：大壁虎的“伪品” 在中国广东、海南一带有这样一种穴居性蜥蜴，广东湛江叫它“沙神”，海南则叫它“坡马”，这就是蜡皮蜥。 蜡皮蜥非常喜欢在开阔的沙质土地中生活，虽然环境没那么优美，但它们长得很让人“赏心悦目”——蜡皮蜥拥有一身漂亮的花纹，又被叫作“蝴蝶蜥”。 但在过去的几十年里，蜡皮蜥被人类大量捕捉，以大壁虎“伪品”的身份入药，导致其种群数量急速减少，2021年2月被列为国家二级保护动物。 在以前的文献中，珠海的淇澳岛是没有蜡皮蜥分布的，但在2023年初，人类惊奇地在珠海淇澳岛发现了它们，这次发现创造了珠海物种的新纪录！
