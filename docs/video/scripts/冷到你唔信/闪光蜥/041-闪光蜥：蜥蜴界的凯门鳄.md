# 闪光蜥：蜥蜴界的凯门鳄

闪光蜥：蜥蜴界的“凯门鳄” 闪光蜥：蜥蜴界的“凯门鳄” 每个人都有自己特别爱吃的东西，蜥蜴界也有这样的吃货——生活在巴西、哥伦比亚、厄瓜多尔、秘鲁和圭亚那等国家的闪光蜥，特别爱吃带壳的东西，什么螃蟹、小龙虾、蛤蜊，甚至是蜗牛也不放过。可问题来了，这些东西那么硬，蜥蜴又不可能像人类一样用灵动的手或合适的工具去剥壳，它们怎么吃呢？看看闪光蜥的“配置”就知道了，它们的头大而粗壮，“下巴”的肌肉十分发达，还长了一嘴又圆又钝的牙齿。与穿透或撕裂皮肉的尖牙不同，这种牙齿可是压碎那些食物外壳的“神器”！ 这些带壳食物里，闪光蜥尤其爱吃蜗牛，它们吃蜗牛的时候，先咬住蜗牛，然后把脑袋一仰，用那粗壮的牙齿把蜗牛“咔”地压碎，吃掉里面的肉，最后吐出蜗牛壳。有些粗暴的闪光蜥甚至连壳都不吐，直接连壳带肉一起吞进肚里！
