# 斑睑虎：夜行侠还是向光者

斑睑虎：夜行侠还是向光者 斑睑虎：夜行侠还是向光者 它们是身上如同穿着“豹纹衫”的斑睑虎——皮肤上有很多像豹纹一样的黑色斑点。有些科学家认为，斑睑虎是个不折不扣的“夜行侠”，因为在2016年以前，尼泊尔从来没发现过斑睑虎，很可能因为它们在晚上出没。有些人不同意，因为他们根据常年饲养斑睑虎的经验，发现斑睑虎在圈养的环境下，可以利用紫外线合成维生素D3！在实际的饲养中，饲养人员也确实常用紫外线为斑睑虎照射。我们都知道，太阳是自然界最主要的紫外线光源，这么说，斑睑虎的生长是要有光吗？斑睑虎其实是不“排斥”太阳的吗？虽然暂未有定论，但有一些人声称，他们在白天日光强烈的沙漠中，曾看到斑睑虎悠闲地晒太阳。
