# 冷到你唔信 - 褶伞蜥

今日继续冷到你唔信！

好，我哋从头讲起。今日吹下水，讲只超级神奇嘅蜥蜴——褶伞蜥！

你知唔知褶伞蜥几把炮？

块膜有几劲？撑开之后简直就係一把"大伞"，阔度可以系身体嘅两三倍！远远望过去，唔知道你嘅仲以為係乜嘢外星生物入侵。

你知唔知？呢把"伞"唔单止大，兼夹颜色劲鲜艳——有橙红、有黄、有紫，就似被人落重料咁！敌人一睇，哇呢个係咩坚嘢？未食得咸鱼唔怕抵仔——但係呢个就係褶伞蜥嘅生存智慧：一撑伞，敌人就脚软。

讲到扮大只，褶伞蜥就係高手。

呢招叫"吓你唔死都吓你一身仔"。敌人一睇，哇呢个唔知係咩怪物，把口大成咁，唔通想一口吞咗我？算喇走先！

呢招叫"不战而屈人之兵"——懂吗？就係你还未郁手，敌人已经跑嗮。

---

但係褶伞蜥最估你唔到嘅，唔单止係把伞，而係——走！

呢只嘢跑起身劲到你唔信！虽然係爬行动物，但係逃跑嗰时，佢会用两条后腿嚟"狂奔"！

几快？ scientists发现，褶伞蜥逃跑嗰时，时速可以去到20几公里！相当于一台摩托车喺你身边掠过！

而且呢个唔係普通嘅跑，而係"双足狂奔"——就似恐龙跑法咁款！用后腿跑，前腿就凌空度摆——型到呢！

你知唔知？呢种跑法有几犀利？就係喺干旱草原同灌木地带，佢可以跑得劲快劲稳——就似你踩住风火轮咁款！

所以褶伞蜥嘅终极生存之道就係：撑伞——张大口——跑！三连招！敌人未郁手，佢已经走嗮。
