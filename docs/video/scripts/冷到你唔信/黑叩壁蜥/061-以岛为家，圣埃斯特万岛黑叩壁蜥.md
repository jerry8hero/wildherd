# 以岛为家，圣埃斯特万岛黑叩壁蜥

以岛为家，圣埃斯特万岛黑叩壁蜥 以岛为家，圣埃斯特万岛黑叩壁蜥 你听说过一座岛的名字与一种蜥蜴有关系吗？在加利福尼亚湾，有座圣埃斯特万岛，这座岛最初被命名为“大蜥蜴的岛屿”，这个名字源于岛上特有的一种大型蜥蜴——圣埃斯特万岛黑叩壁蜥。该岛有很多圣埃斯特万岛黑叩壁蜥，在岛外却丝毫没有这种蜥蜴的踪迹。 很久以前，有人曾把圣埃斯特万岛黑叩壁蜥带到岛外饲养，可这些蜥蜴在离开自己的“故乡”后没能成功地存活下来。圣埃斯特万岛当地的居民，将黑叩壁蜥视作重要的食物，还骄傲地称自己是“大型蜥蜴的人民”。由此可见，圣埃斯特万岛黑叩壁蜥和这座岛屿以及岛上的居民，真的是有斩不断的羁绊。
