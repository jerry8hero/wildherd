## 搜索建议

### 推荐图片网站
1. **Wikimedia Commons** - 免费高清图片，物种照片丰富
2. **iNaturalist** - 野生动物照片
3. **Unsplash** - 高质量免费图片
4. **Pexels** - 免费素材图片
5. **Pixabay** - 免费素材图片

### 搜索技巧
- 使用学名搜索更精准，如 `Chelonia mydas`
- 加上 "HD" 或 "high resolution" 获得高清图
- 加上 "wild" 获得野生环境照片
- 避免使用 "aquarium" 或 "captive" 获得野生个体图
