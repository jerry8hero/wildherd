# 马达加斯加大头侧颈龟：我在这边，亲戚在那边

马达加斯加大头侧颈龟：我在这边，“亲戚”在那边 马达加斯加大头侧颈龟：我在这边，“亲戚”在那边 这个长着大脑袋，有着黑褐色外壳的家伙，是马达加斯加大头侧颈龟，2018年，它被评为“世界上最濒危的龟类”。它不仅珍稀，身世更是离奇，因为它属于南美侧颈龟科这个家族。咦？马达加斯加是非洲的一个岛屿，而南美洲和非洲隔着大洋呢，马达加斯加大头侧颈龟的“亲戚”怎么会在大洋彼岸的南美大陆呢？一些科学家推测，原本马达加斯加岛和南美洲有大陆桥相连，大约在7800万年前的白垩纪晚期，连接南美洲与南极洲和马达加斯加的大陆桥沉没了，马达加斯加大头侧颈龟就成了南美洲以外唯一被发现的南美侧颈龟科家族成员，真是沧海桑田的变化呀！
