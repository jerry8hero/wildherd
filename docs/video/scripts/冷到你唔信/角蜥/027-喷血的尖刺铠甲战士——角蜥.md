# 喷血的尖刺铠甲战士——角蜥

喷血的“尖刺铠甲”战士——角蜥 喷血的“尖刺铠甲”战士——角蜥 哎哟，这浑身长刺、胖乎乎的家伙竟然是一只蜥蜴？没错，这就是主要生活在美洲沙漠的角蜥！它披着一身长满硬刺的“铠甲”，谁要敢来惹它，它一定让来犯者头破血流！ 很多爬行动物都有鳞片，角蜥也是爬行动物，它们的这些硬刺就是爬行动物界一种十分特别的鳞片。 一些角蜥还有一项技能：喷血！它们能让自己头部的血压增高，眼角破裂，喷出鲜血，这样敌人就吓得不敢靠近了。角蜥可以把血喷到1米以外！为什么捕食者一看到血就不敢靠近呢？是晕血吗？有些科学家认为，角蜥平时吃了很多收获蚁，收获蚁体内的一些化学元素在角蜥身体里堆积，让它们的血液带着一股难闻的气味，喷出来会把来犯的犬类、猫科动物“熏走”。
