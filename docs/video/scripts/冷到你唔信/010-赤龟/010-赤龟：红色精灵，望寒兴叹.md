# 赤龟：红色精灵，望寒兴叹

赤龟：红色精灵，望寒兴叹 赤龟：红色精灵，望寒兴叹 赤蠵龟的“赤”是指红色，“蠵”在我国古代有“大龟”的意思。赤蠵龟担得起这个名字吗？赤蠵龟的壳一般是红棕色的，符合“赤”意。它们还是世界上最大的硬壳海龟——有报告显示，有的赤蠵龟能有2米多长，500多千克重！说它们是“蠵”（大龟）一点儿也不夸张。 就是这么巨大的“红色精灵”，“身世”还有点儿可怜呢！在大约300万年前的上新世，我们今天的中美洲浮出海面，把大西洋与印度洋—太平洋之间的海流阻隔了。地球进入冰川循环，海流因为气候变化产生新的流向，冷水上升到好望角附近，把好望角的海水温度大大降低。这对需要迁徙的赤蠵龟是个坏消息，冷水让它们“望寒兴叹”，最终，大西洋的赤蠵龟和太平洋的赤蠵龟被完全隔开。
