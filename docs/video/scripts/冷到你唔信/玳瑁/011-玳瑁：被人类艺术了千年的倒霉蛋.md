# 玳瑁：被人类艺术了千年的倒霉蛋

玳瑁：被人类“艺术”了千年的“倒霉蛋” 玳瑁：被人类“艺术”了千年的“倒霉蛋” 玳瑁在海龟里真的很另类。它们的上颚弯曲成钩状。所以人们也常叫它们“鹰嘴海龟”。它们也是目前人类已知的唯一以海绵为食的爬行动物。再看它们那色彩斑斓的甲壳，不仅你看着养眼，古人也是——玳瑁的甲壳被古人采来进行加工，雕上花纹，做成首饰，象征祥瑞幸福、健康长寿，以“海金”的名声享誉世界。在中国，玳瑁工艺有上千年的历史，唐代时已经炉火纯青。 古人还发现，把玳瑁抓上来，剥离甲壳后再放回大海，玳瑁有时还能再长出龟甲。实际上，龟甲密布神经和血管，剥离角质会对龟甲造成极大的伤害，大多数情况下会导致玳瑁感染死亡。人类对玳瑁的过度需求，已经让它们被列为极危物种和国家一级保护动物了。
