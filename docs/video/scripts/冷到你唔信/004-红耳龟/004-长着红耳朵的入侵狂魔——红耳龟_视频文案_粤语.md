# 冷到你唔信 - 红耳龟

今日继续冷到你唔信！

好，我哋从头讲起。今日吹下水，讲只劲犀利嘅淡水龟——红耳龟！

你可能話，红耳龟？哦，就嗰种宠物店成日见到嘅巴西龟啰！你如果係咁諗，就晒你啦。呢只嘢，虽然名叫"巴西龟"，但係唔係来自巴西，而係美国同墨西哥！而且佢有个劲犀利嘅名——被列入《全球100种最具威胁的外来物种》，劲唔劲？

你知唔知红耳龟个名係点嚟嘅？呢个就劲搞笑——佢对耳仔同附近位置，有红色嘅条纹！就似挂住对红耳环咁样，係佢嘅标志性特征。但係你以为佢净系靠样呃人？错喇！佢哋适应能力劲到呢，本土龟根本唔够打！

讲到红耳龟有几能顶，就係一个"劲"字可以形容。

佢哋係半水栖龟类，咩环境都掂——池塘、湖泊、河流、甚至沼泽地，统统都係佢哋嘅主场。适应能力强到咁上下，新环境？湿湿碎啦！正因为佢咁打得，我哋中国嘅本土龟喺生存竞争入面，完全败下阵嚟。你话几惨。

但係更恐怖嘅係，佢繁殖劲快！一只红耳龟一年可以生几窝蛋，每窝十零只，而且佢哋携带病毒，可以传播沙门氏菌等多种疾病。本土龟一感染，成个族群都麻烦。你话係唔係劲？

你知唔知？红耳龟喺好多国家係犯法嘅！

澳大利亚、欧洲部分国家，贩卖红耳龟係犯法嘅！日本更加在2015年宣布禁止进口！点解？因為佢哋实在太劲喇——适应能力强、繁殖快、带病毒，係外来物种入面嘅"战斗机"。一旦进入新环境，本土物种就好难企理。

所以下次见到红耳龟，记住——佢唔係普通嘅宠物龟，係个劲犀利嘅入侵者！

讲到红耳龟最犀利之处，就梗係佢嘅"全球扩张"——宠物市场嘅超级明星！

话说当年美国宠物市场兴起红耳龟热，商人见到有利可图，就将大量红耳龟当作商品，卖到全世界各个国家。结果点？红耳龟凭借超强适应性，喺好多地方站稳阵脚，慢慢取代本土龟种。

呢个就係"宠物贸易"嘅代价——你以为买只龟好小事，但係一旦佢走失或者被放生，佢就可以摧毁成个生态系统。你话几阴功。

想象下，你喺宠物店买咗只红耳龟，觉得得意就养大咗，后来觉得麻烦就放生。唔知就放生一只"生态炸弹"！佢会抢本土龟嘅食物同栖息地，传播疾病，繁殖又多——呢个就係好心做坏事喇。

各位，跟住落嚟讲嘅，就係红耳龟最犀利嘅入侵能力。

红耳龟之所以成为入侵物种，就係因为佢有三个必杀技：第一，适应能力强，咩环境都生存到；第二，繁殖快，一年几窝蛋；第三，带病毒，传播疾病给本土物种。

有研究显示，喺引入红耳龟的地区，本土龟类数量普遍下降30%以上，有啲地区甚至跌成个位数。呢个就係"外来物种入侵"嘅威力——肉眼睇唔到，但係破坏力劲大。

所以我哋绝对唔可以掉以轻心！如果见到红耳龟，千祈唔好放生，应该联系当地动物园或者野生动物救护机构。

呢个就係大自然最残酷嘅生存法则：劲就係原罪，做得太叻都係问题。
