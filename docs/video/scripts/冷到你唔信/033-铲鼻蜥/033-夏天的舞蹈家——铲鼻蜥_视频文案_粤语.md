# 冷到你唔信 - 铲鼻蜥

各位，今日继续冷到你唔信！

好，我哋从头讲起。今日吹下水，讲只沙漠入面嘅舞蹈家——铲鼻蜥！

你可能話，沙漠咁热，边个得闲喺度跳舞喎？但係铲鼻蜥就唔同，佢真系会跳，而且跳得几落力——不过呢个唔系表演，而係佢嘅续命大法。

你知唔知铲鼻蜥点解叫"铲鼻"？就係因为个鼻似把铲仔——不过呢把"铲"唔系用来整嘢食，而係用来挖沙嘅。佢一嘢就可以成身插入沙入面，即刻消失不见，就似你玩变色龙噉款隐身术。

但係今日重点唔系个鼻，而係佢哋嘅舞姿。

你知唔知？铲鼻蜥热嘅时候会跳"恰恰"！

沙漠嘅沙，晏嗰可以烫到五六十度——想象下你赤脚行喺沙滩度，烫到你嗌救命。铲鼻蜥就精仔，佢哋唔会呆呆哋忍受，而係使出独门绝技——抬脚舞！

具体係咁嘅：佢先抬起左前脚加右后脚，离地大概两厘米——hold住大约十秒——然后再换右前脚加左后脚。交替噉抬，就似跳紧某种奇怪嘅恰恰舞。

你可能会问：抬脚有咩用？重点嚟啦——就系为咗唔畀个脚仔烫熟！

铲鼻蜥靠住条尾保持平衡，一次过抬起两只脚，可以让脚仔暂时离开滚烫嘅沙面，得到短暂降温。就似你喺猛太阳下面，梗係想抬下只脚离地降下温啰——虽然你唔可以真系做到。

呢种舞步唔系几时都跳，只系夏天或者天气真系热到顶唔顺嗰阵先会出绝招。呢个就叫做——热系我哋都会，降温有方法。

讲到铲鼻蜥嘅沙漠生活，就唔可以唔提佢哋嘅速度——虽然个样睇落似呆呆哋，但係佢哋可以每小时移动30公里！就系为咗追捕猎物或者逃避捕食者。毕竟沙漠入面，慢一秒都可能热到瓜老衬。

你知唔知？铲鼻蜥唔单止识跳舞，仲识"水上漂"！

呃你唔信？呢个"水上漂"唔系轻功，而系靠住飞快嘅速度同特殊嘅脚掌结构，佢哋可以短暂噉喺沙面上面滑行——就似你用脚尖点水面噉款，唔知几咁似神仙！

想象下沙漠入面，一只铲鼻蜥高速移动，脚仔抬上抬落，就似跳紧舞——呢个就係沙漠入面最靓嘅舞蹈家啰！

各位，跟住落嚟讲嘅，就係铲鼻蜥最犀利嘅生存技能。

佢唔单止识降温跳舞，仲有一个超级能力——变色！唔同时间、唔同温度，个身颜色会郁下。热嘅时候会比较浅色，cold嗰阵就深色啲。就似你着黑色衫喺沙漠行，热嘞！但係佢哋可以自己变色做凉浸浸色——呢个先叫做真㗎！

铲鼻蜥：谁说我丑？我只系未热身啫。
