# 假装自己是毒蛇——加州山王蛇

假装自己是毒蛇——加州山王蛇 假装自己是毒蛇——加州山王蛇 提到蛇，一般人们对它的印象是：会咬人、很凶、很可怕。但你不知道吧，世界上也有温顺的蛇，这就是生活在美国加利福尼亚州的加州山王蛇。它们在蛇类大家族里不仅算温顺的，而且长得很漂亮。它们的皮肤由红、白、黑等颜色组成，远远看上去像极了用好几种颜色的彩线编织成的艺术品。 加州山王蛇没有毒，但是，有一种和它颜色很近似的家伙叫“珊瑚蛇”，可是凶猛的毒蛇！和毒蛇珊瑚蛇长得像，反而成了加州山王蛇的“护身符”。你想呀，它们又温顺，又没毒，在大自然里很容易被欺负，但是想欺负它们的动物一看：咦，这家伙是不是那种剧毒的珊瑚蛇？就吓得不敢轻易靠近它们了！
