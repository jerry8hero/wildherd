# 史上最憋屈的莫顿水巨蜥：赢了鳄鱼，输给蟾蜍

史上最憋屈的莫顿水巨蜥：赢了鳄鱼，输给蟾蜍 史上最憋屈的莫顿水巨蜥：赢了鳄鱼，输给蟾蜍 在与五爪金龙生活的亚洲隔洋相望的澳大利亚，也有一种水生巨蜥，它们就是生活在澳大利亚北部的莫顿水巨蜥。它们比圆鼻巨蜥更依赖水，晒日光浴都要在水中找块石头或浮木来当床。 说起莫顿水巨蜥，可真是让人哭笑不得，虽然它们的体长一般不超过1米，比不过圆鼻巨蜥，但它们也是极其凶悍的角色，不少鱼类、两栖动物、鸟类、哺乳动物都会成为它们的“盘中餐”，甚至有人拍到它们捕杀年幼的淡水鳄鱼的画面！ 可就是这样的家伙，谁也没想到它们会“输”给不起眼的蟾蜍。根据科学家的调查，自从1935年蔗蟾蜍引入澳大利亚，莫顿水巨蜥的数量大幅减少，被蔗蟾蜍毒死成为莫顿水巨蜥近百年来最常见的死因之一。
