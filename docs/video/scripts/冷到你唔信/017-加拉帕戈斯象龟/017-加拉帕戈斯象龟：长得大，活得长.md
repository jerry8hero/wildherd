# 加拉帕戈斯象龟：长得大，活得长

加拉帕戈斯象龟：长得大，活得长 加拉帕戈斯象龟：长得大，活得长 科幻节目《狂想曲：未来世界物种大揭秘》描述了一亿年以后的地球上会演化出一种比大象要重20多倍、比一般人高4倍的巨龟。 虽然目前地球的龟类家族中没有这么大的家伙，但也有让人“大”开眼界的龟，它们就是现存体形最大的陆龟——加拉帕戈斯象龟。 成年加拉帕戈斯象龟身长可达1.5米，平均体重175千克，相当于一个人类的超级大胖子的体重。不光是体形大得吓人，加拉帕戈斯象龟的寿命也长得吓人。据统计，野生加拉帕戈斯象龟的平均寿命超过100岁，而2006年6月23日在澳大利亚动物园因心脏衰竭去世的一只名叫哈丽的加拉帕戈斯象龟，其寿命居然有170多岁，是当时发现的世界上最长寿的动物。
