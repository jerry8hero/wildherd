# 垃圾堆里的精美丝袜——东部拟眼镜蛇

垃圾堆里的“精美丝袜”——东部拟眼镜蛇 垃圾堆里的“精美丝袜”——东部拟眼镜蛇 在澳大利亚东部、中部和新几内亚南部干燥的桉树林中，在沿海地区的荒地里，在热带草原的林地、干旱的灌木丛，甚至是农田和偶尔洪水泛滥的地方，你都可以看到东部拟眼镜蛇。别惊讶！你甚至能在郊区、农场看到它们，或许它们正躲在建筑物后面、大石头下面，甚至还可能在垃圾堆里…… 别看它们“无孔不入”，但别担心，它们不会主动伏击人，受到惊扰后才会变得危险，是澳大利亚第二毒的蛇（仅次于内陆太攀蛇）和伤人最多的毒蛇。 东部拟眼镜蛇最引人注目的是它们身体表面网纹般的图案，动物学家在1846年收集它们的标本时被这些美丽的图案震撼到了，感叹道：“哇，这真像精美的丝袜！”
