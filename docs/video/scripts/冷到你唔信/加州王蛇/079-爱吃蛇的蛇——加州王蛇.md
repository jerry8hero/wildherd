# 爱吃蛇的蛇——加州王蛇

爱吃蛇的蛇——加州王蛇 爱吃蛇的蛇——加州王蛇 加州王蛇没有毒，为什么敢称“王”？啮齿动物、鸟类、两栖动物、其他种类的蛇……甚至像响尾蛇这样的毒蛇，都在加州王蛇的“食谱”上，它们基本见什么吃什么。加州王蛇怎么敢吃有毒的响尾蛇？它们不怕被毒死吗？那是因为加州王蛇对响尾蛇的毒液有天生的免疫力！厉害吧？可能也是因为这个，它们的名字中带一个“王”字。 加州王蛇爱吃蛇的习惯，让一些养它们的人们很头疼，于是人们就总把它们单独关在一个爬箱里，免得它们吃别的蛇。加州王蛇猎食时，会在缠住猎物后，通过收缩自己的身体把猎物绞死，它们缠住猎物时的压力比人的血压还高！所以，很难有猎物能在它们的“绞杀”下活下来。
