# 评论区置顶

📌 草龟混养安全组合：

✅ 推荐组合：
- 斑马鱼：游速快，草龟抓不到
- 斑马螺：淡水繁殖困难不会爆缸，清理藻类

❌ 不推荐：
- 小型灯鱼：会被吃掉
- 苹果螺：繁殖太快会爆缸
- 小型金鱼：会变零食

⚠️ 重要前提：
- 只适合龟苗和亚成体
- 成年草龟不建议混养
