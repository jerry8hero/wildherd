# 学生党养龟指南，在宿舍也能养好龟

哈喽兄弟们！欢迎回到我的频道！

是不是每次刷到室友的猫猫狗狗，只能默默点赞？宿舍不让养大型宠物，又觉得自己手残养不活？

兄弟萌！是你懂太少！养龟了解一下！省心省力还解压，宿舍巴掌大点地方就能养！这期视频就来说说学生党养龟的正确姿势，手把手教你从零开始！

准备好，我们发车！

很多兄弟觉得养龟是有钱有闲的人才能玩得转的。其实完全不是！养龟比养猫狗省事太多了。

首先，时间成本低。不需要每天遛，没有早晚喂食的压力。龟龟很安静，不会发出噪音影响室友休息，更不用担心被宿舍管理员发现。

其次，情感价值高。学习累了看一眼龟龟爬来爬去，贼解压。养龟还能培养责任心，是很好的情感寄托。

但是！学生党养龟有几个必须解决的痛点：宿舍空间小怎么办？预算有限怎么选？假期龟怎么办？这些坑我当年都踩过，今天统统告诉你！

第一站：准备工作——养龟前必须确认的事！

养龟不是头脑一热就买的冲动消费。在入手之前，这几件事必须先确认：

1. 宿舍政策：先打听清楚宿舍到底让不让养宠物，有些学校明令禁止，别到时候龟龟来了你被迫送走。

2. 室友关系：这个很重要！一定要提前跟室友沟通好。龟虽然安静，但换水的时候万一有点味道，室友介意的话后期会很麻烦。

3. 预算规划：学生党入门设备一百到两百块就能搞定，后续每月龟粮开销也就几十块。比养猫狗便宜多了。

4. 责任心：龟龟寿命很长，养了就要负责到底。放假要带回家，不能遗弃。想清楚再入手！

第二站：装备选择——学生党性价比之王！

宿舍空间寸土寸金，装备必须精简实用。

龟缸：二十到四十厘米的整理箱就够用，方形比圆形省空间。直接放书桌角落或者阳台上，不占地方。不需要买什么高大上的生态缸，够用就好。

加热棒：这个钱不能省，一定要买带温控的！安全第一，不然龟龟煮熟了你就等着哭吧。

晒背灯：不用买太贵的，普通的 UVA 灯就行。学生党预算有限，该省省该花花。

过滤：小瀑布过滤就足够了，简单实用，能减少换水频率。

记住原则：设备在精不在多，够用就行！

第三站：龟种选择——哪些龟适合学生党？

学生党选龟，体型小、便宜、皮实是关键。

草龟：国货之光，价格实惠，体型适中，互动性好。入门首选！

巴西龟：常见易得，生命力顽强。但注意有些品种是入侵物种，不要随意放生。

蛋龟：体型超小，超级可爱，适合小缸。但有些品种需要加温，入手前做好功课。

新手建议从草龟开始，价格便宜生命力强，练习手感最好。

第四站：日常照顾——忙成狗也能养好龟！

学生党最大的问题就是：课业繁忙，没时间照顾龟龟怎么办？

喂食：龟粮最方便，一袋能吃好久。新鲜食物偶尔喂点就行。记住八字真言：少量多次，宁饿勿多！喂多了反而容易肠炎。

换水：宿舍没有水管，用桶装水或晾晒过的自来水就行。一周换一次，小缸换水也就几分钟的事。也可以用过滤减少换水频率。

温度：冬天有暖气问题不大，夏天开空调要注意昼夜温差。温差太大龟龟容易感冒。

忙的时候怎么办：龟龟超级耐饿，一周不喂都没事！提前把食物和水备好，周末集中照顾就行。养龟不会占用你太多学习时间。

第五站：假期攻略——龟龟安置问题！

假期是学生党养龟的大难题。

短期假期（3-5天）：龟龟自己在家完全没问题，提前换好水、加够食物，放心浪！

中期假期（一周左右）：可以让靠谱的同学帮忙照看一下，记得写个饲养指南。

长期假期（寒暑假）：要么带回家，要么寄养在亲戚朋友家。运输的时候注意保温，别让龟龟冻着了。

实在不行咋办？跟室友搞好关系，假期让室友帮你看几天，请人家吃顿饭就解决了！

终极总结时间！

换水一周一次就够，喂食少量别贪多；
假期提前安排好，责任心要记心窝。
