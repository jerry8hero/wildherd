## 评论区置顶

📌 视频补充（养草龟基础装备）：

🏠 **饲养容器**
→ 玻璃缸/整理箱/周转箱都可以
→ 容量至少是龟龟体积的5倍
→ 放在安静的地方

☀️ **晒台**
→ 树脂/石头/沉木都可以
→ 必须稳固安全
→ 高度不要太高，方便龟龟上下

💧 **过滤系统**
→ 外置过滤器或瀑布过滤
→ 新手推荐水妖精+打氧泵
→ 可以大大减少换水频率

🌡️ **加热棒**
→ 适宜温度25-30度
→ 水温低于20度食欲下降
→ 选择质量好的防爆加热棒

💡 **晒背灯**
→ 需要UVB灯帮助合成维生素D3
→ 促进钙吸收
→ 每天开灯8小时左右

🏠 **躲避屋**
→ 草龟胆子小
→ 有躲避更有安全感
→ 能减少应激反应

🌡️ **温度计**
→ 随时监测水温
→ 配合加热棒使用
