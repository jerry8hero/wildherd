## 评论区置顶

📌 视频补充（水质管理必知）：

💧 **换水判断**
→ 看水色：明显浑浊或发绿就该换了
→ 闻味道：有异味必须换
→ 测指标：氨氮超标要换
→ 每次换水不超过三分之一

🌡️ **水温控制**
→ 新水和原缸水温差不能超过2度
→ 自来水提前晾晒2-3天除氯
→ 换水时沿缸壁缓慢注入

🔄 **过滤系统**
→ 流量要达到水体体积的5-10倍
→ 物理过滤拦粪便，生化过滤培菌
→ 过滤棉不要用自来水冲，用原缸水漂洗

🌿 **绿水养龟**
→ 嫩绿水：能见度较高，可以继续用
→ 老绿水：像豌豆汤，必须换水
→ 新手建议保持水质清澈透亮最安全
