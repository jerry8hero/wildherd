## 评论区置顶

📌 视频补充（常见病防治必知）：

🩹 **腐皮病**
→ 症状：皮肤白色/黄白色斑块、腐烂
→ 治疗：隔离干养，每天泡水2-3次进食
→ 用药：聚维酮碘消毒 + 金霉素软膏
→ 关键：早发现早治疗，一周左右恢复

🛡 **腐甲病**
→ 症状：龟甲局部发软、变色、有异味
→ 治疗：清理腐烂组织，聚维酮碘消毒
→ 注意：严重的腐甲一定要送医
→ 关键：拖延到穿透甲壳很难治

🦠 **肺炎**
→ 症状：张嘴呼吸、流鼻涕、侧浮
→ 治疗：恒温28-30度，浅水饲养保持湿度
→ 注意：肺炎不能完全干养！
→ 关键：治疗及时成功率很高

💊 **肠炎**
→ 症状：排便稀薄、颜色异常、有恶臭
→ 治疗：停食保温，28度水温
→ 用药：益生菌调理，补充水分防脱水

☀️ **软甲病**
→ 症状：龟甲软化、变形
→ 治疗：多晒太阳，补维生素D3
→ 建议：用UVB灯照射

⚠️ 病情严重一定要找专业异宠兽医！

💡 记住：拖得越久，治疗难度越大，成功率越低！
