# 养龟术语大全，看完不再是小白

哈喽大家好，欢迎回来！养龟有很多专业术语，新手听起来一头雾水。这期视频就来说说养龟术语大全，看完你就不再是小白了。

首先说说基础术语。龟苗是指刚出生不久的龟。亚成是指接近成年的龟。成龟是指已经发育成熟的龟。种龟是指用来繁殖的龟。这些是基本的年龄段称呼。

饲养环境术语。裸缸是指没有底砂和装饰的缸。造景缸是指布置了景观的缸。绿水是指含有藻类的水。困水是指晾晒去除氯气的水。养水是指建立硝化系统的过程。

设备术语。过滤是指净化水质的设备。加热棒是用来加温的设备。晒背灯是提供紫外线和热源的灯。温控器是控制温度的设备。晒台是供龟爬上去休息的地方。

喂食术语。开口粮是指龟苗第一次吃的食物。龟粮是专门配制的龟食物。贴膘是指让龟储备脂肪。停食是指停止喂食。定时定量是喂食的原则。

冬眠术语。冬眠是龟在冬季的自然生理状态。浅水冬眠是在水中冬眠的方式。椰土冬眠是用椰土作为介质。排空是指冬眠前清空肠胃。出眠是指冬眠结束醒来。

繁殖术语。交配是指龟的繁殖行为。产蛋是指母龟下蛋。受精蛋是指能孵化的蛋。白蛋是指未受精的蛋。孵化是指蛋变成龟苗的过程。

疾病术语。腐皮是指皮肤腐烂的疾病。腐甲是指甲壳腐烂的疾病。肺炎是呼吸系统的疾病。肠胃炎是消化系统的疾病。白眼病是眼睛发炎的疾病。

状态术语。浮水是指龟漂浮在水面上。拒食是指不吃东西。缩头是指龟把头缩进壳里。晒背是指龟在阳光下取暖。伸展是指龟伸展四肢。

品相术语。刀刻纹是指甲壳上的深刻纹路。象牙底是指腹甲呈象牙白色。麒麟尾是指尾巴分叉变异。变异是指基因改变导致的特殊外观。品相是指龟的外观品质。

交易术语。直播间的龟是指在网上直播销售的龟。玩家自繁是指玩家自己繁殖的龟。wc是指野生捕获的龟。ch是指人工繁殖的龟。cb是指人工养大的龟。

饲养方式术语。深水养是指水位较深的养法。浅水养是指水位较浅的养法。绿水养是指用绿水养龟。生态养是指模拟自然环境的养法。加温养是指用加热棒保持温度的养法。

其他常用术语。硝化细菌是指净化水质的益生菌。应激是指环境变化导致的紧张状态。调理是指让龟恢复健康的过程。静养是指让龟休息适应环境。干养是指把龟放在干燥环境饲养。
