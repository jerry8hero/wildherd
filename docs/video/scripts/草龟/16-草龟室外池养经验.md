# 草龟室外池养经验，在家也能打造生态龟池

养龟的朋友，你有没有想过——为什么别人的龟颜色鲜艳、状态爆棚，你的龟却越养越像塑料玩具？

答案可能很简单：**它们住在室外，而你住在室内。**

这期视频，草龟室外池养经验，教你在家轻松打造生态龟池！

---

## 一、室外池 vs 室内缸

室外池养龟有四大好处——

第一，**阳光是天然的**。紫外线杀菌、促消化，龟的颜色会更漂亮。第二，**雨水也是天然的**。比纯净水更适合龟的生长。第三，**空间大状态好**。龟在室外活动量大，观赏性比室内缸强不知道多少倍。第四，**省下不少室内空间**。不用整间屋都摆满龟缸。

**总结一句：室外池养出来的龟，就像真的在养龟。**

---

## 二、选址与建设

室外池选址记住三个关键词：**阳光、避风、不要低洼**。

每天能晒到四到六小时太阳最好，但夏季要有遮阳措施。位置要避风，不要放在风口。**千万不要选低洼处**，容易积水滋生细菌。

建设要点记三点——

**第一，池子要够大。** 深度三十到五十厘米，池底做好防渗漏，四周做坡度方便龟爬上岸。

**第二，晒台是刚需。** 没有晒台就等于让龟裸泳，龟必须有地方上来休息和晒太阳。

**第三，过滤不可少。** 池塘过滤或者沼泽过滤都可以，植物可以吸收水中养分，定期清洗维护就行。

---

## 三、水质核心：生态系统这样建立

很多新手纠结换水，其实**生态系统平衡之后，你在养一池水，龟只是顺便活着。**

怎么建立？三步走——

**种植水生植物。** 睡莲、荷花、菖蒲，水草吸收杂质改善水质。

**放螺和虾。** 帮忙清理藻类，吃掉食物残渣。

**控制投喂量。** 雨水会增加水体营养，投喂太多会导致富营养化，水太绿就要换部分水。

换水的时候，每周换四分之一到三分之一就行，用抽水机抽出底层脏水，**新水一定要晾晒之后再用**，除氯之外，还可以避免温差大。换水选晴天进行。

## 四、季节管理：一表看清

| | 夏季 | 冬季 |
| -- | -- | -- |
| **核心** | 防暑降温 | 防寒保温 |
| **措施** | 遮阳网＋加深水位 | 保温材料/塑料大棚 |
| **细节** | 及时清理残饵 | 防水面结冰 |
| **注意** | 高温暴晒会要命 | 北方注意冬眠管理 |

夏季重点：**遮阳网是保命的东西**，水体大温度更稳定。冬季重点：**南方可以自然冬眠，北方建议池底冬眠加保温盖**。

---

## 五、混养与造景（可选项）

**混养推荐：** 锦鲤——观赏性强，龟鱼共处和谐。螺和虾——清理藻类好帮手。

**不太建议：** 金鱼容易成为龟的食物。**大龟和小的要分开养**，大龟会咬小龟。

**造景原则：** 堆石、种草、沉木造躲避，简洁为主。**美观和实用兼顾就行，不用追求奢华。**

---

## 六、成本与心态

材料丰俭由人——水泥池成本低又耐用，周转箱都可以改造。**不用追求奢华，实用就好。**

但养龟最重要的不是多少钱，而是心态。龟在自然环境中状态更好，能观察到它们的自然行为。这就是室外池养龟最大的乐趣。
