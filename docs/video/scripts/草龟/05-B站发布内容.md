## 评论区置顶

📌 视频补充（出眠养护必知）：

⏰ **出眠时机**
→ 气温稳定在18度以上，不是看月份
→ 温度要持续稳定一周以上
→ 南方2-3月，北方（东北、西北）4-5月
→ 让龟自己醒最好，不要强行弄醒

🚫 **出眠后不要马上喂**
→ 肠胃功能未恢复，容易引发肠胃炎
→ 等龟完全清醒、活动自如再喂
→ 等观察到正常排便后再喂

🍽 **第一次喂食**
→ 少量，约为正常量的1/3
→ 首食建议龟粮为主，避开高蛋白（虾肉、鱼肉）
→ 温度低于20度坚决不要喂

💧 **浅水过背**
→ 出眠后体力未恢复，水位一定不要太高
→ 浅水过背即可，避免龟龟体力不支溺水

🔍 **状态检查**
→ 眼睛、鼻孔、嘴巴有无分泌物
→ 甲壳有无腐皮、烂甲
→ 四肢力度是否正常
