简介：
龟龟突然不吃东西、状态变差？很可能是应激反应在作怪。草龟胆子小，应激反应比较常见，这期详细讲解应激的原因和处理方法。

新龟到家最容易发生应激反应。开春入手新龟的朋友一定要仔细看！

这期内容要点：

✅ 什么是应激反应：龟对外界环境变化的生理反应
✅ 6大原因：环境变化、水温水质、温度波动、惊吓、嘈杂、新龟加入
✅ 新龟到家处理：干养过渡→浅水→静养三天→少量喂食
✅ 四个坑不要踩：急于下水、急于换水、喂太多、频繁拿玩
✅ 过温方法：先在室内静置半小时，让龟适应室温，然后再放入水中
✅ 换水灵活：根据龟状态和水质调整，不要机械定时
✅ 升温渐进：每天升2-3度，从20度到28度需要三四天

建议收藏备用！遇到应激不要慌，给龟安静恢复的时间。

## 评论区置顶

📌 视频补充（应激反应处理必知）：

🌡 过温处理
→ 冬季温差大时，先连包装静置半小时
→ 再干养一小时左右让龟适应环境温度
→ 过温时先在室内静置半小时，让龟慢慢适应室温，然后再放入水中
→ 过温做得好，应激能少一半

🚫 四个坑不要踩
→ 急于下水：龟还没适应温度变化，容易应激
→ 急于换水：水质变化是大的应激源，前一周尽量不换
→ 喂太多：肠胃还没恢复，吃多了容易肠炎
→ 频繁拿玩：需要安静恢复，不断打扰延长适应时间

💊 应激严重时
→ 转移到安静稳定的环境
→ 减少干扰，让龟自己安静下来
→ 适当升温到28度帮助恢复
→ 水中可加电解质补充体力
→ 密切观察状态变化

💡 记住：稳定的环境是最好的预防，给龟足够的适应时间！
