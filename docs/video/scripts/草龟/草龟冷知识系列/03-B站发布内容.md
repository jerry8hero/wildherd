## 评论区置顶

 龟骨头冷知识要点回顾：

🦴 第一类：颈椎齿突
→ 位置：第二节颈椎（枢椎）前端
→ 形状：最尖锐的小象牙
→ 为什么留下：卡喉咙、啃不动

🦴 第二类：尾椎脉弧
→ 位置：每节尾椎腹面
→ 形状：V 字形鱼骨状
→ 为什么留下：连着尾椎、像鱼刺

🦴 第三类：甲壳缘板
→ 位置：背甲边缘一圈
→ 形状：带尖角的骨片
→ 为什么留下：缝锯齿嵌合、整体性强

🦴 第四类：髂嵴与乌喙骨
→ 位置：腰带的髂骨、肩带的乌喙骨
→ 形状：匕首状、S 形尖突
→ 为什么留下：藏在深处、肌肉啃光才露出

 三字真言：硬、连、卡
→ 硬：密质骨硬度高
→ 连：韧带嵌合难拆
→ 卡：形状不利吞咽
