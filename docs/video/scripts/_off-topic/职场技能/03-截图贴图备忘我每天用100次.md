比如产品经理要同时看需求文档和原型图，直接截图贴上去，左边看功能描述，右边看界面设计，一个屏幕搞定！

---

第一站：Windows系统——Win+Shift+S，截图神器！

Windows 用户记住这个组合键：Win + Shift + S！

按下之后，屏幕会变灰，你可以选择：
矩形截图：框选一个区域
窗口截图：截取某个窗口
全屏截图：截整个屏幕
自定义截图：想截哪里截哪里

截完之后，图片会自动复制到剪贴板，直接 Ctrl + V 粘贴就行！

那贴图功能呢？按 Win + W 打开 Windows Ink 工作区，里面有截图草图板！

更重要的是！截图之后，你可以在上面标注、涂鸦、画箭头，标注完直接固定在屏幕上！

固定之后，图片会一直在最顶层显示，永远不会因为切换窗口被挡住！

---

第二站：Mac系统——好戏更足！

Mac 用户更爽！系统自带的功能更强！

截图快捷键：
Command + Shift + 3：全屏截图
Command + Shift + 4：区域截图，选中什么截什么
Command + Shift + 4 然后按空格：窗口截图，截哪个窗口点哪个

截完图之后，Mac 会让你直接在截图上做标注！写字、画箭头、涂鸦、裁剪都能做！

做完了点击"完成"，图片就保存了。

贴图的功能在图床上！截图之后，按住 Fn 键不松，然后你就可以把截图"贴"在屏幕上！

Mac 的贴图更厉害，图片可以调整大小、透明度，还能随时隐藏显示，简直是效率神器！

第三站：进阶用法——让贴图效率翻倍！

说完基础功能，再说几个进阶用法！

第一个：给截图分类！
不同项目的截图，建议建不同的文件夹。比如我就有"项目A参考"、"客户沟通"、"需求文档"三个文件夹，每次截图直接 Ctrl + X 切换目录，效率拉满！

第二个：快捷指令自动化！
Mac 用户可以用快捷指令 app，设置一个自动化流程，比如"截图 + 加上日期水印 + 保存到指定文件夹"，一条龙服务，截图自动整理！

第三个：和备忘录配合！
有些需要长期保留的截图，建议直接存到备忘录里。Mac 的备忘录有搜索功能，时间久了想找某张截图，直接搜索关键词，秒找到！
