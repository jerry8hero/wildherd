- 3-5年性成熟
- 水温25度以上
- 产蛋2-4枚/窝

📌 孵化条件：
- 蛭石孵化
- 湿度80%
- 温度27-29度
- 孵化期60-90天

📌 苗子开口：
- 丰年虾
