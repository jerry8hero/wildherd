## 【置顶评论】

🦎 睫角守宫 vs 豹纹守宫快速对比：

| | 豹纹守宫 | 睫角守宫 |
|---|---|---|
| **类型** | 地栖型 | 树栖型 |
| **缸体** | 平缸，低矮就行 | 竖缸，要高不要宽 |
| **温度** | 30-32℃热区 | 22-26℃（怕热也怕冷） |
| **湿度** | 30-40%（干燥） | 60-80%（湿润+通风） |
| **性格** | 温顺，随便抓 | 活泼，胆小会跳 |
| **食物** | 纯肉食（虫子） | 杂食（果泥+虫子） |
| **喂食频率** | 一周2-3次 | 幼体2果泥+3蟋蟀/成体2+2 |
| **价格** | 几十到一百 | 150-500+ |
| **新手推荐** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

⚠️ 睫角守宫断尾不可再生！千万不要抓尾巴，上手从侧面托住腹部和四肢
⚠️ 怕热也怕冷：冬天低于20度必须加温，加热垫贴缸体侧面/背面，不贴底部
⚠️ 高湿+不通风=真菌感染（烂尾烂屁屁），喷水后必须保证通风，建议设干湿分区
✅ 怕虫子也能养：睫角可以纯果泥喂养，比豹纹守宫友好得多
⚠️ 蜕皮不完整是常见问题，手指脚趾的皮要重点检查

❓ 你选了豹纹还是睫角？评论区晒出你的小可爱，看到会回～
