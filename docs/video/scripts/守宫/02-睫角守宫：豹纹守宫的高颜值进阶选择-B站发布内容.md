## 【置顶评论】

🦎 睫角守宫 vs 豹纹守宫快速对比：

| | 豹纹守宫 | 睫角守宫 |
|---|---|---|
| **类型** | 地栖型 | 树栖型 |
| **缸体** | 平缸，低矮就行 | 竖缸，要高不要宽 |
| **温度** | 30-32℃（怕冷） | 22-26℃（怕热） |
| **湿度** | 30-40%（干燥） | 60-80%（湿润） |
| **性格** | 温顺，随便抓 | 活泼，胆小会跳 |
| **食物** | 纯肉食（虫子） | 杂食（果泥+虫子） |
| **价格** | 几十到一百 | 150-500+ |
| **新手推荐** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

⚠️ 睫角守宫蜕皮不完整是常见问题，手指脚趾的皮要重点检查
