# WildHerd 优化路线图

> 爬宠饲养管理应用的持续迭代指南

---

## 一、核心功能强化

### 1. 成长追踪系统
- [ ] **体重/体长曲线图表** - 使用 fl_chart 可视化生长趋势
- [ ] **蜕皮记录** - 记录蜕皮日期、状态、周期分析
- [ ] **年龄计算** - 自动计算爬宠月龄/年龄
- [ ] **阶段标记** - 幼体、亚成、成体的阶段划分

### 2. 喂食管理系统
- [ ] **喂食计划** - 设定喂食频率（每周/每月）
- [ ] **食物库存** - 记录冻鼠、乳鼠、昆虫等库存
- [ ] **拒食记录** - 追踪拒食情况，分析原因
- [ ] **食物类型统计** - 了解饮食结构

### 3. 环境监控强化（habitat 模块）
- [ ] **温湿度历史曲线** - 每日记录走势图表
- [ ] **灯具使用计时** - UVB/加热灯更换提醒
- [ ] **环境评分优化** - 根据物种标准自动评估
- [ ] **季节性建议** - 冬天/夏天的温湿度调整提醒

### 4. 医疗健康系统强化
- [ ] **用药记录** - 药物名称、剂量、疗程追踪
- [ ] **体检记录** - 定期体检、粪便检查记录
- [ ] **疾病历史** - 完整的患病-治疗档案
- [ ] **异宠医院库** - 附近爬宠医院信息（可手动添加）

---

## 二、数据可视化与报告

### 5. 统计仪表盘
- [ ] **饲养概览** - 爬宠数量、物种分布饼图
- [ ] **月度报告** - 喂食次数、医疗支出统计
- [ ] **年度总结** - 成长对比、饲养支出分析

### 6. 相册与时间线
- [ ] **成长相册** - 按时间线展示照片
- [ ] **对比视图** - 去年vs今年同一时期对比
- [ ] **照片分类** - 成长/蜕皮/环境/医疗分类相册

---

## 三、实用功能

### 7. 财务记录
- [ ] **支出追踪** - 购买、食物、医疗费用记录
- [ ] **物种花费统计** - 每种爬宠的饲养成本分析
- [ ] **年度支出报表**

### 8. 知识库扩展
- [ ] **物种饲养指南** - 每种爬宠的详细饲养方案
- [ ] **常见问题 FAQ** - 新手常见问题解答
- [ ] **饲养术语解释** - 新手入门术语表
- [ ] **饲养心得文章** - 用户可添加自己的饲养经验

### 9. 提醒系统
- [ ] 喂食提醒（可设置间隔）
- [ ] 换水/清洁提醒
- [ ] 灯具更换提醒（UVB 建议6个月）
- [ ] 驱虫/体检提醒
- [ ] 疫苗接种提醒

---

## 四、架构与体验优化

### 10. 数据导出与备份
- [ ] **数据导出** - 导出为 Excel/CSV 格式
- [ ] **数据导入** - 从备份恢复数据
- [ ] **本地加密存储** - 保护隐私数据

### 11. 桌面端优化
- [ ] **窗口大小自适应**
- [ ] **键盘快捷键**
- [ ] **深色/浅色主题切换**
- [ ] **平板模式布局**

### 12. 用户体验优化
- [ ] **快速添加入口** - 首页快捷添加爬宠/记录
- [ ] **批量操作** - 批量添加记录、批量编辑
- [ ] **搜索优化** - 全局搜索爬宠、记录、知识
- [ ] **离线优先** - 完全离线可用

---

## 优先级建议

### 第一优先级（高）
1. 体重/体长曲线图表
2. 喂食计划与记录
3. 温湿度历史曲线
4. 用药记录

### 第二优先级（中）
5. 蜕皮记录
6. 财务支出统计
7. 统计仪表盘
8. 数据导出/备份

### 第三优先级（低）
9. 食物库存管理
10. 异宠医院库
11. 饲养知识库扩展
12. 桌面端优化

---

## 技术实现提示

- **图表库**: 项目已集成 `fl_chart`，可直接使用
- **状态管理**: 使用 Riverpod，可创建专门的 providers
- **数据存储**: 现有 SQLite + SharedPreferences 方案足够
- **图片处理**: 桌面端需处理文件选择（Web/桌面端兼容）
