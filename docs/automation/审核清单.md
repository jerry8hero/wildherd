# 内容审核清单

## 发布前必查项目

### 一、知识准确性
- [ ] 物种名称、学名正确
- [ ] 饲养方法符合该物种特性
- [ ] 数据引用（如体型、寿命、温度等）有据可查
- [ ] 安全提示准确无误

### 二、内容合规性
- [ ] 无政治敏感内容
- [ ] 无低俗色情内容
- [ ] 无虚假信息
- [ ] 无侵权内容（音乐、画面等）
- [ ] 涉及动物需符合动物福利要求

### 三、标题和封面
- [ ] 标题无夸张虚假宣传
- [ ] 标题无侮辱性词语
- [ ] 封面图片清晰
- [ ] 封面与内容相关
- [ ] 封面无过度PS导致误导

### 四、B站特定要求
- [ ] 标签设置完整（20个）
- [ ] 简介包含核心信息
- [ ] 分区选择正确
- [ ] 章节时间轴设置合理
- [ ] 视频无广告嫌疑内容

### 五、互动引导
- [ ] 开头有吸引力（留住观众）
- [ ] 内容有干货（增加收藏）
- [ ] 结尾有互动引导（点赞投币）
- [ ] 置顶评论内容准备就绪

---

## 分品类审核重点

### 鳄龟类
- [ ] 强调外来入侵物种不能放生
- [ ] 强调安全操作（不要徒手抓取）
- [ ] 准确说明体型潜力（避免误导以为长不大）
- [ ] 混养风险提示

### 新手入门类
- [ ] 难度评估客观
- [ ] 设备清单完整
- [ ] 入门品种推荐合理
- [ ] 避免过于乐观的描述

### 健康医疗类
- [ ] 强调及时就医
- [ ] 提供基本判断方法
- [ ] 避免绝对化的治疗承诺
- [ ] 免责声明适当

### 价格类
- [ ] 价格信息标注时间（避免过时）
- [ ] 强调价格波动正常
- [ ] 不推荐具体购买渠道（避免广告嫌疑）
- [ ] 品质鉴别方法客观

## 常见问题修正

### 标题常见问题
- 过度标题党 → 改为客观描述
- 含有绝对化词语 → 改为相对表述
- 与内容不符 → 调整标题或内容

### 内容常见问题
- 知识点有误 → 查阅权威资料修正
- 表达不清晰 → 简化语言
- 节奏拖沓 → 删减冗余内容

### 安全警示
- 未提及风险 → 补充安全提示
- 警示不够醒目 → 强调强调再强调
