# 爬宠博主自媒体运营工作流手册

## 一、内容创作流程

```
选题 → 资料收集 → 脚本撰写 → AI优化 → 人工审核 → 定稿发布
```

### 1.1 选题阶段
- 来源：热点追踪、社区讨论、竞品分析、用户反馈
- 工具：Notion选题看板
- 评分：热度×0.3 + 内容匹配度×0.3 + 制作可行性×0.2 + 差异化×0.2

### 1.2 资料收集
- 爬宠知识库
- 权威资料交叉验证
- 素材整理归档

### 1.3 脚本撰写
- 按照标准模板撰写
- 保持个人风格
- 控制时长节奏

### 1.4 AI优化（使用prompts/模板）
- 标题生成：10个备选 → 人工挑选
- 标签推荐：20个标签
- 简介撰写：B站风格引导互动
- 脚本润色：保持知识准确

### 1.5 人工审核
- 知识准确性核对
- 敏感内容检查
- 风格一致性确认

---

## 二、发布流程

### 2.1 B站发布
1. 视频剪辑完成 → 生成封面
2. 登录B站创作者平台
3. 上传视频 → 设置定时发布
4. 填写标题、简介、标签
5. 设置章节时间轴
6. 发布前检查清单确认
7. 提交发布

### 2.2 小红书发布
1. 运行 `python3 scripts/md2xiaohongshu.py` 生成配图
2. 提取视频核心要点
3. 撰写小红书风格文案
4. 发布图文/短视频

## 三、AI辅助工具使用指南

### 3.1 Kimi AI（推荐，新手友好）
- 使用场景：脚本润色、标题生成、标签推荐
- 访问：https://kimi.moonshot.cn
- 操作：复制prompts/中的模板，填入内容，发送给Kimi

### 3.2 通义千问
- 使用场景：批量内容处理、知识整理
- 访问：https://tongyi.aliyun.com
- 适合：结构化知识提取

### 3.3 智谱清言
- 使用场景：内容优化、多语言
- 访问：https://www.zhipuai.cn
- 适合：专业术语检查

## 四、定时任务配置

### 4.1 创建定时提醒
```bash
# 每天早上8点检查待发布内容
0 8 * * * cd /path/to/wildherd && python3 scripts/check_pending.py

# 每周一三五 19:00 推送新内容提醒
0 19 * * 1,3,5 cd /path/to/wildherd && python3 scripts/schedule_reminder.py
```

### 4.2 查看定时任务
```bash
crontab -l
```

### 4.3 编辑定时任务
```bash
crontab -e
```

## 五、数据复盘

### 5.1 周复盘（每周日）
- 本周发布视频数据汇总
- 对比上周数据变化
- 发现问题制定改进方案

### 5.2 月复盘（每月最后一天）
- 本月数据趋势分析
- 内容效果排名
- 下月内容计划

### 5.3 核心指标
| 指标 | 合格线 | 优秀线 |
|------|--------|--------|
| 播放完成率 | >40% | >60% |
| 点赞率 | >3% | >8% |
| 投币率 | >1% | >3% |
| 收藏率 | >5% | >10% |
| 评论数/千播 | >2 | >5 |

## 六、互动管理

### 6.1 评论回复优先级
- P0：负面评论/争议（30分钟内）
- P1：专业问题咨询（2小时内）
- P2：常规评论互动（当天内）

### 6.2 常见问题自动回复
- 参考 `configs/auto_replies.yml`
- 根据实际场景调整话术
- 保持个人风格一致
