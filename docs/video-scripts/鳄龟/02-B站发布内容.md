特别提醒：

- 鳄龟生长快，缸要一步到位
- 水位可以很深（40-60cm）
- 不需要晒台
- 长柄镊子是必备安全工具
- 加热棒必须配温控插座

📌 补充一些视频里没讲到的：

1️⃣ 预算参考：
- 60cm缸：100-200元
- 1.5米缸：300-500元
- 2米缸：500-1000元
- 过滤设备：200-500元
- 加热设备：100-300元

2️⃣ 设备品牌：
- 加热棒：创星、森森
- 温控插座：国内品牌
- 过滤：滴流盒最佳

3️⃣ 鳄龟不需要晒台的原因：
- 原生环境很少晒太阳
- 深水龟不太需要UVB
- 室内饲养保证恒温即可

4️⃣ 防咬手套选择：
- 加厚橡胶手套
- 皮革手套也可以
- 手套会降低手感
