📌 补充一些视频里没讲到的：

1️⃣ 被咬急救步骤：
- 保持冷静，不要硬拽
- 把手和龟一起放回水中
- 用流水冲龟鼻尖/眼睛
- 敲击龟背让它松口
- 伤口消毒就医

2️⃣ 静养环境设置：
- 暗光环境最佳
- 水位刚没过背甲
- 温度28度恒温
- 避免一切震动和噪音

3️⃣ 开食首选：
- 活小鱼苗（激发本能）
- 死鱼虾肉（切小块）
- 泥鳅段（营养丰富）
- 龟粮（可作为辅助）

4️⃣ 初期观察重点：
- 是否缩头不动
- 是否有腐皮腐甲
- 眼睛是否明亮
- 四肢是否有力
