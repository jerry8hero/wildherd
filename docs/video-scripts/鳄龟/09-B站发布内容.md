置顶评论：
📌 补充一些视频里没讲到的：

1️⃣ 被咬紧急处理：
- 保持冷静
- 和龟一起放回水中
- 流水冲龟鼻尖/眼睛
- 敲击龟背让它松口
- 万不得已可以用工具撬嘴

2️⃣ 伤口处理：
- 大量清水冲洗（至少5分钟）
- 碘伏消毒
- 纱布包扎
- 口服抗生素
- 每天换药观察

3️⃣ 必须就医的情况：
- 骨头可能受伤
- 大量出血不止
- 伤口很深
- 发烧或感染迹象

4️⃣ 安全工具：
- 长柄镊子（喂食用）
- 防咬手套（操作时戴）
- 长柄夹子（换水用）
- 切忌空手操作
