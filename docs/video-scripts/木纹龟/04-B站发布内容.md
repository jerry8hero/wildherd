📌 过温步骤：

1. 连袋子一起放进缸里
2. 泡30-60分钟
3. 让水温慢慢过渡
4. 不要直接放龟下水

📌 第一周注意：
- 不换水
- 不喂食
- 减少打扰
- 保持26度
