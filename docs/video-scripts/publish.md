置顶评论：
 📌 补充一些视频里没讲到的：

 1️⃣ 过温关键点：
  - 找一个干净的塑料盆或整理箱，盆里不要放水
  - 将龟连同包裹它的湿苔藓或湿布一起取出，放入空盆中
  - 静置30到60分钟，让龟的体温缓慢从运输低温过渡到室内常温
  - 这一步能有效预防肺炎！

 2️⃣ 静养环境设置：
  - 水位：刚没过龟背1到2厘米
  - 晒台：必须设置一个坡度平缓的晒台
  - 躲避：水中放置躲避穴

 3️⃣ 三不原则：
  - 一不喂食：静养3到5天，排空肠胃
  - 二不换水：除非水脏臭，否则只补水不换水
  - 三不打扰：不要频繁拿起来看

 4️⃣ 开食小贴士：
  - 水温必须保持在25度以上
  - 首选鲜虾肉，气味浓郁诱食效果好
  - 开食时间因个体差异不同，有的龟一周后开食，有的需要两周甚至更久
  - 一定要有耐心！切忌填鸭式强喂！
