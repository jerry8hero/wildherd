侧颈龟常见病防治，新手必看！

哈喽大家好，欢迎回来！养龟最怕的就是龟龟生病，这期视频给大家说说侧颈龟的常见病防治，建议收藏备用。

先说最常见的腐皮病。症状是龟皮肤出现白色或黄白色的斑块，有腐烂现象。原因是水质不好或龟龟受伤后细菌感染。治疗方法是隔离干养，每天用聚维酮碘消毒，涂抹金霉素软膏，保持环境干燥清洁。

腐甲病也很常见。症状是龟甲局部发软、变色、有异味。原因是外伤或水质问题导致细菌感染。治疗需要先清理腐烂组织，用双氧水消毒，然后涂抹抗生素软膏，严重的需要口服抗生素。

肺炎是致命性疾病，一定要重视。症状是张嘴呼吸、流鼻涕、侧浮、不爱动。原因是温差太大或水质问题。治疗方法是隔离加温到30度，用阿莫西林或头孢类抗生素药浴，严重的话需要注射治疗。

肠炎也是高发疾病。症状是排便稀薄、颜色异常、有异味，龟龟食欲不振。原因是喂食不当或水质问题。治疗方法是停食几天，保持水温28度左右，用益生菌调理肠胃，严重的话用抗生素治疗。

软甲病需要注意。症状是龟甲软化、变形。原因是缺乏紫外线照射或维生素D3不足。治疗方法是提供适当的光照，补充维生素D3。侧颈龟虽然对光照要求不高，但也要适当晒背。

最后说说预防。保持水质干净、控制好温度、合理喂食是预防疾病的关键。侧颈龟是热带品种，加温饲养时更要注意水质，因为水温高更容易滋生细菌。新龟到家要隔离观察，平时多观察龟龟的状态，发现异常及时处理。
