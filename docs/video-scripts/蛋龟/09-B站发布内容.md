置顶评论：
📌 补充一些视频里没讲到的：

1️⃣ 产蛋池设置：
- 椰土或沙土混合物
- 湿度以手握成团为宜
- 深度15cm以上
- 放置在安静角落

2️⃣ 孵化盒制作：
- 透明塑料盒打孔
- 蛭石铺底，蛋埋一半
- 盖子半开保持透气
- 放在恒温28度环境

3️⃣ 性别温度对照：
- 30度以上：胚胎死亡率高
- 28-30度：公龟为主
- 26-28度：母龟为主
- 25度以下：发育缓慢

4️⃣ 苗子开口饲料：
- 丰年虾（首选）
- 红虫
- 龟粮泡软
- 蛋黄水（应急用）
