📌 补充一些视频里没讲到的：

1️⃣ 换水细节：
- 提前24小时困水
- 新旧水温差控制在2度内
- 每次换1/3到1/2
- 不要一次性全换

2️⃣ 绿水制作：
- 阳光直射 + 少量有机物
- 嫩绿水养龟好，老绿水容易腐皮
- 绿水太浓需要换掉一部分

3️⃣ 温控设备测试：
- 加热棒设置比目标温度低2度
- 温控插座设置在目标温度
- 双重保险确保安全

4️⃣ 春秋季管理：
- 减少换水频率
- 降低喂食量
- 密切关注温差预报
- 准备加温设备以防倒春寒
