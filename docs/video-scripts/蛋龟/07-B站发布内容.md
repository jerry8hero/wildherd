置顶评论：
📌 补充一些视频里没讲到的：

1️⃣ 缸体选择：
- 超白缸：60cm起步，方便观察
- 周转箱：性价比高，尺寸灵活
- 整理箱：适合苗子期

2️⃣ 过滤方案：
- 滴流盒：效果最好，但占空间
- 滤桶：隐蔽性好，适合成品缸
- 背滤/侧滤：一体化美观

3️⃣ 底沙选择：
- 细沙：容易清理，不藏污垢
- 溪流沙：美观但价格高
- 不建议用泥底，容易坏水

4️⃣ 水草推荐：
- 水榕：皮实耐咬
- 皇冠草：生长快
- 绿萝：可水可陆
- 避免贵重水草，蛋龟会啃
