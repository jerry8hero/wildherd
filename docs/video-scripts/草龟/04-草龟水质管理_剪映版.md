# 草龟水质管理：避开这些误区，你的龟才能健康长寿

哈喽大家好，欢迎回来！
养龟先养水，水质是养好草龟的核心。
今天帮大家避开几个常见的坑！

换水频率不是固定几天换一次，而是看两点：
水体是否浑浊、有异味。
过滤系统是否正常工作。
检测出氨氮或亚硝酸盐超标，就必须换水了。
每次换水不超过三分之一，避免龟应激。

水温控制非常重要！
新水和原缸水温差不能超过两度，否则龟容易感冒甚至肺炎。
自来水要提前晾晒两到三天，既除氯又调温。
换水时沿着缸壁缓慢注入，不要直接冲刷龟的身体。

过滤系统非常关键！
草龟虽然吃得不多，但排泄量不小，对水质污染较快。
过滤器流量要达到水体体积的五到十倍。
物理过滤拦截粪便，生化过滤培养硝化细菌。
过滤系统必须二十四小时开启，停摆会导致有益菌大量死亡。

开缸时添加优质硝化细菌制剂，可以加速建立生态循环。
不要用自来水洗过滤棉，会杀死有益菌。
用原缸水轻轻漂洗即可。

绿水养龟有好处，但新手慎用浓绿水！
夜间藻类会消耗氧气，浓度过高会导致龟窒息。
水太绿看不清龟的状态，容易延误治疗。
藻类死亡分解还会产生毒素。
保持嫩绿水，隐约可见龟即可。
新手建议清澈透亮的水质最安全。

水位控制是重点！
幼龟水位过背两到三厘米，防止溺水。
成龟完全可以深水饲养，三十到五十厘米甚至更深。
有稳固晒台和坡道就行。
深水好处是活动空间大、水质稳定、减少腐皮。
无论水多深，必须保证龟能轻松爬出水面呼吸休息！

每天清理水面油膜和残渣。
每周检查过滤，观察龟的排便和精神状态。
养水是养平衡，不要频繁大换水。

总结：强大过滤加合理深水配晒台，加稳定温差，加正确认识绿水。
别让错误的浅水论和小过滤限制了龟的健康成长！
