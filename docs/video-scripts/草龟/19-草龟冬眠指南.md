# 草龟冬眠全攻略，新手必看

哈喽大家好，欢迎回来！天气转凉，又到了草龟准备冬眠的季节。草龟是中国原生龟种，耐寒能力比较强，是比较适合冬眠的品种。这期视频详细给大家说说草龟冬眠的那些事。

首先说说哪些龟可以冬眠。健康的草龟成龟，背甲五厘米以上、体质健康的可以进行冬眠。草龟耐寒能力较强，比其他热带龟种更适合冬眠。当年出生的苗子、弱龟、病龟建议加温过冬，不要冬眠。

冬眠前的准备很关键。秋末开始就要给龟龟贴膘，多喂一些高蛋白的食物，让龟龟储备足够的脂肪。冬眠前两周要停止喂食，让龟龟排空肠胃，避免冬眠期间肠胃炎。这是冬眠最重要的一步，带食物冬眠很容易导致龟龟死亡。

常见的冬眠方式有几种。第一是浅水冬眠，把龟龟放在过背的水里，放在阴暗安静的地方。这种方法简单，但要注意水质，一周换一次水。第二是椰土冬眠，用湿润的椰土或蛭石让龟龟钻进去，定期喷水保持湿度。第三是湿沙冬眠，用干净的河沙让龟龟钻进去，保持沙子湿润。

冬眠期间的维护不能少。定期检查龟龟的状态，看看有没有异常。每周换一次浅水，保持水质。椰土或沙子要保持湿润，但不能积水。还要注意防鼠防虫。

冬眠温度要控制好。最适宜的温度是五到十五度，草龟比较耐寒，短时间低温可以承受，但长时间低温有冻伤风险。可以用温度计监测，有条件的话放在地下室或冰箱冷藏室都可以。

什么时候结束冬眠？一般到来年春天，气温稳定在十八度以上时，龟龟会自然苏醒。苏醒后不要马上喂食，等龟龟完全清醒、排便后再开始喂食。刚开始喂食要少量，让龟龟的肠胃慢慢适应。

最后特别提醒：第一年的龟龟苗不要冬眠，一定要加温过冬。加温棒调到二十八到三十度，正常喂食，等明年体质好了再考虑冬眠。
