# 草龟的互动训练，让你的龟龟更亲人

哈喽大家好，欢迎回来！很多养龟的朋友都希望自己的草龟能像猫狗一样亲人，这期视频就来说说草龟的互动训练。草龟虽然不如哺乳动物聪明，但只要方法得当，完全可以训练出良好的互动性。

首先说说互动的理论基础。草龟是爬行动物，智商有限，不能像猫狗那样训练。但它们可以通过条件反射形成习惯。草龟的视力很好，能识别主人。它们的记忆力也不错，能记住喂食的时间和环境。

建立信任是关键。新到家的龟不要急于互动，先静养几天让它适应环境。不要经常把龟拿在手里玩，这会让它感到紧张。喂食时动作要轻柔，不要突然伸手吓到龟。让龟慢慢熟悉你的气味和声音。

喂食互动是最有效的训练方法。每次喂食前发出固定的声音，比如敲敲缸壁或者叫它的名字。坚持一段时间后，龟就会形成条件反射，一听到声音就会过来。这个方法简单有效，非常适合新手。

手势训练可以尝试。用手指轻轻点点水面，龟会以为有食物。长期坚持，龟就会追着手指跑。可以在喂食时用手指引导龟龟游动方向。手势要简单固定，不要太复杂。龟的注意力很短，训练时间不要太长。

上手训练需要耐心。开始时先把手放在水里，让龟熟悉你的手。等龟不怕手后，可以轻轻托起它的腹甲。把龟拿在手里时，要让它感受到你的体温。刚开始龟可能会挣扎，不要强迫。等它慢慢适应后，就能轻松上手了。

正确的抓握方式很重要。用手指轻轻抓住龟的背甲边缘，不要太用力。拇指和中指分别放在龟的前后腿处，提供支撑。不要抓住龟的四肢，容易造成骨折。抓取前要洗干净手，避免把脏东西传到龟身上。

互动时要注意的事项。龟四肢乱动时不要强行按住，容易受伤。龟张嘴想咬人时不要把手凑上去。互动时间不要太长，龟会感到疲劳。不要在龟睡觉时打扰它，容易受到惊吓。

如何让龟追着手游。把食物放在手心里，沉入水中喂给龟吃。龟会靠近你的手来吃食物，逐渐就不怕手了。可以用镊子夹着食物逗引龟，让它追着游动。这个游戏很有趣，龟也很喜欢。

训练龟听名字。给龟取一个简单的名字，比如小草或者龟龟。每次喂食前都叫它的名字，轻轻敲敲缸壁。坚持几天后，龟就会对自己的名字有反应。训练要有耐心，不能急于求成。龟的反应可能比较慢，不要放弃。

互动游戏的玩法。可以在缸里放一个小球，让龟推动玩耍。有些草龟会推着球滚，非常有趣。可以用温度计轻轻碰触龟的鼻子，训练它的反应。还可以设置一些小障碍，让龟穿越过去。

影响互动性的因素。龟的年龄越小，训练效果越好。野生龟比人工养殖的龟更难训化。公龟通常比母龟更活泼。龟的健康状况直接影响训练效果，生病时要停止训练。

不同阶段的训练重点。幼龟时期是训练的黄金阶段，要多进行互动。亚成龟可以加强上手训练，培养感情。成龟以维持互动为主，不要强迫它做不喜欢的事。年老的龟互动性会下降，尊重它的习惯就好。

最后总结一下。草龟的互动训练需要耐心和恒心，用食物引诱最有效。不要急于求成，让龟慢慢适应。建立信任后，龟会很乐意和你互动的。
