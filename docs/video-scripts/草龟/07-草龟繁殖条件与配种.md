# 草龟繁殖条件与配种，学会了就能自己繁殖

哈喽大家好，欢迎回来！很多养草龟的朋友都希望能体验繁殖的乐趣，看着小龟破壳而出，特别有成就感。但草龟繁殖需要满足一定的条件，这期视频就详细给大家讲讲草龟繁殖的那些事。

首先说说繁殖的前提条件。草龟通常需要五年以上才能性成熟，雄龟三年以上可以尝试繁殖。雌龟需要达到五百克以上，背甲十二厘米以上才有繁殖能力。年龄和体重达标后，繁殖成功率才会高。

关于种龟的选择。雄龟要选体型修长、尾巴粗长、指甲完整的。雌龟要选体型圆润、背甲厚实、反应灵敏的。建议一雄多雌的比例饲养，两到三只雌龟配一只雄龟最合适。种龟的健康状况直接影响受精率，一定要选择健康活泼的个体。

如何区分公母。成年草龟很容易区分，公龟体型较小，尾巴粗长，泄殖孔在背甲外缘内侧。母龟体型较大，尾巴短小，泄殖孔靠近背甲。最好选择年龄相近的种龟，避免体型差异过大导致交配困难。

交配时间与环境。草龟的交配期在每年的九月份到十一月份，还有来年的三月份到五月份。交配需要在水中进行，水深要没过龟背十厘米以上。环境要安静，不要经常打扰。可以在缸底铺一些光滑的鹅卵石，方便公龟固定身体。

交配期间的饲养。交配期要保证种龟营养充足，多喂一些高蛋白的食物，比如鱼虾、螺蛳。可以适当增加喂食频率，每天喂一次。每次喂食量以龟龟头部大小为宜。还可以添加一些维生素E和钙粉，提高繁殖能力。

交配成功的表现。交配成功后，雌龟的食欲会明显增加，这是为了储备能量产蛋。交配后一到两个月，雌龟会开始产蛋。产蛋前雌龟会变得焦躁不安，到处爬动寻找合适的位置。发现这些信号后，就要准备好产蛋池了。

产蛋池的准备。雌龟产蛋需要松软的沙土，椰土和蛭石混合都可以。沙土湿度要适中，手捏成团但不滴水。产蛋池要放在安静阴暗的地方，避免惊扰。产蛋池的深度要十厘米以上，方便雌龟挖坑。

最后提醒几点。第一，不要过早让龟繁殖，年龄不够繁殖成功率低，还会损伤龟的身体。第二，交配期不要频繁换水，避免惊扰。第三，繁殖需要耐心，可能第一年不会成功，慢慢积累经验。
