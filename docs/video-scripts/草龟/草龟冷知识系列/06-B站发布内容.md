简介：
草龟繁殖充满了神奇和有趣的冷知识！性别居然是温度决定的？母龟能存精子好几年？公龟会累死母龟？这期视频全面解析那些让你惊呼"原来如此"的繁殖秘密，繁殖爱好者必看！

这期内容要点：

✅ 性别控制：孵化温度高于30℃多母龟，低于26℃多公龟
✅ 受精斑：24小时左右出现，会随时间扩散，是胚胎发育信号
✅ 孵化时间：28-30℃需55-60天，高温快但畸形率高
✅ 破壳靠头顶：破壳齿在头顶，破壳需1-2天，不要急于帮忙
✅ 小龟不开食：卵黄囊是第一顿饭，静养1-2天后再下水
✅ 母龟存精：一次交配可存储精子数年，无需每次都交配
✅ 受精率非100%：取决于公龟质量和交配频率，健康种龟可达80%+
✅ 1公配1母：小水体强烈建议，大水体才1配3，否则母龟会被累坏
✅ 产蛋坑深：沙土厚度需15-20厘米，挖浅了蛋会暴露
✅ 补钙靠晒太阳：必须配合阳光合成D3，没条件用UVB灯

## 评论区置顶

📌 视频补充（繁殖要点）：

🌡️ 温度性别控制
→ 高于30℃：多母龟
→ 低于26℃：多公龟
→ 28℃左右：公母比例接近

🥚 受精斑
→ 24小时左右出现
→ 白色斑点/斑块
→ 没有=白蛋，无法孵化

⏱️ 孵化参数
→ 温度：28-30℃
→ 湿度：手捏成团不滴水
→ 时间：55-80天
→ 温差不超过2度

🐢 破壳后
→ 脐带收好再下水
→ 水位：刚过背甲一半
→ 几天内不喂食

💡 记住：繁殖是耐心修行，第一年不成功很正常！
