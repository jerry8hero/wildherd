## 评论区置顶

📌 视频补充（行为判断要点）：

😮 缩头
→ 不是怕你，是本能
→ 新龟到家静养几天
→ 等它适应后会放松

💧 浮水判断
→ 短暂浮着休息：正常
→ 侧浮+呼吸声重：肺炎信号
→ 及时加温治疗

👀 挠眼睛
→ 偶尔：正常清洁
→ 频繁：眼睛发炎或水质刺激
→ 检查眼睛有无分泌物

🎙️ 叫声
→ 龟一般不叫
→ 叫声=呼吸困难=可能是肺炎
→ 及时检查治疗

💩 吃屎
→ 正常行为
→ 粪便有未消化营养
→ 补充营养用的
→ 及时清理即可

💡 记住：多观察，才能读懂你的龟！
