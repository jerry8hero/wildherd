# 学生党养龟指南，在宿舍也能养好龟

哈喽大家好，欢迎回来！很多学生朋友想养龟，但宿舍条件有限，不知道该怎么养。这期视频就来说说学生党养龟的攻略，在宿舍也能养好龟。

首先说说宿舍养龟的优势。养龟比养猫狗省事多了。不需要每天遛，花费时间少。龟很安静，不会影响室友。养龟还能缓解学习压力。观察龟龟很有趣，是很好的放松方式。

宿舍养龟的准备工作。确认宿舍是否让养宠物。跟室友沟通好，避免矛盾。准备好龟缸和基本设备。规划好养龟的开销。确定自己能负责到底。

龟缸的选择。宿舍空间小，要选择小缸。二十到四十厘米的整理箱就可以。不需要太复杂，够用就好。方形缸比圆形缸省空间。可以放在书桌或阳台上。

适合学生养的龟。草龟是很好的选择，价格实惠。巴西龟也很常见，容易买到。蛋龟体型小，适合小缸。但要注意品种是否合规。

饲养设备的选择。学生党预算有限，选择性价比高的。加热棒选择带温控的，安全第一。晒背灯可以用uva替代，不用太贵。过滤可以用小瀑布，简单实用。

喂食问题怎么解决。龟粮是最方便的。买一袋能吃很久。新鲜食物可以偶尔喂。周末回家可以补充喂食。喂食量不要太多，避免浪费。

换水问题怎么处理。宿舍没有专门的水管。换水可以用桶装水或晾晒的自来水。小缸换水比较方便，一周一次即可。也可以用过滤减少换水次数。注意不要影响室友。

宿舍温度问题。宿舍有暖气冬天不用担心。夏天开空调要注意温差。没有暖气可以用加热棒。昼夜温差大要注意保温。温度稳定龟才能健康。

学习忙的时候怎么照顾。龟很耐饿，一周不喂也没事。提前准备好食物和水。周末可以集中照顾。学习再忙也要看一眼龟的状态。养龟不会占用太多时间。

假期龟怎么办。短期假期可以留校照顾。长期假期可以寄养在同学家。或者让家人帮忙照看。也可以把龟带回家。注意运输途中的安全。

如何隐蔽地养龟。如果不能公开养，可以选隐蔽的缸。放在柜子上或书架上。用收纳箱养，外表看不出来。跟室友沟通好还是最好的。偷偷养容易出问题。

养龟的费用预算。入门设备一百到两百元。每月的龟粮花费很少。换水用的水费不多。总花费比养猫狗少很多。是学生党也能承受的宠物。

养龟对学习的帮助。养龟可以放松心情，缓解压力。观察龟龟能培养耐心。养宠物是很好的情感寄托。课余时间有事情做，不会无聊。但不要沉迷，耽误学习。

和室友相处的注意事项。提前沟通好，获得同意。保持卫生，定期换水。不影响室友的生活。龟缸放在不碍事的地方。关系处好很重要。

学生党养龟的心态。养龟是长期的承诺。不能三分钟热度。放假要带回家，不能遗弃。学习重要，养龟也要用心。龟龟也是一条生命。

最后总结一下。学生党完全可以养龟。准备好基本的设备就行。龟很省心，不占用太多时间。但要有责任心，不能半途而废。养龟是很好的经历。
