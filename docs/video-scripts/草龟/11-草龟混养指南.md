# 草龟混养指南，这些搭配安全又好看

哈喽大家好，欢迎回来！很多朋友不满足于只养草龟，想在龟缸里放一些鱼螺藻，形成一个完整的生态系统。这期视频就来说说草龟混养的那些事，什么能混，什么不能混，一次讲清楚。

首先说说混养的好处。混养可以增加观赏性，让缸里更热闹。鱼可以吃掉水中的浮游生物和藻类，螺可以清理缸壁的藻类和粪便。形成一个小的生态循环，减少换水频率。龟缸也会变得更加生动有趣。

适合与草龟混养的鱼类。第一推荐的是斑马鱼，价格便宜，适应能力强。斑马鱼游动速度快，草龟不容易抓到。黑壳虾也是很好的选择，能清理缸中的藻类和食物残渣。但是黑壳虾容易被草龟吃掉，数量要放够。草金鱼可以混养，但要选择体型大一点的。小金鱼容易被草龟当成食物。

适合与草龟混养的螺类。苹果螺是经典选择，繁殖能力强，能吃掉缸壁的藻类。斑马螺汽水螺不会繁殖，适合喜欢螺的朋友。神秘螺颜色漂亮，但会长毛毛虫。螺类可以有效保持缸壁清洁，是很好的清洁工。

可以尝试的混养生物。泥鳅可以清理底层食物残渣，但要选体型大的。灯鱼比如红绿灯、宝莲灯，体型小游动快，不太容易被抓住。水草缸里可以放一些黑壳虾，既能吃藻又能当活饵。

绝对不能混养的生物。凶猛的鱼类不能和草龟混养，比如地图鱼、罗汉鱼。体型太小的鱼会被草龟吃掉，没有意义。热带鱼不能和草龟混养，水温要求不同。有攻击性的龟类不能混养，比如鳄龟、巴西龟。

混养的水质要求。草龟喜欢偏碱性的水，水质要求不高。但混养鱼螺后，水质要求就高了。氨氮和亚硝酸盐不能超标，否则鱼螺会死。需要强大的过滤系统来维持水质。可以放一些硝化细菌制剂，帮助建立菌群。

混养的喂食技巧。喂食要先喂鱼，再喂龟。龟粮要放在龟经常活动的地方。食物不要一次投喂太多，避免污染水质。可以选择沉底性的龟粮，让龟在底层吃。活饵比如小鱼苗要控制数量，不要放太多。

空间分配要合理。龟缸要有深水区和浅水区，让鱼有躲避空间。放一些水草和躲避物，让鱼有地方藏身。晒台要稳固，龟能爬上去休息。不要放太多造景素材，占用太多空间。

混养的数量控制。鱼的数量不要太多，根据缸的大小来决定。一般六十厘米的缸养五到十条斑马鱼就够了。螺的数量也要控制，繁殖太快会爆缸。草龟的数量根据缸的大小来定，通常一只成龟需要六十厘米以上的缸。

混养的风险与注意事项。草龟虽然温和，但饿极了也会捕食鱼。混养初期要密切观察，看龟是否攻击鱼。如果龟一直追着鱼抓，要及时隔离。混养鱼死后要及时捞出，防止污染水质。

适合新手的混养组合。推荐斑马鱼加苹果螺的组合，安全又省心。斑马鱼便宜，死了也不心疼。苹果螺能清理藻类，保持缸壁干净。这个组合对水质要求不高，适合新手尝试。

最后总结一下。草龟混养要选择合适的搭档，控制好数量，注意喂食和水质。混养是个循序渐进的过程，先从简单的组合开始，慢慢增加。养好了非常有意思，能体验到生态缸的乐趣。
