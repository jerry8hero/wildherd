# 草龟混养指南，这些搭配安全又好看

哈喽大家好，欢迎回来！很多朋友不满足于只养草龟，想在龟缸里放一些鱼螺藻，形成一个完整的生态系统。这期视频就来说说草龟混养的那些事，什么能混，什么不能混，一次讲清楚。

首先说说混养的好处。混养可以增加观赏性，让缸里更热闹。鱼可以吃掉水中的浮游生物和藻类，螺可以清理缸壁的藻类和粪便。形成一个小的生态循环，减少换水频率。龟缸也会变得更加生动有趣。

适合与草龟混养的鱼类。首推斑马鱼，价格便宜，适应能力强，游动速度快，草龟不容易抓到。草金鱼可以混养，但要选择体型大一点的，小金鱼容易被草龟当成食物。灯鱼比如红绿灯、宝莲灯也可以尝试，它们体型小游动快，不太容易被抓住，但只能和亚成体草龟混养，成年草龟一口一个。

适合与草龟混养的螺类。苹果螺是经典选择，能吃掉缸壁的藻类，但繁殖能力极强，容易爆缸。斑马螺汽水螺不会繁殖，适合喜欢螺的朋友。神秘螺颜色漂亮，但会产下粉色的卵块，孵化后数量会暴增。螺类可以有效保持缸壁清洁，是很好的清洁工。

可以尝试的混养生物。水草缸里可以放一些黑壳虾，但必须明确：黑壳虾对草龟来说是极品零食，不是清洁工，放多少都会被吃光。苹果螺和斑马螺是更好的选择。

绝对不能混养的生物。凶猛的鱼类不能和草龟混养，比如地图鱼、罗汉鱼。热带鱼不能和草龟混养，水温要求不同。有攻击性的龟类不能混养，比如鳄龟、巴西龟。另外特别注意：清道夫等带刺的异形鱼不能养，它们会吸附在龟壳上吸食粘液，导致龟甲溃烂。

混养的水质要求。草龟是出了名的"造粪机器"，排泄量巨大，对水质的污染远超鱼类。混养鱼螺后，水质要求就更高了。氨氮和亚硝酸盐不能超标，否则鱼螺会死。必须使用强大的过滤系统，建议过滤流量要达到水体体积的5到10倍。可以放一些硝化细菌制剂，帮助建立菌群。普通小鱼缸的过滤根本不够，最好用沼泽过滤或周转箱过滤。

混养的喂食技巧。正确的做法是把龟拿出来单独喂食，或者在固定位置投喂龟粮，等龟吃完后及时捞出残渣。千万不要把龟粮直接撒进缸里，龟粮泡久了会严重坏水。龟粮要放在龟经常活动的地方，让龟自己吃。食物不要一次投喂太多，避免污染水质。如果龟粮有剩余，要及时捞出。

空间分配要合理。龟缸要有深水区和浅水区，让鱼有躲避空间。造景方面不建议放真水草，草龟是"推土机"，会连根拔起咬烂水草，而且草龟排泄量大，真水草在重污染水体中很难存活。建议用沉木、鹅卵石或假水草来提供躲避。晒台要稳固，龟能爬上去休息。不要放太多造景素材，占用太多空间。

混养的数量控制。鱼的数量不要太多，根据缸的大小来决定。一般六十厘米的缸养五到十条斑马鱼就够了。螺的数量也要控制，繁殖太快会爆缸。草龟的数量根据缸的大小来定，通常一只成龟需要六十厘米以上的缸。

混养的风险与注意事项。第一，体型是关键：龟苗时期不要和成体鱼混养，鱼可能咬龟尾巴；成年草龟不要和小型鱼混养，龟一口一个。第二，草龟虽然温和，但饿极了也会捕食鱼，混养初期要密切观察。第三，混养鱼死后要及时捞出，防止污染水质。

适合新手的混养组合。推荐斑马鱼加斑马螺的组合，安全又省心。斑马鱼便宜游得快，死了也不心疼。斑马螺不会繁殖爆缸，能清理藻类。这个组合对水质要求不高，适合新手尝试。注意：如果草龟已经是成体，就不要混养任何小型鱼了，成体草龟的嘴能塞进什么就会吃什么。

最后总结一下。草龟混养要选择合适的搭档，控制好数量，注意喂食和水质。混养是个循序渐进的过程，先从简单的组合开始，慢慢增加。养好了非常有意思，能体验到生态缸的乐趣。
