简介：
龟龟不吃东西急坏了很多新手朋友？拒食是养龟过程中最常见的问题，但这背后原因各不相同。这期视频全面解析草龟拒食的9大原因和对策，让你快速找到问题所在，对症处理！

这期内容要点：

✅ 新环境适应：1-2周不吃正常，别经常打扰
✅ 温度是关键：低于20℃少吃，低于15℃停食；加温到25℃以上
✅ 肠胃问题：停食3-5天+益生菌调理，严重的需要隔离药浴
✅ 应激反应：新龟到家、换缸、换水都会应激，保持环境稳定
✅ 疾病信号：眼睛发红/浮水/张嘴呼吸=生病了，要对症治疗
✅ 挑食问题：饿治法最有效，停食一周再喂龟粮
✅ 发情期拒食：公龟常见，正常现象，无需处理
✅ 冬眠期不吃：低于15℃进入冬眠，正常；但5cm以下幼龟建议加温过冬
✅ 母龟卡蛋：焦躁爬动、后腿挖掘状，可能是卡蛋，需及时检查

龟不吃东西先别慌！仔细观察、分析原因才是关键。建议收藏备用！

## 评论区置顶

📌 视频补充（拒食问题快速排查）：

🌡 温度先行
→ 低于20℃少吃，低于15℃基本不吃
→ 解决方案：加温到25-28℃

🤒 观察症状
→ 眼睛发红/有分泌物 → 可能是感冒或肺炎
→ 浮水/张嘴呼吸 → 呼吸道问题
→ 拉稀/拖便 → 肠胃炎

💊 肠胃问题处理
→ 停食3-5天
→ 轻微：益生菌调理
→ 严重：隔离浅水药浴，别乱用猛药

😰 应激反应
→ 新龟到家、换缸、换水都会应激
→ 保持环境稳定，别经常打扰

🍖 挑食问题
→ 饿治法：停食一周再喂龟粮
→ 龟饿了自然会吃

⚠️ 注意
→ 小龟苗（5cm以下）体质弱，不建议冬眠
→ 15℃左右的低温水最伤龟，消耗大却无法进食

💡 记住：龟不吃东西先别急！仔细观察、分析原因才是关键！
