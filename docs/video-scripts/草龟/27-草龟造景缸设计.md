# 草龟造景缸设计，这样做既美观龟也住得舒服

哈喽大家好，欢迎回来！很多养龟的朋友不满足于简单的周转箱，想要一个漂亮的生态造景缸。这期视频就来说说草龟造景缸的设计要点，既要美观更要实用。

首先说说缸体的选择。草龟造景缸建议使用超白玻璃缸，透明度高方便观赏。缸的尺寸根据龟的数量和大小来定，一只成龟建议用六十厘米以上的缸。缸不要太高，方便换水和清理。方形缸比圆形缸更适合造景，空间利用率高。

底材的选择与铺设。底材可以选择火山石、水草泥或者专门养龟的底砂。火山石透气性好，还能培养硝化细菌。铺设厚度五到八厘米即可，前低后高形成坡度。可以在缸底铺一层陶粒作为底层，再铺火山石。

过滤系统的配置。造景缸的过滤很重要，推荐使用底滤或侧滤。过滤泵的流量要达到水体体积的六到八倍。物理过滤用过滤棉，生化过滤用陶瓷环和细菌屋。过滤出水口要设计成雨淋式，增加水中溶氧。

晒台的设计。晒台是造景缸必不可少的部分，可以用石头、沉木或专门的晒台设备。沉木是最推荐的选择，既是晒台又能提供躲避。晒台要放在灯光能照射到的地方。晒台高度要能让龟轻松爬上去，背甲能完全露出水面。

水草的种植。适合草龟缸的水草有很多，比如水榕、皇冠草、金鱼藻等。水草可以吸收水中杂质，改善水质。种植时要用镊子把根部埋入底砂中。水草需要光照，可以用专业的水草灯每天照射六到八小时。

躲避物的设置。草龟胆子小，需要躲避物来增加安全感。躲避可以用陶瓷罐、椰壳或沉木洞穴。躲避要放在阴暗的角落，不要放在太显眼的位置。躲避的口不要太大，龟能进出即可。

造景素材的选择。可以用石头、沉木、杜鹃根来造景。素材要提前清洗浸泡，去除杂质。摆放时遵循前低后高的原则，层次分明。避免使用尖锐的石头，防止龟甲受伤。沉木可以选造型自然的，造景效果更好。

灯光系统的配置。造景缸需要专业的水族灯，既能照亮又能促进水草生长。选择LED灯，省电又耐用。每天开灯八到十二小时，要有定时器控制。灯具要选择防水的，安全第一。

维护与清洁。造景缸维护比裸缸复杂一些，每周换水三分之一即可。水泵要定期清洗，保持出水畅通。水草需要定期修剪，去除烂叶。底砂可以用洗沙器定期清理粪便。造景素材表面会长藻，要及时刷洗。

造景的注意事项。不要放太多素材，会占用太多活动空间。避免使用会改变水质的素材，比如石灰石。造景要留出足够的游动空间。不要放有毒的素材，比如有些木头会释放毒素。

适合混养的生物。造景缸可以放一些工具螺，比如苹果螺、斑马螺。可以放一些工具鱼，比如黑壳虾、小金鱼。注意混养生物不要太多，否则会影响水质。混养生物要以工具类为主，不要放太多观赏鱼。

最后总结一下。草龟造景缸设计的原则是美观实用兼顾，过滤要强大，晒台要稳固，水草要选好养的。造景是个循序渐进的过程，可以慢慢完善。
