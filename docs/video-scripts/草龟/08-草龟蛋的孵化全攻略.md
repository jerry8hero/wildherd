# 草龟蛋的孵化全攻略，看完就能自己孵化

哈喽大家好，欢迎回来！上期我们讲了草龟繁殖的条件与配种，这期来说说最关键的环节，草龟蛋的孵化。孵化成功率直接决定了繁殖的成败，新手朋友一定要仔细看。

首先说说如何收集龟蛋。雌龟产蛋后要及时收集，最好在产蛋后两小时内完成。龟蛋不能见强光，移动时要轻拿轻放。收集时不要旋转龟蛋的方向，保持原来的摆放姿势。龟蛋上有白色的受精斑，要在二十四小时内标记清楚，方便后续管理。

如何判断受精。龟蛋产出后二十四小时左右，会出现白色的受精斑。受精斑通常出现在蛋的中间位置，呈圆形或椭圆形。没有受精斑的是白蛋，无法孵化。受精斑会随着时间推移慢慢扩散，这是正常的。

孵化介质的选择。常用的孵化介质有蛭石、椰土和蛭石混合土。推荐使用蛭石，保湿透气性好。蛭石要用开水烫一下消毒，晾凉后使用。湿度以手捏成团但不滴水为准。孵化盒要有透气孔，保持空气流通。

温度对孵化的影响。温度决定了龟苗的性别和孵化时间。孵化温度在二十八到三十度之间，通常五十五到六十天就能出壳。温度高于三十度，孵化时间会缩短，但畸形率会增加。温度低于二十六度，孵化时间会延长到八十天以上。最好使用孵化箱恒温孵化，温差不能超过两度。

湿度管理很重要。孵化期间要保持介质湿度稳定。每周检查一次，用喷壶适当补水。孵化盒内可以放一杯水，保持环境湿度。湿度太低会导致龟苗干死，湿度太高会导致霉蛋。孵化后期可以适当降低湿度，帮助龟苗顺利破壳。

孵化箱的使用方法。孵化箱要提前设置好，温度稳定后再放入龟蛋。孵化箱内放一个温湿度计，随时监测。孵化盒要放在孵化箱中间位置，远离加热管。孵化期间尽量不要打开孵化箱，避免温差过大。

龟蛋的摆放有讲究。龟蛋要横放在孵化介质中，埋入三分之二。不要竖着放，也不要压在一起。每个龟蛋之间要保持一定距离，方便观察。每个孵化盒不要放太多蛋，一般不超过十枚。

孵化期间的检查。孵化第一周不要翻动龟蛋，避免影响胚胎发育。从第二周开始，可以每周检查一次。检查时轻轻取出，观察蛋的变化。如果发现发霉的蛋要及时清除，防止传染其他蛋。发育正常的蛋会慢慢变大，颜色变深。

龟苗破壳的信号。孵化后期，蛋表面会出现裂纹，这是龟苗即将破壳的信号。龟苗会用头部的破壳齿顶开蛋壳。破壳过程可能持续一到两天，不要急于帮忙。破壳后不要立即取出，让它在孵化盒中待几个小时，脐带完全收好后再移出。

龟苗出壳后的处理。龟苗出壳后要在孵化盒中静养一到两天，等卵黄囊完全吸收再下水。下水时水位要浅，只没过背甲一半。刚开始不要喂食，等龟苗适应后再投喂开口粮。

最后提醒几点。第一，孵化期间不要频繁打开孵化箱。第二，发现死蛋要及时清除，防止发霉污染。第三，龟苗体质较弱，第一周最好加温到二十八度饲养。
