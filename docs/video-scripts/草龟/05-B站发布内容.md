## 评论区置顶

📌 视频补充（出眠养护必知）：

⏰ **出眠时机**
→ 气温稳定在18度以上
→ 龟龟会自然苏醒
→ 南方2-3月，北方4-5月

🚫 **出眠后不要马上喂**
→ 肠胃功能未恢复
→ 容易引发肠胃炎
→ 等完全清醒、排便后再喂

🍽 **第一次喂食**
→ 少量，约为正常量的1/3
→ 先喂好消化的食物（龟粮泡软/虾肉）
→ 温度低于20度不要喂

🔍 **状态检查**
→ 眼睛、鼻孔、嘴巴有无分泌物
→ 甲壳有无腐皮、烂甲
→ 四肢力度是否正常
