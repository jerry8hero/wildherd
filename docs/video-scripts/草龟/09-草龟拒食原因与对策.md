# 草龟拒食原因与对策，看完再也不愁

哈喽大家好，欢迎回来！养龟过程中最常见的问题就是拒食，龟不吃东西急坏了很多新手朋友。这期视频就来说说草龟拒食的原因和对策，让你的龟龟早日恢复食欲。

首先说说最常见的原因，新环境适应期。新龟到家后拒食是正常的。需要一到两周时间来适应环境。期间不要经常打扰它。保持水质清洁，水温适宜。适应后会慢慢开始进食。

第二个原因是温度太低。草龟是变温动物，温度低就没有食欲。水温低于二十度开始少吃。低于十五度基本不吃东西。解决方法很简单，加温到二十五度以上。温度上去后食欲就会恢复。

第三个原因是肠胃问题。
喂食不当会导致肠胃炎。症状除了拒食，还有拉稀、粪便不成型甚至拖便。治疗方法：先停食静养三到五天，保持水温稳定。轻微的用益生菌调理，严重的出现张嘴呼吸或浮肿，需要隔离进行浅水药浴，切勿乱用猛药，咨询专业兽医或有经验的玩家更为稳妥。

第四个原因是应激反应。环境变化或受到惊吓会拒食。新龟到家、换缸、换水都会应激。保持环境稳定，不要经常打扰。等龟慢慢适应后会恢复进食。

第五个原因是疾病。肺炎、感冒等疾病会导致拒食。观察龟有没有其他症状。眼睛是否发红、有分泌物。是否浮水、呼吸困难。发现问题要对症治疗。

第六个原因是挑食。长期喂一种食物龟会吃腻。可以尝试换一种龟粮。或者喂一些新鲜的食物，比如虾肉、鱼肉。多样化的食物可以提高食欲。

这里教大家一个狠招——饿治法。草龟非常喜欢吃活食，一旦吃过鱼虾就不爱吃龟粮了。解决方法很简单：先停食一周，然后再尝试喂龟粮。龟饿了自然会吃，这是解决挑食最有效的方法，观众用过都说好。

第七个原因是发情期。成龟在发情期会拒食，尤其是公龟。这是正常现象，持续时间不会太长。发情期过后会自动恢复进食。不需要特别处理。

第八个原因是冬眠期。气温低于十五度，龟会进入冬眠，这期间不吃是正常的。但要注意，如果你的龟太小（背甲小于五厘米）或者体质弱，建议不要冬眠，使用加热棒加温过冬，正常喂食。千万不要让龟在十五度左右的低温水里硬扛，那样最伤龟，新陈代谢紊乱却无法进食，很容易消耗殆尽甚至死亡。

第九个原因是难产或卡蛋。如果是养了多年的母龟，且近期有焦躁爬动、后腿呈挖掘状，可能是卡蛋了，这会导致拒食，需要及时检查。

如何判断龟是否生病。这里给大家几个判断要点，画面上可以配合展示正常龟和病龟的对比图。观察精神状态，是否有活力。检查眼睛是否明亮、有无分泌物（健康的眼睛应该是清澈有神的）。看看甲壳有没有异常，比如腐甲、发软。排便是否正常。综合判断才能确定原因。

促进食欲的方法。画面上可以展示加温设备示意。提高水温到二十八度左右，这是草龟最活跃的温度。提供躲避处，让龟有安全感。用虾肉或鱼腥味重的食物引诱，腥味对草龟来说是无法抗拒的。喂食前先逗引一下龟，增加运动也能促进食欲。保持环境安静，减少干扰。

喂食的技巧。可以尝试把食物放在晒台附近。或者把食物放在龟的头部前方。不要一次放太多，吃不完会污染水质。可以尝试用手喂，增加互动。

什么情况下需要就医。这里画面可以展示几张典型症状的龟照供大家对照。拒食超过一个月没有改善。伴随其他症状比如浮水、粪便异常。龟看起来精神很差，越来越瘦，这时候就要考虑找兽医看看了。

最后总结一下。拒食原因有很多，要仔细观察分析。常见原因是温度、环境、应激和疾病。针对不同原因采取不同对策。保持耐心，龟会慢慢好起来的。
