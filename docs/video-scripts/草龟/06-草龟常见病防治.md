# 草龟常见病防治，新手必看

哈喽大家好，欢迎回来！

很多新手朋友养龟，最怕的就是龟龟生病。而且往往等到病情严重了才发现问题，结果错过了最佳治疗时间。

特别要提醒大家的是：有些龟龟冬眠前明明很健康，但出眠时却发现生病了。这是因为冬眠期间也存在很多风险，比如水质恶化、温度波动、介质不干净等。所以出眠时一定要仔细检查龟龟的状态，发现异常及早处理。

在说疾病之前，说个题外话。这是我个人的养龟理念：龟龟不是用价格来衡量的。既然选择养了，就应该对它负责到底。当然，这只是我自己的想法，不强求大家都认同。

防病重于治。如果龟龟病情严重，一定要及时找专业兽医处理，不要自己乱用药。

先说腐皮病。症状是皮肤出现白色或黄白色的斑块，有腐烂现象。草龟皮壳薄，容易受伤后细菌感染。治疗方法是隔离干养，但注意不是完全离开水，每天泡水两三次用来进食和补水，其他时间擦干上药，用聚维酮碘消毒，涂抹金霉素软膏。早发现早治疗，一般一周左右就能恢复。

再说腐甲病。症状是龟甲局部发软、变色、有异味。原因是外伤或水质问题。治疗需要先清理腐烂组织，用聚维酮碘消毒，然后涂抹抗生素软膏。注意，严重的腐甲一定要送医处理。腐甲初期很好治疗，但如果拖延到腐烂穿透甲壳，治疗就很麻烦了。

肺炎是致命性疾病，一定要重视。症状是张嘴呼吸、流鼻涕、侧浮、不爱动。原因是温差太大。治疗方法是隔离加温到28到30度，用益生菌和电解质多维浴增强体力。千万注意，肺炎龟不能完全干养，需要浅水饲养保持湿度。肺炎治疗及时的话成功率很高，但如果拖到后期就很危险了。

肠炎也很常见。症状是排便稀薄、颜色异常、有异味。原因是喂食不当或水质问题。治疗方法是停食几天，保持水温28度左右，用益生菌调理肠胃。肠炎期间要特别注意补充水分，防止脱水。

软甲病要注意。症状是龟甲软化、变形。原因是缺乏紫外线照射或缺钙。治疗方法是多晒背，补充维生素D3，有条件的可以用UVB灯照射。

最后说说预防。保持水质干净、控制好温度、合理喂食是预防疾病的关键。草龟胆子小，应激反应也可能导致生病。新龟到家要隔离观察两周以上，平时多观察龟龟的状态，发现异常及时处理。

记住，发现龟龟异常，及早处理是关键。拖得越久，治疗难度越大，成功率越低。
