# 如何挑选健康的草龟，记住这几点就不会被坑

哈喽大家好，欢迎回来！买龟是养龟的第一步，但很多新手朋友不会挑，往往买到病龟或者体质弱的龟。这期视频就详细说说如何挑选健康的草龟全是干货，建议收藏。

首先说说购买渠道的选择。推荐去当地信誉好的水族店或者爬宠店购买。能现场挑选，看到龟的真实状态。网购虽然价格便宜，但运输风险大，容易在路上出问题。如果选择网购，要选信誉高的店铺，看看评价再决定。

什么时间买龟最好。春秋季是买龟的最佳时间，气温适宜，龟的状态好。尽量避开夏季高温和冬季寒冷的时候。买龟要选择气温稳定在二十度以上的日子。天气不好时不要买龟，运输容易出问题。

挑选时的第一点，看精神状态。健康的龟爬动有力，四肢挣扎活跃。把龟翻过来，它能迅速翻回去。健康的龟眼睛明亮有神，眼皮不肿胀。不健康的龟眼神呆滞，经常闭眼。

挑选时的第二点，看甲壳状况。健康的草龟甲壳完整，没有裂纹或缺口。甲壳表面光滑，纹路清晰。腹甲平整，没有发软或腐烂的地方。用手指轻压甲壳，感觉坚硬有弹性。如果甲壳发软，说明缺钙或者有其他问题。

挑选时的第三点，看皮肤和四肢。皮肤干净，没有斑点、溃烂或白色附着物。四肢完整，指甲齐全。指甲如果有断裂，会影响捕食和攀爬。检查腋下和腿窝处，这些地方容易藏寄生虫。

挑选时的第四点，看嘴巴和鼻子。嘴巴张开时对称，没有流涎或黏液。鼻子干净通畅，没有分泌物。如果鼻子冒泡或者有黏液，很可能是感冒或肺炎。嘴巴经常张开的龟不要买，可能是呼吸道问题。

挑选时的第五点，看排便情况。如果有条件，可以观察龟的粪便。健康的龟粪便成型，呈深绿色或黑色。没有异常的臭味。如果龟拉稀或者粪便异常，说明肠胃有问题。

挑选时的第六点，测试反应速度。把龟放在地上，看它爬行的速度。健康的龟爬动迅速，方向感强。反应迟钝的龟可能是体质弱或者生病了。健康的龟看到有人靠近会把头缩进去。

挑选时的第七点，沉浮测试。把龟放入水中，看它是沉底还是浮水。健康的龟会沉入水底，游泳有力。浮水的龟可能有肺炎或者其他疾病。侧浮的龟绝对不能要，这是肺炎的典型症状。

挑选苗子要注意的事项。苗子体质弱，挑选要更仔细。看肚脐是否收好，没有收好的容易感染。苗子要选活泼好动的，不要选总是缩着的。苗子太小不建议买，最好选三厘米以上的。

挑选成龟要注意的事项。成龟要选体型饱满的，太瘦的不要。背甲颜色均匀，没有发白或褪色。检查生殖器是否外露，外露的可能是脱肛。成龟的年龄可以通过年轮判断。

如何判断商家的龟是否靠谱。看商家的饲养环境是否干净。健康的龟和病龟分开养，这是基本的。看商家能否提供饲养指导。价格太低的龟不要买，很可能有问题。

买龟后的处理。新龟到家不要急于下水，先干养一到两小时。水的深度不要超过龟的背甲。静养三天后再开始喂食，这期间让它适应新环境。密切观察龟的状态，发现异常及时处理。

最后总结一下。挑选草龟要一看精神二看壳，三看皮肤四看嘴。鼻子眼睛要干净，爬行沉浮要正常。记住这些要点，就不会被商家坑了。
