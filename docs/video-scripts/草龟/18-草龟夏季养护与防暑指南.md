# 草龟夏季养护与防暑指南，让龟安全度夏

哈喽大家好，欢迎回来！夏季高温酷暑，对草龟来说是个考验。温度太高容易中暑，水质也容易变坏。这期视频就来说说草龟夏季养护的注意事项，让你的龟安全度过夏天。

首先说说夏季的温度控制。草龟最适宜的温度是二十五到二十八度。超过三十二度就会有中暑风险。温度超过三十五度非常危险，可能导致死亡。有条件的话最好使用空调或降温设备。保证龟在舒适的环境中生活。

如何给龟缸降温。第一种方法是使用风扇，帮助空气流通。风扇不要对着龟直吹，容易着凉。第二种方法是换水降温，方法是换掉部分热水，换入凉水。第三种方法是在缸上覆盖遮阳网，避免阳光直射。第四种方法是放置冰袋，但不要直接放入水中。

水位调整有讲究。夏季蒸发快，要及时补充水分。水位可以适当加深，水体大温度更稳定。但要注意晒台不能被淹没，龟需要随时出水呼吸。加深水位的同时要确保龟能轻松游到水面。

夏季水质容易变坏。高温环境下细菌繁殖加快，龟的排泄物也更多。换水频率要比其他季节高一些。可以每两到三天换一次水。每次换水不超过三分之一，保持水质稳定。

夏季喂食的注意事项。高温会影响龟的食欲，喂食量要适当减少。食物残渣要及时清理，防止腐败水质。可以选择在早晚凉爽的时候喂食。喂食时间避开正午高温时段。

夏季适合喂的食物。新鲜的鱼虾富含蛋白质，是很好的选择。蔬菜可以多喂一些，比如南瓜叶、生菜。适当减少肉类，防止消化不良。西瓜等水果可以偶尔喂，但不能太多。

夏季常见疾病与预防。肠胃炎是夏季高发病，和喂食不当有关。腐皮病也常见于水质差的环境。中暑是最危险的情况，会危及龟的生命。预防关键是控制温度和保持水质。

如何判断龟是否中暑。中暑的龟会表现得无精打采。龟会张大嘴巴呼吸，呼吸急促。龟身发软，没有力气爬动。严重时会出现漂浮在水面无法下沉的情况。中暑要立即抢救，及时降温。

中暑的急救方法。发现中暑要立即把龟移到阴凉处。用凉水慢慢降温，不要用冰水。用风扇帮助散热，补充水分。严重时要及时就医，不要拖延。

夏季晒背要注意。夏季阳光强烈，晒背时间要控制好。建议早上九点前或下午四点后晒背。晒背时要有遮阳处，让龟自己选择。中午不要让龟直接晒太阳，容易中暑。

室内养龟的注意事项。不要把龟缸放在阳光直射的地方。空调不要对着龟缸吹，温差大会生病。使用加热棒要注意检查，防止干烧。夏季电器使用多，注意用电安全。

室外养龟的注意事项。室外龟池要搭建遮阳棚。防止雨水进入，导致水质变化。要防止蚊虫滋生，影响龟的健康。野生天敌比如鸟类要防范。定期检查环境，确保安全。

夏季繁殖的注意事项。夏季是草龟的交配季节。如果准备繁殖，要提供合适的环境。交配后要注意观察母龟是否产蛋。高温会影响龟蛋的孵化，注意控制温度。

最后总结一下。夏季养龟关键是控制温度，保持水质。喂食要适量，及时清理残渣。防止中暑，注意晒背时间。做好了这些，龟龟就能安全度夏。
