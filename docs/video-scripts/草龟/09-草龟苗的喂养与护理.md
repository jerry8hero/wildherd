# 草龟苗的喂养与护理，新手必看

哈喽大家好，欢迎回来！上期我们讲了龟蛋的孵化，这期来说说出壳后的龟苗该怎么养。草龟苗非常脆弱，护理不当很容易夭折，新手朋友一定要认真看。

首先说说龟苗到家后的处理。如果是网购的龟苗，收到后不要急于下水。先把龟苗放在阴暗安静的地方休息一到两小时。让它慢慢适应环境温度，然后放入浅水中。水位只没过背甲三分之一即可，水太深会导致龟苗体力不支溺水。

开口粮的选择。龟苗第一次吃东西叫开口，常用的开口粮有寸金、育归等幼龟粮。也可以喂煮熟的鸡蛋黄，捏碎后投喂。丰年虾幼虫也是很好的选择，营养丰富。用温水泡软龟粮后再投喂，方便龟苗吞咽。

喂食频率与方法。龟苗消化能力弱，每天喂一到两次即可。每次喂食量以龟苗头部大小为宜，在十五分钟内吃完最好。喂食时把食物放在龟苗前方，让它自己采食。如果龟苗不吃，可以尝试换一种食物。

水质管理要仔细。龟苗对水质非常敏感，水质不好很容易腐皮。小水体要每天换水，大水体可以两到三天换一次。换水时水温差不要超过两度。自来水要提前晾晒两天再用。换水时不要一次性全部换掉，留三分之一原水。

温度控制很关键。龟苗需要加温饲养，水温控制在二十八到三十二度。使用加热棒要选择质量好的，最好带温控器。加热棒要完全浸入水中，放在安全的位置。昼夜温差不要超过三度，防止龟苗感冒。

幼龟粮的选择。随着龟苗长大，可以慢慢过渡到普通幼龟粮。选择正规品牌的龟粮，营养配比更均衡。颗粒大小要根据龟苗的口型选择，太大的粮它吃不了。可以偶尔喂一些鱼虾肉，补充蛋白质。

晒背的重要性。龟苗需要晒背来促进钙质吸收。每天晒一到两小时，太阳光不要太强烈。早上九点到十点的阳光最温和。晒背时要有遮阳的地方，让龟苗自己选择。室内饲养要用专业的UVB晒背灯。

疾病预防。龟苗最常见的疾病是腐皮和肠炎。保持水质清洁是预防腐皮的关键。喂食要适量，吃不完的食物及时捞出。温差变化不要太大，避免龟苗感冒。新买的龟苗要隔离观察一周再合缸。

冬眠注意事项。当年的龟苗建议加温过冬，不要冬眠。体质弱的龟苗冬眠风险很高。使用加热棒把水温维持在二十八度左右，正常喂食。等到来年体质健壮后再考虑冬眠。

分箱饲养很重要。如果一次孵化多只龟苗，建议分箱饲养。体型差异大的龟苗要分开养，避免大龟欺负小龟。争抢食物时小龟容易吃不到，影响生长。分开饲养每只龟苗都能健康成长。

环境布置。龟苗缸不需要太复杂，一个整理箱就能满足。底材可以用火山石或裸缸，方便清理。放一块小石头当晒台，让龟苗能够爬上去休息。不要放太多装饰，避免藏污纳垢。

最后总结一下。龟苗饲养要把握几个重点，加温饲养、保持水质、稳定喂食。只要用心照顾，龟苗存活率会大大提高。不要急于求成，给龟苗一点时间成长。
