## 使用说明

1. 打开 runwayml.com，登录账号
2. 点击 Create → Generate Video
3. 选择 Gen-2 或 Gen-3 Alpha
4. 复制上方提示词到输入框
5. 可上传参考图片保持风格一致
6. 生成后下载 MP4
7. 导入剪映添加配音和字幕
