## 评论区置顶

📌 视频补充（龟龟喂食必知）：

🐢 **龟粮选择**
→ 幼龟粮蛋白高，成龟粮更均衡
→ 颗粒大小要适合龟龟大小

⏰ **喂食时间**
→ 春秋：中午喂 | 夏：早晚喂 | 冬：冬眠不喂

🍽 **喂食量**
→ 头部大小那么多，15分钟内吃完
→ 宁少勿多！

⚠️ **这些绝对不能喂！**
→ 变质食物、含盐含油食物（肾衰竭、水肿）
→ 牛奶、面包、巧克力、菠菜、洋葱、韭菜

🌞 **补钙与晒背**
→ 每周食物撒一点爬宠钙粉
→ 多晒太阳或配UVB灯，缺光龟壳易软变形

💧 **重要提醒**
→ 草龟必须在水中进食，靠水吞咽
→ 变温动物，低于20度不要喂，易引发肠炎
