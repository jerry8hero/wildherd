# 新手养草龟常见误区大盘点，看看你中招了没

哈喽大家好，欢迎回来！很多新手朋友养龟时都会犯一些错误，有些错误甚至会导致龟龟死亡。这期视频就来说说新手常见的误区，看看你中招了没。

第一个误区是频繁换水。很多新手怕水脏，每天换水甚至一天换几次。频繁换水会破坏水质稳定，让龟应激。水质不稳定是很多疾病的根源。正确的做法是根据水质情况，三到五天换一次水即可。

第二个误区是水位太深。新手往往以为龟水性好，把水位加得很深。草龟虽然水性不错，但苗子容易溺水。幼龟水位过背两三厘米就行。成龟也要有晒台，能轻松爬出水面。深水需要慢慢过渡，不能急于求成。

第三个误区是喂得太多。新手担心龟饿着，总是喂很多。喂太多会导致消化不良，肠胃炎。草龟不知道饥饱，有多少吃多少。每次喂食量以龟头部大小为宜。宁可饿着也不要喂多。

第四个误区是乱喂食物。新手觉得龟什么都能吃，这是大错特错。人类食物绝对不能喂，含盐含油会要命。生的肥肉会导致消化不良。冰箱里的食物不能直接喂，要彻底解冻。喂食要选择专门的龟粮和安全的食物。

第五个误区是温差忽视。温差大是龟的最大杀手。空调直吹、换水温差大都会导致感冒。天气变化时要注意保温。加热棒要选择质量好的，带温控的。水温要稳定在适宜的范围。

第六个误区是暴晒太阳。晒太阳对龟有好处，但暴晒会中暑。夏季中午不能晒，要选择早晚。晒背要有遮阳处，让龟自己选择。室内养龟用晒背灯就行，不用晒自然光。

第七个误区是混养不加考虑。新手觉得多养几只龟热闹，不考虑体型差异。大龟会欺负小龟，甚至咬伤咬死。不同品种的龟性格不同，混养要谨慎。养在一起之前要先隔离观察。

第八个误区是不了解就繁殖。新手觉得养龟很简单，想让它繁殖。龟没到年龄繁殖成功率很低。体质弱的龟繁殖会伤身体。繁殖需要很多知识，新手不要急于尝试。先把龟养好再说。

第九个误区是乱用药。新手一看到龟有问题就慌，用各种药。有些药对龟有副作用。剂量掌握不好会出问题。是药三分毒，能不用药就不用。用药前最好咨询有经验的人。

第十个误区是忽略静养。新手买到龟就急着上手玩。新龟到家需要静养，适应环境。经常上手会导致龟应激，生病。静养几天后再开始喂食。给龟时间，它会慢慢信任你。

第十一个误区是龟缸布置太复杂。新手喜欢把龟缸装饰得很漂亮。太多装饰品容易藏污纳垢。增加清理难度，还影响水质。简单的环境更利于养龟。养好了再慢慢布置也不迟。

第十二个误区是不了解冬眠。以为所有龟都必须冬眠。苗子体质弱，冬眠容易死亡。有些龟不适合冬眠，要加温饲养。冬眠前要做好准备，否则会出龟命。不了解冬眠知识前不要轻易尝试。

第十三个误区是轻信商家。新手容易被商家误导。商家说龟好养，其实可能有问题。商家推荐的套餐不一定适合。买龟前先做好功课。找信誉好的商家购买。

第十四个误区是不愿意学习。以为养龟很简单，不用学。养龟需要很多知识。不学习就会犯错误，龟会遭罪。现在网络发达，学习很方便。多看多问，慢慢就懂了。

最后总结一下。养龟看似简单，其实门道很多。新手容易犯的错误也很多。但犯错不可怕，可怕的是不改。多学习，多观察，有耐心。慢慢积累经验，龟会养得越来越好。
