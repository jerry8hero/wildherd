## 评论区置顶

📌 视频补充（龟圈黑话速查表）：

🔤 **入圈基础类**
→ 接龟=买龟带回家（含过温过水仪式感）
→ 龟坑=养龟这个坑，入坑容易出坑难
→ 龟团=龟群聚晒背，也指团购

🔤 **交易类**
→ 刀=砍价、不刀=不讲价、可小刀=可砍运费
→ 包邮=问清楚普快还是顺丰
→ 拼龟=合伙买、跳车=反悔、秒没=瞬间卖完

🔤 **龟种类**
→ 小青/奶耶/南石=黄喉拟水龟苗（颜色像奶茶）
→ 大三线=三线闭壳龟
→ 黑巴=黑颈乌龟
→ 台草=台湾草龟血统，价格较贵

🔤 **品相类**
→ 全品=完美、通货=普通
→ 断尾=尾巴断、错甲=甲壳排列异常
→ 翘甲=背甲边缘上翘、塌背=背甲中间凹陷

🔤 **避雷类（新手必看！）**
→ 温室龟=体质差易暴毙
→ 药浴龟=泡药假象勿轻信
→ 侧漂=肺炎症状
→ 腐皮腐甲=龟在求救

🔤 **状态类**
→ 发色=颜色变好、爆纹=新纹路长出
→ 团灭=全死、升天=死了

🔤 **身份类**
→ 压成=接近成年、成体=完全发育
→ 种公/种母=繁殖用龟
