# 草龟出眠养护指南，新手必看

哈喽大家好，欢迎回来！天气转暖，又到了草龟出眠的季节。草龟是中国原生龟种，经过漫长的冬眠，现在终于苏醒过来了。这期视频详细给大家说说草龟出眠后的养护要点。

首先说说如何判断龟龟是否出眠。气温稳定在十八度以上时，草龟会自然从冬眠中苏醒。这是判断出眠时机的主要依据。不同地区温度回升时间不同，南方可能二三月就醒了，北方可能要四五月份。

出眠后不要急着喂食，这是最重要的！龟龟刚苏醒，肠胃功能还没有完全恢复，马上喂食容易引发肠胃炎。正确的做法是等龟龟完全清醒、活动自如，再观察有没有正常排便，之后再开始少量喂食。

出眠后如何喂食有讲究。第一次喂食要少量，约为正常量的三分之一，让龟龟的肠胃慢慢适应。可以先喂一些好消化的食物，比如龟粮泡软或者少量虾肉。温度不稳定时不要喂，保持在二十度以上再喂食比较安全。

出眠后的状态检查不能少。检查龟龟的眼睛、鼻孔、嘴巴有没有异常分泌物。检查甲壳有没有腐皮、烂甲。检查四肢力度，强壮的龟会用力挣扎。如果发现异常要及时处理。

出眠后的水质管理要重视。冬眠期间的水质可能已经变差，要逐渐换水，但不能全部换掉，每次换三分之一左右。自来水一定要提前晾晒，去除氯气。水温要和室温接近，避免温差过大。

哪些龟龟出眠后需要特别照顾？体质虚弱、有伤病史的龟龟，出眠后要密切观察。当年没有冬眠的龟苗，第一年春天可能比较弱，需要加温照顾。

出眠后常见问题如何处理？不吃食：可能是温度不够，继续等待。精神萎靡：检查是否有病。排便异常：可能是肠胃炎，要停食观察。

总结一下：出眠不等同于马上喂食，等龟龟完全清醒、排便后再喂。第一次少量喂食，好消化为主。密切观察状态，发现问题及时处理。
