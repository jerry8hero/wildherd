# 草龟出眠养护指南，新手必看

哈喽大家好，欢迎回来！天气转暖，又到了草龟出眠的季节。

但是！草龟出眠不能急！春天虽然来了，但气温忽冷忽热很不稳定。如果不看温度就急着把龟弄醒，龟龟很容易生病甚至死亡。很多龟友就是没注意这点，结果反而害了自己的龟龟。

这期视频详细给大家说说草龟出眠的正确时机和方法。

首先说说如何判断龟龟是否可以出眠。关键只有一个：气温稳定在十八度以上。这里说的是稳定，不是今天二十度明天又降到十度这种。要持续一周以上温度都比较平稳才行。

不同地区情况不一样。南方回暖早，二三月份可能就稳定了。北方地区，特别是东北和西北，可能要到四五月份才具备出眠条件。所以不要看别人家的龟醒了就着急，每个地方气候不同。

什么时候可以出眠？让龟自己醒最好。如果龟还在冬眠，但气温已经稳定在十八度以上了，可以尝试把龟拿出来检查一下状态，但不要急着放进水里。

出眠后不要马上喂食！这是最重要的。龟龟刚苏醒，肠胃功能还没有完全恢复，马上喂食容易引发肠胃炎。正确的做法是等龟龟完全清醒、活动自如，再观察有没有正常排便，之后再开始少量喂食。

出眠后如何喂食？第一次喂食要少量，约为正常量的三分之一。让龟龟的肠胃慢慢适应。可以先喂一些好消化的食物，比如龟粮泡软或者少量虾肉。温度不稳定时坚决不要喂，至少要保持在二十度以上才能喂食。

出眠后的状态检查不能少。检查龟龟的眼睛、鼻孔、嘴巴有没有异常分泌物。检查甲壳有没有腐皮、烂甲。检查四肢力度，强壮的龟会用力挣扎。如果发现异常要及时处理。

出眠后的水质管理要重视。冬眠期间的水质可能已经变差，要逐渐换水，但不能全部换掉，每次换三分之一左右。自来水一定要提前晾晒，去除氯气。水温要和室温接近，避免温差过大。

出眠后常见问题如何处理？不吃食：可能是温度不够，继续等待观察。精神萎靡：检查是否有病，必要时就医。排便异常：可能是肠胃炎，要停食观察。

总结一下：出眠的时机看温度，不看月份；出眠后不要马上喂食，等龟完全清醒再说；第一次少量喂，好消化为主；密切观察状态，有问题及时处理。
