# 草龟的生理结构与习性，了解你的龟龟

哈喽大家好，欢迎回来！很多养龟的朋友对自己养的龟龟并不够了解。这期视频就来说说草龟的生理结构与习性，帮助你更好地认识你的龟龟。

首先说说草龟的外形特征。草龟体型中等，成年背甲十五到二十厘米。背甲呈椭圆形，有三条纵棱。背甲颜色为灰褐色或黑褐色。腹甲为黄色，有对称的斑点。头部较小，嘴呈喙状。

草龟的甲壳结构。甲壳分为背甲和腹甲两部分。背甲由十三块盾片组成。腹甲由九块骨板组成。甲壳是骨骼的变形，非常坚硬。甲壳起到保护和支撑作用。

草龟的皮肤特点。草龟皮肤较薄，容易受伤。皮肤颜色与环境有关。皮肤上有鳞片保护。皮肤呼吸功能有限，主要靠肺部。皮肤可以吸收少量水分。

草龟的呼吸系统。草龟主要用肺呼吸。呼吸时头部会伸出水面。肺容量不大，需要频繁浮出水面。有些龟可以用泄殖腔呼吸，但草龟不太擅长。呼吸系统健康对龟很重要。

草龟的消化系统。草龟的消化系统比较简单。食道很长，可以储存食物。胃部有胃酸帮助消化。肠道负责吸收营养。排泄通过泄殖腔完成。

草龟的视觉。草龟的视力不太好，看不太远。但对移动的物体敏感。能分辨白天和夜晚。养龟时动作要轻，不要吓到它。视觉不是草龟的主要感知方式。

草龟的听觉。草龟几乎没有外耳廓。有内耳，可以听到声音。对振动比声音更敏感。养龟时不用太担心噪音。但也要保持环境安静，避免惊吓。

草龟的嗅觉。嗅觉是草龟最主要的感知方式。靠嗅觉寻找食物和伴侣。能感知水质的变化。鼻子是感知环境的主要器官。养龟时要注意水质清洁。

草龟的触觉。皮肤可以感知温度和水流。甲壳有触觉，能感知震动。环境变化会引起龟的反应。养龟时要注意水温变化。轻拿轻放，避免弄伤龟龟。

草龟的繁殖习性。草龟是卵生动物。繁殖期在每年的春秋两季。母龟产蛋后需要孵化。孵化温度决定龟苗性别。公龟需要三到五年性成熟。

草龟的冬眠习性。草龟是变温动物，会冬眠。冬眠是自然的生理需求。冬眠期间不进食，代谢降低。冬眠对龟的健康有益。冬眠需要合适的环境。

草龟的食性。草龟是杂食性动物。幼龟偏肉食，长大后偏素食。喜欢吃的食物有鱼虾、螺蛳、蔬菜等。知道饥饱，不会撑死。但喂食也要适量。

草龟的活动规律。草龟白天活动较多。喜欢晒背，阳光充足时会出来。夜晚会安静休息。活动受温度影响很大。温度合适龟会比较活跃。

草龟的寿命。草龟寿命较长，一般二三十年。精心饲养可以活到四十年以上。影响寿命的因素包括饮食、环境等。养得好可以陪伴主人很久。龟是长寿的宠物。

草龟的性格特点。草龟性格温和，不主动攻击人。胆子比较小，遇到危险会缩头。公龟发情期可能会有攻击性。互动性比猫狗差，但也能认识主人。养龟需要耐心。

草龟的社交行为。草龟不是群居动物。但可以和其他龟混养。体型差异大的要分开养。混养需要注意空间和食物。龟之间也会有领地意识。

最后总结一下。了解草龟的生理结构才能养好它。草龟是杂食性变温动物。嗅觉是最主要的感知方式。繁殖和冬眠是重要的习性。养龟要尊重它的习性。
