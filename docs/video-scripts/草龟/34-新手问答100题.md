# 新手问答100题，看完养龟不求人

哈喽大家好，欢迎回来！养龟过程中新手会遇到很多问题，这期视频总结了100个常见问题，看完你养龟就不需要问别人了。

基础类问题。草龟吃什么？龟粮、鱼虾、蔬菜都吃。草龟怎么养？保持水质，控制温度，定期喂食。草龟寿命多长？三四十年没问题。草龟好养吗？比猫狗省事，很好养。

喂食类问题。多长时间喂一次？苗子每天喂，成龟两三天喂一次。喂多少合适？头部大小那么多。不吃食怎么办？检查温度和环境，找原因。能喂肉吗？可以，但不要喂肥肉。能喂狗粮猫粮吗？不能，营养不同。

水质类问题。水多久换一次？根据水质，三到五天换一次。换水注意什么？注意温差，不超过两度。自来水能直接用吗？需要困水去氯。水变绿了还能用吗？绿水可以继续用。

温度类问题多少度合适？二十五到二十八度最合适。冬天怎么办？用加热棒加温。需要晒背吗？需要，每天一到两小时。

冬眠类问题。能冬眠吗？健康的成龟可以冬眠。冬眠怎么养？浅水冬眠或椰土冬眠。苗子能冬眠吗？建议加温过冬。冬眠要注意什么？保持湿度，定期检查。

疾病类问题。龟生病了什么表现？不吃不动、浮水、粪便异常怎么办？隔离治疗，对症下药。腐皮怎么治？干养消毒涂药。肺炎怎么治？加温药浴。肠胃炎怎么治？停食调理。

繁殖类问题。草龟怎么繁殖？需要异性龟，繁殖期交配。龟多大能繁殖？五年以上。蛋怎么孵化？恒温孵化箱，五十到六十天。苗子怎么养？浅水加温，细心照顾。

设备类问题。需要什么设备？龟缸、加热棒、晒台、过滤。需要买晒背灯吗？室内养需要。需要过滤器吗？建议配置。设备在哪买？爬宠店或网上都可以。

进阶类问题。能混养吗？可以，但注意体型差异。能和鱼一起养吗？可以，但要选合适的鱼。能深水养吗？可以，要有晒台。造景缸怎么做？可以种水草，放沉木。

新手常见问题。新手适合养什么龟？草龟、巴西龟最适合。新手容易犯什么错？喂太多、换水太勤。养龟需要多少预算？一百到五百元足够。从哪里买龟好？实体店或信誉好的网店。

这些问题覆盖了养龟的方方面面。如果还有其他问题，可以在评论区问我。养龟是一个不断学习的过程。多看多问，经验就慢慢积累了。
