简介：
龟龟拉稀、便秘、食欲不振？肠道问题是草龟最常见的健康隐患。现在是春季，龟刚苏醒，肠道最脆弱的时候，特别容易出问题。这期视频详细讲解草龟肠道健康的核心要点和益生菌的正确使用方法。

这期内容要点：

✅ 温度是核心：25-28℃消化最好，低于15℃必须停食
✅ 粪便判断：黑色成型最健康，白色粉末是尿酸（正常），白色粘液是肠炎（警惕）
✅ 清肠关键：冬眠前1-2周停食，开春先泡澡排便再开食
✅ 益生菌定位：本质是调理剂，不是急救药，严重肠炎必须用药
✅ 益生菌使用：水温25-35℃，不与抗生素同用，停药2-3天后再补充
✅ 草龟饮食：幼体鱼虾龟粮为主，成体可增植物性，不要喂人类熟食
✅ 开春定心丸：泡澡几天不排便也正常，继续观察，饿不死

春季开食一定要慢！建议收藏备用！

## 评论区置顶

📌 视频补充（肠道健康必知）：

🌡 温度是核心
→ 25-28℃消化最好，菌群最活跃
→ 低于20℃消化下降，低于15℃必须停食
→ 春季早晚温差大，开食一定要慢

💩 粪便判断
→ 黑色成型：最健康
→ 白色粉末沉淀：尿酸，正常现象
→ 白色胶状粘液：肠炎症状，要警惕
→ 水样便：肠胃有炎症

🧹 清肠是关键
→ 冬眠前1-2周停食，排空肠道
→ 开春后先温水泡澡+益生菌，刺激肠道蠕动
→ 等排便了、状态稳定了再开食，第一顿要少
→ 泡澡几天不排便也正常，继续观察，饿不死

💊 益生菌使用
→ 水温25-35℃，不超过40℃
→ 本质是调理剂，不是急救药
→ 不与抗生素同用，停药2-3天后再补充
→ 日常调理每周1-2次，生病恢复期可每天

🚫 饮食禁忌
→ 不要喂人类吃的熟食，高盐高油伤肾
→ 白水煮熟鱼虾可以，但不加调料
→ 龟粮经过熟化工艺，最安全

💡 记住：养龟先养肠，预防比治疗更重要！
