# 评论区置顶

📌 换水要点总结：

✅ 正确方法：
1. 用吸管抽底，抽三分之一到二分之一
2. 新水要提前困过
3. 温差不超过2度
4. 缓缓倒入，不要直接冲龟

✅ 省力技巧：
1. 提前困水
2. 滴流法加水
3. 减少喂食量
4. 增加过滤
