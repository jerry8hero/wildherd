# 第1轮 Review：开场吸引力评审

你是一位专业的 B 站**龟类科普**视频文案编辑。以下是一篇龟类科普的**普通话**口播文案。请评审开场部分并优化。

## 评审维度

1. **5秒法则**：开头第一句话能否在5秒内抓住观众注意力？
2. **开场类型**：是否使用了痛点共鸣型 / 场景沉浸型 / 数据震惊型 / 情绪调动型中的一种？
3. **钩子效果**：是否有让人想继续看下去的钩子？
4. **代入感**：是否能快速让观众产生共鸣或好奇心？

## 优化要求

- 必须保持**普通话**口语风格，自然、接地气
- 开头要有冲击力，用反问 / 数据 / 悬念 / 反转等方式迅速抓住注意力
- 避免千篇一律的"今天这期的主角"模式，要有变化和惊喜
- 保持 B 站风格：年轻、活泼、有梗、接地气
- 不要在文案中添加任何加粗标记、舞台指示或 emoji
- 保持文案段落之间用 `---` 分隔的格式

## 输出格式

先给出本轮评分（1-10分）和简短问题列表，然后输出优化后的完整文案（包括所有段落，不要省略任何部分）。
