# 锯缘摄龟饲养指南

> 帮你养好最偏陆栖的半水龟品种

---

## 一、基本信息

| 项目 | 数据 |
|------|------|
| 原产地 | 中国（湘粤桂琼滇）、越南、泰国、缅甸、印度 |
| 体型 | 中型半水龟 |
| 水性 | 偏陆栖，水位不超过背甲一半 |
| 冬眠 | 可冬眠，但建议北方加温 |
| 性格 | 凶猛、胆大、互动性好 |

## 二、环境设置

### 2.1 饲养容器

| 阶段 | 建议尺寸 |
|------|----------|
| 幼龟 | 60cm 以上 |
| 成龟 | 120cm 以上（建议120×50×45cm） |

**环境布局建议：**
- 垫材：椰土、苔藓、腐殖土（保湿）
- 躲避处：提供躲避洞穴或植物遮蔽
- 水盘：供饮水和泡澡，水位不超过背甲高一半
- 植物：种植耐践踏的草本植物
- 光照：散射光，避免强光直射

### 2.2 温度控制

| 温度范围 | 状态 |
|----------|------|
| 22-30°C | 正常生长、进食 |
| 25-28°C | 最佳状态 |
| 19°C 以下 | 冬眠 |
| 32°C 以上 | 夏眠 |

**关键点：**
- 最佳温度：**25-28°C**
- 喜暖怕寒
- 北方建议加温饲养
- 冬眠温度保持在 8°C 以上，时间不超过 2 个月

### 2.3 湿度要求

| 项目 | 要求 |
|------|------|
| 空气湿度 | 70-85% |
| 垫材湿度 | 潮湿但不积水 |
| 水盘 | 定期更换 |

**注意：** 湿度过低会导致头皮发白

### 2.4 光照

- 喜阴，忌强光
- 每天散射光照射 2-4 小时
- 或使用 UVB 灯每天照射 8-10 小时
- 促进钙质吸收和健康成长

## 三、喂食指南

### 3.1 食物选择

| 食物类型 | 推荐度 | 说明 |
|----------|--------|------|
| 活饵（蚯蚓、杜比亚蟑螂） | ⭐⭐⭐ | 最爱，活性强 |
| 面包虫/大麦虫 | ⭐⭐⭐ | 常用活饵 |
| 蜗牛/螺肉 | ⭐⭐⭐ | 野生环境主要食物 |
| 蟋蟀/蝗虫 | ⭐⭐⭐ | 昆虫类营养丰富 |
| 龟粮 | ⭐⭐ | 营养均衡，可作为主食 |
| 水果（香蕉、西瓜、火龙果） | ⭐⭐ | 补充维生素 |
| 熟南瓜 | ⭐ | 偶尔投喂 |
| 动物肝脏 | ⭐ | 偶尔补充营养 |

### 3.2 喂食频率

| 阶段 | 频率 | 量 |
|------|------|------|
| 幼龟 | 每天 1 次 | 头部大小 |
| 成龟 | 每周 3 次 | 体重的 2-3% |
| 夏眠/冬眠 | 停止喂食 | - |

### 3.3 饲养食谱建议

**人工饲养最佳食谱搭配：**
1. 主食：龟粮 + 果蔬
2. 活饵：蚯蚓、杜比亚蟑螂作为零食
3. 偶尔：毛蛋等高蛋白食物

**这样养出的锯缘：**
- 生长纹色泽沉淀好
- 活动性强
- 交配勤、产蛋量高

## 四、日常护理

### 4.1 健康检查

| 检查项 | 正常 | 异常 |
|--------|------|------|
| 眼睛 | 清亮、无分泌物 | 发红、有分泌物 → 感染 |
| 皮肤 | 光滑、无斑点 | 发白、溃烂 → 腐皮 |
| 龟壳 | 坚硬、色泽均匀 | 软甲、斑点 → 腐甲/软甲 |
| 粪便 | 成型 | 不成形 → 肠胃炎 |
| 食欲 | 积极 | 拒食 → 检查健康 |

### 4.2 常见疾病预防

| 疾病 | 预防措施 |
|------|----------|
| 寄生虫病 | 新龟到家检查驱虫，注意环境卫生 |
| 腐甲病 | 保持环境适宜湿度，避免外伤 |
| 肠胃炎 | 食物新鲜、控制喂食量、温度稳定 |
| 肺炎 | 温差不超过 3°C，避免低温 |
| 螨虫 | 定期检查腋窝、胯窝、颈窝 |

### 4.3 清洁要点

- 勤清理粪便
- 定期更换水盘用水
- 每周检查垫材湿度
- 每月整体消毒一次

## 六、成熟判断

### 达到繁殖条件的标准

| 项目 | 公龟 | 母龟 |
|------|------|------|
| 体重 | 400g+ | 500g+ |
| 年龄 | 4-5 年 | 5-6 年 |
| 背甲 | 15cm+ | 15cm+ |

### 促进成熟的建议

1. **营养跟上** - 高蛋白食物为主，活饵定期投喂
2. **空间充足** - 不要养在太小的容器
3. **环境稳定** - 温湿度恒定
4. **食物多样化** - 果蔬搭配，保证维生素

---

## 七、注意事项

### 必知要点

1. **偏陆栖** - 水位绝不可超过背甲高度一半
2. **喜潮湿** - 保持环境湿度，但避免积水
3. **性格凶猛** - 群养雄龟易打架咬伤
4. **喜阴怕光** - 避免强光直射
5. **喜暖怕寒** - 温度低于 19°C 会冬眠
6. **耐饿能力强** - 4-5 个月不喂不易饿死

### 新手常见错误

| 错误 | 后果 |
|------|------|
| 水位太深 | 溺水 |
| 强光直射 | 应激、拒食 |
| 湿度太低 | 脱水、皮肤病 |
| 低温冬眠时间过长 | 体质下降甚至死亡 |
| 群养雄龟 | 打架咬伤 |

## 八、繁殖要点

### 繁殖基本信息

| 项目 | 数据 |
|------|------|
| 繁殖季节 | 4-6 月 |
| 产卵数量 | 4-8 枚 |
| 孵化温度 | 28-31°C |
| 孵化周期 | 60-80 天 |

### 繁殖注意事项

1. 产蛋后及时采收，避免被吃掉
2. 孵化期间保持温度稳定
3. 稚龟出壳后单独饲养
