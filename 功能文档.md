# WildHerd 爬宠助手 - 功能文档与优化建议

## 一、项目概述

**WildHerd（爬宠助手）** 是一款专为爬宠爱好者打造的 Flutter 移动应用，帮助记录爬宠的日常生活、健康状况，同时提供丰富的爬宠知识百科和社区交流功能。

---

## 二、已实现功能

### 2.1 首页模块 🏠

| 功能 | 说明 |
|------|------|
| 欢迎卡片 | 展示爬宠数量统计 |
| 爬宠快捷入口 | 横向列表快速访问 |
| 推荐物种 | 根据用户等级推荐 |
| 快捷功能按钮 | 喂食记录、健康记录、知识助手 |
| 今日提醒 | 湿度、温度等环境提醒 |
| 添加爬宠 | FAB 快速入口 |

### 2.2 爬宠管理 🦎

| 功能 | 说明 |
|------|------|
| 添加爬宠档案 | 名字、种类、性别、出生日期、体重、体长 |
| 爬宠类型 | 蛇类、守宫、蜥蜴、龟类、两栖、蜘蛛、昆虫、哺乳、鸟类、鱼类 |
| 爬宠详情页 | 完整档案信息展示 |
| 删除爬宠 | 档案管理 |

### 2.3 记录管理 📝

| 功能 | 说明 |
|------|------|
| 喂食记录 | 时间、食物类型、食物量 |
| 健康记录 | 体重、体长、状态、排便情况 |
| 成长相册 | 照片、描述、日期 |

### 2.4 知识百科 📚

| 搜索功能 | 支持中英文和学名搜索 |

### 2.5 知识库 📖

## 四、优化建议

### 4.1 功能增强

| 优先级 | 建议 | 说明 |
|--------|------|------|
| **高** | 喂食提醒功能 | 定时提醒用户喂食 |
| **高** | 健康数据图表 | 体重、体长趋势可视化 |
| **高** | 云端数据同步 | 多设备数据同步 |
| **中** | 用户登录系统 | 账户体系支持 |
| **中** | 消息通知 | 社区回复、系统通知 |
| **中** | 爬宠种类扩展 | 增加更多物种数据 |
| **低** | 社区图片上传 | 动态支持图片 |
| **低** | 分享功能 | 分享到社交平台 |

### 4.2 技术优化

| 优先级 | 建议 | 说明 |
|--------|------|------|
| **高** | 状态管理统一 | 统一使用 Riverpod 或 Provider，避免混用 |
| **高** | 数据层重构 | 考虑使用 Isar/Drift 替代手写 SQL |
| **中** | 单元测试 | 添加核心业务逻辑测试 |
| **中** | 性能优化 | 列表懒加载、图片缓存优化 |
| **低** | 代码分割 | 按模块拆分，减少首屏加载 |
| **低** | PWA 增强 | 离线支持、推送通知 |

### 4.3 用户体验优化

| 优先级 | 建议 | 说明 |
|--------|------|------|
| **高** | 引导页 | 新用户首次使用引导 |
| **高** | 空状态设计 | 无数据时的友好提示 |
| **中** | 暗黑模式 | 深色主题支持 |
| **中** | 手势操作 | 滑动返回、长按菜单 |
| **低** | 动画效果 | 页面切换动画增强 |

### 4.4 数据扩展

| 优先级 | 建议 | 说明 |
|--------|------|------|
| **高** | 疾病库扩展 | 增加更多疾病数据 |
| **中** | 物种数据完善 | 补充繁殖、环境参数 |
| **中** | 饲养视频 | 接入饲养教程视频 |
| **低** | 商家数据 | 附近爬宠店推荐 |
