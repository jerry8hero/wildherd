# 红面蛋龟饲养指南

> 帮你把红面蛋龟养得健康漂亮，为繁殖做准备

---

## 一、基本信息

| 项目 | 数据 |
|------|------|
| 原产地 | 中美洲、南美洲（热带） |
| 体型 | 中型蛋龟 |
| 水性 | 一般 |
| 冬眠 | **不冬眠**，需全年加温 |
| 性格 | 凶猛、互动性好 |

---

## 二、环境设置

### 2.1 饲养容器

| 阶段 | 建议尺寸----------|
|  |
|------|幼龟 | 40cm 以上 |
| 成龟 | 60-100cm（建议 200L 水箱） |

**环境布局建议：**
- 深水区：40-60cm 水深
- 浅水区：20-40cm 水深
- 陆地/晒台：30% 面积
- 躲避处：提供躲避空间

### 2.2 温度控制

| 温度范围 | 状态 |
|----------|------|
| 24-32°C | 正常生长、交配 |
| 24°C 以下 | 活动减少，不进食 |
| 18°C 以下 | 冬眠状态（危险！） |
| 32°C 以上 | 可能夏眠 |

**关键点：**
- 最佳温度：**25-28°C**
- 必须加温饲养
- 使用加热棒/加热灯
- 冬季尤为重要

### 2.3 水质要求

| 项目 | 要求 |
|------|------|
| 水源 | 晾晒 1-2 天的自来水 |
| 换水 | 每周 1-2 次，或根据过滤情况 |
| 水位 | 15-25cm（不过背） |
| 过滤 | 建议安装过滤系统 |

**注意：** 红面蛋龟对水质敏感，水质差易患：
- 白眼病
- 腐皮病
- 肠胃炎

### 2.4 光照

- 每天晒太阳 1-2 小时（无玻璃阻隔）
- 或使用 UVB 灯每天照射 10-12 小时
- 促进发色、钙质吸收

## 三、喂食指南

### 3.1 食物选择

| 食物类型 | 推荐度 | 说明 |
|----------|--------|------|
| 龟粮 | ⭐⭐⭐ | 主食，营养均衡 |
| 虾 | ⭐⭐⭐ | 发色关键 |
| 小鱼 | ⭐⭐⭐ | 优质蛋白 |
| 昆虫 | ⭐⭐ | 零食，营养丰富 |
| 螺肉 | ⭐⭐ | 补钙 |
| 蔬菜 | ⭐ | 偶尔投喂 |

### 3.2 喂食频率

| 阶段 | 频率 | 量 |
|------|------|------|
| 幼龟 | 每天 1 次 | 头部大小 |
| 成龟 | 隔天 1 次 | 体重的 2-3% |

### 3.3 发色技巧

红面蛋龟以红色面部为美，发色建议：

1. **多喂虾** - 虾红素促进发色
2. **白色环境** - 白色箱子饲养
3. **充足光照** - 自然光或 UVB
4. **少喂猪肉** - 脂肪多易褪色

## 四、日常护理

### 4.1 健康检查

| 检查项 | 正常 | 异常 |
|--------|------|------|
| 眼睛 | 清亮、无分泌物 | 发红、有白膜 → 白眼病 |
| 皮肤 | 光滑、无斑点 | 发白、溃烂 → 腐皮 |
| 龟壳 | 坚硬、色泽均匀 | 软甲、斑点 → 腐甲 |
| 粪便 | 成型 | 不成形 → 肠胃炎 |
| 食欲 | 积极 | 拒食 → 检查健康 |

### 4.2 常见疾病预防

| 疾病 | 预防措施 |
|------|----------|
| 白眼病 | 保持水质、晒太阳 |
| 腐皮病 | 定期换水、保持清洁 |
| 肠胃炎 | 食物新鲜、控制量 |
| 软甲病 | 补钙、充足光照 |

## 六、成熟判断

### 达到繁殖条件的标准

| 项目 | 公龟 | 母龟 |
|------|------|------|
| 体重 | 350g+ | 400g+ |
| 年龄 | 3-4 年 | 4-5 年 |
| 腹甲 | 13cm+ | 13cm+ |

### 促进成熟的建议

1. **营养跟上** - 高蛋白食物为主
2. **空间充足** - 不要养在太小的容器
3. **环境稳定** - 温度恒定
4. **公母分开养** - 合养会频繁交配影响生长

---

## 七、注意事项

### 必知要点

1. **必须加温** - 温度低于 18°C 有生命危险
2. **水质清洁** - 对水质敏感
3. **不要混养** - 性格凶猛，会咬伤其他龟
4. **小心咬人** - 喂食时用镊子，不要手直接喂
5. **控制喂食** - 容易肥胖

### 新手常见错误

| 错误 | 后果 |
|------|------|
| 冬季不加温 | 死亡/冬眠不醒 |
| 喂食过量 | 肥胖、肠胃炎 |
| 水质不换 | 腐皮、白眼 |
| 混养其他龟 | 咬伤、咬死 |
